# <a name="what-is-phone-system"></a>¿Qué es sistema telefónico?

Una central de conmutación (PBX) es un sistema telefónico dentro de una empresa. Sistema telefónico le ofrece funcionalidades PBX, pero sin el equipamiento caro y complicado. 

Sistema telefónico le permite reemplazar su sistema PBX local existente con un conjunto de características entregadas desde Microsoft 365 u Office 365 que está estrechamente integrado en su experiencia en la nube. Para obtener más información sobre las características, vea [esto es lo que obtiene con Sistema telefónico.](here-s-what-you-get-with-phone-system.md)

## <a name="phone-system-technology"></a>Tecnología de Sistema telefónico

Sistema telefónico habilita el control de llamadas y las capacidades de PBX en la nube con Microsoft Teams y Skype Empresarial Online. 
  
Con Sistema telefónico, los usuarios pueden usar Teams o Skype Empresarial Online para realizar y recibir llamadas, transferir llamadas y desactivar o activar el audio de las llamadas. Los usuarios del sistema telefónico pueden hacer clic en un nombre de su libreta de direcciones y realizar llamadas de Teams o Skype Empresarial Online a esa persona. Para realizar y recibir llamadas, los usuarios de Sistema telefónico pueden usar sus dispositivos móviles, auriculares con portátil o PC, o uno de los muchos teléfonos IP que funcionan con Teams y Skype Empresarial Online. Los administradores del sistema telefónico pueden administrar las opciones y la configuración de llamadas desde la misma consola que se usa para mensajería, colaboración, entre otros.
  
Las llamadas entre usuarios de su organización se gestionan internamente en Sistema telefónico y nunca van a la red telefónica conmutada (RTC). Esto se aplica a las llamadas entre usuarios de su organización ubicados en diferentes áreas geográficas, quitando los costes de larga distancia en estas llamadas internas.

Para llamadas fuera de su organización, Microsoft ofrece varias opciones para conectar Sistema telefónico a la red telefónica conmutada (RTC).

## <a name="connect-to-the-public-switched-telephone-network-pstn"></a>Conectarse a la red telefónica conmutada (RTC)
  
Sistema telefónico puede estar conectado a RTC de una de estas dos maneras:
  
- Compra un plan de llamadas de Microsoft (nacional o nacional e internacional). El plan de llamadas de Microsoft es una solución todo en la nube con Microsoft como su operador de RTC. Para obtener más información, vea [Sistema telefónico y Planes de llamadas.](calling-plan-landing-page.md)

- Use su infraestructura de telefonía existente para la conectividad con RTC local.

  Para los usuarios de Teams, puede conectar su infraestructura de telefonía local a Sistema telefónico mediante el enrutamiento directo. Para obtener más información, vea [Enrutamiento directo de sistema telefónico.](direct-routing-landing-page.md)

  Para obtener más información sobre todas las soluciones de telefonía de Microsoft, consulte [soluciones de telefonía de Microsoft.](https://docs.microsoft.com/SkypeForBusiness/hybrid/msft-telephony-solutions)


## <a name="phone-system-with-services"></a>Sistema telefónico con servicios

 Sistema telefónico puede usarse para servicios y correo de voz, como:

- **Operadores automáticos:** los operadores automáticos se pueden usar para crear un sistema de menús para su organización que permita a los autores de llamadas internos y externos desplazarse por el sistema para localizar y realizar o transferir llamadas a usuarios de la compañía o departamentos de su organización. Vea [¿Qué son los operadores automáticos en la nube?](what-are-phone-system-auto-attendants.md).

- **Colas de llamadas:** los saludos de la cola de llamadas se pueden usar cuando alguien llama a un número de teléfono de su organización. Estos saludos incluyen la capacidad de poner automáticamente las llamadas en espera y la capacidad de buscar al siguiente agente de llamadas disponible para que se ocupará de la llamada mientras las personas que realiza la llamada escuchan música en espera. Puede crear una o varias colas de llamadas para su organización. Vea [Crear una cola de llamadas en la nube.](create-a-phone-system-call-queue.md)

- **Correo** de voz: cuando obtiene una licencia de Sistema telefónico para un usuario, ese usuario puede obtener correo de voz que los autores de llamadas han dejado. El correo de voz en la nube se configura y aprovisiona automáticamente para los usuarios después de asignarles una licencia de Sistema telefónico y un número de teléfono. Consulte [Configurar el correo de voz en la nube.](set-up-phone-system-voicemail.md)

Para obtener más información sobre las características, vea [esto es lo que obtiene con Sistema telefónico.](here-s-what-you-get-with-phone-system.md) Si está listo para empezar, consulte [Configurar sistema telefónico en su organización.](setting-up-your-phone-system.md)
