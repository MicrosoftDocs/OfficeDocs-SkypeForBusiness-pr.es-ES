# <a name="sharing-files-in-microsoft-teams"></a>Uso compartido de archivos en Microsoft Teams

> [!INCLUDE [new feature coming soon](includes/new-feature-coming-soon-article.md)]

El archivo de uso compartido de las características de los equipos permite a los usuarios compartir contenido con otros usuarios de los equipos de su organización. El uso compartido en los equipos se basa en la configuración establecida en SharePoint y OneDrive, por lo que todo lo que configure para SharePoint y OneDrive para controlar el uso compartido en los equipos así como.

Los equipos de uso compartido permite a los usuarios hacer lo siguiente:

- Compartir archivos de OneDrive.

- Establecer permisos para los archivos que desean compartir con otros usuarios.

- Compartir archivos entre los equipos.

- Compartir archivos desde su lista de archivos usados recientemente (normalmente, estos son los archivos de los usuarios están más interesados en el uso compartido).

- Permanecer dentro de los equipos cuando hacen clic en un nombre de archivo para abrir un archivo.

Los equipos se acorta largo SharePoint URLS y direcciones URL que apuntan a un archivo de explorador. Los equipos se usa sólo el nombre del archivo para vincular a un archivo. Además, la opción de **obtener el vínculo** se cambió para **Copiar vínculo** a cualquier confusión que los usuarios pueden tener sobre la concesión a otros usuarios acceso a un archivo.

## <a name="configure-sharing-in-onedrive-and-sharepoint"></a>Configurar el uso compartido en OneDrive y SharePoint

Para obtener más información sobre el uso compartido de archivos de OneDrive y SharePoint, incluido cómo configurar el uso compartido y cómo activar y desactivar el uso compartido, vea:

- [Información general de uso compartido externo](https://docs.microsoft.com/sharepoint/external-sharing-overview) - describe lo que sucede cuando los usuarios compartir, dependiendo de lo que está compartiendo y con quién.

- [Activar o desactiva el uso compartido externo](https://docs.microsoft.com/sharepoint/turn-external-sharing-on-or-off) - describe cómo global y los administradores de SharePoint pueden cambiar su configuración de uso compartido de nivel de la organización para SharePoint y OneDrive.

- [Cambio externo de uso compartido para un sitio](https://docs.microsoft.com/sharepoint/change-external-sharing-site) – describe cómo global y los administradores de SharePoint pueden activar el uso compartido de activado o desactivado para un sitio externos.

- [Cambiar el tipo de vínculo predeterminado cuando los usuarios obtener vínculos para compartir](https://docs.microsoft.com/sharepoint/change-default-sharing-link) - describe cómo establecer el tipo de vínculo predeterminado para que sea más restrictivo.
