<a name="enhance-existing-microsoft-365-groups-with-microsoft-teams"></a>Mejore los Microsoft 365 existentes con Microsoft Teams
=======================================================

Microsoft Teams usuarios pueden mejorar un grupo de Microsoft 365 existente con Teams funcionalidad. Al buscar mejorar un grupo de Microsoft 365 público, los usuarios pueden hacerlo si el número de miembros es igual o menor que 5 000.

Para ello, los usuarios deben pasar por el flujo de creación de un nuevo equipo desde el Microsoft Teams cliente. Seleccione **Crear desde Microsoft 365** y, a continuación, elija el grupo existente que desea mejorar con  >  Teams. Los miembros del grupo existentes se agregarán como miembros al equipo automáticamente.

> [!IMPORTANT]
> Solo Microsoft 365 propietarios del grupo tienen permiso para mejorar un grupo existente con Teams y ese grupo debe contener menos de 5.000 usuarios. Teams que ya se han mejorado no se mostrarán en la lista.
>
>La opción para crear un equipo desde un grupo Microsoft 365 no estará disponible a menos que se hayan cumplido estas condiciones.

Los usuarios también pueden invitar a una lista de distribución a un equipo y los miembros de esa lista de distribución se agregarán al equipo. Se trata de un complemento de una sola vez y los cambios posteriores en la pertenencia a grupos en la lista de distribución no se replicarán ni sincronizarán con Teams.

![Captura de pantalla de la opción para crear un equipo a partir de Microsoft 365 grupo.](media/Enhance_Existing_Office_365_groups_with_Microsoft_Teams_image2.png)

También puede agregar grupos de seguridad habilitados para correo como miembros de un equipo. Pero si después agrega más miembros al grupo de seguridad, estos no se agregarán automáticamente al equipo. Debe agregar los nuevos miembros por separado o volver a agregar el grupo de seguridad al equipo. (Si vuelve a agregar el grupo de seguridad, con la desduplicación se asegurará de que los miembros se agreguen una sola vez).

Hay dos tipos de configuración de privacidad con Microsoft 365, **público y privado.** Mientras que ambos tipos de grupo se pueden habilitar para Teams, hay una ligera diferencia cuando se trata de autoservicio.

-   Los usuarios pueden buscar grupos públicos y pueden unirse por su cuenta sin necesidad de la aprobación del propietario del equipo.

-   No se pueden realizar búsquedas en los grupos privados. Los usuarios no se pueden unir a un grupo privado a menos que el propietario del equipo los agregue como miembros.

Al crear un nuevo equipo en Teams, un propietario de un grupo de Microsoft 365 privado existente tiene una opción para usar la pertenencia al grupo de Microsoft 365 para crear el equipo. Los usuarios pueden agregar sus archivos SharePoint y OneNote mediante la adición de una pestaña para SharePoint y la combinación de OneNote archivos.
