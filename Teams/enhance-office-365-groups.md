<a name="enhance-existing-microsoft-365-groups-with-microsoft-teams"></a>Mejorar los grupos existentes de Microsoft 365 con Microsoft Teams
=======================================================

Los usuarios de Microsoft Teams pueden mejorar un grupo de Microsoft 365 existente con la funcionalidad de Teams. Cuando se analiza la mejora de un grupo público de Microsoft 365, los usuarios pueden hacerlo si el número de miembros es igual o menor que 5.000.

Para ello, los usuarios deben pasar por el flujo de creación de un equipo nuevo desde el cliente de Microsoft Teams. Seleccione **Crear desde** el grupo de Microsoft  >  **365** y, a continuación, elija el grupo existente que desea mejorar con Teams. Los miembros del grupo existentes se agregarán como miembros al equipo automáticamente.

> [!IMPORTANT]
> Solo los propietarios de grupos de Microsoft 365 tienen permiso para mejorar un grupo existente con Teams, y ese grupo debe contener menos de 5 000 usuarios. Los equipos que ya se han mejorado no se mostrarán en la lista.
>
>La opción para crear un equipo desde un grupo de Microsoft 365 no estará disponible a menos que se hayan cumplido estas condiciones.

Los usuarios también pueden invitar a una lista de distribución a un equipo y los miembros de esa lista de distribución se agregarán al equipo. Se trata de una adición única y los cambios posteriores en la pertenencia a grupos en la lista de distribución no se replicarán ni se sincronizarán en Teams.

![Captura de pantalla de la opción para crear un equipo desde un grupo de Microsoft 365.](media/Enhance_Existing_Office_365_groups_with_Microsoft_Teams_image2.png)

También puede agregar grupos de seguridad habilitados para correo como miembros de un equipo. Pero si después agrega más miembros al grupo de seguridad, estos no se agregarán automáticamente al equipo. Debe agregar los nuevos miembros por separado o volver a agregar el grupo de seguridad al equipo. (Si vuelve a agregar el grupo de seguridad, con la desduplicación se asegurará de que los miembros se agreguen una sola vez).

Hay dos tipos de configuración de privacidad con los grupos de Microsoft 365: **públicos y privados.** Mientras que ambos tipos de grupo se pueden habilitar para Teams, hay una pequeña diferencia en lo que se respecta al autoservicio.

-   Los usuarios pueden buscar grupos públicos y pueden unirse ellos mismos sin necesidad de la aprobación del propietario del equipo.

-   No se pueden realizar búsquedas en los grupos privados. Los usuarios no se pueden unir a un grupo privado a menos que el propietario del equipo los agregue como miembros.

Al crear un equipo nuevo en Teams, un propietario de un grupo privado de Microsoft 365 existente tiene una opción para usar la pertenencia al grupo de Microsoft 365 para crear el equipo. Los usuarios pueden agregar sus archivos existentes de SharePoint y OneNote agregando una pestaña para SharePoint y combinando archivos de OneNote.
