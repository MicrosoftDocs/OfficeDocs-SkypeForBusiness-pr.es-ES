# <a name="whats-the-status-of-your-port-orders"></a>¿Cuál es el estado de mis solicitudes de portabilidad?

Para ver el estado de la solicitud de portabilidad, en el panel de navegación izquierdo del Centro de administración de Microsoft Teams, vaya a >**Solicitudes de portabilidad** de **voz** >  y, a continuación, haga clic en **Historial de pedidos**. El estado de cada solicitud de portabilidad se muestra en la columna **Estado**. Vea [cuánto tiempo se tarda en realizar la portabilidad de los números](../phone-number-calling-plans/port-order-overview.md#how-long-does-it-take-to-port-numbers) para obtener información sobre el proceso de pedido. 

En la tabla siguiente se enumeran los estados de las solicitudes de portabilidad, así como las acciones que puede realizar si es necesario.

|**Estado**|**¿Puede ver el pedido?**|**¿Puedes editar el pedido?**|**¿Puedes cancelar el pedido?**|**¿Puede eliminar el pedido?**|**Descripción**|
|:-----|:-----|:-----|:-----|:-----|:-----|
|**Tratamiento** <br/> |Sí  <br/> |No  <br/> |Sí  <br/> |No  <br/> |El administrador ha creado el pedido y Microsoft lo ha recibido.  <br/> |
|**Ponerse en contacto con el operador** <br/> |Sí  <br/> |No  <br/> |Sí  <br/> |No  <br/> |Microsoft ha recibido y aprobado el pedido y estamos trabajando con el operador que ha perdido para obtener la aprobación.  <br/> |
|**Transferencia aprobada** <br/> |Sí  <br/> |No  <br/> |Sí  <br/> |No  <br/> |La solicitud ha sido aceptada por el operador que ha perdido y se ha establecido la fecha de compromiso de pedido en firme (FOC).  <br/> |
|**Transferencia pendiente** <br/> |Sí  <br/> |No  <br/> |No  <br/> |No  <br/> |La transferencia está a menos de 24 horas de distancia, por lo que el pedido ya no se puede editar ni cancelar.  <br/> |
|**Error** <br/> |No  <br/> |Sí  <br/> |Sí  <br/> |Sí (en este momento, no puede eliminar la solicitud de portabilidad si hay un error. Es necesario volver a crear la solicitud de portabilidad o ponerse en contacto con [el Servicio de asistencia de TNS](../manage-phone-numbers-for-your-organization/contact-tns-service-desk.md).  <br/> |El operador que perdió rechazó la orden.  <br/> |
|**Completado** <br/> |Sí  <br/> |No  <br/> |No  <br/> |No  <br/> |Los números se han transferido correctamente.  <br/> |
|**Cancelado** <br/> |No  <br/> |Sí  <br/> |No  <br/> |No  <br/> |El administrador canceló el pedido.  <br/> |

Para obtener instrucciones paso a paso, vea [Transferir números de teléfono a Teams](transfer-phone-numbers-to-teams.md).

Si necesita ayuda o necesita obtener más números de teléfono, póngase en contacto con el [Servicio de asistencia de TNS](../manage-phone-numbers-for-your-organization/contact-tns-service-desk.md).
