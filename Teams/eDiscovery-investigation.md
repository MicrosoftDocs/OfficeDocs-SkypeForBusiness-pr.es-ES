## <a name="ediscovery-of-guest-to-guest-chats"></a>exhibición de documentos electrónicos de chats de invitado a invitado

Sin un buzón de correo, chats invitado a invitado (en el que no hay ningún usuario particular inquilinos de chats 1xN) no se puede indizar y como resultado, no se incluirá en la exhibición de documentos electrónicos. Para facilitar la exhibición de documentos electrónicos para chats de invitado a invitado, se crea un buzón de correo basados en la nube (o buzón fantasma) para almacenar los datos de 1xN. Después de que los datos de chat de los equipos se almacenan en el buzón de correo basados en la nube, se indizan para la búsqueda de contenido de exhibición de documentos electrónicos y cumplimiento de normas.
