# <a name="toll-free-dialing-restrictions-in-mexico"></a>Restricciones de marcado de número gratuito en México

[Busque otro país o región](../toll-free-dialing-limitations-and-restrictions.md), u obtenga más información sobre restricciones y limitaciones de marcado gratuitas.


|**Detalles**|**Descripción**|
|:-----|:-----|
|¿Hay disponible una llamada gratuita?  <br/> |Sí  <br/> |
|Formato (s) de marcado  <br/> | 001 800 XXX XXXX <br/>  01 800 XXX XXXX <br/> |
|Código ISO de país o región  <br/> |MX  <br/> |
|Isla/territorio cubierta  <br/> |No aplicable  <br/> |
|Isla/territorio no cubierto  <br/> |No aplicable  <br/> |
|Disponibilidad de línea fija/por cable  <br/> |Parcial. Ver comentarios.  <br/> |
|Disponibilidad de red móvil  <br/> |Parcial. Ver comentarios.  <br/> |
|Comentarios, restricciones y problemas de acceso  <br/> |No se garantiza el acceso desde redes en itinerancia internacionales.  <br/> Para el formato de número 001-800: Para el acceso fijo - Telmex. Para el acceso móvil - solo Telcel. Se cobra el tiempo de llamada. Sin cargos adicionales si el cliente final tiene la Telmex como proveedor local y de larga distancia.  <br/> Para el formato de número 01-800: No se cobra el tiempo de llamada de acceso móvil.  <br/> |
