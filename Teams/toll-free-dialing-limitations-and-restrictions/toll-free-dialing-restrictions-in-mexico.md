# <a name="toll-free-dialing-restrictions-in-mexico"></a>Restricciones de marcado de número gratuito en México

[Busque otro país o región,](../toll-free-dialing-limitations-and-restrictions.md)o obtenga más información sobre las limitaciones y restricciones de marcado gratuitos.


|**Detalles**|**Descripción**|
|:-----|:-----|
|¿Hay llamadas gratuitas disponibles?  <br/> |Sí  <br/> |
|Formatos de marcado  <br/> | 001 800 XXX XXXX <br/>  01 800 XXX XXXX <br/> |
|Código ISO de país o región  <br/> |MX  <br/> |
|Isla o territorio cubierto  <br/> |No aplicable  <br/> |
|Isla o territorio no cubierto  <br/> |No aplicable  <br/> |
|Disponibilidad de líneas fijas o cableadas  <br/> |Parcial. Ver comentarios.  <br/> |
|Disponibilidad de red móvil  <br/> |Parcial. Ver comentarios.  <br/> |
|Comentarios, restricciones y problemas de acceso  <br/> |No se garantiza el acceso desde redes en itinerancia internacionales.  <br/> Para el formato de número 001-800: Para el acceso fijo - Telmex. Para el acceso móvil - solo Telcel. Se cobra el tiempo de llamada. No hay ningún cargo adicional si el cliente final está con Telmex como su proveedor local y de larga distancia.  <br/> Para el formato de número 01-800: No se cobra el tiempo de llamada de acceso móvil.  <br/> |
