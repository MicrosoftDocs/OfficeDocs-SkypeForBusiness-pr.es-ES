# <a name="toll-free-dialing-restrictions-in-panama"></a>Restricciones de marcado de número gratuito en Panamá

[Busque otro país o región,](../toll-free-dialing-limitations-and-restrictions.md)o obtenga más información sobre las limitaciones y restricciones de marcado gratuitos.


|**Detalles**|**Descripción**|
|:-----|:-----|
|¿Hay llamadas gratuitas disponibles?  <br/> |Sí  <br/> |
|Formatos de marcado  <br/> | 00-800-XXX-XXXX <br/> |
|Código ISO de país o región  <br/> |PA  <br/> |
|Isla o territorio cubierto  <br/> |No aplicable  <br/> |
|Isla o territorio no cubierto  <br/> |No aplicable  <br/> |
|Disponibilidad de líneas fijas o cableadas  <br/> |Solo de C&amp;.  <br/> |
|Disponibilidad de red móvil  <br/> |No  <br/> |
|Comentarios, restricciones y problemas de acceso  <br/> |El cliente debe marcar 011 (el código internacional de Panamá de larga distancia) antes del número gratuito.  <br/> |
