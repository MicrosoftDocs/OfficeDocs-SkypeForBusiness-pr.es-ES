## <a name="dial-plans"></a><span data-ttu-id="76135-685">Planes de marcado</span><span class="sxs-lookup"><span data-stu-id="76135-685">Dial plans</span></span>

<span data-ttu-id="76135-686">Un [plan de marcado](https://docs.microsoft.com/SkypeForBusiness/what-are-calling-plans-in-office-365/what-are-dial-plans) en la característica de Sistema telefónico de Office 365 es un conjunto de reglas de normalización con las que los números de teléfono que se marcan se traducen a un formato alternativo (normalmente, el formato [E.164](https://go.microsoft.com/fwlink/?linkid=859014)) para poder autorizar las llamadas y enrutarlas.</span><span class="sxs-lookup"><span data-stu-id="76135-686">A [Dial Plan](https://docs.microsoft.com/SkypeForBusiness/what-are-calling-plans-in-office-365/what-are-dial-plans), in the Phone System feature of Office 365, is a set of normalization rules that translates dialed phone numbers into an alternate format (typically [E.164](https://go.microsoft.com/fwlink/?linkid=859014) format) for call authorization and call routing.</span></span> 

<span data-ttu-id="76135-687">Con un plan de marcado, los usuarios pueden marcar números de teléfono de la forma habitual, como, por ejemplo, omitiendo el código de área para las llamadas locales, omitiendo el prefijo internacional para las llamadas nacionales o incluso usando una marcación de dígitos breve cuando se realizan llamadas telefónicas.</span><span class="sxs-lookup"><span data-stu-id="76135-687">A dial plan allows users to dial phone numbers the way they are accustomed to, such as omitting area code for local calls, omitting country code for domestic calls, or even using short digit dialing when placing a phone call.</span></span>

<span data-ttu-id="76135-688">Dentro de la característica de Sistema telefónico de Office 365, existen dos tipos de planes de marcado:</span><span class="sxs-lookup"><span data-stu-id="76135-688">Within the Phone System feature of Office 365, there are two types of dial plans:</span></span>

-   <span data-ttu-id="76135-689">**Plan de marcado de servicio**.</span><span class="sxs-lookup"><span data-stu-id="76135-689">**Service dial plan**.</span></span> <span data-ttu-id="76135-690">Es el plan de marcado predeterminado y se aplica a los usuarios en función de la ubicación de uso de Office 365. No se puede modificar.</span><span class="sxs-lookup"><span data-stu-id="76135-690">This is the default dial plan and applied to users based on Office 365 usage location, and it cannot be modified.</span></span>
-   <span data-ttu-id="76135-691">**Plan de marcado de inquilino**.</span><span class="sxs-lookup"><span data-stu-id="76135-691">**Tenant dial plan**.</span></span> <span data-ttu-id="76135-692">Este plan de marcado se puede personalizar dentro de un inquilino y se puede volver a dividir en dos tipos:</span><span class="sxs-lookup"><span data-stu-id="76135-692">This is a customizable dial plan within a tenant, and further divided into two types:</span></span>

> [!NOTE]

<span data-ttu-id="76135-696">El plan de marcado efectivo que se asigna a los usuarios es la combinación del plan de marcado de servicio (basado en la ubicación de uso de Office 365 del usuario) y el plan de marcado de inquilino (ya sea el de inquilino global o el de usuario de inquilino).</span><span class="sxs-lookup"><span data-stu-id="76135-696">The effective dial plan assigned to users is the combination of service dial plan (based on user’s Office 365 usage location) and tenant dial plan (can be either tenant-global dial plan or tenant-user dial plan).</span></span>
