# <a name="managing-room-software-stack"></a>Administrar la pila de software de sala  
En cada sala, las distintas partes de la pila de software trabajan juntas para ofrecer la experiencia. A un nivel alto, se pueden resumir como sigue: 

![Captura de pantalla del resumen de la pila de software](../media/update-management-006.jpg)

Para garantizar una experiencia de reunión positiva todos los días en cada sala, es importante **estandarizar cada parte del software** que se ejecuta en cada una de las salas e implementar correcciones en las salas según la configuración. También es habitual que los clientes configuren su propio proceso de validación para ganar confianza en las correcciones pendientes, comprobando que realmente solucionan un problema que podrían estar experimentando o las características de vista previa para que la organización esté preparada para el cambio.  

Servicios administrados para Salas de Microsoft Teams controla todo lo anterior para que no tenga que preocuparse por obtener una actualización o solucionar problemas con una actualización. En esta sección, describiremos cómo funcionan las actualizaciones de software administrado.  

## <a name="managing-to-a-good-state"></a>Administrar a un estado correcto 
Nuestro objetivo principal es mantener las salas en funcionamiento y disponibles para su organización. Los expertos de Servicios administrados trabajan constantemente en su nombre para ajustar, validar e implementar actualizaciones en sus salas de producción. En algunos casos, puede significar que priorizamos el estado del salón sobre las nuevas características y versiones que se validarán antes de que se implementen.

Lo más importante es que no tiene que preocuparse por invertir tiempo para obtener información detallada sobre estas actualizaciones y validarlas usted mismo. Si hay problemas con alguna actualización específica y con cualquier sala específica, trabajaremos con usted para solucionarlos y resolverlos.  

Por lo tanto, el proceso de implementación de las actualizaciones se puede dividir en lo siguiente:

- El equipo de Servicios administrados identificará de forma proactiva las actualizaciones, las validará y las promoverá a su entorno.
- Novedades entregados a sus habitaciones se ordenarán para evitar conflictos e interacciones y, a continuación, se programarán para evitar horas de funcionamiento regulares.
- El sistema de anillos le proporciona la capacidad de estar al tanto de las actualizaciones que se aplican a sus habitaciones sin que usted haga ninguna de las tareas de las piernas.
- Servicios administrados supervisará las actualizaciones y avisará automáticamente a su centro de operaciones de salas para que realice un seguimiento de los problemas y trabaje con usted para resolverlos.
