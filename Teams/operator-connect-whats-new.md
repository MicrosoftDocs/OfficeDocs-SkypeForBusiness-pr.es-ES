# <a name="whats-new-for-operator-connect-operators"></a>Novedades para operadores de conexión de operadores

En este artículo se describen las novedades de Operator Connect. Vuelve a comprobarlo con frecuencia si hay actualizaciones.

## <a name="dr-to-oc-migration-api"></a>API de migración de DR a OC

El 1 de julio de 2022, Microsoft publicó una API para los operadores donde pueden migrar los números de Enrutamiento directo a Operador Connect. 

Para obtener más información, consulte [Operador Connect Swagger](https://operatorconnect.microsoft.com/api/swagger#tag/TnMigration-(New)).

## <a name="number-status"></a>Estado del número

El 1 de julio de 2022, Microsoft publicó una API para operadores que se usa para proporcionar información sobre el estado del operador sobre los números dados con respecto al inquilino consentido.

Para obtener más información, consulte [Operador Connect Swagger](https://operatorconnect.microsoft.com/api/swagger#operation/NumberStatus_GetNumberStatus).
