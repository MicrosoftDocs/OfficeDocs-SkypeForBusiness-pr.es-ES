# <a name="direct-routing-media-path-country-codes"></a><span data-ttu-id="3f2a4-103">Códigos de países de ruta de medios de enrutamiento directo</span><span class="sxs-lookup"><span data-stu-id="3f2a4-103">Direct Routing media path country codes</span></span>

<span data-ttu-id="3f2a4-104">Cuando se elige una ruta de enrutamiento para los medios, el enrutamiento directo, de forma predeterminada, asigna siempre un centro de recursos basado en la dirección IP pública del controlador de borde de sesión (SBC) y siempre selecciona la ruta más cercana al centro de recursos de SBC.</span><span class="sxs-lookup"><span data-stu-id="3f2a4-104">When choosing a routing path for media, Direct Routing, by default, always assigns a datacenter based on the public IP address of the Session Border Controller (SBC), and always selects the path closest to the SBC datacenter.</span></span>

<span data-ttu-id="3f2a4-105">Sin embargo, en algunos casos, la ruta de acceso a los medios predeterminados puede no ser la ruta de acceso de medios óptima; por ejemplo, una IP pública de un intervalo de Estados Unidos puede asignarse a un SBC ubicado en Europa.</span><span class="sxs-lookup"><span data-stu-id="3f2a4-105">However, in some cases the default media path might not be the optimal media path; for example, a public IP from a United States range might be assigned to an SBC located in Europe.</span></span> 

<span data-ttu-id="3f2a4-106">Si usas el parámetro-MediaRelayRoutingLocationOverride con los cmdlets New-CsOnlinePSTNGateway y set-CsOnlinePSTNGateway, puedes especificar la región preferida para el tráfico de multimedia.</span><span class="sxs-lookup"><span data-stu-id="3f2a4-106">By using the -MediaRelayRoutingLocationOverride parameter with the New-CsOnlinePSTNGateway and Set-CsOnlinePSTNGateway cmdlets, you can specify the preferred region for media traffic.</span></span> <span data-ttu-id="3f2a4-107">Por ejemplo, el siguiente comando especifica que la región preferida es Alemania:</span><span class="sxs-lookup"><span data-stu-id="3f2a4-107">For example, the following command specifies that the preferred region is Germany:</span></span>

<span data-ttu-id="3f2a4-108">Set-CSOnlinePSTNGateway-Identity sbc1.contoso.com – MediaRelayRoutingLocationOverride DE</span><span class="sxs-lookup"><span data-stu-id="3f2a4-108">Set-CSOnlinePSTNGateway -Identity sbc1.contoso.com –MediaRelayRoutingLocationOverride DE</span></span> 

<span data-ttu-id="3f2a4-109">Tenga en cuenta que Microsoft solo recomienda establecer este parámetro si el registro de llamadas indica claramente que la asignación predeterminada del centro de recursos para la ruta multimedia no usa la ruta más cercana al centro de trabajo de SBC.</span><span class="sxs-lookup"><span data-stu-id="3f2a4-109">Note that Microsoft only recommends setting this parameter if the call logs clearly indicate that the default assignment of the datacenter for the media path does not use the path closest to the SBC datacenter.</span></span>
