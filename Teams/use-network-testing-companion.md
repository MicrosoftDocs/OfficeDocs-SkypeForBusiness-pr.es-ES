<a name="use-the-network-testing-companion"></a>Use el Asistente para pruebas de red
=================================

El [Asistente de prueba de redes](https://www.powershellgallery.com/packages/NetworkTestingCompanion/1.5.4) de Skype empresarial y Microsoft Teams le ofrece una forma sencilla de probar la calidad de la red y la conexión con Skype empresarial online o Microsoft Teams. Puede interpretar fácilmente los resultados y compartirlos con administradores de red para obtener información sobre posibles problemas de red.

Puede usar esta herramienta en la fase de planeamiento de una implementación de Skype empresarial o Microsoft Teams. Si actualmente usa Teams o Skype empresarial online, puede usar esta herramienta para solucionar problemas de calidad de voz o simplemente comprobar la conexión de red antes de hacer una llamada o después de tener una experiencia de mala calidad.

> [!NOTE]
> No necesita una suscripción de Office 365 para descargar e instalar el Asistente para pruebas de red.

## <a name="download-the-tool-and-documentation"></a>Descargar la herramienta y la documentación

Descargue el complemento de prueba de redes de la [Galería de PowerShell](https://www.powershellgallery.com/packages/NetworkTestingCompanion/1.5.4).

Descargue la guía de [instalación, la guía para usuarios y](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/network-testing-companion.zip?raw=true)la guía de resultados.
