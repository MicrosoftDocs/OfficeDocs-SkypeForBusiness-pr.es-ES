# <a name="whats-the-status-of-my-port-orders-in-teams"></a>¿Qué es el estado de Mis pedidos de puerto en los equipos?

Puede ver el estado de su pedido de puerto yendo al **Centro de administración de equipos de Microsoft** > **portal heredado** > **voz** > **pedidos de puerto**. Cada estado de orden de puerto se mostrará en la columna **estado** . Si necesita ayuda, [soporte técnico de contacto para los productos de negocio - ayuda de administración](https://docs.microsoft.com/office365/admin/contact-support-for-business-products).

En la siguiente tabla se enumera los Estados de orden de puerto, así como las acciones que puede realizar si es necesario.

|**Estado**|**¿Puede ver el orden?**|**¿Puede editar el orden?**|**¿Puede cancelar el orden?**|**¿Puede eliminar el orden?**|**Descripción**|
|:-----|:-----|:-----|:-----|:-----|:-----|
|**Tratamiento** <br/> |Sí  <br/> |No  <br/> |Sí  <br/> |Compatibilidad para reuniones privadas con id. de conferencia de reunión dinámicos  <br/> |El administrador ha creado el pedido y se han recibido por Microsoft.  <br/> |
|**Ponerse en contacto con el operador** <br/> |Sí  <br/> |No  <br/> |Sí  <br/> |Compatibilidad para reuniones privadas con id. de conferencia de reunión dinámicos  <br/> |El orden se ha recibido y aprobado por Microsoft, y estamos trabajando con el operador de perder para obtenerlo apruebe.  <br/> |
|**Transferencia de aprobado** <br/> |Sí  <br/> |Firme orden Commitment(FOC)  <br/> |Sí  <br/> |Compatibilidad para reuniones privadas con id. de conferencia de reunión dinámicos  <br/> |El orden se ha aceptado por el operador de perder, y se ha establecido la fecha FOC.  <br/> |
|**Transferencia pendiente** <br/> |Sí  <br/> |Compatibilidad para reuniones privadas con id. de conferencia de reunión dinámicos  <br/> |No  <br/> |No  <br/> |La transferencia es menor que 24 horas, por lo que ya no se puede editar o cancelar el orden.  <br/> |
|**Error** <br/> |No  <br/> |Sí  <br/> |Sí   <br/> |Sí (en este momento, no se puede eliminar el orden de puerto si se ha producido un error. El orden de puerto debe volver a crearse o necesita para [admitir de contacto para productos de negocio: Ayuda de administración](https://docs.microsoft.com/office365/admin/contact-support-for-business-products).  <br/> |El operador de perder rechazó el orden.  <br/> |
|**Completado** <br/> |Sí  <br/> |Compatibilidad para reuniones privadas con id. de conferencia de reunión dinámicos  <br/> |No  <br/> |No  <br/> |Los números se han transferido correctamente.  <br/> |
|**Cancelar** <br/> |No  <br/> |Sí  <br/> |Compatibilidad para reuniones privadas con id. de conferencia de reunión dinámicos  <br/> |No  <br/> |El administrador ha cancelado el orden.  <br/> |
   
 **Para obtener instrucciones paso a paso, vea [transferir los números de teléfono para Office 365](/microsoftteams/transfer-phone-numbers-to-office-365).**
 
> [!NOTE]
> Si necesita obtener más números de teléfono, [póngase en contacto con el soporte de productos para empresas: ayuda para administradores](https://docs.microsoft.com/office365/admin/contact-support-for-business-products).
