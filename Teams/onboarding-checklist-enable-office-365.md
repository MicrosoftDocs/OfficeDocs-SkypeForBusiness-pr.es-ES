| 1  | Crear una organización de Microsoft 365 u Office 365| Antes de que su organización pueda disfrutar de las ventajas de Teams, debe configurar una organización de Microsoft 365 o Office 365. Si no tiene microsoft 365 o Office 365 organización, consulte las instrucciones en la columna **Información adicional**. | | [Configurar Microsoft 365 o Office 365 para empresas](https://support.office.com/article/Set-up-Office-365-for-business-6a3a29a0-e616-4713-99d1-15eda62d04fa) <br/><br/>[Si necesita asistencia adicional, el Microsoft FastTrack para Microsoft 365 o Office 365 equipo está disponible para ayudar](https://fasttrack.microsoft.com/office) |

| 3  | Asignar licencias de Microsoft 365 o Office 365 a la organización | Normalmente, usted o el administrador de licencias recibirá un vínculo de licencias de Microsoft para agregar sus licencias a Microsoft 365 o Office 365 organización. <br/><br/>Según el canal que haya usado para comprar licencias, es posible que deba visitar una de las guías que aparecen en la columna **Información adicional** . | | [Agregar licencias a una suscripción pagada por usar una clave de producto](https://support.office.com/article/Add-licenses-to-a-subscription-paid-for-using-a-product-key-4fb4bd7e-3920-4ce0-98fb-0c06e3fedf53) <br/><br/>[Agregar licencias a una suscripción comprada a través del Centro de servicios de licencias por volumen](https://support.office.com/article/Add-licenses-to-a-subscription-purchased-through-the-Volume-Licensing-Service-Center-82ba88fa-ebdf-4d44-a7b3-cea82b25d71a) |
| 4  | OPCIONAL: Habilitar créditos de comunicaciones | Créditos de comunicaciones es una característica opcional que se usa para proporcionar acceso gratuito a los números de teléfono puente de conferencia de su organización, o para dar a un organizador de conferencia la posibilidad de llamar a los participantes de la reunión internacional. <br/><br/>Además de la licencia estándar de Audioconferencia por usuario, las organizaciones de licencias por volumen pueden elegir una oferta de pago por minuto para habilitar las capacidades de Audioconferencia. <br/><br/>Decida si se necesitan créditos de comunicaciones para su organización y, si es así, habilítalos agregando la licencia complementaria Créditos de comunicaciones a su inquilino. | | [¿Qué son los Créditos de comunicaciones?](what-are-communications-credits.md) <br/><br/>[Configurar Créditos de comunicaciones para su organización](set-up-communications-credits-for-your-organization.md) <br/><br/>[Audioconferencia con pago por minuto](audio-conferencing-pay-per-minute.md) |
| 5  | Confirmar que las licencias están disponibles para Microsoft 365 o Office 365 organización | Vaya a la Centro de administración de Microsoft 365 y compruebe que se muestran las SKU de licencia y la cantidad esperada en su organización de Microsoft 365 o Office 365. <br/><br/>Use la referencia en **Información adicional** para que se le guiará a través de este proceso. | | [¿Qué suscripción de Microsoft 365 o Office 365 para empresas tengo?](https://support.office.com/article/What-Office-365-for-business-subscription-do-I-have-092252f8-08df-4cdb-a8d2-b8653caa29a1) |
| 6  | Configure el entorno para las identidades. | Los usuarios pueden crearse en Microsoft 365 o Office 365 directamente (en el modelo de implementación en línea) o sincronizarse desde Active Directory local a Microsoft 365 o Office 365 organización. <br/><br/>Determine si debe usar identidades de nube, identidades sincronizadas o identidades federadas. Determinar el tipo de identidad adecuado está fuera del ámbito de esta lista de comprobación; sin embargo, encontrará vínculos a información sobre estas opciones en la columna **Información adicional** . <br/><br/>**Nota:** Si usa identidades federadas o sincronizadas, asegúrese de que los nombres principales de usuario (UPN) locales coincidan con los UPN de Microsoft 365 o Office 365 y que todos los atributos necesarios estén configurados para sincronizarse con Azure AD Connect. Para los atributos necesarios para Teams, use la lista de atributos de Skype for Business Online. | | [Información sobre la identidad de Microsoft 365 o Office 365 y Azure Active Directory](https://support.office.com/article/Understanding-Office-365-identity-and-Azure-Active-Directory-06a189e7-5ec6-4af2-94bf-a22ea225a7a9) <br/><br/>[Prepararse para aprovisionar usuarios mediante la sincronización de directorios en Microsoft 365 o Office 365](https://support.office.com/article/Prepare-to-provision-users-through-directory-synchronization-to-Office-365-01920974-9e6f-4331-a370-13aea4e82b3e) <br/><br/>[Sincronización de Azure AD Connect: atributos sincronizados con Azure Active Directory](/azure/active-directory/connect/active-directory-aadconnectsync-attributes-synchronized) |
| 7  | Confirmar administradores de inquilinos | Trabaje con sus equipos de seguridad para desarrollar un modelo de administración de Microsoft 365 o Office 365. <br/><br/>Asegúrese de identificar y documentar a todos los administradores de inquilinos y servicios. | | [Acerca de los roles de administrador de Microsoft 365 o Office 365](https://support.office.com/article/About-Office-365-admin-roles-da585eea-f576-4f55-a1e0-87090b6aaa9d) |
| 8  | Implementar roles administrativos para su inquilino | Valide que el modelo de administración satisface las necesidades de su organización y asigne los roles de administrador de Microsoft 365 o Office 365 a sus administradores. | | [Asignar roles de administrador en Microsoft 365 o Office 365 para empresas](https://support.office.com/article/Assign-admin-roles-in-Office-365-for-business-eac4d046-1afd-4f1a-85fc-8219c79e1504) |
