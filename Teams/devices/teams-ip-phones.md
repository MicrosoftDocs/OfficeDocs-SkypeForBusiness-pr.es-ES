# <a name="ip-phones-certified-for-microsoft-teams"></a>Teléfonos IP certificados para Microsoft Teams

El programa de certificación de los teléfonos IP garantiza que los clientes que utilicen teléfonos de escritorio y teléfonos de conferencia certificados disfrutarán de una experiencia de alta calidad en llamadas de audio y reuniones.

## <a name="announcing-certified-for-microsoft-teams"></a>Anuncio de Certified for Microsoft Teams

Además de ofrecer a los clientes una gama de teléfonos IP certificados para Skype Empresarial, como parte del programa de certificación de teléfonos IP de Teams, anunciamos la disponibilidad de nuevo hardware para teléfonos IP certificado para Microsoft Teams. Este programa de certificación garantiza una barra de calidad más alta con objetivos de rendimiento y métricas de calidad de audio más elevados. Los proveedores de teléfonos están trabajando activamente para cumplir con esta barra superior y se cumplirán los nuevos requisitos este año natural. Algunos de nuestros socios también están trabajando para ofrecer soporte técnico para el servicio de Skype Empresarial en el mismo hardware que los teléfonos IP de Teams. Entre las ventajas del teléfono IP certificado de Teams se incluyen las siguientes:

- Experiencia nativa de Teams con integración de botones de hardware y notificaciones LED
- Integración con el calendario y unirse a una reunión con un solo toque
- Implementado y administrado a través de la solución de administración de dispositivos de Microsoft con integración de Intune
