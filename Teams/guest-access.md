-   <span data-ttu-id="9250a-130">Con acceso como invitado, un usuario invitado tiene acceso a un equipo específico de los equipos que se les invite al usuario y puede tener acceso a recursos específicos de equipo como canales y los archivos.</span><span class="sxs-lookup"><span data-stu-id="9250a-130">With guest access, a guest user has access to a specific Teams team that the user is invited into, and can access team-specific resources such as channels and files.</span></span> <span data-ttu-id="9250a-131">El usuario invitado también puede charlar con y llamar a los miembros del equipo.</span><span class="sxs-lookup"><span data-stu-id="9250a-131">The guest user can also chat with and call members of the team.</span></span>

-   <span data-ttu-id="9250a-132">Con la federación o el acceso externo, un usuario externo puede tener charlas de 1:1 y las llamadas con un usuario de otra organización mediante el uso de los equipos o Skype para la empresa.</span><span class="sxs-lookup"><span data-stu-id="9250a-132">With federation or external access, an external user can have 1:1 chats and calls with a user in another organization by using either Teams or Skype for Business.</span></span> <span data-ttu-id="9250a-133">El usuario federado no puede tener acceso a los recursos del equipo.</span><span class="sxs-lookup"><span data-stu-id="9250a-133">The federated user cannot access any team resources.</span></span> <span data-ttu-id="9250a-134">En comparación con acceso como invitado, el acceso externo permite la comunicación de toda la organización (si admitidas por directivas) sin una invitación explícita, pero se limita a conversaciones y llamar sólo a.</span><span class="sxs-lookup"><span data-stu-id="9250a-134">Compared to guest access, external access allows cross organization communication (if permitted by policies) without an explicit invitation, but is limited to chat and call only.</span></span>
