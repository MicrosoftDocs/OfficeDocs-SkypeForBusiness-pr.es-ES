## <a name="unclassified-streams"></a><span data-ttu-id="e85b8-244">Transmisiones sin clasificar</span><span class="sxs-lookup"><span data-stu-id="e85b8-244">Unclassified Streams</span></span>

<span data-ttu-id="e85b8-245">En CQD, una transmisión se marca como _Sin_ clasificar cuando se produce un error en la conectividad de establecimiento de conectividad interactiva (ICE) o cuando no se notifican todas las métricas necesarias para calcular la clasificación de la transmisión.</span><span class="sxs-lookup"><span data-stu-id="e85b8-245">In CQD, a stream is marked _Unclassified_ when Interactive Connectivity Establishment (ICE) connectivity fails or when all the metrics required to compute the stream classification are not reported.</span></span>

<span data-ttu-id="e85b8-246">Para verificar los errores de conectividad de ICE, examine las dimensiones "ICE de primera conectividad" e "ICE de segunda conectividad" para obtener un valor "FALLO".</span><span class="sxs-lookup"><span data-stu-id="e85b8-246">To check for ICE connectivity failures, examine the dimensions "First Connectivity Ice" and "Second Connectivity Ice" for a "FAILED" value.</span></span> <span data-ttu-id="e85b8-247">Si alguno de los valores indica un error, la transmisión se marca como _Sin clasificar._</span><span class="sxs-lookup"><span data-stu-id="e85b8-247">If either value indicates a failure, the stream is marked as _Unclassified_.</span></span>

<span data-ttu-id="e85b8-248">Si la conectividad de ICE se realiza correctamente para _una_ transmisión sin clasificar, es probable que la transmisión se considere _sin_ clasificar porque no se han notificado las métricas clave de la transmisión.</span><span class="sxs-lookup"><span data-stu-id="e85b8-248">If ICE connectivity succeeded for an _Unclassified_ stream, the stream is likely considered _Unclassified_ because key stream metrics were not reported.</span></span> <span data-ttu-id="e85b8-249">Hay algunas razones por las que puede que no se notifiquen estas métricas:</span><span class="sxs-lookup"><span data-stu-id="e85b8-249">There are a few reasons these metrics may not be reported:</span></span>
