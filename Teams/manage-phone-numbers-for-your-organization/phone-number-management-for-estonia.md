# <a name="phone-number-management-for-estonia"></a>Teléfono de números para Estonia

Use esta tabla para buscar información sobre cómo obtener y administrar números de teléfono en Estonia para Microsoft Teams.
  
Para obtener más información, consulte [Administrar los números de teléfono para su organización](manage-phone-numbers-for-your-organization.md).
  
|**¿Qué acción desea realizar?**|**Descargue los formularios correctos**|
|:-----|:-----|
|Obtenga nuevos números de teléfono de usuario o servicio que no aparecen en el Centro de administración de Microsoft Teams. |En Estonia, puede obtener números de teléfono para su organización con el centro Microsoft Teams administración. También puede enviar una nueva solicitud de número de teléfono descargando el formulario y ponerse en contacto con el Servicio de atención al cliente de [PTSN para obtener instrucciones.](contact-pstn-service-desk.md)<br/><br/> [Nueva solicitud de número de teléfono para Estonia (v.1.0).](<https://download.microsoft.com/download/4/2/c/42c560b3-c108-4f99-966f-674f19f7073a/New phone number request for Estonia (v.1.0) (en-us).pdf>) <br/><br/>Los nuevos pedidos pueden tardar de 1 a 4 semanas en completarse en este mercado dependiendo de si tenemos acciones de número en la región específica. |
|Transferirnos números de teléfono. | Descargue este formulario. <br/><br/>[Carta de autorización (LOA) para Estonia (todos los números) (v.1.0)](<https://download.microsoft.com/download/0/a/c/0ac70736-9dbd-4cd9-a195-d98ee5c5ae35/Estonia LOA.pdf>)<br/><br/>Rellene y póngase en contacto con el Servicio de soporte técnico de [PTSN para obtener instrucciones.](contact-pstn-service-desk.md) También debe incluir una factura de utilidad de la dirección de empresa a la que desea transferir números.<br/><br/>Transferir números puede tardar hasta 3 semanas en completarse en este mercado. Estos números no se pueden usar para conferencias en este momento.|
|Publicar números de teléfono en la guía telefónica nacional o cancelar su publicación.  <br/> |[Póngase en contacto con el Servicio de soporte técnico de PTSN](contact-pstn-service-desk.md) para obtener instrucciones sobre la lista de números de teléfono que desea publicar o anular la publicación. <br/> |
|Cambie el número de un usuario a un número de servicio.  <br/> |Póngase en contacto con el Servicio de soporte técnico de [PTSN](contact-pstn-service-desk.md) para obtener instrucciones sobre la lista de números de usuario que desea convertir en números de servicio. <br/> |
|Realizar actualizaciones del servicio de directorio para números de cola de llamadas o de operador automático|Para agregar o cambiar números de cola de llamadas o operadores automáticos, póngase en contacto con [el Servicio de asistencia RTC para obtener instrucciones.](contact-pstn-service-desk.md) |
