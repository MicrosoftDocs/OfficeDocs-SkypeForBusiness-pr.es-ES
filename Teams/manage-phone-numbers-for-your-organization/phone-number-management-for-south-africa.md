# <a name="phone-number-management-for-south-africa"></a>Teléfono de números para Sudáfrica

Use esta tabla para buscar información sobre cómo obtener y administrar números de teléfono en Sudáfrica para Microsoft Teams.
  
Para obtener más información, consulte [Administrar los números de teléfono para su organización](manage-phone-numbers-for-your-organization.md).
  
|**¿Qué acción desea realizar?**|**Descargue los formularios correctos**|
|:-----|:-----|
|Obtenga nuevos números de teléfono de usuario o servicio que no aparecen en el Centro de administración de Microsoft Teams. |En Sudáfrica, puede obtener números de teléfono para su organización con el centro Microsoft Teams administración. También puede enviar una nueva solicitud de número de teléfono descargando el formulario y ponerse en contacto con el Servicio de [atención al cliente de TNS para obtener instrucciones](contact-tns-service-desk.md).<br/><br/> [Nueva solicitud de número de teléfono para Sudáfrica (v.1.0)](https://download.microsoft.com/download/5/a/b/5abce7fe-6e62-4be0-8012-86e694dcf642/New-phone-number-request-South-Africa-(v.1.0)-(en-us)[59].pdf). <br/><br/>Los nuevos pedidos pueden tardar de 1 a 4 semanas en completarse en este mercado dependiendo de si tenemos acciones de número en la región específica. <br/><br/> Antes de que se le puedan asignar números de teléfono, debe descargar el formulario Conocer al [cliente,](https://download.microsoft.com/download/c/d/b/cdbdff1e-5aa6-4ec9-86ce-df6f0cfd6805/South-Africa-Know-Your-Customer-%28v.1.0%29-%28en-us%29%5b59%5d.pdf) completarlo en su totalidad y enviarlo a nosotros.|
|Transferirnos números de teléfono. | Descargue este formulario. <br/><br/>[Carta de autorización (LOA) para Sudáfrica (todos los números) (v.1.0)](<https://download.microsoft.com/download/e/9/5/e959053a-4b35-49f9-a23e-7085575c46f2/letter-of-authorization-(loa)-for-south-africa-(v-1.0)-(en-za).docx>)<br/><br/>Rellene el formulario y [póngase en contacto con el Servicio de atención al cliente de TNS para obtener instrucciones](contact-tns-service-desk.md). También debe proporcionar la factura más reciente del proveedor de servicios actual.<br/><br/>Transferir números puede tardar hasta 6 semanas en completarse en este mercado. Estos números no se pueden usar para conferencias en este momento.|
|Publicar números de teléfono en la guía telefónica nacional o cancelar su publicación.  <br/> |[Póngase en contacto con el Servicio de](contact-tns-service-desk.md) atención al cliente de TNS para obtener instrucciones sobre la lista de números de teléfono que desea publicar o anular la publicación. <br/> |
|Cambie el número de un usuario a un número de servicio.  <br/> |[Póngase en contacto con el Servicio de](contact-tns-service-desk.md) atención al cliente de TNS para obtener instrucciones sobre la lista de números de usuario que desea convertir en números de servicio. <br/> |
|Realizar actualizaciones del servicio de directorio para números de cola de llamadas o de operador automático|Para agregar o cambiar números de cola de llamadas o operadores automáticos, póngase en contacto [con el Servicio de asistencia de TNS para obtener instrucciones](contact-tns-service-desk.md). |
