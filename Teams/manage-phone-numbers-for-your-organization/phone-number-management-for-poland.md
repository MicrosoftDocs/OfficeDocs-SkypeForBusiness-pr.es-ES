# <a name="phone-number-management-for-poland"></a>Teléfono de números para Polonia

Use esta tabla para encontrar información sobre cómo obtener y administrar números de teléfono en Polonia para Microsoft Teams.
  
Para obtener más información, consulte [Administrar los números de teléfono para su organización](manage-phone-numbers-for-your-organization.md).
  
| ¿Qué acción desea realizar? | Descargue los formularios correctos |
|:-----|:-----|
|Obtenga nuevos números de teléfono de usuario o servicio que no aparecen en el Centro de administración de Microsoft Teams.  |En Polonia, puede obtener números de teléfono para su organización con el centro Microsoft Teams administración. También puede enviar una nueva solicitud de número de teléfono descargando el formulario y ponerse en contacto con el Servicio de atención al cliente de [TNS para obtener instrucciones.](contact-tns-service-desk.md)<br/><br/>[Nueva solicitud de número de teléfono para Polonia (v.1.0) (en-us).](<https://download.microsoft.com/download/a/1/3/a130f4bd-3d48-4918-aea2-62cc24874c78/New phone number request for Poland (v.1.0) (en-us).pdf>) <br/><br/> Antes de que se le puedan asignar números de teléfono, debe descargar el formulario Conocer al [cliente,](<https://download.microsoft.com/download/f/f/a/ffa2afce-f85a-44e7-be2b-f571f63b422d/Poland - Know Your Customer.pdf>)completarlo en su totalidad y ponerse en contacto con el Servicio de asistencia de [TNS](contact-tns-service-desk.md)para obtener instrucciones.<br/><br/>Teléfono números se ordenarán en bloques de 10. Los clientes solo se cobrarán por las licencias del plan de llamadas suscritas.<br/><br/>Los nuevos pedidos pueden tardar de 1 a 12 semanas en completarse en este mercado dependiendo de si tenemos acciones de número en la región específica.|
|Transferirnos números de teléfono.| Descargue este formulario. <br/>[Carta de autorización (LOA) para Polonia (todos los números) (v.1.0) (pl-pl)](<https://download.microsoft.com/download/f/e/1/fe1e6c18-d72c-4327-86c8-4cfb247c2c3f/Poland LOA.pdf>) <br/> <br/>Rellene y póngase en contacto con el Servicio de atención al cliente de [TNS para obtener instrucciones.](contact-tns-service-desk.md) <br/><br/>Transferir números puede tardar hasta tres semanas en completarse en este mercado. Estos números no se pueden usar para conferencias en este momento.  |
|Publicar números de teléfono en la guía telefónica nacional o cancelar su publicación.  <br/> |Póngase en contacto con el Servicio de atención al cliente de [TNS](contact-tns-service-desk.md) para obtener instrucciones sobre la lista de números de teléfono que desea publicar o anular la publicación. <br/> |
|Cambie el número de un usuario a un número de servicio.  <br/> |Póngase en contacto con el Servicio de atención al cliente de [TNS](contact-tns-service-desk.md) para obtener instrucciones sobre la lista de números de usuario que desea convertir en números de servicio. <br/> |
|Realizar actualizaciones del servicio de directorio para números de cola de llamadas o de operador automático|Para agregar o cambiar números de cola de llamadas o operadores automáticos, póngase en contacto con el Servicio de asistencia de [TNS para obtener instrucciones.](contact-tns-service-desk.md) |
