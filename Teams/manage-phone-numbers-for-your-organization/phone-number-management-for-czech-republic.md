# <a name="phone-number-management-for-the-czech-republic"></a>Administración de números de teléfono para la República Checa

[!INCLUDE [preview-feature](../includes/preview-feature.md)]

Use esta tabla para obtener información sobre cómo obtener y administrar números de teléfono en la República Checa para Microsoft Teams.

Para obtener más información, consulte [Administrar los números de teléfono para su organización](manage-phone-numbers-for-your-organization.md).

|**¿Qué acción desea realizar?**|**Descargue los formularios correctos**|
|:-----|:-----|
|Obtenga nuevos números de teléfono de usuario o servicio que no aparecen en el Centro de administración de Microsoft Teams. |En la República Checa, puede obtener números de teléfono para su organización mediante el Centro de administración de Microsoft Teams. También puede enviar una nueva solicitud de número de teléfono descargando el formulario Conocer el formulario de cliente y [ponerse en contacto con el Servicio de asistencia de TNS para obtener instrucciones](contact-tns-service-desk.md).<br/><br/>Para poder asignarle los números, debe descargar el formulario de información del cliente: <br/><br/><ul><li>[conocer el formulario de cliente de República Checa](https://download.microsoft.com/download/6/f/6/6f66c134-5ad4-42d8-b7f9-f857fb5fd4ca/KYC-CZ-CZ.docx)</ul></li> <br/>Los pedidos nuevos pueden tardar de dos a tres semanas en llenarse en este mercado.
|Transferirnos números de teléfono. | Descargue este formulario. <br/><br/>[Carta de autorización (LOA) para la República Checa (todos los números) (v.2.0) (cz-cz)](https://download.microsoft.com/download/6/f/6/6f66c134-5ad4-42d8-b7f9-f857fb5fd4ca/LOA-CZ-GEO-EN.pdf)<br/><br/>Rellénalo y [ponte en contacto con el servicio de asistencia de TNS para obtener instrucciones](contact-tns-service-desk.md). También debe incluir la factura más reciente de su operador actual para que se transfieran los números.<br/><br/>La transferencia de números puede tardar de dos a tres semanas en completarse en este mercado. <br/><br/>Estos números no se pueden usar para conferencias en este momento. |
|Publicar números de teléfono en la guía telefónica nacional o cancelar su publicación.  <br/> |[Póngase en contacto con el Servicio de asistencia de TNS para obtener instrucciones](contact-tns-service-desk.md) sobre la lista de números de teléfono que desea publicar o cancelar la publicación. <br/> |
|Cambie el número de un usuario a un número de servicio.  <br/> |[Póngase en contacto con el Servicio de asistencia de TNS para obtener instrucciones](contact-tns-service-desk.md) sobre la lista de números de usuario que desea convertir a números de servicio. <br/> |
|Realizar actualizaciones del servicio de directorio para números de cola de llamadas o de operador automático|Para agregar o cambiar los números de la cola de llamadas o del operador automático, póngase en contacto con [el Servicio de asistencia de TNS para obtener instrucciones](contact-tns-service-desk.md). |
