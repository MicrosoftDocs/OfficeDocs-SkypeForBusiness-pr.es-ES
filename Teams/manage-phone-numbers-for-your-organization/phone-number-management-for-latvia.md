# <a name="phone-number-management-for-latvia"></a>Teléfono de números para Letonia

Use esta tabla para encontrar información sobre cómo obtener y administrar números de teléfono en Letonia para Microsoft Teams.
  
Para obtener más información, consulte [Administrar los números de teléfono para su organización](manage-phone-numbers-for-your-organization.md).
  
|**¿Qué acción desea realizar?**|**Descargue los formularios correctos**|
|:-----|:-----|
|Obtenga nuevos números de teléfono de usuario o servicio que no aparecen en el Centro de administración de Microsoft Teams. |En Letonia, puede obtener números de teléfono para su organización mediante el centro Microsoft Teams administración. También puede enviar una nueva solicitud de número de teléfono descargando el formulario y ponerse en contacto con el Servicio de atención al cliente de [PTSN para obtener instrucciones.](contact-pstn-service-desk.md)<br/><br/>[Nueva solicitud de número de teléfono para Letonia (v.1.0).](<https://download.microsoft.com/download/7/4/7/7477191d-1b6d-4d6b-a97f-c958ae204db5/New phone number request for Latvia (v.1.0) (en-us).pdf>) <br/><br/> Antes de que se le puedan asignar números de teléfono, debe descargar el formulario Conocer al [cliente,](<https://download.microsoft.com/download/1/8/d/18d3c082-984f-4271-ba23-80225904c5e2/Latvia - Know Your Customer.pdf>)completarlo en su totalidad y enviarlo a nosotros.<br/><br/>Los nuevos pedidos pueden tardar de 1 a 4 semanas en completarse en este mercado dependiendo de si tenemos acciones de número en la región específica. |
|Transferirnos números de teléfono. | Descargue este formulario. <br/><br/>[Carta de autorización (LOA) para Letonia (todos los números) (v.1.0)](<https://download.microsoft.com/download/d/d/6/dd67eefd-1886-44fd-a845-127e5cadc1ee/Latvia LOA.pdf>) <br/><br/>Rellene y póngase en contacto con el Servicio de soporte técnico de [PTSN para obtener instrucciones.](contact-pstn-service-desk.md) También debe incluir una factura de utilidad de la dirección de empresa a la que desea transferir números.<br/><br/>Transferir números puede tardar hasta 3 semanas en completarse en este mercado. Estos números no se pueden usar para conferencias en este momento.|
|Publicar números de teléfono en la guía telefónica nacional o cancelar su publicación.  <br/> |[Póngase en contacto con el Servicio de soporte técnico de PTSN](contact-pstn-service-desk.md) para obtener instrucciones sobre la lista de números de teléfono que desea publicar o anular la publicación. <br/> |
|Cambie el número de un usuario a un número de servicio.  <br/> |Póngase en contacto con el Servicio de soporte técnico de [PTSN](contact-pstn-service-desk.md) para obtener instrucciones sobre la lista de números de usuario que desea convertir en números de servicio. <br/> |
|Realizar actualizaciones del servicio de directorio para números de cola de llamadas o de operador automático|Para agregar o cambiar números de cola de llamadas o operadores automáticos, póngase en contacto con [el Servicio de asistencia RTC para obtener instrucciones.](contact-pstn-service-desk.md) |
