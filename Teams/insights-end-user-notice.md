# <a name="education-insights-end-user-notice"></a>Insights para Educación aviso de usuario final

Microsoft proporciona esta aplicación, pero no forma parte de Microsoft Teams.
El uso de esta aplicación puede requerir una suscripción para Microsoft 365 otras servicios Microsoft la aplicación ("Servicios de aplicación").
El acceso a esta aplicación y a los servicios de aplicación correspondientes se rige por los términos existentes para dichos Servicios de aplicación que se aplican a usted a través de (a) su acuerdo con el Contrato de servicios de Microsoft o (b) el contrato de suscripción de su organización con Microsoft.
Es posible que pierda el acceso a esta aplicación si: i) la suscripción de su organización expira o ii) su organización decide finalizar el acceso a su suscripción, a la aplicación o a los Servicios de aplicación. La aplicación puede incluir componentes de terceros con licencia para usted en términos independientes.
Los avisos de terceros para dichos componentes de terceros pueden ser accesibles en línea en el aviso [de terceros.](https://eduinsights.microsoft.com/thirdpartynotice.txt)
