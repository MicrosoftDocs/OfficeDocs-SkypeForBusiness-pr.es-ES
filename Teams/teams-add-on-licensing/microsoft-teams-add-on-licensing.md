# <a name="microsoft-teams-add-on-licenses"></a>Microsoft licencias de complementos de Teams

## <a name="which-capabilities-can-i-get-with-add-on-licenses"></a>¿Qué funcionalidades puedo obtener con licencias de complementos?

Las licencias de complementos son licencias para funcionalidades específicas de Microsoft Teams. Le ofrecen la flexibilidad de agregar funcionalidades solo para los usuarios de su organización que las necesiten. Para agregar una funcionalidad, compre una licencia de complemento para cada usuario que la vaya a usar.

Las opciones de licencias complementarias disponibles dependen del plan de Microsoft 365 que tenga. Para ver qué opciones de llamadas de voz y audioconferencia están disponibles, consulte la sección [¿Qué capacidades de Microsoft están disponibles con mi plan?](#which-microsoft-capabilities-are-available-with-my-plan) de este artículo.

Esta es una lista de licencias de complementos de Teams y sus capacidades que puede agregar a su suscripción de Teams.

## <a name="which-microsoft-capabilities-are-available-with-my-plan"></a>¿Qué capacidades de Microsoft están disponibles con mi plan?

Todos los planes de Microsoft 365 permiten a los usuarios de Teams realizar llamadas de punto a punto dentro de su organización mediante la tecnología VoIP. Sin embargo, es posible que desee agregar distintas licencias para funcionalidades como realizar llamadas a números RTC o configurar un PBX.

### <a name="voice-and-pstn-connectivity"></a>Conectividad de voz y RTC

Tanto si es una pequeña empresa, una organización empresarial, una institución educativa, una agencia gubernamental o una organización sin ánimo de lucro, tiene tres maneras de obtener Teams Phone System y la conectividad con RTC:

- Compre un plan de Microsoft 365 Empresa, E1/E3, A1/A3 o G1/G3 y agregue **Teams Phone con** licencias de paquetes del plan de llamadas.
  - Este modelo le proporcionará capacidades de PBX del sistema telefónico de Teams y un plan de llamadas nacionales Microsoft.
  - Elija esta configuración si quiere que Microsoft controle toda la administración y facturación de números de teléfono.
  - Compruebe si la licencia del [paquete Teléfono de Teams con plan de llamadas está disponible en su región](../country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md).

- Compre un plan E1/E3, A1/A3 o G1/G3, agregue licencias **de Teléfono Teams Estándar** y elija un Microsoft plan de llamadas independiente u operador RTC de terceros.
  - Este modelo le proporcionará Teams Phone System, pero seguirá necesitando conectarse a la RTC.
  - Para ver todas las opciones de conectividad de RTC, consulte [Opciones de conectividad de RTC](../pstn-connectivity.md).
  - Para obtener información sobre todas las soluciones de voz, consulte [Planear la solución de voz de Teams](../cloud-voice-landing-page.md).

- Compre un plan E5/A5/G5 y agregue **planes de llamadas Microsoft** o conéctese a un proveedor de conectividad RTC de terceros.
  - Los planes de Microsoft 365 E5, A5 y G5 incluyen Teams Phone System, pero las organizaciones siguen necesitando elegir una [opción de conectividad con RTC](../pstn-connectivity.md).

## <a name="unassign-licenses-from-users"></a>Anular la asignación de licencias de usuarios

Existen dos formas de quitar el acceso de los usuarios a determinados productos: quitar la asignación de licencias de productos de usuarios específicos o cancelar la asignación de licencias de usuario de un producto específico.

Para obtener instrucciones sobre cómo cancelar la asignación de licencias, vea [Cancelar la asignación de licencias de usuarios](/microsoft-365/admin/manage/remove-licenses-from-users).
