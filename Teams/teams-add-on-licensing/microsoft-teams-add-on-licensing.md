# <a name="microsoft-teams-add-on-licenses"></a>Licencias de complementos de Microsoft Teams

## <a name="what-are-add-on-licenses"></a>¿Qué son las licencias complementarias?

Las licencias de complementarias son licencias para características específicas de Microsoft Teams. Le ofrecen la flexibilidad de agregar características solo para los usuarios de su organización que las necesiten. Para agregar una característica, compre una licencia complementaria para cada usuario que la usará.

## <a name="what-voice-features-are-available-with-my-plan"></a>¿Qué características de voz están disponibles con mi plan?

Vea qué licencias de complementos necesita para obtener características de voz en Teams, según el plan que tenga.

Dependiendo de las características de voz que desee, le recomendamos que compare si es más rentable cambiar a un plan que incluya esas características. Por ejemplo, para la mayoría de las organizaciones grandes, comprar un paquete de software en un plan de Microsoft 365 Enterprise dará como resultado un costo general más bajo. Por lo general, cuando compra varias licencias de complementos individualmente en lugar de como parte de un plan, es posible que termine con un costo combinado más alto.

#### <a name="small-business"></a>[**Empresa pequeña**](#tab/small-business/)

Si es una pequeña o mediana empresa (menos de 300 usuarios), tiene cuatro formas de obtener características de voz:

- Compre un plan de Microsoft 365 Empresa y Microsoft 365 Business Voice.
- Compre un plan Enterprise E1 o E3 y Microsoft 365 Business Voice.
- Compre un plan Enterprise E1 o E3 y agregue características de voz individualmente.
- Compre un plan Enterprise E5, que incluye características de voz.

Para Microsoft 365 Business Voice, tiene dos opciones entre las que elegir:

- **Microsoft 365 Business Voice con** un plan de llamadas: Obtén esto si quieres que Microsoft controle toda la administración de números de teléfono y la facturación.
- **Microsoft 365 Business Voice sin un plan** de llamadas (enrutamiento directo): Obtén esto si quieres que toda la administración de números de teléfono y la facturación sean manejadas por tu proveedor de telefonía de terceros. Tendrá que configurar e implementar [](../direct-routing-landing-page.md) enrutamiento directo para conectar los controladores de borde de sesión (SBC) locales a Microsoft 365 u Office 365.

Para obtener más información, vea [¿Qué necesito comprar para usar Microsoft 365 Business Voice?](../business-voice/what-to-buy.md).

Este es un resumen, basado en el plan de licencias.

  ![Opciones de plan de pequeña empresa](../media/licensing-smallbusiness-options.png)

Si las audioconferencias no son compatibles en su país o región, tiene la opción de obtener un plan de Microsoft 365 Enterprise E5 sin un plan de audioconferencia o un plan de Office 365 Enterprise E5 sin un plan de audioconferencia.

Si ya tiene un plan de Microsoft 365 Enterprise u Office 365 Enterprise, aún puede comprar licencias de complementos para características de voz individuales. Sin embargo, tiene la opción de cambiar a un plan [de Microsoft 365 Empresa](https://www.microsoft.com/microsoft-365/compare-all-microsoft-365-products?&activetab=tab:primaryr2) y a Microsoft [365 Business Voice.](https://www.microsoft.com/microsoft-365/business/business-voice) En algunos casos, dependiendo de las características que desee, puede ser más rentable obtener un plan de Microsoft 365 Empresa con Microsoft 365 Business Voice.

> [!NOTE]
> Si necesita usar un servicio telefónico de [terceros,](../direct-routing-landing-page.md) enrutamiento directo se implementa en su organización, o si los planes de llamadas no están disponibles en su país o región, debe usar enrutamiento directo para las características de voz.

#### <a name="trial-offers"></a>[**Ofertas de prueba**](#tab/trial/)

### <a name="trial-offers-for-licensing-plans"></a>Ofertas de prueba para planes de licencias

En respuesta a una necesidad creciente de que las personas trabajen de forma remota (o trabajen desde casa), Microsoft tiene varias ofertas de prueba gratuitas de Teams para organizaciones gubernamentales, educativas y sin ánimo de lucro. Las pruebas están disponibles para estos planes:

- Prueba de Office 365 Enterprise [E1](../e1-trial-license.md) (interrumpida a partir del 1 de julio de 2020), prueba de E3 y prueba de E5
- Prueba de Office 365 Educación [A1](../teams-edu-licensing.md)
- Prueba de Office 365 Government [G1](../g1-trial-license.md) (interrumpida a partir del 1 de julio de 2020)

Para suscribirse a una oferta de prueba, vea [Registrarse para obtener una oferta de prueba.](https://transform.microsoft.com/customer-trials?tab=requestnewtrial)

### <a name="trial-offers-for-voice-features"></a>Ofertas de prueba para características de voz

Este es un resumen de las ofertas de prueba de características de voz.

  ![Opciones basadas en pruebas](../media/licensing-trial-offers.png)

 El plan base no tiene por qué ser una oferta de prueba para usar ninguna de estas pruebas para las características de voz. Puede usar un plan de Microsoft 365 Empresa o un plan Enterprise, Government, Education o Nonprofit con cualquiera de las ofertas de prueba para características de voz. Para ver las ofertas de prueba y los complementos disponibles, en el Centro de administración de Microsoft 365, vaya a Servicios de compra de facturación, seleccione Otras categorías que le puedan interesar y, a continuación, seleccione  >   **Complementos.**
