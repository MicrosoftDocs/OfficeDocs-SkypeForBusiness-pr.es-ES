# <a name="office-365-enterprise-e5-without-audio-conferencing"></a>Office 365 Enterprise E5 (sin conferencias de audio)

Si ya ha comprado el plan Office 365 Enterprise E5 (sin conferencias de audio), estas son las licencias de complementos de Microsoft teams que necesita comprar para obtener más características.

|Para agregar esta característica|Esto es lo que debe hacer |
|:------------------|:--------------------------|
|Audioconferencia <br/> | Para obtener este complemento, use el [botón cambiar de plan](https://support.office.com/article/73318661-8f33-478b-bcc7-fb8d69dbb22a), si está disponible, para desplazarse al plan Office 365 Enterprise E5 que incluye audioconferencia. Además, obtendrá otras características. </li></ol> |
|¡Sistema telefónico incluido! <br/> | Compre un [plan de llamadas](../calling-plans-for-office-365.md) para Office 365. <br/>  O bien, use el [servicio telefónico de su proveedor existente](microsoft-teams-add-on-licensing.md#bkmk_existing).  <br/> |
|Se incluye el buzón de voz de nube. <br/> |No es necesario hacer nada más.  <br/> |
