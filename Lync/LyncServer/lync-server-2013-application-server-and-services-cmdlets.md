# <a name="application-server-and-services-cmdlets-in-lync-server-2013"></a>Cmdlets de servicios y servidor de aplicaciones en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-06-20_

Las aplicaciones de servidor hacen referencia a los programas individuales que se ejecutan en Microsoft Lync Server 2013. Puede administrar estas aplicaciones mediante cmdlets desde el shell de administración de Lync Server.

<div>

## <a name="application-server-and-services-cmdlets"></a>Cmdlets de servicios y servidor de aplicaciones

Use los siguientes cmdlets para administrar aplicaciones de Lync Server 2013.

**Servidor de aplicaciones y servicios**

  - <span></span>  
    [Set-CsApplicationServer](https://technet.microsoft.com/en-us/library/Gg398562(v=OCS.15))

<!-- end list -->

  - <span></span>  
    [Get-CsServerApplication](https://technet.microsoft.com/en-us/library/Gg425948(v=OCS.15))

  - <span></span>  
    [Nuevo: CsServerApplication](https://technet.microsoft.com/en-us/library/Gg398096(v=OCS.15))

  - <span></span>  
    [Remove-CsServerApplication](https://technet.microsoft.com/en-us/library/Gg398366(v=OCS.15))

  - <span></span>  
    [Set-CsServerApplication](https://technet.microsoft.com/en-us/library/Gg412850(v=OCS.15))

</div>
