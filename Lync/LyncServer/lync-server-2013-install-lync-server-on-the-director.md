Siga estos pasos para instalar los componentes de Lync Server 2013 en un director.

<div>

## <a name="to-install-lync-server-components-on-a-director"></a>Para instalar componentes de Lync Server en un director

1.  En el Asistente para la implementación de Lync Server, en la página Lync Server 2013, junto a **paso 2: configurar o quitar componentes de Lync Server**, haga clic en **Ejecutar**.

2.  En la página **instalar componentes de Lync Server** , haga clic en **siguiente** para configurar los componentes tal y como se definen en su topología publicada.

3.  Una vez completada la instalación de los componentes de Lync Server, haga clic en **Finalizar**.
