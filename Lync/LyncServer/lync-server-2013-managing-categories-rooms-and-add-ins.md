# Administrar categorías, salones de chat y complementos en Lync Server 2013

 

_**Última modificación del tema:** 2012-10-06_

En el Panel de control de Lync Server 2013 o con los cmdlets de Windows PowerShell, los administradores de Chat persistente pueden usar la página **Chat persistente** para crear categorías y complementos. Para administrar salones de Chat persistente, los administradores pueden usar los cmdlets de Windows PowerShell. Además, si el administrador de Chat persistente también está habilitado para SIP, puede usar el cliente de Lync para iniciar una página web y administrar salones de chat.

En los temas siguientes se describe cómo crear y trabajar con categorías y salones de chat.
