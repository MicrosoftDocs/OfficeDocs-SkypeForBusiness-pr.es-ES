# <a name="managing-categories-rooms-and-add-ins-in-lync-server-2013"></a>Administrar categorías, salones y complementos en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-10-06_

En el panel de control de Lync Server 2013 o mediante los cmdlets de Windows PowerShell, los administradores de chat persistente pueden usar la página **chat persistente** para crear categorías y complementos. Para administrar salones de chat persistente, los administradores pueden usar los cmdlets de Windows PowerShell. Como alternativa, si el administrador de chat persistente también está habilitado para SIP, puede usar el cliente de Lync para iniciar una página web para crear y administrar salones de chat.

En los temas siguientes se describe cómo crear y trabajar con categorías y salones de chat.
