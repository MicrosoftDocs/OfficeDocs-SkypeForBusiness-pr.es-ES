_**Última modificación del tema:** 2012-10-01_

Los servicios Lync Server 2013 y servidor de chat persistente usan una configuración de grupo de servidores *extendido* para la recuperación ante desastres. Un grupo de servidores extendidos es un grupo que tiene equipos que se distribuyen entre dos centros de datos físicos, pero que están dentro de un único sitio lógico de Lync Server.
