</div>

<div id="mainSection">

<span> </span>

_**Última modificación del tema:** 2012-10-01_

Los servicios Lync Server 2013, servidor de chat persistente usan ** una configuración de agrupación extendida para la recuperación ante desastres. Una agrupación extendida es una agrupación que tiene equipos que se distribuyen entre dos centros de datos físicos, pero que se encuentran dentro de un único sitio lógico de Lync Server.
