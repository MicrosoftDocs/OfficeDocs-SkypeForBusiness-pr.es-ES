# <a name="backing-up-lync-server-2013"></a>Copia de seguridad de Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2013-02-17_

En los procedimientos de esta sección se describe cómo realizar una copia de seguridad de Lync Server para que pueda recuperar el servicio en caso de una interrupción o un error.

Debe desarrollar una estrategia de copia de seguridad y recuperación y planear la organización, como se describe en [desarrollar una estrategia de copia de seguridad y restauración y planear Lync Server 2013](lync-server-2013-developing-a-backup-and-restoration-strategy-and-plan.md). Esta estrategia y el plan deben incluir los procedimientos específicos que tenga previsto usar. Use los procedimientos incluidos en los temas de esta sección, junto con las hojas de [cálculo de copias de seguridad y restauración de Lync server 2013](lync-server-2013-backup-and-restoration-worksheets.md), para documentar cómo planea realizar una copia de seguridad de la implementación de Lync Server específica.
