# Creación de una topología perimetral y de director Lync Server 2013

 

_**Última modificación del tema:** 2012-09-08_

La creación de la topología implica las siguientes tareas de planeación e implementación:

  - **Planificación**    Debe definirse una topología adecuada para la organización y deben identificarse los componentes necesarios para implementarla. Estos son los pasos habituales del proceso de planificación. Microsoft Lync Server 2013, Herramienta de planeación suministrado por Lync Server 2013 facilita el proceso de planificación, así como también incluye la posibilidad de introducir cambios a medida que los requisitos y los planes se van completando.

  - **Implementación**   La topología que defina con Generador de topologías es esencial para la implementación de cualquier servidor de Lync Server 2013. Si no finaliza la definición y publicación de la topología usando Generador de topologías como parte de sus esfuerzos de planeación, deberá completarla y hacer disponible la información desde los servidores perimetrales antes de implementar sus servidores perimetrales.

No puede implementar componentes del servidor perimetral hasta que haya implementado, como mínimo, un grupo de servidores interno, y deberá instalar Generador de topologías para implementar un grupo de servidores interno. En esta sección no se aborda la instalación de Generador de topologías porque forma parte del proceso de instalación del grupo de servidores interno.

Para obtener más información acerca de estas herramientas, consulte [Lista de comprobación de la implementación del acceso de usuarios externos en Lync Server 2013](lync-server-2013-deployment-checklist-for-external-user-access.md).


> [!NOTE]
> Si ha usado Generador de topologías anteriormente para definir una topología completa, incluida la topología perimetral, puede omitir las tareas <A href="lync-server-2013-define-your-edge-topology.md">Definir la topología perimetral en Lync Server 2013</A> y <A href="lync-server-2013-publish-your-topology.md">Publicar una topología en Lync Server 2013</A> de esta sección, deberá completar la tarea <A href="lync-server-2013-export-your-topology-and-copy-it-to-external-media-for-edge-installation.md">Exportar la topología de Lync Server 2013 y copiarla en un medio externo para la instalación perimetral</A>.
