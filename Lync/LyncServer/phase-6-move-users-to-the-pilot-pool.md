# Fase 6: Mover usuarios al grupo piloto

 

_**Última modificación del tema:** 2012-10-16_

Puede mover un solo usuario o grupos de usuarios a la nueva implementación de Lync Server 2013 mediante los dos métodos siguientes: el Panel de Control de Lync Server y el Shell de administración de Lync Server. En los temas de esta sección se describen las tareas que debe completar durante la implementación piloto, así como antes de mover la implementación de Lync Server 2013 desde una implementación piloto a una implementación de nivel de producción.
