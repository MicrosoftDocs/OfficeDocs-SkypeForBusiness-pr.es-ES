# <a name="planning-and-deploying-unified-contact-store-in-lync-server-2013"></a>Planeación e implementación de almacén de contactos unificados en Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-06-14_

El almacenamiento de contactos unificados es una característica que ofrece una experiencia con los contactos consistente en todos los productos de Microsoft Office. Esta característica permite a los usuarios almacenar toda la información de contacto en Exchange 2013 de modo que la información esté disponible globalmente en Lync, Exchange, Outlook y Outlook Web Access.
