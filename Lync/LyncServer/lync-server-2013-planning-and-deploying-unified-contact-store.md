# <a name="planning-and-deploying-unified-contact-store-in-lync-server-2013"></a>Planear e implementar el almacén de contactos unificado en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-06-14_

El almacén de contactos unificado es una característica que proporciona una experiencia de contacto coherente en todos los productos de Microsoft Office. Esta característica permite a los usuarios almacenar toda la información de contacto en Exchange 2013 de modo que la información esté disponible en todo el mundo a través de Lync, Exchange, Outlook y Outlook Web Access.
