# <a name="configuring-call-park-in-lync-server-2013"></a>Configuración del estacionamiento de llamadas en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-10-30_

El estacionamiento de llamadas permite a un usuario de Enterprise Voice poner una llamada en espera desde un teléfono y, después, recuperar la llamada más adelante marcando un número interno (conocido como una *órbita*de estacionamiento de llamadas) desde cualquier teléfono.

Los componentes que llaman a los usos de estacionamiento se instalan y habilitan automáticamente en el servidor front-end o el servidor Standard Edition al implementar la telefonía IP empresarial. Sin embargo, debe configurar el estacionamiento de llamadas antes de que esté disponible para los usuarios.

Esta sección le guiará a través de la configuración del estacionamiento de llamadas.
