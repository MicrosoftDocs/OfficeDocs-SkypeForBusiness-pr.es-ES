# <a name="managing-response-group-queues-in-lync-server-2013"></a>Administrar colas de grupo de respuesta en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-10-02_

Las colas contienen llamadas a un grupo de respuesta hasta que un agente responde a la llamada. Cuando administra una cola, asigna uno o varios grupos de agentes a la cola y especifica la configuración de la cola, como el número de llamadas que la cola puede mantener antes de realizar una acción de desbordamiento y el período de tiempo durante el que una llamada espera un agente antes de realizar un tiempo de espera ction. Cuando la aplicación de grupo de respuesta busca un agente disponible, busca en los grupos de agentes en el orden en que se enumeren.
