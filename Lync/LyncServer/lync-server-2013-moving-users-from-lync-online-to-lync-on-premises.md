# Mover usuarios de Lync Online a Lync local en Lync Server 2013

 

_**Última modificación del tema:** 2014-05-08_

¿Su organización ha implementado Lync Online antes de implementar Lync Server local? ¿Y ahora quiere mover cuentas de usuario habilitadas originalmente para Lync en Lync Online a la implementación de Lync Server local? Esta sección le puede ayudar a hacerlo.


> [!NOTE]
> Esta sección solo trata sobre mover usuarios habilitados originalmente para Lync en Lync Online. ¿Tiene cuentas de usuario que primero habilitó para Lync local y después movió a Lync Online? Si es así y quiere volverlas a mover a la implementación local, el proceso es sencillo: <A href="lync-server-2013-administering-users-in-a-hybrid-deployment.md">Administrar usuarios en una implementación híbrida de Lync Server 2013</A>.
