# <a name="replication-cmdlets-in-lync-server-2013"></a>Cmdlets de replicación en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-10-09_

Los cmdlets de replicación proporcionan una forma de supervisar y administrar la replicación de Lync Server. Estos cmdlets pueden usarse para configurar opciones de replicación, supervisar el progreso de la replicación y forzar manualmente la replicación en un servidor.

<div>

A continuación se presenta una lista de cmdlets directamente relacionados con la administración de replicaciones:
