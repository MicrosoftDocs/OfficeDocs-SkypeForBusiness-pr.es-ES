# Administrar la infraestructura de red de Lync Server 2013

 

_**Última modificación del tema:** 2014-02-11_

Microsoft Lync Server 2013 incluye compatibilidad para el servicio de control de admisión de llamadas (CAC) y el desvío de medios. Para implementar estas características, debe configurar una red de regiones, sitios, subredes, etc., que le permitan administrar el ancho de banda en situaciones en las que las transmisiones de audio y vídeo deban restringirse. También puede usar la tecnología de red Calidad de servicio (QoS) para ayudar a proporcionar una experiencia de usuario final óptima en las comunicaciones de audio y vídeo.

Puede usar el Panel de control de Lync Server para administrar el CAC y el desvío de medios. En los temas siguientes se indican los pasos para hacerlo.
