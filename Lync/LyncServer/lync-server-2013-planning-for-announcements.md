# Planear los anuncios en Lync Server 2013

 

_**Última modificación del tema:** 2012-09-13_

La Lync ServerAplicación de anuncio permite configurar la administración de llamadas telefónicas entrantes cuando el número marcado es válido para la organización, pero no se ha asignado a ningún usuario ni teléfono. Puede transferir estas llamadas a un destino predeterminado (número de teléfono, URI de SIP o correo de voz) o reproducir un anuncio de audio, o ambas opciones. La aplicación Anuncio contribuye a evitar situaciones en las que el autor de la llamada marca un número equivocado y oye el tono de línea ocupada o el cliente SIP recibe un mensaje de error. En esta sección se recoge información de planificación específica de la Aplicación de anuncio.
