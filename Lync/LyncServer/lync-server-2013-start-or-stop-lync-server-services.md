# Inicio o detención de los servicios de Lync Server 2013

 

_**Última modificación del tema:** 2014-02-05_

Puede usar Panel de control de Lync Server para iniciar o detener todos los servicios de Lync Server 2013 que se estén ejecutando en un equipo específico o iniciar y detener un servicio específico.

## Para iniciar o detener todos los servicios de Lync Server en un equipo

1.  Desde una cuenta de usuario que sea miembro del grupo RTCUniversalServerAdmins (o tenga derechos de usuario equivalentes), o se asigne al rol CsServerAdministrator o CsAdministrator, inicie sesión en cualquier equipo que esté en la red en la que haya implementado Lync Server 2013.

2.  Abra una ventana del explorador y después introduzca la dirección URL de administración para abrir el panel de control de Lync Server. Para más información sobre los diferentes métodos que puede usar para iniciar el panel de control de Lync Server, consulte [Abrir las herramientas administrativas de Lync Server](lync-server-2013-open-lync-server-administrative-tools.md).

3.  En la barra de navegación izquierda, haga clic en **Topología** y después en **Estado**.

4.  En la página **Estado**, ordene la lista o realice una búsqueda para encontrar el equipo que está ejecutando los servicios que desea iniciar o detener y después haga clic en los mismos.

5.  Haga clic en **Acción**.

6.  Haga clic en **Iniciar todos los servicios** o **Detener todos los servicios**.

## Para iniciar o detener un servicio específico

1.  Desde una cuenta de usuario que se asigne al rol CsUserAdministrator o CsAdministrator, inicie sesión en cualquier equipo en la implementación interna.

2.  Abra una ventana del explorador y después introduzca la dirección URL de administración para abrir el panel de control de Lync Server. Para más información sobre los diferentes métodos que puede usar para iniciar el panel de control de Lync Server, consulte [Abrir las herramientas administrativas de Lync Server](lync-server-2013-open-lync-server-administrative-tools.md).

3.  En la barra de navegación izquierda, haga clic en **Topología** y después en **Estado**.

4.  En la página **Estado**, ordene la lista o realice una búsqueda para encontrar el equipo que está ejecutando el servicio que desea iniciar o detener y después haga clic en el servicio.

5.  Haga clic en **Propiedades**.

6.  Ordene la lista de servicios, si fuera necesario, y haga clic en el servicio que desea iniciar o detener.

7.  Haga clic en **Acción**.

8.  Haga clic en **Iniciar servicio** o **Detener servicio**.

9.  Haga clic en **Cerrar**.
