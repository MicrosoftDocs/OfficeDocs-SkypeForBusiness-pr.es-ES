<span data-ttu-id="c4b91-p101">El informe de DNS, que forma parte de los informes de administrador, detalla todas las entradas recomendadas y conocidas del sistema de nombres de dominio (DNS) para las redes interna, perimetral y externa. Si el diseñador ha completado las ediciones del diagrama de red y ha establecido todas las direcciones IP y los nombres de dominio completos (FQDN) en sus valores de producción, el informe de DNS representa un recurso de configuración excelente. This report can also serve as an operational troubleshooting document.</span></span>
