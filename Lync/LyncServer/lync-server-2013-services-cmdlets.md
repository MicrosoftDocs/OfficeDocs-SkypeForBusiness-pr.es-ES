# <a name="services-cmdlets-in-lync-server-2013"></a>Cmdlets de servicios en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-06-20_

Los cmdlets de servicios y roles de servidor que se suministran con Microsoft Lync Server 2013 permiten devolver información sobre los distintos componentes que se han instalado como parte de la implementación de Lync Server 2013. Además, estos cmdlets facilitan la detención e inicio de los servicios de Lync Server.

<div>

A continuación se presenta una lista de cmdlets directamente relacionados con la administración de servicios:
