# <a name="services-cmdlets-in-lync-server-2013"></a>Cmdlets de servicios en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-06-20_

El rol de servidor y los cmdlets de servicios que se incluyen en Microsoft Lync Server 2013 le permiten devolver información acerca de los distintos componentes que se han instalado como parte de su implementación de Lync Server 2013. Además, estos cmdlets hacen que sea más fácil detener e iniciar los servicios de Lync Server.

<div>

## <a name="services-cmdlets"></a>Cmdlets de servicios

A continuación se muestra una lista de cmdlets que se relacionan directamente con la administración de servicios:

**Servicios**

  - <span></span>  
    [Get-CsService](https://technet.microsoft.com/en-us/library/Gg413038(v=OCS.15))
