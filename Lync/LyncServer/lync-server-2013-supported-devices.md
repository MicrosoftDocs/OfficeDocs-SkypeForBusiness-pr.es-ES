# Dispositivos admitidos

 

_**Última modificación del tema:** 2016-12-08_

Lync Server admite todos los teléfonos que se muestran en la tabla siguiente. Tenga en cuenta que no todos los teléfonos compatibles pueden ejecutar Lync Phone Edition. Use la tabla para saber qué móviles pueden ejecutar Lync Phone Edition y dónde puede obtener información detallada acerca de todos los teléfonos compatibles.


> [!NOTE]
> Se admiten tres tipos de teléfonos: teléfonos de escritorio (teléfono IP auricular o dispositivos USB que están diseñados para que los usen los empleados en su escritorio), dispositivos de conferencias (IP manos libres o teléfonos USB que están diseñados para usarse en las salas de reuniones) y teléfonos de área común (teléfonos IP que están diseñados para usarse en áreas compartidas (como vestíbulos, cocinas, plantas de fabricación y áreas de trabajo compartidas) y se pueden personalizar para proporcionar diferentes conjuntos de características de Lync Phone Edition.



Para obtener información detallada sobre todos los dispositivos optimizados para Lync, consulte "Teléfonos y dispositivos certificados para Microsoft Lync" en [http://go.microsoft.com/fwlink/?linkid=208938\&clcid=0xC0A](http://go.microsoft.com/fwlink/?linkid=208938%26clcid=0xc0a). Las tablas de comparación de teléfonos de este sitio web incluyen información sobre precios y especificaciones técnicas.
