# <a name="run-best-practices-analyzer"></a>Ejecutar el analizador de procedimientos recomendados

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-10-11_

La herramienta analizador de procedimientos recomendados de Office Communications Server 2007 R2 reúne información de configuración de una implementación de Office Communications Server 2007 R2 y determina si la configuración se establece de acuerdo con las prácticas recomendadas de Microsoft. Puede instalar la herramienta en un equipo cliente que ejecute Microsoft .NET Framework 2,0 o directamente en el servidor que ejecuta Office Communications Server 2007 R2. Le recomendamos que instale y ejecute esta herramienta en un equipo cliente. Las herramientas administrativas de Office Communications Server 2007 R2 también deben instalarse de forma local en el equipo cliente, de modo que el analizador de procedimientos recomendados pueda recopilar un conjunto completo de datos.

Puede descargar el analizador de procedimientos recomendados de Office Communications Server 2007 R2 del centro de descarga de [http://go.microsoft.com/fwlink/p/?LinkId=268702](http://go.microsoft.com/fwlink/p/?linkid=268702)Microsoft en.
