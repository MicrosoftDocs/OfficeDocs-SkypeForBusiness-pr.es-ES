# <a name="setting-up-the-infrastructure-for-archiving-in-lync-server-2013"></a>Configuración de la infraestructura para el archivado en Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-10-01_

Los requisitos de infraestructura para el archivado son los mismos que para la implementación de Lync Server, excepto para el almacenamiento. No se necesita ninguna configuración de infraestructura adicional, excepto la configuración del almacenamiento mediante el almacenamiento de Exchange 2013, las bases de datos de archivado o ambos. Para obtener más información sobre los requisitos de infraestructura para Lync Server 2013, consulte [determining Your Infrastructure Requirements for Lync server 2013](lync-server-2013-determining-your-infrastructure-requirements.md) en la documentación de planeación y [preparación de la infraestructura y los sistemas para Lync Server 2013](lync-server-2013-preparing-the-infrastructure-and-systems.md) en la documentación sobre implementación. Para obtener más información sobre los requisitos de almacenamiento para el archivado, consulte [Technical Requirements for archiving in Lync server 2013](lync-server-2013-technical-requirements-for-archiving.md) en la documentación de planeación, [configurar plataformas del sistema para el archivado](lync-server-2013-setting-up-system-platforms-for-archiving.md) en Lync Server 2013 en la documentación sobre implementación y [configurar el almacenamiento para el archivado en Lync Server 2013](lync-server-2013-setting-up-storage-for-archiving.md) en la documentación sobre implementación.
