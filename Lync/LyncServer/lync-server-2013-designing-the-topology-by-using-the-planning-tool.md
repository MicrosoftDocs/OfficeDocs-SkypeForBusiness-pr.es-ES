# Diseño de la topología para Lync Server 2013 con la herramienta de planeación

 

_**Última modificación del tema:** 2016-12-08_

La Microsoft Lync Server 2013, herramienta de planeación consiste en una herramienta de tipo encuesta controlada por un asistente que formula preguntas sobre la topología de Lync Server 2013 que se está diseñando. La Herramienta de planeación hace uso de la información provista y de las prácticas más adecuadas de capacidad y diseño de la topología para sugerir una topología en función de las respuestas proporcionadas. Puede descargar la Herramienta de planeación desde el Centro de descarga de Microsoft ([http://go.microsoft.com/fwlink/?LinkID=282725](http://go.microsoft.com/fwlink/?linkid=282725)).

El fin último de la Herramienta de planeación reside en simplificar las posibles dificultades que entraña el diseño de una topología de Lync Server 2013 completa. De igual modo, esta herramienta proporciona referencias contextuales a la documentación de planeación e implementación de la propia herramienta, siempre y cuando haya una conexión de Internet disponible que permita conectarse al sitio web de Microsoft TechNet.

Tras personalizar la topología con las direcciones TCP/IP y los nombres de dominio completos (FQDN) de la infraestructura, la Herramienta de planeación pone a disposición un conjunto de informes que cubren temas como, entre otros, la nomenclatura del Sistema de nombres de dominio (DNS), las reglas de firewall y los certificados.

La Herramienta de planeación también ofrece la posibilidad de exportar información en dos formatos:

  - Microsoft Excel

  - Microsoft Visio
