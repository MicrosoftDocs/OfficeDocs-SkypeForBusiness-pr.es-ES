# <a name="analyzing-and-resolving-issues-identified-by-best-practices-analyzer-in-lync-server-2013"></a>Analizar y resolver problemas identificados por Best Practices Analyzer en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-06-25_

El analizador de procedimientos recomendados no realiza cambios en su entorno para resolver problemas. Solo detecta posibles problemas y muestra información sobre cómo resolverlos. Si identifica los problemas que necesita resolver, debe determinar la solución adecuada. Los temas de esta sección le ayudan a identificar y resolver algunos de los problemas potenciales más importantes.
