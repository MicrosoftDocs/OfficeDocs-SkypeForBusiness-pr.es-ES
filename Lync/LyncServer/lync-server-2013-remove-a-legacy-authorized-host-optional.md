# Quitar un host autorizado heredado en Lync Server 2013 (opcional)

 

_**Última modificación del tema:** 2012-09-05_

Cuando migre un control remoto de llamadas a una implementación de Lync Server, debe quitar las entradas de host autorizado existentes (conocidas como *entradas de aplicación de confianza* en Lync Server) para cualquier puerta de enlace SIP/CSTA de su implementación heredada. Debe usar las herramientas administrativas para quitar las entradas de host autorizado de dichas implementaciones, respectivamente.
