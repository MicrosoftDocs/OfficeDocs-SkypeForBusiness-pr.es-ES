# <a name="configure-clients-for-migration"></a>Configurar los clientes para la migración

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-09-24_

Este tema contiene los pasos de implementación de cliente recomendados que debe realizar antes de migrar a Lync Server 2013. Estos cambios de configuración deben realizarse en Lync Server 2010.

<div>

## <a name="to-configure-clients-before-migration"></a>Para configurar los clientes antes de la migración:

1.  Implemente las actualizaciones más recientes de servidor, cliente y dispositivo (hotfixes) para Lync Server 2010. Para obtener la información más actualizada sobre Lync Server 2010, consulte actualizaciones del centro de recursos para Lync en <http://go.microsoft.com/fwlink/p/?linkid=232630>.

2.  En Lync Server 2010, use el filtrado de versiones de cliente para permitir solo a los clientes que tengan instaladas las actualizaciones más recientes.
