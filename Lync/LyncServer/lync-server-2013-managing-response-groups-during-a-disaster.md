# <a name="managing-response-groups-in-lync-server-2013-during-a-disaster"></a>Administración de grupos de respuesta en Lync Server 2013 durante un desastre

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-11-01_
