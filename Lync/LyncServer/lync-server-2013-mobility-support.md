# <a name="mobility-support-in-lync-server-2013"></a>Compatibilidad con movilidad en Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-06-29_

Con la movilidad de Lync Server 2013, los usuarios tienen acceso a la funcionalidad de Lync 2013 en sus dispositivos móviles. Entre los dispositivos móviles admitidos se incluyen los siguientes:

  - Dispositivos Apple iOS admitidos

  - Windows Phone

  - Android

  - Nokia

Para más detalles sobre las características y los requisitos de movilidad, consulte [Planning for Mobility in Lync Server 2013](lync-server-2013-planning-for-mobility.md) en la documentación referente a la planeación.
