# <a name="topology-builder-supports-sql-server-mirroring-in-lync-server-2013"></a>El generador de topologías admite la creación de reflejos de SQL Server en Lync Server 2013

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-09-26_

Lync Server 2013 admite la creación de reflejo de los servidores back-end para alta disponibilidad. Puede usar el generador de topologías para configurar la creación de reflejos, incluida la configuración de un testigo.
