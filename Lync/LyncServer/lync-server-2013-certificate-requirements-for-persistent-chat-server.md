# <a name="certificate-requirements-for-persistent-chat-server-in-lync-server-2013"></a>Requisitos de certificado para el servidor de chat persistente en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-10-03_

Para instalar el servidor de chat persistente, debe tener un certificado emitido por la misma entidad de certificación que el que usan los servidores internos de Lync Server 2013 para cada servidor que ejecuta los servicios Web de chat persistente para la carga y descarga de archivos. Asegúrese de que dispone de los certificados necesarios antes de iniciar la instalación del chat persistente, especialmente si está usando una CA externa.
