# <a name="managing-persistent-chat-user-access-in-lync-server-2013"></a>Administración del acceso de usuarios de chat persistente en Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

_**Última modificación del tema:** 2012-10-06_

En los temas siguientes se describe cómo usar el panel de control de Lync Server 2013 y la página de **chat persistente** para administrar el acceso de usuarios en chat persistente. Para administrar el servidor de chat persistente, debe disponer de derechos y permisos de administrador de chat persistente en el sistema. Al instalar el servidor de chat persistente, se especifican uno o más usuarios que tienen los permisos y derechos de administrador de chat persistente necesarios. Para conceder derechos y permisos de administrador a cuentas de usuario adicionales, use **administrar usuarios y grupos**de usuarios. Para obtener más información, consulte [habilitar a un usuario para administrar categorías, salones de chat y derechos y permisos de usuario en Lync Server 2013](lync-server-2013-enabling-a-user-to-manage-categories-chat-rooms-and-user-rights-and-permissions.md).
