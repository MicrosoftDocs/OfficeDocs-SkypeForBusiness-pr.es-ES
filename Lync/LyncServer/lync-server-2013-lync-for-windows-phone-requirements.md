# Requisitos de Lync para Windows Phone

 

_**Última modificación del tema:** 2016-12-08_

Microsoft Lync 2013 para Windows Phone proporciona mensajería instantánea (MI), presencia mejorada y telefonía para usuarios de su organización que se conectan desde un smartphone o un dispositivo móvil con Windows Professional. Los dispositivos móviles permiten a los usuarios ampliar el alcance de Lync 2013. En este tema se describen las consideraciones de planeación de Lync 2013 para Windows Phone que incluyen la identificación de requisitos previos y requisitos técnicos, los componentes requeridos y la guía de implementación.

## Requisitos previos de Lync para Windows Phone

A continuación se enumeran los requisitos previos de Lync 2013 para Windows Phone.

  - Windows Phone 8, con nombre en código “Apollo”, o la última versión.

  - El dispositivo Windows Phone debe tener las últimas actualizaciones disponibles en Microsoft. Si desea conocer más detalles, consulte el historial de actualización de Windows Phone 8 en <http://go.microsoft.com/fwlink/?linkid=281961>.

  - El dispositivo debe tener 12 MB de espacio disponible en disco.

  - El usuario debe tener un plan de voz y de datos de un operador.
