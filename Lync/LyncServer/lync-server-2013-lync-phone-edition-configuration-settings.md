# Opciones de configuración de Lync Phone Edition en Lync Server 2013

 

_**Última modificación del tema:** 2012-10-10_

Las opciones de configuración para dispositivos que ejecutan Lync Phone Edition son de aplicación global, pero también se pueden crear nuevas colecciones de configuraciones que se apliquen a un sitio concreto. Las colecciones incluyen opciones sobre seguridad SIP, bloqueo de dispositivos, etc.
