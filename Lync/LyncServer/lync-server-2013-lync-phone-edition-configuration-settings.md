# <a name="lync-phone-edition-configuration-settings-in-lync-server-2013"></a>Configuración de Lync Phone Edition en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-10-10_
