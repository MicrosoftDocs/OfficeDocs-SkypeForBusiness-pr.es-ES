# Modificar el filtro para direcciones URL predeterminado

 

_**Última modificación del tema:** 2012-06-26_

Al usar el filtro de mensajería instantánea, Lync Server 2013 proporciona un filtro global de direcciones URL que bloquea direcciones URL específicas que se incluyen en conversaciones de mensajería instantánea entre usuarios en la totalidad de la implementación de Lync Server 2013. Si usa Panel de control de Lync Server, puede llevar a cabo las siguientes acciones:

  - Bloquear todas las direcciones URL o un subconjunto de las mismas en conversaciones de mensajería instantánea.

  - Permitir todas las direcciones URL. A modo de opción, puede crear una notificación que se inserta al inicio de cada mensaje instantáneo con una dirección URL.

  - Permitir URL específicas e incluir una advertencia en cada mensaje instantáneo que incluya una dirección URL.

Por otra parte, puede elegir bloquear direcciones URL que contengan tipos de archivo específicos o bloquear solamente direcciones URL de Internet permitiendo direcciones URL que estén comprendidas en la zona de intranet local del servidor (direcciones URL de intranet) para que puedan pasar por el servidor. Para obtener más información acerca del filtrado de direcciones URL, consulte [Configuración de la transferencia de archivos y el filtrado de direcciones URL para la mensajería instantánea (MI) en Lync Server 2013](lync-server-2013-configuring-file-transfer-and-url-filtering-for-instant-messaging-im.md).
