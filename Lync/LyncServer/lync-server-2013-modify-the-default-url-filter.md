# <a name="modify-the-default-url-filter-in-lync-server-2013"></a>Modificar el filtro de direcciones URL predeterminado en Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-06-26_

Mediante el filtro de mensajería instantánea (mi), Lync Server 2013 proporciona un filtro de URL global que bloquea direcciones URL específicas contenidas en conversaciones de mensajería instantánea entre los usuarios de la implementación de Lync Server 2013. Mediante el uso del panel de control de Lync Server, puede hacer lo siguiente:

  - Bloquear todas las direcciones URL o un subconjunto de las mismas en conversaciones de mensajería instantánea.

  - Permitir todas las direcciones URL. A modo de opción, puede crear una notificación que se inserta al inicio de cada mensaje instantáneo con una dirección URL.

  - Permitir URL específicas e incluir una advertencia en cada mensaje instantáneo que incluya una dirección URL.

Por otra parte, puede elegir bloquear direcciones URL que contengan tipos de archivo específicos o bloquear solamente direcciones URL de Internet permitiendo direcciones URL que estén comprendidas en la zona de intranet local del servidor (direcciones URL de intranet) para que puedan pasar por el servidor. Para obtener más información sobre el filtrado de URL, consulte [configuración de transferencia de archivos y filtrado de URL para la mensajería instantánea (mi) en Lync Server 2013](lync-server-2013-configuring-file-transfer-and-url-filtering-for-instant-messaging-im.md).
