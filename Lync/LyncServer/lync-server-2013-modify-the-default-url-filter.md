# <a name="modify-the-default-url-filter-in-lync-server-2013"></a>Modificar el filtro de dirección URL predeterminado en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-06-26_

Al usar el filtro de mensajería instantánea (mi), Lync Server 2013 proporciona un filtro de URL global que bloquea las direcciones URL específicas contenidas en las conversaciones de mensajería instantánea entre los usuarios de toda la implementación de Lync Server 2013. Mediante el panel de control de Lync Server, puede hacer lo siguiente:

  - Bloquear todo o un subconjunto de direcciones URL en conversaciones de mensajes instantáneos.

  - Permitir todas las direcciones URL. Como opción, puede crear un aviso que se inserte al principio de cada mensaje instantáneo que contenga una dirección URL.

  - Permitir direcciones URL específicas e incluir una advertencia con cada mensaje instantáneo que contenga una dirección URL.

Además, puede elegir bloquear las direcciones URL que contienen determinados tipos de archivo o bloquear solo las direcciones URL de Internet permitiendo que las direcciones URL que se encuentran dentro de la zona de Intranet local del servidor (direcciones URL de la intranet) pasen a través del servidor. Para obtener más información sobre el filtrado de URL, consulte [configuración de la transferencia de archivos y filtrado de URL para mensajería instantánea (mi) en Lync Server 2013](lync-server-2013-configuring-file-transfer-and-url-filtering-for-instant-messaging-im.md).
