# <a name="trusted-applications-cmdlets-in-lync-server-2013"></a>Cmdlets de aplicaciones de confianza en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-06-20_

Una aplicación de confianza es una aplicación desarrollada por un tercero que recibe el estado de confianza para ejecutarse como parte de Microsoft Lync Server 2013, pero que no es una parte integrada del producto. Lync Server 2013 proporciona cmdlets que se pueden usar para configurar y administrar aplicaciones de confianza.

<div>
