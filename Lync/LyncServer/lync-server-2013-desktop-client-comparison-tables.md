# Tablas de comparación de clientes para Lync Server 2013

 

_**Última modificación del tema:** 2017-05-03_

En las tablas siguientes se comparan las características y prestaciones de clientes de Lync 2013 por las siguientes categorías:

  - Compatibilidad con presencia ampliada

  - Compatibilidad con contactos y grupos de contactos

  - Compatibilidad con la mensajería instantánea

  - Compatibilidad con conferencias

  - Compatibilidad con funciones de telefonía

  - Compatibilidad con usuarios externos

  - Compatibilidad con archivado, cumplimiento y registro

Estas tablas indican cuáles son las características que los usuarios de Lync tienen a su disposición en una implementación local de Lync Server 2013. Las mismas características están también disponibles para los usuarios de Lync Online y Office 365 a menos que se indique lo contrario.

> [!IMPORTANT]  
> Consulte <a href="lync-server-2013-mobile-client-comparison-tables.md">Tablas de comparación de clientes móviles para Lync Server 2013</a> para obtener información detallada sobre la disponibilidad de funciones en clientes móviles de Lync 2013.




> [!NOTE]
> <UL>
> <LI>
> <P>Para obtener detalles sobre los planes de suscripción de Lync Online, consulte la descripción del servicio de Lync Online en <A href="http://go.microsoft.com/fwlink/p/?linkid=282430">http://go.microsoft.com/fwlink/p/?LinkId=282430</A>.</P>
> <LI>
> <P>Lync Server 2013 es compatible con las siguientes versiones anteriores de clientes: Lync 2010, Lync 2010 Mobile, Lync Phone Edition y Lync 2010 Attendant. Para obtener información acerca de estos clientes, consulte la versión Lync 2010 de la Tabla de comparación de clientes en <A href="http://go.microsoft.com/fwlink/p/?linkid=213798">http://go.microsoft.com/fwlink/p/?LinkID=213798</A>.</P>
> <LI>
> <P>El cliente Lync 2013 admite audio y vídeo en un entorno de Infraestructura de escritorio virtual (VDI). Para obtener detalles acerca de las funciones de Lync 2013 en un entorno de VDI, consulte <A href="lync-server-2013-supported-virtualization-technologies-and-known-limitations.md">Tecnologías de virtualización admitidas y limitaciones comunes en Lync Server 2013</A>.</P></LI></UL>
