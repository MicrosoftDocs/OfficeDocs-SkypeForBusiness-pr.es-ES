# Configurar planes de marcado en Lync Server 2013

 

_**Última modificación del tema:** 2013-02-22_

Una plan de marcado de Lync Server 2013 es un conjunto determinado de reglas de normalización que convierten números de teléfono para una ubicación determinada, usuario individual u objeto de contacto con fines de autorización telefónica y enrutamiento de llamadas.
