# <a name="installing-the-operation-manager-agent-files-on-a-watcher-node-in-lync-server-2013"></a>Instalar los archivos del agente Operations Manager en un nodo de monitor en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-10-20_

De forma similar a la configuración de un servidor Lync para informar de alertas de componentes, un nodo de monitor de Lync Server 2013 requiere que se instalen los archivos del agente de System Center Operations Manager. Esto permitirá la ejecución de transacciones sintéticas y que se informen de los avisos al servidor de administración raíz de System Center Operations Manager.

Para instalar los archivos del agente, siga los mismos procedimientos que se describen en las secciones siguientes:

1.  [Instalar un certificado en un nodo de monitor ubicado fuera de la red perimetral de Lync Server 2013](lync-server-2013-installing-a-certificate-on-a-watcher-node-located-outside-the-perimeter-network.md)

2.  [Instalar los archivos del agente Operations Manager en Lync Server 2013](lync-server-2013-installing-the-operation-manager-agent-files.md)

3.  [Configuración del equipo de Lync Server 2013 para participar en la detección de System Center](lync-server-2013-configuring-the-lync-server-computer-to-participate-in-system-center-discovery.md)
