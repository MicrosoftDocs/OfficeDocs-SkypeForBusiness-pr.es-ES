# Instalación de los archivos del agente Operations Manager en un nodo de monitor

 

_**Última modificación del tema:** 2012-10-20_

Al igual que para configurar un Lync Server para que informe de avisos de componentes, el nodo de supervisión de Lync Server 2013 requiere que los archivos del agente System Center Operations Manager estén instalados. Esto permitirá la ejecución de transacciones sintéticas y que se informen de los avisos al servidor de administración raíz de System Center Operations Manager.

Para instalar los archivos del agente, siga los mismos procedimientos que se describen en las secciones siguientes:

1.  [Instalación de un certificado en un nodo de monitor localizado fuera de la red perimetral](lync-server-2013-installing-a-certificate-on-a-watcher-node-located-outside-the-perimeter-network.md)

2.  [Instalación de los archivos del agente Operations Manager](lync-server-2013-installing-the-operation-manager-agent-files.md)

3.  [Configuración del equipo de Lync Server para participar de la detección en System Center](lync-server-2013-configuring-the-lync-server-computer-to-participate-in-system-center-discovery.md)
