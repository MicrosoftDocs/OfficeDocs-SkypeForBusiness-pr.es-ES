# Quitar Lync para iPhone y iPad

 

_**Última modificación del tema:** 2013-02-19_

Para quitar Lync 2013 para iPhone o Microsoft Lync 2010 para iPad del dispositivo, realice los siguientes pasos:

1.  En la pantalla de inicio del dispositivo móvil, puntee y mantenga presionado el icono de Lync.

2.  Cuando los mosaicos empiecen a agitarse y aparezca la **X**, puentee la **X** para eliminar la aplicación.
