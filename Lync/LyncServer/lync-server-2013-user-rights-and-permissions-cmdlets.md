# <a name="user-rights-and-permissions-cmdlets-in-lync-server-2013"></a>Cmdlets de permisos y derechos de usuario en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

Los cmdlets de permisos de usuario se usan principalmente para administrar el control de acceso basado en roles (RBAC), la nueva tecnología para delegar el control administrativo de Microsoft Lync Server 2013.
