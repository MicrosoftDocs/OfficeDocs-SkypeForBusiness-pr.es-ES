# Configurar los intervalos numéricos de respuesta de llamadas en grupo

 

_**Última modificación del tema:** 2013-02-22_

La atención a llamadas grupales se basa en la Aplicación de estacionamiento de llamadas. Cuando la atención a llamadas grupales se implementa, la tabla de órbitas de estacionamiento de llamadas se debe configurar con los intervalos de números de teléfono que se hayan designado como números de grupo de atención de llamadas. Estos números de grupo son los números que los usuarios marcan para atender las llamadas que están sonando para otro usuario.

Al igual que sucede con los números de órbita de estacionamiento de llamadas, los números de grupo de atención a llamadas deben ser extensiones virtuales que no tengan asignado ningún usuario o teléfono. Cada grupo de servidores front-end en el que se implemente la atención a llamadas grupales puede tener uno o más intervalos de números de grupo de atención a llamadas. Estos intervalos pueden ser únicos en toda la implementación de Lync Server.
