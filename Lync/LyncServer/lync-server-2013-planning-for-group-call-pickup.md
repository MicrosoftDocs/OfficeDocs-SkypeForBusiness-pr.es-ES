# <a name="planning-for-group-call-pickup-in-lync-server-2013"></a>Planeación de la recogida de llamadas grupales en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2013-02-01_

La actualización acumulativa para Lync Server 2013: febrero de 2013 presenta la llamada de grupo como una nueva característica de telefonía IP empresarial. La recopilación de llamadas de grupo permite a los usuarios seleccionar llamadas que se llamen a otro usuario marcando un número de grupo de atención de llamadas. En esta sección se incluye información de planeación específica para la recogida de llamadas grupales.
