# Hacer una copia de seguridad de los datos y los sistemas

 

_**Última modificación del tema:** 2016-12-08_

Antes de iniciar la migración a Lync Server 2013, se recomienda realizar una copia de seguridad completa del sistema y documentar el sistema existente, incluido un inventario de las cuentas de usuario hospedadas en cada grupo de servidores, de modo que se pueda revertir a Office Communications Server 2007 R2 cuando sea necesario. Existen varios programas y herramientas para realizar copias de seguridad y restaurar datos, configuraciones y sistemas. Para obtener información detallada y los procedimientos, consulte [http://go.microsoft.com/fwlink/?linkid=168162\&clcid=0xC0A](http://go.microsoft.com/fwlink/?linkid=168162%26clcid=0xc0a)
