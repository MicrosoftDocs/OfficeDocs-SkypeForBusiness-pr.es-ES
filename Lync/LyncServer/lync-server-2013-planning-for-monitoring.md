# Planeamiento de la supervisión en Lync Server 2013

 

_**Última modificación del tema:** 2012-09-05_

El servicio de supervisión de Microsoft Lync Server 2013 ofrece una manera para que los administradores recopilen el uso, la tendencia y los datos de calidad de servicio para las sesiones de comunicación que tienen lugar en su organización. La supervisión en Lync Server 2013 ya no requiere un rol de servidor independiente; en su lugar, el servicio de supervisión se integra en cada servidor front-end. Sin embargo, de manera predeterminada la supervisión no está habilitada en Lync Server 2013. Este documento le ayudará a determinar si la supervisión debería estar o no habilitada en su organización.
