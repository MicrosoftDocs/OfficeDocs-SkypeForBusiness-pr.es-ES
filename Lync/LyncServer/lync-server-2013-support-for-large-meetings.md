# Compatibilidad con reuniones grandes en Lync Server 2013

 

_**Última modificación del tema:** 2012-10-01_

Lync Server 2013 admite reuniones de hasta 1000 participantes mediante conferencia de audio y video (A/V) y permite compartir presentaciones de PowerPoint. Esta compatibilidad requiere un grupo dedicado configurado para admitir reuniones grandes y administrado de manera que garantice que solo hospedará una reunión grande al mismo tiempo.

En esta sección se describe cómo admitir reuniones grandes mediante un grupo de Lync Server 2013 dedicado. Describe las consideraciones de escalabilidad y los requisitos de implementación para un grupo dedicado, incluidos los requisitos de topología, hardware, software y configuración. También proporciona un conjunto de procedimientos recomendados para admitir reuniones numerosas, un resumen de los métodos de prueba y los resultados de las pruebas de escalabilidad del servidor realizadas por el equipo de ingenieros de Lync Server, y respuestas a las preguntas más frecuentes sobre la compatibilidad para reuniones grandes.
