# <a name="whats-new-for-devices-in-lync-server-2013"></a>Novedades para dispositivos en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-06-22_

Lync Server 2013 incluye Lync Phone Edition, software que se ejecuta en dispositivos calificados y proporciona características de telefonía tradicionales y avanzadas, seguridad integrada, facilidad de administración y más. Lync Phone Edition funciona de la misma manera con Lync Server 2013 que con Lync Server 2010. Para obtener más información sobre las características más recientes relacionadas con los dispositivos, consulte [novedades para los dispositivos](http://go.microsoft.com/fwlink/p/?linkid=256490) en la biblioteca de TechNet de 2010 de Lync Server.
