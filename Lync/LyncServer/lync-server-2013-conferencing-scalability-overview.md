# <a name="overview-of-conferencing-scalability-in-lync-server-2013"></a>Información general sobre la escalabilidad de conferencias en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-10-01_

El tamaño máximo de una conferencia hospedada en Lync Server 2013 en un grupo compartido (es decir, un grupo de servidores que hospede todas las cargas de trabajo de Lync Server 2013, incluida la mensajería instantánea (mi) y la presencia, la Conferencia y la telefonía IP empresarial) es 250 usuarios.

Antes de analizar el uso de Lync Server para admitir reuniones más grandes, vamos a analizar la metodología de prueba de escalabilidad que usamos y la forma en que determinamos el modelo de 250-User para la prueba del grupo compartido.
