</div>

Los extremos de la aplicación son objetos de contacto dentro de los servicios de dominio de Active Directory. Puede usar cmdlets desde el shell de administración de Lync Server para administrar estos contactos en Microsoft Lync Server 2013.

Use los siguientes cmdlets para administrar los puntos de conexión de la aplicación.

**Puntos de conexión de la aplicación**

  - <span></span>  
    [Get-CsApplicationEndpoint](https://technet.microsoft.com/en-us/library/Gg398655(v=OCS.15))

  - <span></span>  
    [Move-CsApplicationEndpoint](https://technet.microsoft.com/en-us/library/Gg398188(v=OCS.15))
