# Accesibilidad en Lync Server 2013

 

_**Última modificación del tema:** 2016-12-08_

Microsoft se compromete a hacer más sencillos sus productos y servicios para cualquiera que los use. Las siguientes secciones ofrecen información acerca de las características, productos y servicios que ayudan a hacer Lync Server 2013 más accesible para las personas con discapacidades.

## Características de accesibilidad de Lync Server 2013

Las siguientes características de Lync Server 2013 ayudan a hacerlo más accesible para personas con discapacidades:

  - Métodos abreviados de teclado.

  - Texto alternativo para ilustraciones

Además, algunas herramientas y características de accesibilidad de Windows pueden beneficiar a usuarios de Lync Server con discapacidades. Los cambios de tamaño y color de Windows PowerShell ofrecen opciones de accesibilidad al usar el Shell de administración de Lync Server. Para obtener detalles acerca de las opciones de accesibilidad de Windows PowerShell, vea “Accessibility in Windows PowerShell 2.0” en la biblioteca de TechNet en [http://go.microsoft.com/fwlink/?linkid=98964\&clcid=0xC0A](http://go.microsoft.com/fwlink/?linkid=98964%26clcid=0xc0a).

## Texto alternativo para ilustraciones

Todas las ilustraciones de la Ayuda de Lync Server 2013, incluidas las capturas de pantalla, los diagramas, los diagramas de flujo y otras ilustraciones, tienen texto alternativo asociado. Los usuarios con dificultad para ver ilustraciones pueden pausar el cursor en la ilustración para leer el texto alternativo. El texto alternativo describe lo que se muestra en la ilustración.

## Características de accesibilidad de Windows

El sistema operativo de Windows tiene muchas características de accesibilidad integradas que son útiles para individuos que tienen dificultad a la hora de escribir o de usar un mouse, son invidentes o tiene problemas de visión, o tienen discapacidades auditivas. Las características se instalan durante la instalación. Para obtener detalles acerca de estas características, vea la Ayuda de Windows o la accesibilidad de Microsoft en [http://go.microsoft.com/fwlink/?linkid=18139\&clcid=0xC0A](http://go.microsoft.com/fwlink/?linkid=18139%26clcid=0xc0a).

  - **Tutoriales detallados gratuitos**   Microsoft ofrece una serie de tutoriales detallados que proporcionan procedimientos detallados para ajustar los valores y las opciones de accesibilidad en el equipo. Esta información se presenta en un formato paralelo de manera que puede obtener información acerca de cómo usar el mouse, el teclado o una combinación de ambos.
    
    Para encontrar tutoriales detallados para productos de Microsoft, vea la accesibilidad de Microsoft en [http://go.microsoft.com/fwlink/?linkid=18139\&clcid=0xC0A](http://go.microsoft.com/fwlink/?linkid=18139%26clcid=0xc0a).

  - **Productos de tecnología de asistencia para Windows**   Hay disponible una amplia variedad de productos de tecnología de asistencia para facilitar el uso de los equipos para personas con discapacidades. Puede buscar en un catálogo de productos de tecnología de asistencia que se ejecutan en Windows en el sitio web de accesibilidad de Microsoft en [http://go.microsoft.com/fwlink/?linkid=18139\&clcid=0xC0A](http://go.microsoft.com/fwlink/?linkid=18139%26clcid=0xc0a).
    
    Si usa tecnología de asistencia, asegúrese de ponerse en contacto con su distribuidor de tecnología de asistencia antes de actualizar el software o hardware para comprobar que no haya problemas de compatibilidad.

Microsoft quiere ofrecer la mejor experiencia posible para todos nuestros clientes, incluso para las personas con discapacidades. Si necesita ayuda, póngase en contacto con nuestro equipo de soporte técnico de accesibilidad, que está entrenado para ayudar a las personas con discapacidades, ya sea por teléfono o correo electrónico.

## Servicio al cliente para personas con dificultades auditivas

Si tiene discapacidades auditivas, hay disponible un acceso completo a los servicios al cliente y productos de Microsoft a través de un servicio de teléfono de texto (TTY/TDD):

  - En el caso del servicio al cliente, póngase en contacto con Microsoft Sales Information Center en el (800) 892-5234 entre 6:30 y 17:30 P.M. (hora occidental de los Estados Unidos) de lunes a viernes, excepto festivos.

  - Para la asistencia técnica en Estados Unidos, póngase en contacto con los Servicios de soporte técnico de Microsoft en el (800) 892-5234 entre las 6:00 de la mañana y las 6:00 de la tarde, hora del Pacífico, de lunes a viernes, excluyendo vacaciones. En Canadá, marque (905) 568-9641 entre las 8 de la mañana y las 8 de la tarde, hora oriental, de lunes a viernes, excluyendo vacaciones.

Los servicios de soporte técnico de Microsoft están sujetos a los precios, términos y condiciones implementados en el momento en que se usa el servicio. Para obtener detalles, vea el sitio web Soporte Microsoft en [http://go.microsoft.com/fwlink/?linkid=18142\&clcid=0xC0A](http://go.microsoft.com/fwlink/?linkid=18142%26clcid=0xc0a).
