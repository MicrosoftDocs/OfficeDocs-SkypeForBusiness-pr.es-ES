# <a name="configure-iis-for-lync-server-2013"></a>Configurar IIS para Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2011-12-16_

La configuración de Internet Information Services (IIS) para Lync Server 2013 implica la instalación de los componentes correctos para admitir los servicios web que necesita Lync Server 2013. Para obtener más información sobre cómo instalar IIS, consulte [configuración de IIS en Lync Server 2013](lync-server-2013-iis-configuration.md). Si tiene una directiva para ejecutar el Asistente para configuración de seguridad en los servidores antes de ponerlos en servicio o como una parte típica del mantenimiento, consulte [reactivate Server After Security Configuration Wizard cierra los puertos en IIS](lync-server-2013-re-activate-server-after-security-configuration-wizard-closes-ports-in-iis.md) para obtener información sobre un efecto secundario de la ejecución del asistente que cerrará los puertos en una configuración de IIS de Lync Server 2013.
