# Administración de llamadas a números no asignados en Lync Server 2013

 

_**Última modificación del tema:** 2012-11-01_

Lync Server le permite configurar cómo se tratan las llamadas telefónicas entrantes si el número marcado es válido para su organización, pero no está asignado a un usuario o teléfono. Puede usar la Aplicación de anuncio para transferir dichas llamadas a un destino predeterminado (número de teléfono, SIP URI o buzón de voz), reproducir un anuncio de audio o ambas cosas. También puede transferir esas llamadas a un número de teléfono de operador automático de MU de Exchange. Al tratar las llamadas a números no asignados siguiendo uno de los métodos anteriores, contribuirá a evitar situaciones en las que el autor de la llamada marca el número equivocado y oye un tono de línea ocupada o el cliente SIP recibe un mensaje de error.

En esta sección se explica cómo administrar intervalos de números no asignados para tratar llamadas a números de teléfono no asignados. Aquí también se explica cómo administrar anuncios durante la recuperación ante desastres si desea utilizar esta funcionalidad cuando se produce una interrupción del servicio.


> [!NOTE]
> No es obligatorio tratar los números no asignados durante una interrupción del servicio.
