# <a name="preparing-active-directory-domain-services-for-lync-server-2013"></a>Preparación de los servicios de dominio de Active Directory en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2013-02-21_

Antes de implementar y operar Lync Server 2013, debe preparar los servicios de dominio de Active Directory extendiendo el esquema y, a continuación, creando y configurando objetos. Las extensiones de esquema agregan las clases y atributos de Active Directory que necesita Lync Server.

En los temas de esta sección se describe cómo preparar AD DS para la implementación de Lync Server y cómo asignar los permisos de configuración y unidad organizativa (OU). Para obtener más información sobre los cambios de esquema necesarios para Lync Server, vea [extensiones de esquema de Active Directory, clases y atributos usados por Lync server 2013](lync-server-2013-active-directory-schema-extensions-classes-and-attributes-used-by-lync-server.md).
