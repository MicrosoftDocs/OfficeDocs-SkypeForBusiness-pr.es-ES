# Nuevas características de supervisión en Lync Server 2013

 

_**Última modificación del tema:** 2012-11-08_

Lync Server 2013 no tiene un rol de servidor de supervisión diferente. La supervisión es una característica importante disponible en todos los servidores front-end de una implementación de Enterprise Edition y en los servidores de Standard Edition, que se pueden implementar y configurar para un grupo de servidores o un sitio. La novedad en Lync Server 2013 es que puede se habilitar la creación de reflejo de base de datos de SQL Server para la base de datos de supervisión.
