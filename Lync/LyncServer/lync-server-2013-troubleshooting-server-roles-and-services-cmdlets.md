# <a name="troubleshooting-server-roles-and-services-cmdlets-in-lync-server-2013"></a>Solución de problemas de los cmdlets de servicios y roles de servidor en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-08-27_

Los cmdlets de solución de problemas proporcionan distintas formas de comprobar que Microsoft Lync Server 2013 funciona según lo esperado. Por ejemplo, los cmlets CsHealthMonitoringConfiguration permiten configurar cuentas de prueba para los grupos de registradores y de directores. A su vez, estas cuentas de prueba son válidas para comprobar que los usuarios puedan completar correctamente tareas comunes como iniciar sesión en el sistema, intercambiar mensajes instantáneos o realizar llamadas a teléfonos de la red telefónica conmutada (RTC).
