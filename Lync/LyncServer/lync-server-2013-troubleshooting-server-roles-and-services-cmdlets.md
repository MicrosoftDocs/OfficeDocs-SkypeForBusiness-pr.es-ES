# <a name="troubleshooting-server-roles-and-services-cmdlets-in-lync-server-2013"></a>Solución de problemas de roles de servidor y cmdlets de servicios en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-08-27_

Los cmdlets de solución de problemas proporcionan distintas maneras de comprobar que Microsoft Lync Server 2013 funciona según lo esperado. Por ejemplo, los cmdlets CsHealthMonitoringConfiguration le permiten configurar cuentas de prueba para los grupos de registradores y directores. A su vez, puede usar estas cuentas de prueba para comprobar que los usuarios pueden completar correctamente tareas comunes como iniciar sesión en el sistema, intercambiar mensajes instantáneos o hacer llamadas a un teléfono que se encuentra en la red de telefonía pública conmutada (RTC).
