# Requisitos de puerto

 

_**Última modificación del tema:** 2013-03-27_

Lync Server requiere que estén abiertos determinados puertos del firewall. Además, si se ha implementado el protocolo de seguridad de Internet (IPSec) en la organización, IPSec debe estar deshabilitado en el intervalo de puertos que se usa para la entrega de audio, vídeo y vídeo panorámico.
