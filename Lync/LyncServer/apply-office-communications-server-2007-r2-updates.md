# Aplicar actualizaciones de Office Communications Server 2007 R2

 

_**Última modificación del tema:** 2016-12-16_

Antes de migrar a Lync Server 2013, aplique las actualizaciones en el entorno de Office Communications Server 2007 R2. Para obtener la información más reciente sobre Office Communications Server 2007 R2, vea las **Actualizaciones paraCommunications Server 2007 R2** en [http://go.microsoft.com/fwlink/p/?linkid=3052\&kbid=968802](http://go.microsoft.com/fwlink/p/?linkid=3052%26kbid=968802).

Para instalar actualizaciones de Office Communications Server 2007 R2, recomendamos que siga el procedimiento **Método 1: Programa de instalación de actualizaciones acumulativas del servidor** descrito en el artículo 968802 de Microsoft Knowledge Base que trata sobre las actualizaciones de Communications Server 2007 R2 en [http://go.microsoft.com/fwlink/p/?linkid=3052\&kbid=968802](http://go.microsoft.com/fwlink/p/?linkid=3052%26kbid=968802).
