# <a name="apply-office-communications-server-2007-r2-updates"></a>Aplicar las actualizaciones de Office Communications Server 2007 R2

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-10-19_

Antes de migrar a Lync Server 2013, es necesario aplicar las actualizaciones en el entorno de Office Communications Server 2007 R2. Para obtener la información más actualizada sobre Office Communications Server 2007 R2, consulte **actualizaciones para** **Communications Server 2007 R2** en [http://go.microsoft.com/fwlink/p/?linkid=3052\&kbid=968802](http://go.microsoft.com/fwlink/p/?linkid=3052%26kbid=968802).

Para instalar las actualizaciones de Office Communications Server 2007 R2, le recomendamos que siga el **método 1 de instalación de actualización de servidor acumulativa** que se describe en el artículo 968802 de Microsoft Knowledge base, "actualizaciones para Communications Server 2007 R2", en [http://go.microsoft.com/fwlink/p/?linkid=3052\&kbid=968802](http://go.microsoft.com/fwlink/p/?linkid=3052%26kbid=968802).
