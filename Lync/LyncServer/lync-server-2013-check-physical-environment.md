# <a name="performing-physical-environmental-checks"></a>Realizar comprobaciones físicas del entorno

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2014-04-30_

Antes de comprobar el rendimiento, la disponibilidad y la funcionalidad de la implementación de Lync Server 2013, debe comprobar el entorno físico. Por ejemplo, es posible que haya que reducir la temperatura del salón del servidor o que sea necesario reemplazar un cable de red. Para obtener los mejores resultados, realice los siguientes controles físicos del entorno:

  - **Medidas de seguridad física**   se debe proteger la protección de seguridad física, como bloqueos, puertas y salas de acceso restringido. Compruebe si hay entradas no autorizadas, o indicios de daños en el equipo.

  - ****   La temperatura y la humedad alta temperatura, el flujo de aire deficiente y la humedad pueden provocar el sobrecalentamiento de los componentes de hardware. Compruebe la temperatura y la humedad para asegurarse de que los sistemas de entorno, como el calentamiento y el aire acondicionado, pueden mantener condiciones aceptables y funcionar dentro de las especificaciones del fabricante del hardware. Cuando se haya instalado recientemente un nuevo equipo, compruebe también que el flujo de aire hacia y desde los servidores no está obstaculizado y cumple con las especificaciones del fabricante.

  - **Dispositivos y componentes**   la organización de Lync Server 2013 depende de una red física que funcione y de hardware relacionado. Asegúrese de que los enrutadores, conmutadores, concentradores, cables físicos y conectores estén operativos.

La información específica sobre cómo realizar estas comprobaciones dependerá en gran medida del sitio de instalación y del hardware del servidor que se haya elegido. La primera vez que realice esta comprobación, consulte la documentación del hardware y tenga en cuenta los parámetros deseados para futuras referencias.
