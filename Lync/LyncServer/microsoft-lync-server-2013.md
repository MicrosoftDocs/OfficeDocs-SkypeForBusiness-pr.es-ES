# Microsoft Lync Server 2013

 

_**Última modificación del tema:** 2016-12-08_

Microsoft Lync Server 2013 software de comunicaciones ofrece soluciones de mensajería instantánea (MI), presencia, conferencias y telefonía que cumplen los requisitos de colaboración necesarios en un entorno de empresa.

¿Quiere descargar una copia de la documentación de Lync Server 2013? Puede descargar el archivo de ayuda compilado **LyncServer2013\_ITPro** (.chm) del Centro de descarga de Microsoft en [http://go.microsoft.com/fwlink/p/?LinkID=331103](http://go.microsoft.com/fwlink/p/?linkid=331103).

En las siguientes tablas se enumeran los temas de la biblioteca de documentación de Lync Server 2013.
