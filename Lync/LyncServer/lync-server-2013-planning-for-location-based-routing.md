# <a name="planning-for-location-based-routing-in-lync-server-2013"></a>Planeación del enrutamiento de Location-Based en Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

_**Última modificación del tema:** 2013-07-31_

La información de este tema se refiere a las actualizaciones acumulativas para Lync Server 2013: febrero de 2013.

Location-Based enrutamiento permite restringir el enrutamiento de las llamadas entre los extremos de VoIP y los extremos de RTC en función de la ubicación de las partes en la llamada. El enrutamiento Location-Based forma parte de la infraestructura de Lync Server 2013 Enterprise Voice. El enrutamiento Location-Based es una característica de administración de llamadas que controla cómo se enrutan las llamadas por parte de Lync Server 2013 CU1. Aplica las reglas de autorización de llamadas en cuanto a si las llamadas se pueden enrutar a puntos de conexión de PBX o RTC en función de la ubicación geográfica del autor de la llamada de Lync.
