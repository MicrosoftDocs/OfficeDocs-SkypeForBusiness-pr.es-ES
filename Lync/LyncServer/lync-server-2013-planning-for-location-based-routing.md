# Planificar el enrutamiento basado en ubicación en Lync Server 2013

 

_**Última modificación del tema:** 2013-07-31_

La información de este tema se refiere a las actualizaciones acumulativas de Lync Server 2013: febrero de 2013.

El enrutamiento basado en ubicación permite restringir el enrutamiento de las llamadas entre extremos de VoIP y extremos de RTC según la ubicación de los interlocutores. El enrutamiento basado en ubicación forma parte de la infraestructura de Telefonía IP empresarial de Lync Server 2013. El enrutamiento basado en ubicación es una característica de administración de llamadas que controla la manera en la que CU1 de Lync Server 2013 redirige las llamadas. Exige la aplicación de reglas de autorización de llamadas que determinan si las llamadas se pueden redirigir a extremos de PBX o RTC, en función de la ubicación geográfica del autor de la llamada de Lync.
