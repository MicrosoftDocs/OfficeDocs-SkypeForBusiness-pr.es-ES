# <a name="archiving-and-monitoring-cmdlets-in-lync-server-2013"></a>Cmdlets de archivado y supervisión en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-10-04_

Los cmdlets de archivado y supervisión permiten a los administradores administrar los mensajes instantáneos y el archivado de sesiones de conferencia; para grabar información de detalles de llamadas; y para supervisar Microsoft Lync Server 2013 mediante la calidad de la experiencia (QoE).


> [!NOTE]
> Para obtener más información sobre los cmdlets, vea el&nbsp;blog de Windows PowerShell <A href="http://go.microsoft.com/fwlink/p/?linkid=263432">http://go.microsoft.com/fwlink/p/?linkId=263432</A>de Lync Server en. El contenido de los blogs y sus URL están sujetos a cambios sin previo aviso.
