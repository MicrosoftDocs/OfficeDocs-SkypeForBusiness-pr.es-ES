# Plataformas admitidas en Lync Web App en Lync Server 2013

 

_**Última modificación del tema:** 2017-02-07_

Para usar Lync Web App, debe disponer de una de las siguientes combinaciones de sistema operativo y explorador compatibles.


> [!NOTE]
> Algunas características de Lync Web App, como las de voz basada en equipo, vídeo, uso compartido y visualización de uso compartido de pantalla continuo, requieren un complemento. Puede instalar el complemento de uso compartido al unirse a la reunión o al iniciar una de estas características. 1

## Requisitos de hardware

Los requisitos de hardware del equipo los determinan el sistema operativo y el explorador. Las características de voz y telefonía requieren micrófono y altavoces, auriculares con micrófono o un dispositivo equivalente compatible con el equipo. Las características de vídeo necesitan un dispositivo de vídeo compatible con el equipo. Para información detallada sobre la compatibilidad del hardware de vídeo y la calidad de vídeo esperada, vea [Requisitos de vídeo del cliente de Lync para Lync Server 2013](lync-server-2013-lync-client-video-requirements.md).
