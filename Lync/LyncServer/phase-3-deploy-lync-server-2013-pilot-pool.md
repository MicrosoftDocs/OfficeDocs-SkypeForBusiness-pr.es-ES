# Fase 3: Implementar un grupo piloto de Lync Server 2013

 

_**Última modificación del tema:** 2012-10-19_

Esta sección abarca los pasos requeridos para implementar una implementación piloto del Lync Server 2013. La implementación de Lync Server 2013 requiere el uso del Generador de topologías para definir la topología y los componentes que desea implementar al preparar el entorno para la implementación de los componentes de Lync Server 2013, importar y publicar el diseño de topología en el primer servidor front-end y, luego, instalar y configurar el software de Lync Server 2013 para los componentes de la implementación. La implementación de grupos piloto de Lync Server 2013 coexistirá con el grupo de Lync Server 2010 existente.
