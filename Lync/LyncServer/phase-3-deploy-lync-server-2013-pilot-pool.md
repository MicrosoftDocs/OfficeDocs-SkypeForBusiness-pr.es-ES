# <a name="phase-3-deploy-lync-server-2013-pilot-pool"></a>Fase 3: implementar el grupo de pruebas piloto de Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-10-19_

En esta sección se describen los pasos necesarios para implementar un grupo piloto de Lync Server 2013. La implementación de Lync Server 2013 requiere el uso de generador de topologías para definir la topología y los componentes que desea implementar, preparar el entorno para la implementación de los componentes de Lync Server 2013, publicar el diseño de la topología en el primer front-end Servidor y, a continuación, instalar y configurar el software de Lync Server 2013 para los componentes de su implementación. Cuando se complete, la implementación del grupo piloto de Lync Server 2013 coexistirá con un grupo de servidores de Lync Server 2010 existente.
