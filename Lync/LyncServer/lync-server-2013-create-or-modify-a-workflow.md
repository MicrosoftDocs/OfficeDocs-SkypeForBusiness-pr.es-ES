# Creación o modificación de un flujo de trabajo en Lync Server 2013

 

_**Última modificación del tema:** 2012-10-02_

Lync Server 2013 admite dos tipos de flujos de trabajo: grupo de extensiones y respuesta interactiva de voz (IVR). Al crear un flujo de trabajo, se usa la Herramienta de configuración de grupo de respuesta para especificar la cola que se van a usar y otras opciones, como el mensaje de bienvenida, música en espera, horas de oficina y preguntas que la Aplicación de grupo de respuesta hace al autor de la llamada.


> [!NOTE]
> Debe crear grupos de agentes y colas antes de crear un flujo de trabajo que use dichos elementos. Si desea crear horas de oficina y vacaciones predefinidas que pueda usar en varios flujos de trabajo, también deberá definir dichas horas y vacaciones antes de crear un flujo de trabajo que las vaya a usar.
