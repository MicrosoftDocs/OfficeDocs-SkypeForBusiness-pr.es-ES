# Proporcionar conectividad RTC en un sitio de sucursal en Lync Server 2013

 

_**Última modificación del tema:** 2012-10-05_

Se recomienda usar la Microsoft Lync Server 2013, herramienta de planeación para agregar sitios de sucursal a su topología y para configurar la infraestructura de voz de sitios de sucursal.

Si no usa la Herramienta de planeación, siga los procedimientos descritos en los temas de esta sección, en primer lugar, para agregar los sitios de sucursal y, en segundo lugar, para configurar la infraestructura de voz mediante la definición de la puerta de enlace IP-RTC y la configuración del tronco SIP (con o sin desvío de medios). Otra opción consiste en conectar una central de conmutación (PBX) al sitio de sucursal.


> [!NOTE]
> Si desea proporcionar resistencia de sitios de sucursal, debe implementar una Aplicación de sucursal con funciones de supervivencia, un Servidor de sucursal con funciones de supervivencia o un Servidor Standard Edition en el sitio de sucursal. Para más información, vea <A href="lync-server-2013-deploying-a-survivable-branch-appliance-or-server.md">Implementar una aplicación o servidor de sucursal con funciones de supervivencia con Lync Server 2013</A> o <A href="lync-server-2013-deploying-lync-server.md">Implementar Lync Server 2013</A>, según corresponda, en la documentación sobre implementación.
