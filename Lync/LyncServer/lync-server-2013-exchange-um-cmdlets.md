# <a name="exchange-um-cmdlets-in-lync-server-2013"></a>Cmdlets de mensajería unificada de Exchange en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-06-26_

Microsoft Lync Server 2013 funciona junto con la mensajería unificada (UM) de Exchange para implementar el operador automático y el acceso de suscriptores para el correo de voz hospedado. Estas características se pueden administrar a través de cmdlets en el shell de administración de Lync Server.
