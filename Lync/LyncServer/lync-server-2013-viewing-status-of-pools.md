# <a name="viewing-status-of-pools-in-lync-server-2013"></a>Ver el estado de los grupos de servidores en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2014-05-01_

Para cada grupo de servidores Enterprise y el grupo de servidores de cada servidor Standard Edition, puede ver información sobre el estado del grupo de servidores y el estado de otros componentes usados por el grupo de servidores a través del panel de control de Lync Server 2013. Se recomienda implementar System Center Operations Manager 2012 con el panel de control de Lync Server para ayudar a los administradores a ver el estado de los grupos de servidores dentro de su entorno.

<div>

## <a name="view-pools"></a>Ver grupos

1.  Mediante el panel de control de Lync Server, abra el panel de control de Lync Server.

2.  Haga clic en topología y compruebe el estado de todos los servidores de la topología.
