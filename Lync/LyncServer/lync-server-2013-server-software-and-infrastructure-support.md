# <a name="server-software-and-infrastructure-support-in-lync-server-2013"></a>Software de servidor y compatibilidad con la infraestructura en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-06-29_

La compatibilidad de software para componentes de servidor, incluidos todos los roles de servidor y el almacenamiento de Lync Server 2013, incluye sistemas operativos compatibles, software de base de datos, software de infraestructura y otro software necesario para admitir funcionalidades específicas. También incluye la virtualización de los componentes de servidor.
