# <a name="voice-application-cmdlets-in-lync-server-2013"></a>Cmdlets de la aplicación de voz en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-10-09_

Las aplicaciones de voz se ejecutan en el servicio de aplicación en el servidor front-end. Estas aplicaciones le ayudan a administrar las llamadas telefónicas entrantes y salientes permitiéndole detener llamadas, tratar números sin asignar, coordinarse con mensajería unificada de Exchange (UM) y ejecutar la aplicación de grupo de respuesta.
