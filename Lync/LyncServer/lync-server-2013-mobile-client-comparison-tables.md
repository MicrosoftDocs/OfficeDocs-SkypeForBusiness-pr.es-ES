# <a name="mobile-client-comparison-tables-for-lync-server-2013"></a>Tablas de comparación de clientes móviles para Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2015-09-21_

En las siguientes tablas se comparan las características y capacidades de los clientes móviles de Lync 2013 y del cliente de escritorio de Lync 2013 en las siguientes categorías:

  - Inicio de sesión, notificaciones de inserción y características generales

  - Presencia mejorada

  - Contactos y grupos de contactos

  - Mensajería instantánea (MI)

  - Audio y vídeo entre usuarios de Lync

  - Conferencia

  - Telefonía

  - Usuarios externos

  - Archivado y cumplimiento

Estas tablas indican las características que están disponibles para los usuarios de Lync en una implementación local de Lync Server 2013. Las mismas características también están disponibles para usuarios de Skype empresarial online y Microsoft Office 365, a menos que se indique lo contrario en las notas al pie de la tabla.

> <LI>

> <LI>
