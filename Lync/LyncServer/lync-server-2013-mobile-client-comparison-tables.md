# Tablas de comparación de clientes móviles para Lync Server 2013

 

_**Última modificación del tema:** 2016-12-08_

Las siguientes tablas comparan las características y las capacidades entre clientes móviles de Lync 2013 y el cliente de escritorio de Lync 2013 en las siguientes categorías:

  - Inicio de sesión, notificaciones de inserción y características generales

  - Presencia mejorada

  - Contactos y grupos de contactos

  - Mensajería instantánea (MI)

  - Audio y vídeo de Lync-a -Lync

  - Conferencia

  - Telefonía

  - Usuarios externos

  - Archivado y cumplimiento

Estas tablas indican cuáles son las características que los usuarios de Lync tienen a su disposición en una implementación local de Lync Server 2013. Las mismas características están también disponibles para los usuarios de Skype Empresarial Online y de Microsoft Office 365, a menos que en las notas al pie de las tablas se indique lo contrario.


> [!NOTE]
> <UL>
> <LI>
> <P>Para obtener ayuda y recursos en línea para usuarios finales, vea "Microsoft Lync 2013 para clientes móviles" en <A href="http://go.microsoft.com/fwlink/?linkid=286237">http://go.microsoft.com/fwlink/?LinkId=286237</A>.</P>
> <LI>
> <P>Para comparar las características disponibles en otros clientes de Lync 2013, vea <A href="lync-server-2013-desktop-client-comparison-tables.md">Tablas de comparación de clientes para Lync Server 2013</A>.</P>
> <LI>
> <P>Lync Server 2013 también admite aplicaciones móviles de Lync 2010. Para más información, vea las tablas de comparación de los clientes móviles de la documentación de Lync Server 2010 en <A class=uri href="http://go.microsoft.com/fwlink/?linkid=234777">http://go.microsoft.com/fwlink/?linkid=234777</A>.</P></LI></UL>
