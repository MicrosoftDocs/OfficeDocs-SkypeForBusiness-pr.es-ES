# <a name="mobile-client-comparison-tables-for-lync-server-2013"></a>Tablas de comparación de clientes móviles para Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

</div>

<div id="mainSection">

<span> </span>

_**Última modificación del tema:** 2015-09-21_

En las tablas siguientes se comparan las características y capacidades entre los clientes móviles de Lync 2013 y el cliente de escritorio de Lync 2013 en las siguientes categorías:

  - Inicio de sesión, notificaciones de inserción y características generales

  - Presencia ampliada

  - Contactos y grupos de contactos

  - Mensajería instantánea

  - Audio y vídeo Lync a Lync

  - Conferencia

  - Telefonía

  - Usuarios externos

  - Archivado y cumplimiento

Estas tablas indican las características que están disponibles para los usuarios de Lync en una implementación local de Lync Server 2013. Las mismas características también están disponibles para los usuarios de Skype empresarial online y Microsoft 365 o Office 365, a menos que se indique lo contrario en las notas al pie de la tabla.
