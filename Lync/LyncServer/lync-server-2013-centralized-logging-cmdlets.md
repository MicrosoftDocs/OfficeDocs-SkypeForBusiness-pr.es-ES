# <a name="centralized-logging-cmdlets-in-lync-server-2013"></a>Cmdlets de registro centralizado en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2012-10-08_

Los cmdlets de registro centralizado proporcionan una manera para que los administradores administren y configuren las funciones de registro centralizado que se incluyen en Microsoft Lync Server 2013. El registro centralizado permite a los administradores habilitar o deshabilitar simultáneamente el seguimiento de eventos en varios equipos.

<div>
