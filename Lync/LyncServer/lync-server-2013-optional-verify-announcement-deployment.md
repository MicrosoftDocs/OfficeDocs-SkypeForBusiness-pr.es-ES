# <a name="optional-verify-announcement-deployment-in-lync-server-2013"></a>Faculta Comprobar la implementación de la presentación en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2013-02-25_

Una vez instalada y configurada la opción Anuncio, tiene que comprobar la configuración para asegurarse de que las llamadas a números sin asignar actúen según lo esperado. Como mínimo, compruebe los elementos siguientes:

  - Llamar a un número válido para la organización, pero que esté sin asignar.

  - Llamar a un número sin asignar y comprobar que se reproduzca el anuncio correcto.
