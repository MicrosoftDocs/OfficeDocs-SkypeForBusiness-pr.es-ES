# Tabla Manufacturers en Lync Server 2013

 

_**Última modificación del tema:** 2015-03-09_

La tabla Manufacturers es una tabla de apoyo. Cada registro contiene información sobre el fabricante de un dispositivo (teléfono de escritorio).
