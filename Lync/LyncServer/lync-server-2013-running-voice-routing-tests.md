# <a name="running-voice-routing-tests-in-lync-server-2013"></a>Ejecución de pruebas de enrutamiento de voz en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2013-02-21_

Lync Server 2013 proporciona dos métodos diferentes para probar rutas de voz: puede realizar una prueba informal y ad hoc con cualquier número de teléfono y cualquier ruta de voz; o bien, puede realizar más pruebas formales con los casos de prueba de ruta de voz. Con las pruebas formales, se definen aspectos como el número que se va a marcar y el plan de marcado y la Directiva de voz que se van a usar, y Lync Server puede comprobar que, teniendo en cuenta estas condiciones, el número proporcionado se puede enrutar correctamente a la red RTC. Ambos métodos se describen en las secciones posteriores de esta documentación.
