# <a name="running-voice-routing-tests-in-lync-server-2013"></a>Ejecución de pruebas de enrutamiento de voz en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2013-02-21_

Lync Server 2013 ofrece dos métodos diferentes para probar las rutas de voz: puede realizar una prueba informal y ad hoc con cualquier número de teléfono y cualquier ruta de voz; o bien, puede realizar pruebas más formales con los casos de prueba de ruta de voz. Con las pruebas formales, se define como el número que se marcará y el plan de marcado y la Directiva de voz que se utilizará, de modo que Lync Server pueda comprobar que, dadas estas condiciones, el número proporcionado se pueda enrutar correctamente a la red PSTN. Ambos métodos se describen en secciones posteriores de esta documentación.
