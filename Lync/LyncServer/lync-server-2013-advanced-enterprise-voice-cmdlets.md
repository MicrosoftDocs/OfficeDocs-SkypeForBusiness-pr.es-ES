# <a name="advanced-enterprise-voice-cmdlets-in-lync-server-2013"></a>Cmdlets avanzados de telefonía IP empresarial en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2016-04-12_

Al implementar Enterprise Voice en una organización, también pueden implementarse otras aplicaciones y características para mejorar la experiencia del usuario y la eficiencia de la organización. Entre estas características se encuentran la configuración de medios, el control de admisión de llamadas, E9-1-1 mejorado (E9-1-1) y las llamadas de emergencia.
