# <a name="setting-up-the-director-in-lync-server-2013"></a>Configuración del Director en Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2014-05-05_

Si está habilitando el acceso para los usuarios externos mediante la implementación de servidores perimetrales, una opción es implementar un director. Un director es un servidor que ejecuta Microsoft Lync Server 2013 que autentica solicitudes de usuario, pero no aloja ninguna cuenta de usuario. Ahora, esto no es un requisito, pero es muy útil si está preocupado por el rendimiento y desea ayudar a simplificar las solicitudes de autenticación. Si decide que esta es una buena idea para su organización, los pasos para configurar un director o un grupo de directores son similares a la configuración de un grupo de servidores front-end Enterprise Edition o un servidor Standard Edition. Una vez que haya definido los directores en el generador de topologías, tendrá que realizar los pasos que se indican en esta sección.
