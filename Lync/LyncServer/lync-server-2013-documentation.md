# <a name="documentation-in-lync-server-2013"></a>Documentación de Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Última modificación del tema:** 2015-05-15_

El modelo de MOF se compone de numerosas funciones de administración de servicios. La documentación sobre cómo y cuándo se realizan las tareas puede compartirse con miembros del mismo equipo o con otros equipos. El método de almacenar y compartir la documentación puede variar según el tipo de función. Por ejemplo, los procedimientos para la administración del sistema se pueden almacenar como documentos de Word porque es probable que se impriman y hagan referencia a ellos con frecuencia. La información de administración de configuración puede generarse automáticamente y almacenarse en una base de datos para facilitar la búsqueda y la indización. Puede que la documentación sea delicada y se debe restringir.

<div>

## <a name="document-management-systems"></a>Sistemas de administración de documentos

Un sistema de administración de documentación actúa como un repositorio central para los documentos y ayuda a garantizar que solo esté disponible la última revisión de un documento. También puede considerar el archivado de la versión anterior del documento como referencia. Lync Server proporciona una funcionalidad adecuada para esta tarea.

</div>

<div>

## <a name="databases"></a>Bases de datos

Se han tratado varias funciones de administración y herramientas que son adecuadas para usar bases de datos. Es probable que el proceso de administración de la configuración utilice procesos automatizados que almacenan grandes cantidades de datos que requieren indización y búsqueda. El personal de soporte técnico puede buscar una base de datos de problemas y soluciones anteriores al solucionar problemas nuevos.

Es probable que se usen diferentes bases de datos para distintos fines. Decida si estas bases de datos se deben vincular o combinar. Por ejemplo, si el servicio de asistencia al cliente identifica varios problemas con un tema común (como software nuevo que causa un problema con un determinado adaptador de red), el personal de soporte técnico puede consultar a la base de datos de configuración para predecir cuántos equipos pueden verse afectados.
