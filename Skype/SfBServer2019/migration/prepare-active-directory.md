# <a name="prepare-active-directory-for-skype-for-business-server"></a>Preparar Active Directory para Skype Empresarial Server

Antes de implementar Skype Empresarial Server 2019 en estado de coexistencia, debe realizar algunas tareas adicionales de Active Directory para configurar el esquema, el bosque y el dominio para Skype Empresarial Server 2019. Las extensiones de esquema agregan las clases y atributos de Active Directory necesarios para Skype Empresarial Server 2019. <!-- For additional information, see the topic 
[Preparing Active Directory Domain Services for Skype for Business Server 2019](../deployment/deploying-lync-server-2013/preparing-active-directory-domain-services-for-lync-server-2013.md). -->
  
### <a name="to-prepare-active-directory-for-skype-for-business-server-2019"></a>Para preparar Active Directory para Skype Empresarial Server 2019

1. En el Skype Empresarial Server front-end de 2019, ejecute Skype Empresarial Server instalación de 2019.
    
2. Seleccione **Preparar Active Directory**.
  
3. Complete los pasos del 1 al 5 en el asistente.
