# <a name="download-topology-from-existing-deployment"></a>Descargar una topología desde una implementación existente

Al crear un Skype para el grupo de servidores de Business Server 2019, va a usar el almacén de Administración Central que está asociada con la instalación heredada. Al iniciar sesiones de edición de generador de topologías en utilizar por primera vez y posteriores, se le pedirá la ubicación donde desea que el generador de topología para cargar el documento de configuración actual. Debido a que ya tiene una topología definida y ha establecido el almacén de Administración Central, debe elegir descargar una topología de una implementación existente. Generador de topología leerá la base de datos y recuperar la definición actual. 
  
### <a name="to-download-a-topology-from-an-existing-deployment"></a>Para descargar una topología de una implementación existente

1. Abra el Skype para el Asistente para la implementación de servidor de negocio.
    
2. En la página de **Skype para Business Server 2019 - Asistente para la implementación** , haga clic en **Instalar las herramientas administrativas**.
    
3. Inicie el generador: Haga clic en **Inicio**, haga clic en **Todos los programas**, haga clic en **Microsoft Skype para Business Server 2019**y, a continuación, haga clic en **Skype para Business Server Topology Builder**.
    
4. Seleccione **Descargar topología de la implementación existente**.
  
5. Elija un nombre de archivo y guarde la topología con el tipo de archivo .tbxml predeterminado.
    
6. Expanda el Skype para el nodo de servidor empresarial para revelar los diversos roles de servidor de la implementación.
