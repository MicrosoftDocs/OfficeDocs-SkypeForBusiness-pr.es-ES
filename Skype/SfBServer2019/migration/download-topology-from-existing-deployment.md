# <a name="download-topology-from-existing-deployment"></a>Descargar una topología desde una implementación existente

Al crear un grupo de servidores de Skype empresarial Server 2019, usará el almacén de administración central que está asociado a la instalación heredada. Cuando inicie el Generador de topologías, en el primer uso y en los usos posteriores se le solicitará la ubicación en la que desea que el Generador de topologías cargue el documento de configuración actual. Como ya tiene una topología definida y ha establecido el almacén de administración central, debe elegir descargar una topología de una implementación existente. El Generador de topologías leerá la base de datos y recuperará la definición actual. 
  
### <a name="to-download-a-topology-from-an-existing-deployment"></a>Para descargar una topología de una implementación existente

1. Abra el Asistente para la implementación de Skype empresarial Server.
    
2. En la página del **Asistente para la implementación de Skype empresarial Server 2019** , haga clic en **instalar herramientas administrativas**.
    
3. Inicie el generador de topologías: haga clic en **Inicio**, en **todos los programas**, en **Microsoft Skype empresarial Server 2019**y, a continuación, en **generador de topologías de Skype empresarial Server**.
    
4. Seleccione **Descargar una topología desde una implementación existente**.
  
5. Elija un nombre de archivo y guarde la topología con el tipo de archivo default. tbxml.
    
6. Expanda el nodo Skype empresarial Server para mostrar los distintos roles de servidor en la implementación.
