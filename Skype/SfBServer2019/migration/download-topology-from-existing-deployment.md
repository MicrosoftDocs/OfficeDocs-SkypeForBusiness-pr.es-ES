# <a name="download-topology-from-existing-deployment"></a>Descargar una topología desde una implementación existente

Al crear un grupo Skype Empresarial Server 2019, usarás el Almacén de administración central asociado a la instalación heredada. Cuando inicie el Generador de topologías, en el primer uso y en los usos posteriores se le solicitará la ubicación en la que desea que el Generador de topologías cargue el documento de configuración actual. Dado que ya tiene una topología definida y ha establecido el almacén de administración central, debe elegir descargar una topología de una implementación existente. El Generador de topologías leerá la base de datos y recuperará la definición actual. 
  
### <a name="to-download-a-topology-from-an-existing-deployment"></a>Para descargar una topología de una implementación existente

1. Abra el Asistente Skype Empresarial Server implementación.
    
2. En la **página Skype Empresarial Server 2019 - Asistente** para implementación, haga clic en Instalar herramientas **administrativas**.
    
3. Iniciar generador de topologías: haga clic en Inicio **,** en Todos los **programas,** en **Microsoft Skype Empresarial Server 2019** y, a continuación, haga clic **Skype Empresarial Server Generador de topologías**.
    
4. Seleccione **Descargar una topología desde una implementación existente**.
  
5. Elija un nombre de archivo y guarde la topología con el tipo de archivo .tbxml predeterminado.
    
6. Expanda el Skype Empresarial Server para mostrar los distintos roles de servidor en la implementación.
