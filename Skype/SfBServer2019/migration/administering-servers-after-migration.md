# <a name="administering-servers-after-migration"></a>Administrar los servidores tras la migración

En general, debe utilizar la herramienta administrativa que corresponde a la versión del servidor que desea administrar. No puede instalar herramientas administrativas de distintas versiones en el mismo equipo.
  
> [!IMPORTANT]
> Después de implementar Skype Empresarial Server grupo piloto, no puede usar el Generador de topologías ni el Panel de control para administrar los recursos de 2019. Debe usar las herramientas de 2019 para administrar todos los recursos de versión actual y anterior.
