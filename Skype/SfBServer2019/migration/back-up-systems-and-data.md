# <a name="back-up-systems-and-data"></a>Copia de seguridad de los datos y sistemas

Antes de comenzar la migración, realice una copia de seguridad completa del sistema y documente el sistema existente, incluido un inventario de cuentas de usuario que estén alojados en cada grupo, para que pueda revertir si es necesario. Hay varias herramientas y programas disponibles para realizar copias de seguridad y restaurar los datos, la configuración y los sistemas.

Para obtener detalles y procedimientos, vea procedimientos de copia de seguridad y restauración para su versión.
