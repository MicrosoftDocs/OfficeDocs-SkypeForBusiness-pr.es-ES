# <a name="back-up-systems-and-data"></a>Copia de seguridad de los datos y sistemas

Antes de comenzar la migración, realice una copia de seguridad completa del sistema y documente el sistema existente, incluido un inventario de las cuentas de usuario que están hospedados en cada grupo de servidores, de modo que puede deshacer la migración si es necesario. Varios programas y herramientas están disponibles para la copia de seguridad y restauración de datos, configuración y sistemas.

Para obtener información detallada y los procedimientos, vea copia de seguridad y restauración de procedimientos para su versión.
