### <a name="to-deploy-a-skype-for-business-server-2019-pilot-pool"></a><span data-ttu-id="041c0-115">Para implementar un Skype para Business Server 2019 el grupo piloto</span><span class="sxs-lookup"><span data-stu-id="041c0-115">To deploy a Skype for Business Server 2019 pilot pool</span></span>
