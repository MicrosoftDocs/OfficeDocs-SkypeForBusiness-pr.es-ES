# <a name="skype-for-business-server-2019-management-tools"></a>Skype para herramientas de administración de servidor 2019 empresarial
 
**Resumen:** Obtenga información acerca de las herramientas de administración de servicio de Skype para Business Server 2019.
  
Skype para Business Server 2019 ofrece mensajería instantánea (IM), presencia, conferencias y soluciones de telefonía que admiten los requisitos de colaboración de la empresa. Las herramientas para administrar estos servicios son flexibles y avanzadas.
