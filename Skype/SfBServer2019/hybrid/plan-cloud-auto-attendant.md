# <a name="plan-cloud-auto-attendant"></a>Planeación de operador automático de la nube

Skype para las implementaciones híbridas de Business Server 2019 sólo usar correo de voz en la nube y no se integran con Exchange Online.

Uso de Skype para Business Server 2019 con [Operadores automáticos de teléfono del sistema](/SkypeForBusiness/what-is-phone-system-in-office-365/what-are-phone-system-auto-attendants.md) aún no está disponible.

Si es un usuario actual de Exchange Server 2013 o 2016 de servidor de Exchange, podrá continuar usarlos como implementar Skype para Business Server 2019. Mensajería unificada de Exchange, incluida la funcionalidad de operador automático, se retira en Exchange 2019.

Existe una funcionalidad similar en el sistema telefónico y Skype para permitir la interacción con el sistema telefónico Business Server 2019 es implantar según la característica por característica. Planeación de según corresponda.
