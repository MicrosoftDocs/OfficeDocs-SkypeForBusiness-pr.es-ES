# <a name="the-users-skype-for-business-online-service-plan-isnt-valid"></a>Skype del usuario para el plan de servicio en línea de Business no es válido

Lo sentimos, pero el Skype para plan de servicio en línea de negocio asignado a este usuario no se reconoce o no es válido.
  
Vaya al **Centro de administración de Office 365** > planeación de**usuarios activos** y asignar un servicio válido para este usuario.
