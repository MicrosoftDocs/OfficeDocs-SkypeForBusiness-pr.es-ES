# <a name="manage-the-audio-conferencing-settings-for-a-user"></a>Administrar la configuración de conferencia de Audio para un usuario

Como administrador de Office 365, puede editar la configuración de conferencia de Audio: como proveedor, pago predeterminado o número de teléfono gratuito, Id. de conferencia o PIN, para un usuario individual en la organización. Si desea editar la configuración de su organización, consulte [Administrar la configuración de conferencia de Audio de mi organización](manage-the-audio-conferencing-settings-for-my-organization.md).
  
![Muestra la página de propiedades de conferencia de Audio para un usuario](../images/228550f7-92be-416d-9ab1-7c2ef54dd4e6.png)
  
1. Inicie sesión en Office 365 con su cuenta profesional o educativa.
    
2. Seleccione **Centros de administración** > **Skype Empresarial**.
    
3. En Skype para el centro de administración de negocios, elegir **los usuarios**.
    
4. Seleccione el usuario para el que desea administrar la configuración y, a continuación, en el panel Acción, haga clic en **Editar**![muestra el icono de edición](../images/4d8bea48-be68-4e0e-a54c-73decf7ea4ec.png).
    
5. Elija **conferencias de Audio** en la exploración de la izquierda y, a continuación, en la página de **Propiedades** para el usuario, modificar cualquiera de los siguientes:
    
|**Configuración**|**Descripción**|
|:-----|:-----|
|**Nombre del proveedor** <br/> |Elija el proveedor de la lista.  <br/><br/> **Nota:** El resto de los valores en esta tabla se aplica sólo si selecciona Microsoft como proveedor de conferencia de audio.           |
|**Número de peaje predeterminado** (obligatorio) <br/> |Para los proveedores, estos números de teléfono son los que recibe del proveedor de conferencia de audio. Si el usuario utiliza Microsoft como proveedor de conferencia de audio, estos serán los números que se establecen en el puente de conferencia de audio. Dar formato a los números como desee que aparezcan en Skype para las convocatorias de reunión de negocios y Teams de Microsoft.  <br/> |
|**Número gratuito por defecto** <br/> |Para los proveedores, estos números de teléfono son los que recibe del proveedor de conferencia de audio. Si el usuario utiliza Microsoft como proveedor de conferencia de audio, estos serán los números que se establecen en el puente de conferencia de audio. Dar formato a los números como desee que aparezcan en Skype para las convocatorias de reunión de negocios y Teams de Microsoft.  <br/> |
|**Permitir el uso de números de teléfono gratuitos en el puente de Microsoft de la organización para unirse a las reuniones de este usuario** <br/> |Seleccione esta opción si desea permitir que al usuario de números de teléfono gratuitos para participar en reuniones.  <br/> |
|**Enviar información de conferencia a través de correo electrónico** <br/> |Haga clic en este vínculo sólo si desea enviar inmediatamente un correo electrónico al usuario con su número de identificación y teléfono de conferencia. (Este mensaje no incluye el PIN). Consulte [Enviar un correo electrónico a un usuario con su información de conferencia de Audio](send-an-email-to-a-user-with-their-dial-in-information.md).  <br/> |
|**Id. de conferencia** <br/> |Seleccione **Restablecer** si desea restablecer el Id. de conferencia para el usuario. Para obtener más información, vea [Restablecer un Id. de conferencia para un usuario](reset-a-conference-id-for-a-user.md).  <br/> |
|**PIN** <br/> |Seleccione **Restablecer** si desea restablecer el PIN del usuario. Para obtener más información, consulte [Restablecer el PIN de conferencia de Audio para un usuario](reset-the-audio-conferencing-pin-for-a-user.md).  <br/> |
|**Permitir llamadas sin autenticar ser los primeros en una reunión** <br/> |Seleccione esta opción para permitir que los llamadores no autenticados sea el primero en unirse a reuniones.  <br/> |
|**Restricciones en configuración de acceso telefónico de las reuniones de este usuario** <br/> |Seleccione una opción en esta lista si desea restringir salidas de marcado para llamadas nacionales, o si desea evitar que todas las salidas de marcado de las reuniones.  <br/> |
