 **Usar la versión adecuada de Windows 10**: para los clientes que desean implementar Windows 10 imágenes en sus dispositivos, se requiere una versión específica de Windows 10, y esta versión sólo está disponible para los clientes de licencias por volumen.  Puede obtener una copia desde el [Centro de servicio de licencias por volumen](https://www.microsoft.com/Licensing/servicecenter/). Use las instrucciones en [Configurar una consola de v2 de sistemas de salón de Skype](/Skypeforbusiness/deploy/deploy-clients/console.md) para crear los discos de instalación de software para los sistemas de salón de Skype versión 2.
