# <a name="microsoft-365-apps-for-enterprise-or-skype-for-business-online-standalone-plan-2"></a>Aplicaciones Microsoft 365 para empresas o Skype Empresarial plan independiente en línea 2

[!INCLUDE [sfbo-retirement](../../../Hub/includes/sfbo-retirement.md)]

Si ya ha comprado [Aplicaciones Microsoft 365 para empresas](https://products.office.com/en/business/office-365-proplus-business-software) o Skype Empresarial Plan independiente en línea [2,](https://products.office.com/skype-for-business/online)estas son las licencias de complementos que necesita comprar para obtener más Skype Empresarial características.

> [!IMPORTANT]
> Aunque pueden adquirirse las funciones de Audioconferencia, el Sistema telefónico y los Planes de llamada con Skype Empresarial Online Plan 2 para habilitar las características de llamada de voz, le recomendamos que evalúe un plan Enterprise E5 para comprobar si le es más rentable.

|**Audioconferencia** <br/> Compre 1 **licencia de audioconferencia** para cada persona que vaya a programar o hospedar una reunión de acceso telefónico local. Los asistentes no necesitan licencias.  |Complemento de **audioconferencia** <br/>Una vez adquirida la licencia, Microsoft será su proveedor de audioconferencias. |
|**Números gratuitos para el acceso telefónico a las conferencias y la posibilidad de llamar desde una conferencia para agregar a alguien llamando a cualquier número de teléfono del mundo**  |[Créditos de comunicaciones](/microsoftteams/set-up-communications-credits-for-your-organization)|
