# <a name="microsoft-365-apps-for-enterprise-or-skype-for-business-online-standalone-plan-2"></a>Aplicaciones de Microsoft 365 para empresas o Plan 2 independiente de Skype Empresarial Online

Si ya ha comprado las aplicaciones de [Microsoft 365](https://products.office.com/en/business/office-365-proplus-business-software) para empresas o el [Plan independiente 2](https://products.office.com/skype-for-business/online)de Skype Empresarial Online, estas son las licencias de complementos que debe comprar para obtener más características de Skype Empresarial.

> [!IMPORTANT]
> Aunque pueden adquirirse las funciones de Audioconferencia, el Sistema telefónico y los Planes de llamada con Skype Empresarial Online Plan 2 para habilitar las características de llamada de voz, le recomendamos que evalúe un plan Enterprise E5 para comprobar si le es más rentable.
