# <a name="whats-the-status-of-my-port-orders"></a>¿Cuál es el estado de mis solicitudes de portabilidad?

Puede ver el estado de su pedido de puerto yendo a la **Skype para el centro de administración de negocios** > **voz** > **pedidos de puerto**. Cada estado de la orden de puerto se enumerarán en la columna **estado** . Si ayuda, [soporte para productos de empresa - ayuda de administración de contacto](http://technet.microsoft.com/library/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b%28Office.14%29.aspx).
  
La tabla siguiente muestra estados de orden de puerto, así como las acciones que puede realizar, si es necesario. 
  
|**Estado**|**¿Puede ver el pedido?**|**¿Puede editar el pedido?**|**¿Puede cancelar el pedido?**|**¿Puede eliminar el pedido?**|**Descripción**|
|:-----|:-----|:-----|:-----|:-----|:-----|
|**Procesamiento** <br/> |Sí  <br/> |No  <br/> |Sí  <br/> |No  <br/> |El administrador ha creado el pedido y ha recibido por Microsoft.  <br/> |
|**Ponerse en contacto con el transportista** <br/> |Sí  <br/> |No  <br/> |Sí  <br/> |No  <br/> |El pedido ha sido recibido y aprobado por Microsoft, y estamos trabajando con el transportista para obtener aprobación vencido.  <br/> |
|**Transferencia aprobado** <br/> |Sí  <br/> |Pedido en firme Commitment(FOC)  <br/> |Sí  <br/> |No  <br/> |El pedido ha sido aceptado por el transportista vencido y se ha establecido la fecha de BDC.  <br/> |
|**Pendiente de transferencia** <br/> |Sí  <br/> |No  <br/> |No  <br/> |No  <br/> |La transferencia es menos de 24 horas, por lo que la orden ya no puede ser editada o cancelada.  <br/> |
|**Error** <br/> |No  <br/> |Sí  <br/> |Sí  <br/> |Sí (en este momento, no puede eliminar el orden de puerto si se produce un error. El orden de puerto debe volver a crearse o necesita [soporte de contacto para los productos business - ayuda de Admin](http://technet.microsoft.com/library/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b%28Office.14%29.aspx).  <br/> |El transportista perdedora rechazó el pedido.  <br/> |
|**Completado** <br/> |Sí  <br/> |No  <br/> |No  <br/> |No  <br/> |Los números se han transferido correctamente.  <br/> |
|**Cancelado** <br/> |No  <br/> |Sí  <br/> |No  <br/> |No  <br/> |El administrador ha cancelado el pedido.  <br/> |
   
 **Para obtener instrucciones detalladas paso a paso, consulte [transferir números de teléfono para Office 365](transfer-phone-numbers-to-office-365.md).**
 
> [!NOTE]
> Si necesita obtener más números de teléfono, [póngase en contacto con el soporte de productos para empresas: ayuda para administradores](https://support.office.com/article/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b).
