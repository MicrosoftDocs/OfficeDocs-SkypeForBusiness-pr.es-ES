# <a name="download-a-letter-of-authorization-loa"></a>Descargar una Carta de autorización (LOA) 

La **Carta de autorización (LOA)** nos concede permiso para solicitar en su nombre la transferencia de números existentes desde un proveedor de servicios diferente en caso de disputa. Use una LOA si tiene números de servicio para puentes de audioconferencia, operadores automáticos u otros números de servicio, números de teléfono gratuitos o si tiene más de 999 números de teléfono de usuarios (suscriptores) que debe transferir a Skype Empresarial (o si no quiere usar el **asistente de portabilidad de número local**). Debe [enviar manualmente una solicitud de servicio personalizado](manually-submit-a-custom-service-request.md) con esta LOA para que se transfieran los números de teléfono desde su proveedor de servicios actual.
  
Si tiene números de servicio para puentes de audioconferencia, operadores automáticos u otros números de servicio, números de teléfono gratuitos o más de 999 números de teléfono de usuarios que debe transferir a Skype Empresarial, tendrá que enviar manualmente una solicitud de portabilidad.
  
Si no sabe con certeza para qué tipos de números de teléfono va a solicitar la transferencia, consulte [Diferentes tipos de números de teléfono que se usan para Planes de llamada](different-kinds-of-phone-numbers-used-for-calling-plans.md).
  
> [!IMPORTANT]
> Cuando realice la portabilidad manual de números de teléfono mediante una Carta de autorización (LOA), asegúrese de **seleccionar el tipo de número de teléfono correcto**. Debe **enviar una solicitud de portabilidad independiente** para cada tipo de número de teléfono que desee transferir.
