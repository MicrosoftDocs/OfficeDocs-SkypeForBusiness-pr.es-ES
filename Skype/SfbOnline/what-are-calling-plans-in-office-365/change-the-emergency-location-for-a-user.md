<span data-ttu-id="68185-105">Cada número de teléfono activo debe tener una dirección de emergencia (asociada al obtener un número de teléfono en Office 365 o al transferir un número de teléfono) cuando se asigna el número de teléfono al usuario.</span><span class="sxs-lookup"><span data-stu-id="68185-105">Each active phone number must have an emergency address (associated when you get a phone number in Office 365 or when you transfer a phone number) when the phone number is assigned to the user.</span></span> <span data-ttu-id="68185-106">Cuando el número se asocia con una dirección de emergencia, también puede agregar una ubicación de emergencia para proporcionar una ubicación más exacta dentro de una ubicación física.</span><span class="sxs-lookup"><span data-stu-id="68185-106">When you associate the number with an emergency address, you can also add an emergency location to provide a more exact location within a physical location.</span></span> <span data-ttu-id="68185-107">Una ubicación de emergencia puede ser una planta, el ala de un edificio o un número de oficina donde se encuentra el usuario.</span><span class="sxs-lookup"><span data-stu-id="68185-107">An emergency location can be the floor, building wing, or office number where the user is located.</span></span> <span data-ttu-id="68185-108">Puede tener un número ilimitado de ubicaciones para una determinada dirección de emergencia, y puede cambiar la ubicación de emergencia si el usuario se desplaza a una oficina o edificio diferente: por ejemplo, si el usuario se mueve desde la planta 34 a la planta 35.</span><span class="sxs-lookup"><span data-stu-id="68185-108">You can have an unlimited number of locations for a given emergency address, and you can change the emergency location if the user moves to a different office or building—for example, if the user moves from floor 34 to floor 35.</span></span>

## <a name="change-the-emergency-location-for-a-user"></a><span data-ttu-id="68185-110">Cambiar la ubicación de emergencia de un usuario</span><span class="sxs-lookup"><span data-stu-id="68185-110">Change the emergency location for a user</span></span>

1. <span data-ttu-id="68185-111">Inicie sesión en Office 365 con su cuenta profesional o educativa.</span><span class="sxs-lookup"><span data-stu-id="68185-111">Sign in to Office 365 with your work or school account.</span></span>
    
2. <span data-ttu-id="68185-112">Vaya a **los equipos de Microsoft y Skype para el centro de administración de negocio** > **portal heredado**.</span><span class="sxs-lookup"><span data-stu-id="68185-112">Go to the **Microsoft Teams and Skype for Business Admin Center** > **Legacy portal**.</span></span>
    
3. <span data-ttu-id="68185-113">En el panel de navegación izquierdo, vaya a **voz** > **a los usuarios de voz**.</span><span class="sxs-lookup"><span data-stu-id="68185-113">In the left navigation go to **Voice** > **Voice users**.</span></span>
    
    > [!IMPORTANT]
    > <span data-ttu-id="68185-114">Para que pueda ver la opción de **voz** en la izquierda en la Skype para el centro de administración de negocio, primero debe comprar una licencia adicional para **Conferencias de Audio** , una licencia adicional para **El sistema de teléfono** o al menos una **licencia Enterprise E5**.</span><span class="sxs-lookup"><span data-stu-id="68185-114">For you to see the **Voice** option in the left navigation in the Skype for Business admin center, you must first buy at least one **Enterprise E5 license**, one **Phone System** add-on license, or one **Audio Conferencing** add-on license.</span></span>
    
4. <span data-ttu-id="68185-115">En la página **usuarios de voz** , busque y seleccione el usuario que desea cambiar la ubicación de emergencia.</span><span class="sxs-lookup"><span data-stu-id="68185-115">On the **Voice users** page, locate and select the user you want to change the emergency location for.</span></span>
    
5. <span data-ttu-id="68185-116">En el panel de acciones, en **Ubicación de emergencia**, haga clic en **Cambiar**.</span><span class="sxs-lookup"><span data-stu-id="68185-116">In the Action pane, under **Emergency location**, click **Change**.</span></span>
    
6. <span data-ttu-id="68185-117">En la página **Asignar un número**, haga clic en **Cambiar ubicación**.</span><span class="sxs-lookup"><span data-stu-id="68185-117">On the **Assign number** page, click **Change location**.</span></span> 
    
7. <span data-ttu-id="68185-118">En **Cambiar dirección de emergencia a**, escriba el nombre de la ciudad en el cuadro y haga clic en **Buscar**.</span><span class="sxs-lookup"><span data-stu-id="68185-118">Under **Change emergency address to**, enter the name of the city in the box and click **Search**.</span></span>
    
8. <span data-ttu-id="68185-119">Seleccione la ubicación de emergencia en la lista y, a continuación, haga clic en **Guardar**.</span><span class="sxs-lookup"><span data-stu-id="68185-119">Select the emergency location from the list, and then click **Save**.</span></span>
    
    <span data-ttu-id="68185-120">Si desea agregar una nueva ubicación de emergencia, consulte [Agregar, cambiar o quitar una ubicación de emergencia para su organización](add-change-or-remove-an-emergency-location-for-your-organization.md).</span><span class="sxs-lookup"><span data-stu-id="68185-120">If you want to add a new emergency location, see [Add, change, or remove an emergency location for your organization](add-change-or-remove-an-emergency-location-for-your-organization.md).</span></span>
