# <a name="set-up-your-computer-for-windows-powershell"></a>Configurar un equipo de Windows PowerShell

Administración remota de Skype para los negocios en línea con Microsoft PowerShell sólo se admite en equipos de 64 bits con uno de los siguientes sistemas operativos:
  
- Windows 10
    
- Windows 8.1
    
- Windows 8
    
- Windows Server 2012 R2
    
- Windows Server 2012
    
- Windows Server 2008
    
- Windows 7
    
Además del sistema operativo compatible, el equipo también deben ejecutar lo siguiente:
  
- PowerShell
    
- Skype para el módulo de Business Connector en línea
    
Puede encontrar información acerca de cómo descargar e instalar estos elementos en los siguientes temas:
  
- [Descargue e instale el Skype para el módulo de Business Connector en línea](download-and-install-the-skype-for-business-online-connector.md)
    
- [Descargar e instalar Windows PowerShell 3.0](download-and-install-windows-powershell-3-0.md)
