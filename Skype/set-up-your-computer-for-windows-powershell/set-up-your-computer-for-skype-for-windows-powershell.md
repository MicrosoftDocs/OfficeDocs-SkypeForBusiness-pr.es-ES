# Configurar el equipo para Skype para la administración empresarial Online con Windows PowerShell

> [!IMPORTANT]
> Este artículo se ha traducido con traducción automática; vea la [declinación de responsabilidades](ebf5adf0-d848-4e99-baf4-387d1711e1d0.md#MT_Footer). Para su referencia, puede encontrar la versión en inglés de este artículo [aquí](https://support.office.com/en-us/article/ebf5adf0-d848-4e99-baf4-387d1711e1d0). 
  
Administración remota de Skype Empresarial Online utilizando Microsoft PowerShell sólo se admite en equipos de 64 bits con uno de los siguientes sistemas operativos:
  
- Windows 10
    
- Windows 8.1
    
- Windows 8
    
- Windows Server 2012 R2
    
- Windows Server 2012
    
- Windows Server 2008
    
- Windows 7
    
Además del sistema operativo compatible, el equipo también deben ejecutar lo siguiente:
  
- PowerShell
    
- Módulo de conector de Skype Empresarial Online
    
Puede encontrar información sobre cómo descargar e instalar estos elementos en los siguientes temas:
  
- [Descargue e instale el Skype para módulo Business Connector en línea](download-and-install-the-skype-for-business-online-connector-module.md)
    
- [Descargar e instalar Windows PowerShell 3.0](download-and-install-windows-powershell-3-0.md)
