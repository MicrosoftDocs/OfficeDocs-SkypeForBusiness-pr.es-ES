# Compatibilidad de Skype Empresarial Online en Outlook en la web

> [!IMPORTANT]
> Este artículo se ha traducido con traducción automática; vea la [declinación de responsabilidades](305984ec-3da8-4509-bb2b-6643dcf2cb7d.md#MT_Footer). Para su referencia, puede encontrar la versión en inglés de este artículo [aquí](https://support.office.com/en-us/article/305984ec-3da8-4509-bb2b-6643dcf2cb7d). 
  
Outlook en la web (Outlook Web App) en Office 365 ofrece un básica Skype empresarial Client de web desde la barra de navegación. Este cliente básico está disponible para los usuarios en línea cuya administración no ha configurado una dirección URL de cortesía para su organización de Office 365. Como la cuenta del usuario está en línea y no tiene una dirección URL de cortesía, todavía verán la experiencia incluso si su organización tiene algunas cuentas de usuario que están alojados en local. Los usuarios que dispongan de usuario cuentas locales (si tiene una dirección URL de cortesía o no) o administrados por Microsoft verán la experiencia de Lync en Outlook web app.
  
La siguiente tabla resume las diferentes configuraciones que es posible que tenga y el cliente de web que se utiliza.
  
||||
|:-----|:-----|:-----|
|**Ubicación de la cuenta de usuario** <br/> |**Hay una URL mnemónica configurada o una organización dedicada** <br/> |**Experiencia de Skype Empresarial o Lync** <br/> |
|Online  <br/> |No  <br/> |Experiencia web de Skype Empresarial  <br/> |
|Online  <br/> |Sí  <br/> |Experiencia web de Lync  <br/> |
|Híbrida pero hospedada en línea  <br/> |No  <br/> |Experiencia web de Skype Empresarial  <br/> |
|Híbrida pero hospedada en línea  <br/> |Sí  <br/> |Experiencia web de Lync  <br/> |
|Híbrida pero hospedada localmente  <br/> |No  <br/> |Experiencia web de Lync  <br/> |
|Híbrida pero hospedada localmente  <br/> |Sí  <br/> |Experiencia web de Lync  <br/> |
|Puros en local  <br/> |No  <br/> |Experiencia web de Lync  <br/> |
|Puros en local  <br/> |Sí  <br/> |Experiencia web de Lync  <br/> |
