# <a name="set-up-skype-for-business-online"></a>Configurar Skype para los negocios en línea
  
Debe tener permisos de administrador global de Office 365 para configurar Skype para el negocio. Si tiene un firewall o servidor proxy que restringe el acceso a partes de la web, considere la posibilidad de contratar a un [socio de Microsoft](https://go.microsoft.com/fwlink/?linkid=391089) para configurar Skype para el negocio para usted.
  
## <a name="setting-up-skype"></a>Configuración de Skype

Parece necesita ayuda para configurar Skype con la suscripción de Office 365. Puede seguir los pasos de este artículo para obtener su instalación completada.
  
## <a name="1-plan-for-skype-for-business"></a>1. plan de Skype para empresas

Si tiene **[Office 365 Business Premium](https://products.office.com/en-us/business/office-365-business-premium)** o **Business Essentials**, puede utilizar Skype para empresas para realizar llamadas en línea a otras personas de su empresa que estén incluidos en su suscripción. Por ejemplo, si su empresa tiene 10 personas, podrá [empezar a utilizar Skype para empresa de mensajería instantánea y reuniones en línea](http://support.office.com/article/cc05afa6-1894-4a82-9dd9-6222061f50fd) y [reuniones con Skype para el negocio](https://support.office.com/article/2eed8424-581a-4497-b505-c08c152e5851) mediante Skype para empresa después de realizar los pasos 2 al 6. Y puede [configurar un Skype para reuniones de negocios en Outlook](https://support.office.com/article/b8305620-d16e-4667-989d-4a977aad6556#bkmk_OWA) para reuniones en línea, demasiado!
  
Si desea utilizar Skype para empresas para hacer y recibir **llamadas** de personas *externas* a su empresa:
  
- **Opción 1. Utilice el libre [app de Skype](https://www.skype.com/)**. Si tiene una pequeña empresa (por ejemplo, 1-2 personas), uso de la aplicación de Skype es la mejor manera de ir. Es menos costoso que se utilizará para las llamadas nacionales e internacionales. Todavía puede contener llamadas de conferencia, realizar llamadas de vídeo y compartir el escritorio para presentaciones. [Consulte las tarifas y opciones de pago](https://secure.skype.com/en/calling-rates?wt.mc_id=legacy&amp;expo365=bundled).
    
- **Opción 2. Actualizar el plan y adquirir el sistema telefónico y un Plan de llamada para Office 365**. La forma más sencilla de averiguar cuánto esto cuesta, realizar el cambio, es [en el soporte técnico para los productos business - Admin ayuda](https://support.office.com/article/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b) y tenerlos a hacer todo para usted.
    
Para obtener más información, vea [Planear la instalación de Office 365 para el negocio](https://support.office.com/article/eb926624-018b-4486-bf11-5fba6ee4d645#bkmk_skype).
  
## <a name="2-sign-in-to-office-365"></a>2. Inicie sesión en Office 365
<a name="bkmk_signin"> </a>

Skype para los negocios en línea es parte del conjunto de servicios de Office 365. Para configurar Skype para los negocios en línea, debe iniciar sesión en Office 365. Aquí es cómo se hace:
  
1. Busque su ID de usuario de Office 365 (por ejemplo, *rob@fourthcoffee.com* ). Ha recibido un correo electrónico desde el equipo de Microsoft Online Services que contiene el identificador de usuario de Office 365 que creó cuando adquirió Skype para los negocios en línea. El correo se ve algo como esto:
    
    ![Un ejemplo de mensaje de bienvenida que recibió después de que te registraste en Skype para los negocios en línea. Contiene su id de usuario de Office 365.](../images/977c5c96-29c5-40c0-a4c4-1ba66ba3a1fb.png)
  
2. Iniciar sesión en el centro de administración de Office 365 e introduzca su ID de usuario de Office 365 y contraseña. Después de iniciar sesión, verá el centro de administración de Office 365:
    
    ![Un ejemplo de lo que el centro de administración de Office 365 es similar a cuando haya un Skype para Plan de negocio Online.](../images/ed1d9906-e717-450b-81a3-ce6679bd1be1.png)
  
## <a name="3-set-up-your-domain-and-users"></a>3. configurar tu dominio y usuarios
<a name="bkmk_users"> </a>

Ahora que haya iniciado sesión Office 365, puede configurar el dominio y las personas de su organización para utilizar Skype para los negocios en línea.
  
1. [Agregar un dominio y usuarios a Office 365](https://support.office.com/article/6383f56d-3d09-4dcb-9b41-b5f5a5efd611): usar el Asistente para la instalación de Office 365 para configurar tu dominio personalizado (por ejemplo, *fourthcoffee.com*) con Office 365. **De forma predeterminada, el Asistente para la instalación de Office 365 incluye configurar Skype para los negocios en línea y crear su Skype para identificadores de usuario de negocio.** Si ya utilizó al Asistente para configurar su dominio para Office 365, se ha completado este paso.
    
2. [Compruebe sus conexiones de DNS y dominio](https://support.office.com/article/2b54e1b0-47a7-4018-a1e4-c2b924e7c5a0): utilizar nuestra herramienta - el Solucionador de problemas de dominios - Compruebe que la configuración de DNS y el dominio es correcta. Hacerlo ahora pasará un largo camino para ayudar a descubrir cualquier problema de instalación posterior ya que podrá eliminar la configuración de DNS como el origen de problemas futuros.
    
3. [Las direcciones URL de Office 365 y los intervalos de direcciones IP](https://support.office.com/article/8548a211-3fe7-47cb-abb1-355ea5aa88a2#BKMK_LYO): la mayoría de las empresas pequeñas no es necesario realizar este paso. **Pero si tiene un firewall o servidor proxy restringe el acceso a partes de la web**, debe crear reglas para permitir el acceso a la Skype para los extremos de los negocios en línea. Éste es un paso avanzado realizado mejor un usuario experimentado con la configuración de firewalls y servidores proxy. Si ha hecho esto antes, considere la posibilidad de contratar a un [socio de Microsoft](https://go.microsoft.com/fwlink/?linkid=391089) para configurar Skype para el negocio para usted.
    
## <a name="4-set-up-im-and-presence-in-your-organization"></a>4. configurar mensajería instantánea y presencia en la organización
<a name="bkmk_IM"> </a>

Instant Messaging (IM) y presencia ([controlar el acceso a la información de presencia de Skype para empresas](https://support.office.com/article/fea86e34-60cf-4dd0-bfb2-169a42afd92c)) son características básicas incluidas con Skype para el negocio. De forma predeterminada, las personas de su empresa pueden Skype y mensajería instantánea entre sí.
  
1. **Elija quién más su Skype para usuarios de negocios puede comunicarse con:**
    
  - [Permitir a los usuarios ponerse en contacto con Skype externo para usuarios profesionales](allow-users-to-contact-external-skype-for-business-users.md) Usted *y* el otro negocio tendrá que configurar sus sistemas.
    
    **Importante**: si tiene dos dominios en la empresa, como rob@contosowest.com y ina@contosoeast.com, debe realizar este paso para todos los usuarios pueden comunicarse entre sí.
    
  - [Permiten Skype para usuarios de negocios agregar contactos de Skype](let-skype-for-business-users-add-skype-contacts.md) fuera de su negocio
    
2. **Elegir quién ve si los compañeros están en línea:** La característica de presencia muestra quién está en línea y cuál es su disponibilidad, como disponible, ocupado, ausente o presentación.
    
    ![Ejemplo de estado de conexión de una persona con un mensaje personal.](../images/ab6c10b4-6ad5-453c-bf0e-459b977b6e23.png)
  
    Puede elegir la configuración predeterminada para todos los usuarios de su empresa:
    
  - Mostrar presencia en línea de una persona automáticamente a todas las personas de la organización
    
  - Mostrar presencia en línea de una persona sólo a sus contactos
    
Para obtener instrucciones, consulte [Configurar presencia en Skype para los negocios en línea](configure-presence-in-skype-for-business-online.md).

A veces personas de la organización tendrá que utilizar un teléfono para llamar a una reunión. ¡Skype para empresas incluye la función de conferencia de Audio sólo esta situación! La gente puede llamar a Skype para reuniones de negocios con un teléfono, en lugar de utilizar el Skype para el negocio de la aplicación en un dispositivo móvil o PC.
  
Para obtener instrucciones de instalación paso a paso, consulte [establecer conferencias de Audio de Skype para empresas y equipos de Microsoft](../audio-conferencing-in-office-365/set-up-audio-conferencing.md) Office 365.
  
### <a name="set-up-phone-system-and-the-calling-plans-in-office-365"></a>Configurar el sistema telefónico y los planes de llamada en Office 365

La característica de sistema telefónico en Office 365 proporciona un sistema de teléfono para su negocio. Llamadas a otro Skype para personas de negocios de su organización son gratuitas y los empleados pueden recibir correo de voz entre sí y fuera de los llamadores. Aquí está lo que se obtiene con el sistema de teléfono.
  
Cuando se agrega el servicio Plan de llamadas, sus empleados a obtener un número de teléfono principal de Skype para el negocio. Pueden hacer y recibir llamadas de teléfono fuera de su negocio. Pueden hacer llamadas de voz a través de teléfonos VoIP, PC y dispositivos móviles. Y, en caso de emergencia, puede llamar al 911 para obtener ayuda.
  
Para obtener instrucciones de instalación paso a paso, consulte Set up planes de llamar a.
  
### <a name="set-up-skype-meeting-broadcast"></a>Configurar difusión de reunión de Skype

Difusión de la reunión de Skype es una característica que le permite generar, host, y difundir reuniones con asistentes de hasta 10.000. **Para obtener más información acerca de cómo funciona, consulte [¿Qué es una reunión de Skype difusión?](http://support.office.com/article/c472c76b-21f1-4e4b-ab58-329a6c33757d)**
  
Presentamos una visión general de los pasos para configurar difusión de reunión de Skype:
  
1. [Asignar o quitar licencias para Office 365 para el negocio](http://support.office.com/article/997596b5-4173-4627-b915-36abac6786dc): asignar licencias de **Skype para los negocios en línea** o **Plan de empresa** a todos los usuarios que se va a **host** de una reunión de difusión.
    
2. [Habilitar la difusión de reunión Skype](../set-up-your-network-for-skype-meeting-broadcast/enable-skype-meeting-broadcast.md): de forma predeterminada, esta característica no está habilitada. Una vez activado, los usuarios podrán alojar difusión reuniones con otras personas de su organización.
    
3. [Configurar la red para difundir reunión de Skype](../set-up-your-network-for-skype-meeting-broadcast/set-up-your-network-for-skype-meeting-broadcast.md): si desea host webinars y otras difusiones con asistentes externos a su organización, debe configurar la red.
    
4. [Programar una difusión de reunión Skype](http://support.office.com/article/c3995bc9-4d32-4f75-a004-3bc5c477e553) y tiene una [combinación de una difusión de reunión Skype](http://support.office.com/article/14689da0-821d-48d4-9035-ea762de80ebe): asegúrese de que el trabajo de difusión reuniones programar una difusión de reunión Skype en *https://portal.broadcast.skype.com* y, a continuación, haga que alguien intente unirse a la reunión.
    
## <a name="learn-about-network-connectivity-requirements"></a>Obtenga información acerca de los requisitos de conectividad de red
<a name="bkmk_more"> </a>

La calidad de audio, vídeo y uso compartido de Skype para el negocio de aplicaciones está afectada en gran medida por la calidad de la conectividad de red end-to-end. Para un rendimiento óptimo, es importante asegurarse de que hay una conexión de alta calidad entre la red de la compañía y Skype para los negocios en línea. Para la red y la información de sintonización, vea [Ajustar Skype para el funcionamiento de los negocios en línea](http://support.office.com/article/beec23c2-c5d6-4e84-a8af-e82aefca7802).
