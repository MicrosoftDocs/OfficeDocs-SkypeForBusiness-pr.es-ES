# Solución de problemas con Skype Empresarial Online

A veces, no todo funciona según lo esperado. A continuación, se muestra una lista de temas que pueden ayudarle a solucionar problemas, independientemente de si está administrando Skype Empresarial Online o ayudando a los usuarios a resolver sus problemas.
  
## Solucionar problemas de administración de Skype Empresarial Online

Puede obtener respuestas para la mayoría de sus preguntas de Skype Empresarial Online comprobando [Comprobar causas frecuentes de errores de inicio de sesión de Skype Empresarial Online](troubleshooting-skype-for-business-online-sign-in-errors-for-administrators.md#__toc323194094). Este artículo de Knowledge Base también incluye varios recursos de solución de problemas y referencias de códigos de error: [Cómo solucionar problemas de inicio de sesión en Skype Empresarial Online](https://go.microsoft.com/fwlink/p/?LinkId=747952).
  
Para los problemas de conferencias de acceso telefónico local, consulte [Problemas conocidos y solución de problemas de conferencias audio](../audio-conferencing-in-office-365/audio-conferencing-troubleshooting-and-known-issues.md).

## Ayuda para usuarios de Skype Empresarial Online

La mayoría de los problemas con los que se encuentran los usuarios de Skype Empresarial Online están relacionados con contraseñas olvidadas. Puede resolver la mayor parte de los problemas de inicio de sesión siguiendo las instrucciones en estos dos temas:
  
- [¿Necesita ayuda para iniciar sesión en Skype Empresarial?](http://technet.microsoft.com/library/448b8ea7-5b33-444a-afd4-175fc9930d05%28Office.14%29.aspx)
    
- [¿Necesita ayuda para iniciar sesión en Skype Empresarial?](http://technet.microsoft.com/library/448b8ea7-5b33-444a-afd4-175fc9930d05%28Office.14%29.aspx#bkmk-reset-password)
