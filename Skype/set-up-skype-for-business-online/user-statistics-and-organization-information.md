# <a name="user-statistics-and-organization-information"></a>Información de estadísticas y la organización del usuario

El Skype para Business dashboard le ofrece una instantánea rápida de cómo configurar el servicio para su negocio. Si aún no ha configurado Skype para empresas, consulte [Configurar Skype para los negocios en línea](set-up-skype-for-business-online.md) para obtener instrucciones paso a paso.
  
## <a name="how-to-change-skype-for-business-online-user-statistics"></a>Cómo cambiar de Skype para los negocios en línea estadísticas de usuario

|||
|:-----|:-----|
|**Propiedad** <br/> |**Cómo cambiarlo** <br/> |
|Skype para usuarios de negocios en línea  <br/> |Para agregar o bloquear personas utilicen el Skype para el negocio de la aplicación, vea [asignar o quitar licencias para Office 365 para el negocio](https://support.office.com/article/997596b5-4173-4627-b915-36abac6786dc) o [quitar licencias de usuarios en Office 365 para el negocio](https://support.office.com/article/9b497c85-d0a4-4735-80fa-d3565bc05bd1).  <br/> Para obtener una lista completa de qué hacer cuando un usuario sale de su empresa, vea [quitar a un antiguo empleado de Office 365](https://support.office.com/article/44d96212-4d90-4027-9aa9-a95eddb367d1).  <br/> |
|Usuarios habilitados para conferencias de acceso telefónico  <br/> |Para permitir que las personas para las conferencias de acceso telefónico, vea [Configurar la conferencia de Audio de Skype para empresas y equipos de Microsoft](../audio-conferencing-in-office-365/set-up-audio-conferencing.md).  <br/> |
|Usuarios en la nube  <br/> |Esto es el número de Skype para los usuarios de negocios en línea con cuentas de Office 365.  <br/> Para agregar personas a tu suscripción de Office 365 (es decir, crear cuentas de Office 365 para ellos), vea [Agregar usuarios individualmente o de forma masiva a Office 365 - ayuda de Admin](https://support.office.com/article/1970f7d6-03b5-442f-b385-5880b9c256ec).  <br/> |
|Los usuarios se sincronizan y alojados en línea  <br/> |Esto es el número de Skype para usuarios de negocios en línea que se sincronizan desde un Active Directory local.  <br/> Para configurar Active Directory local, vea [Configurar la sincronización de directorios para Office 365](https://support.office.com/article/1b3b5318-6977-42ed-b5c7-96fa74b08846).  <br/> |
   
> [!NOTE]
> Si aparece no disponible, significa que no hay ninguna información o se han producido errores. 
  
## <a name="how-to-change-information-about-your-organization"></a>Cómo cambiar la información sobre su organización

|||
|:-----|:-----|
|**Propiedad** <br/> |**Cómo cambiarlo** <br/> |
|Nombre de la organización  <br/> |Éste es el nombre que escribiste cuando te registraste para Office 365. Para cambiarla, consulte [cambiar la dirección de su organización, contacto técnico y mucho más](https://support.office.com/article/a36e5a52-4df2-479e-bb97-9e67b8483e10).  <br/> |
|Id. de organización  <br/> |No se puede cambiar el Id.  <br/> |
|Organización creada  <br/> |No puede cambiar este valor.  <br/> |
|Sincronización de Active Directory en  <br/> |Esto es si su organización se sincroniza a un Skype para organización de negocios en línea que es local.  <br/> Para configurar Active Directory local, vea [Configurar la sincronización de directorios para Office 365](https://support.office.com/article/1b3b5318-6977-42ed-b5c7-96fa74b08846).  <br/> |
|Dominios de la organización  <br/> |Cuando configura inicialmente Office 365, estos son los dominios que se registraron.  <br/> Para agregar un dominio, vea [Agregar varios dominios a Office 365](https://support.office.com/article/2d2fa996-b760-411d-a5cc-190d63f13207).  <br/> Para quitar un dominio, vea [quitar un dominio de Office 365](https://support.office.com/article/f09696b2-8c29-4588-a08b-b333da19810c).  <br/> |
