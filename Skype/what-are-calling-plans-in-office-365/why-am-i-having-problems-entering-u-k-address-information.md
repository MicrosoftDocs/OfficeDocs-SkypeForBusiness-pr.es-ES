# ¿Por qué tengo problemas al introducir la información de dirección fuera de los EE. UU.?

> [!IMPORTANT]
> Este artículo se ha traducido con traducción automática; vea la [declinación de responsabilidades](e2376646-f0d6-4d33-a8fd-68ecbd46fb7a.md#MT_Footer). Para su referencia, puede encontrar la versión en inglés de este artículo [aquí](https://support.office.com/en-us/article/e2376646-f0d6-4d33-a8fd-68ecbd46fb7a). 
  
Estas son algunas de las razones por qué puede que tenga problemas para obtener la dirección o direcciones para validar correctamente en el Reino Unido (Reino Unido).
  
## ¿Qué problema hay con el nombre de la compañía?

Si tiene un problema al escribir el nombre de la compañía, recuerde:
  
- Puede tener letras, números y los siguientes caracteres especiales:,-+. ' &amp; "_
    
    > [!CAUTION]
    > En el Reino Unido (Reino Unido), el nombre de la compañía no puede tener el signo +. 
  
- Debe tener 80 caracteres o menos de longitud.
    
- No puede comenzar con un carácter especial. 
    
## ¿Qué problema hay con el número de la casa?

Si tienes problemas para introducir el número de casa, recuerde:
  
- Puede tener letras, números y los siguientes caracteres especiales:-o]
    
- Debe empezar con un número.
    
- Debe tener 20 caracteres o menos de longitud.
    
- Puede tener una mayúscula o minúscula.
    
## ¿Qué problema hay con el nombre del domicilio?

Si tiene un problema al escribir el nombre de la casa, recuerde:
  
- Puede incluir letras, números y los caracteres especiales : , / - + . ' &amp; " _
    
- Debe tener 50 caracteres o menos de longitud.
    
- No puede comenzar con un carácter especial.
    
## ¿Qué problema hay con el nombre de la calle?

Si tiene un problema al escribir el nombre de la calle, recuerde:
  
- Puede tener letras, números y los siguientes caracteres especiales:,-+. ' &amp; "_
    
- Debe tener 50 caracteres o menos de longitud.
    
- No puede comenzar con un carácter especial. 
    
## ¿Qué problema hay con el pueblo o la ciudad?

Si tiene un problema al escribir el nombre de la ciudad o ciudad, recuerde:
  
- Puede tener letras, números y los siguientes caracteres especiales:,-+. ' &amp; "_
    
- Debe tener 50 caracteres o menos de longitud.
    
- No puede comenzar con un carácter especial. 
    
## ¿Qué problema hay con el código postal?

Si tienes problemas para introducir el código postal, recuerde:
  
- Solo puede tener letras, números o espacios.
    
- Debe tener 8 o menos letras, incluidos números y espacios.
    
- **Recuerde que si se encuentra en el Reino Unido (Reino Unido)**, primero **debe** colocar el código postal en uno de los siguientes formatos, donde la letra A = carácter de a Z, el signo # = número de 0 a 9 y la _ (subrayado) = un espacio en blanco. Estos son algunos ejemplos:
    
|**Formato**|**Ejemplo**|
|:-----|:-----|
|AA#A_#AA  <br/> |EC3B 2BD  <br/> |
|A#A_#AA  <br/> |E1W 1AP  <br/> |
|A#_#AA  <br/> |F1 2AG  <br/> |
|A##_#AA  <br/> |C44 8HT  <br/> |
|AA#_#AA  <br/> |DR3 6HG  <br/> |
|AA##_#AA  <br/> |FN35 1TH  <br/> |
