# <a name="what-is-address-validation"></a>¿Qué es la validación de direcciones?

Cuando configura una llamada a planes de Office 365, debe asignar un número de teléfono y dirección de emergencia a cada uno de los usuarios.
  
Validar una dirección de la calle o cívica implica asegurarse de que es correcto y también tienen un formato correcto. Es posible que una dirección de emergencia parcialmente correcta, como el nombre de la ciudad, mal puede pasar todavía paso de validación. El proceso de validación utiliza todas las partes de una dirección determinada para determinar si contiene suficiente información para enrutar la llamada al centro de servicios de emergencia apropiados. Si es así, se devolverá como validado y, a continuación, puede asignarse a un número de teléfono.

> [!NOTE]
> Si necesita obtener más números de teléfono que esto, ponte [en contacto con soporte técnico para productos de empresa - Admin ayuda](https://support.office.com/article/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b)
