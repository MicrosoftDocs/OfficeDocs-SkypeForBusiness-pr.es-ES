# ¿Cuál es el estado de mis solicitudes de portabilidad?

> [!IMPORTANT]
> Este artículo se ha traducido con traducción automática; vea la [declinación de responsabilidades](f651e82d-4237-4e3d-ad74-40fdf11fa8d1.md#MT_Footer). Para su referencia, puede encontrar la versión en inglés de este artículo [aquí](https://support.office.com/en-us/article/f651e82d-4237-4e3d-ad74-40fdf11fa8d1). 
  
Puede ver el estado de su pedido puerto yendo a la **Skype centro de administración de negocio** > **voz** > **pedidos de puerto**. Cada estado de orden de puerto se mostrarán en la columna **estado**. Si ayuda, [Póngase en contacto con el soporte de Office 365 para empresas: ayuda para administradores](http://technet.microsoft.com/library/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b%28Office.14%29.aspx).
  
La siguiente tabla enumera los Estados de orden de puerto, así como las acciones que puede realizar si es necesario.
  
|**Estado**|**¿Puede ver la solicitud?**|**¿Puede editar la solicitud?**|**¿Puede cancelar la solicitud?**|**¿Puede eliminar la solicitud?**|**Descripción**|
|:-----|:-----|:-----|:-----|:-----|:-----|
|**Procesando** <br/> |Sí  <br/> |No  <br/> |Sí  <br/> |No  <br/> |El administrador ha creado el pedido y se han recibido por Microsoft.  <br/> |
|**Contactando con el operador** <br/> |Sí  <br/> |No  <br/> |Sí  <br/> |No  <br/> |El orden se ha recibido y aprobados por Microsoft y estamos trabajando con el transportista perder obtenerlo aprobado.  <br/> |
|**Transferencia aprobada** <br/> |Sí  <br/> |Compromiso de solicitud en firme (FOC)  <br/> |Sí  <br/> |No  <br/> |El orden se ha aceptado el transportista vencida y se ha establecido la fecha FOC.  <br/> |
|**Transferencia pendiente** <br/> |Sí  <br/> |No  <br/> |No  <br/> |No  <br/> |La transferencia es menos de 24 horas, por lo que el orden ya no puede ser editado o cancelado.  <br/> |
|**Error** <br/> |No  <br/> |Sí  <br/> |Sí  <br/> |Sí (en este momento, no puede eliminar el orden de puerto si hay un error. El orden de puerto que se debe volver a crearse o debe ponerse [Póngase en contacto con el soporte de Office 365 para empresas: ayuda para administradores](http://technet.microsoft.com/library/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b%28Office.14%29.aspx).  <br/> |El operador descartado ha rechazado la solicitud.  <br/> |
|**Completada** <br/> |Sí  <br/> |No  <br/> |No  <br/> |No  <br/> |Los números se han transferido correctamente.  <br/> |
|**Cancelada** <br/> |No  <br/> |Sí  <br/> |No  <br/> |No  <br/> |El administrador ha cancelado la solicitud.  <br/> |
   
 **Para obtener instrucciones detalladas paso a paso, consulte [Números de teléfono de transferencia a Office 365](transfer-phone-numbers-to-office-365.md).**
