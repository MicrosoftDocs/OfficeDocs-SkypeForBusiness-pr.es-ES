# <a name="phone-number-management-for-the-us"></a>Administración de número de teléfono para Estados Unidos

Utilice esta tabla para obtener información sobre la obtención y administración de números de teléfono en los Estados Unidos (y Puerto Rico) para Skype para los negocios en línea. 
  
Para obtener más información, vea [administrar números de teléfono para su organización](manage-phone-numbers-for-your-organization.md).
  
|**¿Qué desea hacer?**|**Siga estos pasos y descargue los formularios correctos**|
|:-----|:-----|
|Necesario obtener números de usuario que no se enumeran en el Skype para el centro de administración de negocios.  <br/> |Descargue el [nueva solicitud de número de teléfono de Estados Unidos (números de usuario) (v.3.0(en.us)](../../downloads/new-phone-number-request-for-the-u.s.-(user-numbers)-(v.3.0)-(en.us).pdf) formulario, rellenar hacia fuera y [enviarlo a nosotros](mailto:gcstnmsd@microsoft.com).  <br/> |
|Debe obtener los números de teléfono que se transfieren a nosotros.  <br/> | Descargue una de estas formas: <br/> [Carta de autorización (LOA) para los Estados Unidos (números de servicio y de usuario) (v.3.1)(en.us)](../../downloads/LOA-forms/letter-of-authorization-(loa)-for-the-u.s.-(user-and-service-numbers)-(v.3.1)-(en.us).pdf) <br/> [Carta de autorización (LOA) para los Estados Unidos (números gratuitos) (v.3.1)(en.us)](../../downloads/LOA-forms/letter-of-authorization-(loa)-for-the-u.s.-(toll-free-numbers)-(v.3.1)-(en.us).pdf) <br/><br/>**Nota:** Si está intentando obtener más de 999 números de teléfono de abonado o usuario, necesitará rellenar hacia fuera una carta de autorización (LOA) y envíenoslo. Para obtener más información, consulte [transferencia de números de teléfono para Office 365](../../what-are-calling-plans-in-office-365/transfer-phone-numbers-to-office-365.md) .           Rellenar hacia fuera y [enviarlo a nosotros](mailto:ptn@microsoft.com).  <br/> |
|Debe cambiar el número de un usuario a un número de servicio.  <br/> |[Envíenos](mailto:ptn@microsoft.com) su NIP y la lista de números de usuario que desea convertir en números de servicio. Para configurar un PIN, vaya [aquí](../../what-are-calling-plans-in-office-365/set-your-pin-for-transferring-numbers-to-a-new-service-provider.md).  <br/> |
|Debe transferir números de una organización de Office 365 a otro.  <br/> |[Envíenos](mailto:ptn@microsoft.com) la lista de números de teléfono del usuario (suscriptor) que va a transferir junto con el PIN de la organización de origen que desea transferir los números de. Para configurar un PIN, vaya [aquí](../../what-are-calling-plans-in-office-365/set-your-pin-for-transferring-numbers-to-a-new-service-provider.md).  <br/> |
