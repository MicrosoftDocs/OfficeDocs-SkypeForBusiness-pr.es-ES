# <a name="latest-updates-for-versions-of-skype-for-business-that-use-windows-installer-msi"></a>Actualizaciones más recientes para las versiones de Skype Empresarial que usan Windows Installer (MSI)

Use los vínculos de esta página para obtener más información y descargar las actualizaciones más recientes para las versiones perpetuas de los clientes de Skype Empresarial 2016, Skype Empresarial 2015 y Lync 2013.

> [!NOTE]
> - La información de este artículo solo se aplica a las versiones perpetuas de Skype Empresarial que usan la tecnología de instalación Windows Installer (MSI). Por ejemplo, si instaló una versión con licencia por volumen de Skype Empresarial, como Skype Empresarial se incluye con Office Profesional Plus 2016.
> - La información de este artículo no se aplica Microsoft 365 o Office 365 versiones de Skype Empresarial, como Skype Empresarial se incluyen con Aplicaciones Microsoft 365 para empresas.
> - Para obtener las actualizaciones de cliente móvil más recientes, vaya a la tienda de aplicaciones de su cliente móvil (iTunes, Google Play o Microsoft Store) para ver detalles e instalar actualizaciones.
> - Para averiguar qué versión de Skype Empresarial está usando, vea ¿Qué versión de Skype Empresarial [tengo?](https://support.microsoft.com/help/3195481/how-to-check-the-version-of-skype-for-business-2016)
> - Para obtener más información sobre la instalación de actualizaciones de Office, vea [Instalación de actualizaciones de Office](https://support.office.com/article/2ab296f3-7f03-43a2-8e50-46de917611c5).

Skype Empresarial compatibilidad con Microsoft 365 o Office 365 y el entorno local depende de los requisitos del sistema para la implementación:

- Microsoft 365 y Office 365 funcionan con cualquier versión de Skype Empresarial que esté en soporte estándar, que incluye la última versión de Skype Empresarial 2016. En versiones anteriores de Skype Empresarial, solo las que tienen compatibilidad extendida pueden seguir funcionando con Microsoft 365 o Office 365, aunque con funcionalidad reducida. Para obtener más información, vea [Directiva de ciclo de vida de Microsoft](https://support.microsoft.com/lifecycle).

- Para Skype Empresarial Server implementaciones locales, siga los requisitos del sistema para la versión de Skype Empresarial:

  - [Requisitos para su Skype Empresarial entorno](./plan-your-deployment/requirements-for-your-environment/requirements-for-your-environment.md)

  - [Requisitos del sistema de Lync Server 2013](/previous-versions/office/lync-server-2013/lync-server-2013-system-requirements)
