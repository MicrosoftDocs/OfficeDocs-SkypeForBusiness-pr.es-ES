# <a name="manage-pin-settings-in-skype-for-business-server"></a>Administrar la configuración de PIN en Skype Empresarial Server
 
**Resumen:** Administrar la configuración de PIN en Skype Empresarial Server.
  
Use los procedimientos descritos en las secciones siguientes para administrar LOSPIN en Skype Empresarial Server.
  
Puede administrar directivas de PIN de Skype Empresarial Server desde el Panel de control de Skype Empresarial Server o desde el Shell de administración de Skype Empresarial Server. Use los siguientes procedimientos para configurar directivas de PIN para su organización.
  
- [Ver información de directiva de PIN en Skype Empresarial Server](view-pin-policy-information.md)
    
- [Crear una nueva directiva de PIN en Skype Empresarial Server](create-a-new-pin-policy.md)
    
- [Modificar una directiva de PIN existente en Skype Empresarial Server](modify-an-existing-pin-policy.md)
    
- [Eliminar una directiva de PIN en Skype Empresarial Server](delete-a-pin-policy.md)
    
- [Asignar una directiva de PIN por usuario en Skype Empresarial Server](assign-a-per-user-pin-policy.md)
    
Use los siguientes procedimientos para administrar los NÚMEROS DE  PN de conferencias de acceso telefónico local de los usuarios desde la sección Usuarios del Panel de control de Skype Empresarial Server.
  
- [Establecer el PIN de conferencia de acceso telefónico local de un usuario en Skype Empresarial Server](set-a-user-s-dial-in-conferencing-pin.md)
    
- [Ver información de PIN de usuario en Skype Empresarial Server](view-user-pin-information.md)
    
- [Bloquear o desbloquear un PIN de usuario en Skype Empresarial Server](lock-or-unlock-a-user-pin.md)
