# <a name="manage-pin-settings-in-skype-for-business-server"></a>Administrar la configuración del PIN en Skype empresarial Server
 
**Resumen:** Administrar la configuración del PIN en Skype empresarial Server.
  
Use los procedimientos de las siguientes secciones para administrar los pin de Skype empresarial Server.
  
Puede administrar las directivas de PIN de Skype empresarial Server desde el panel de control de Skype empresarial Server o desde el shell de administración de Skype empresarial. Use los procedimientos siguientes para configurar las directivas de PIN para su organización.
  
- [Ver información de directivas de PIN en Skype empresarial Server](view-pin-policy-information.md)
    
- [Crear una nueva Directiva de PIN en Skype empresarial Server](create-a-new-pin-policy.md)
    
- [Modificar una directiva de PIN existente en Skype empresarial Server](modify-an-existing-pin-policy.md)
    
- [Eliminar una directiva de PIN en Skype empresarial Server](delete-a-pin-policy.md)
    
- [Asignar una directiva de PIN por usuario en Skype empresarial Server](assign-a-per-user-pin-policy.md)
    
Use los procedimientos siguientes para administrar los pin de conferencia de acceso telefónico local de los usuarios desde la sección **usuarios** del panel de control de Skype empresarial Server.
  
- [Configurar el PIN de conferencias de acceso telefónico local de un usuario en Skype empresarial Server](set-a-user-s-dial-in-conferencing-pin.md)
    
- [Ver información de PIN de usuario en Skype empresarial Server](view-user-pin-information.md)
    
- [Bloquear o desbloquear un PIN de usuario en Skype empresarial Server](lock-or-unlock-a-user-pin.md)
