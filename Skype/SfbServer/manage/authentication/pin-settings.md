# <a name="manage-pin-settings-in-skype-for-business-server"></a>Administrar la configuración de PIN en Skype para Business Server
 
**Resumen:** Administrar la configuración de PIN en Skype para Business Server.
  
Use los procedimientos descritos en las secciones siguientes para administrar los PIN de Skype para Business Server.
  
Puede administrar Skype para las directivas de PIN de servidor empresarial de Skype para el Panel de Control de servidor empresarial o Skype para Shell de administración de servidor empresarial. Use los procedimientos siguientes para configurar las directivas de PIN para su organización.
  
- [Ver información de directivas PIN de Skype para Business Server](view-pin-policy-information.md)
    
- [Crear una nueva directiva PIN en Skype para Business Server](create-a-new-pin-policy.md)
    
- [Modificar una directiva PIN existente en Skype para Business Server](modify-an-existing-pin-policy.md)
    
- [Eliminar una directiva de PIN en Skype para Business Server](delete-a-pin-policy.md)
    
- [Asignar una directiva de PIN por usuario en Skype para Business Server](assign-a-per-user-pin-policy.md)
    
Use los procedimientos siguientes para administrar los PIN de conferencia de acceso telefónico de los usuarios de la sección **usuarios** de Skype para el Panel de Control de servidor empresarial.
  
- [Establecer telefónico un usuario PIN en Skype para Business Server](set-a-user-s-dial-in-conferencing-pin.md)
    
- [Vista de usuario información del PIN en Skype para Business Server](view-user-pin-information.md)
    
- [Bloquear o desbloquear un usuario PIN en Skype para Business Server](lock-or-unlock-a-user-pin.md)
