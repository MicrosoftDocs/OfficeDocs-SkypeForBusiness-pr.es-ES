# <a name="manage-pin-settings-in-skype-for-business-server-2015"></a>Administrar la configuración de PIN en Skype Empresarial Server 2015
 
**Resumen:** Administrar la configuración de pines en Skype para Business Server 2015.
  
Utilice los procedimientos en las secciones siguientes para administrar los pines en Skype para Business Server 2015.
  
Puede administrar Skype para Business Server 2015 PIN políticas de Skype para Business Server Control Panel o Skype para el Shell de administración de servidor empresarial. Use los procedimientos siguientes para configurar las directivas de PIN para su organización.
  
- [Ver información de la directiva de NIP en Skype para Business Server 2015](view-pin-policy-information.md)
    
- [Crear una nueva directiva de NIP en Skype para Business Server 2015](create-a-new-pin-policy.md)
    
- [Modificar una directiva existente de PIN en Skype para Business Server 2015](modify-an-existing-pin-policy.md)
    
- [Eliminar una directiva de NIP en Skype para Business Server 2015](delete-a-pin-policy.md)
    
- [Asignar una directiva PIN de cada usuario en Skype para Business Server 2015](assign-a-per-user-pin-policy.md)
    
Utilice los procedimientos siguientes para administrar los pines de conferencia telefónica de los usuarios desde la sección de **usuarios** de Skype para Panel de Control de servidor de Business.
  
- [Establecer conferencias en el marcado de un usuario PIN en Skype para Business Server 2015](set-a-user-s-dial-in-conferencing-pin.md)
    
- [Vista de usuario información del NIP de Skype de Business Server 2015](view-user-pin-information.md)
    
- [Bloquear o desbloquear un PIN en Skype usuario Business Server 2015](lock-or-unlock-a-user-pin.md)
