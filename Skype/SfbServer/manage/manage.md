# <a name="manage-skype-for-business-server"></a>Administrar Skype Empresarial Server 

**Resumen:** Explore estos vínculos para obtener información sobre cómo administrar y mantener Skype para Business Server.
  
Skype para Business Server ofrece mensajería instantánea (IM), presencia, conferencias, vídeo y soluciones de telefonía que pueden satisfacer los requisitos de colaboración de la empresa. Este tema proporciona vínculos a información acerca de cómo configurar, supervisar y mantener su Skype para la implementación de Business Server. 
  
Los autores de los artículos los actualizan de forma continua a medida que se encuentra disponible nueva información y que los usuarios proporcionan comentarios. Esta página se actualiza con frecuencia, de modo que compruébela con regularidad.
