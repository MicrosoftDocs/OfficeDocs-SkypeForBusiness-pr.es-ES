# <a name="system-usage-reports-in-skype-for-business-server-2015"></a>Informes de uso del sistema en Skype Empresarial Server 2015
 
**Resumen:** Obtenga información acerca de los informes de uso del sistema en Skype para Business Server 2015.
  
Los informes de uso del sistema proporcionan información de uso del sistema basado en detalle de llamadas (CDR) los datos recopilados por el Skype para Business Server de grabación.
  
## <a name="in-this-section"></a>En esta sección

- [Informe de registro de usuario en Skype para Business Server 2015](user-registration-report.md)
    
    Proporciona un resumen de conectividad de usuario para el Skype para la implementación de Business Server 2015 según los eventos del registro, como inicios de sesión de usuario. El informe proporciona un modo para ver los inicios de sesión interno y externo y para comparar el número de usuarios que ha iniciado sesión en Skype para Business Server 2015 con el número de usuarios que utiliza realmente el servicio cuando se hubieran iniciado sesión.
    
- [Informe de resumen de actividad de Peer-to-Peer de Skype para Business Server 2015](peer-to-peer-activity-summary-report.md)
    
    Proporciona un resumen de las sesiones punto a punto de mensajería instantánea (MI), audio, vídeo, transferencia de archivos y uso compartido de aplicaciones. Las sesiones punto a punto son sesiones en las que hay solamente dos usuarios.
    
- [Informe de resumen de conferencia en Skype para Business Server 2015](conference-summary-report.md)
    
    Proporciona un resumen de todas las actividades de conferencia. Las conferencias son sesiones en las que hay tres o más usuarios.
    
- [Conferencia PSTN informe resumido en Skype para Business Server 2015](pstn-conference-summary-report.md)
    
    Proporciona un resumen de todas las conferencias RTC. Se trata de conferencias donde se conecta usando la red telefónica pública conmutada (PSTN), que también se conoce como conferencia telefónica de al menos un usuario.
    
- [Informe de uso de grupo de respuesta en Skype para Business Server 2015](response-group-usage-report.md)
    
    Proporciona un resumen de uso de un grupo de respuesta. La aplicación de grupo de respuesta proporciona una manera para que automáticamente enrutar llamadas de teléfono a entidades como una ayuda de línea de soporte técnico de escritorio o cliente.
    
- [Informe de inventario del teléfono IP en Skype para Business Server 2015](ip-phone-inventory-report.md)
    
    Proporciona información sobre los teléfonos IP que actualmente se usan en la organización. El informe se basa en los inicios de sesión y los registros del teléfono. No se tiene que tratar como un inventario completo. Por ejemplo, es posible que ya no utilice teléfonos que aún se mencionan en el informe (porque iniciaron sesión al menos una vez). Del mismo modo, también podría tener teléfonos nuevos que no se muestran en el informe simplemente porque los usuarios no iniciado sesión en Skype para Business Server con sus nuevos teléfonos todavía.
    
- [Llamar al informe de Control de admisión en Skype para Business Server 2015](call-admission-control-report.md)
    
    Proporciona una lista de actividades punto a punto y de conferencia que utilizan el servicio de control de admisión de llamadas. El control de admisión de llamadas (CAC) es una forma de determinar, en función de las restricciones de ancho de banda, si está permitido establecer sesiones de comunicación en tiempo real, como las llamadas de voz o vídeo.
