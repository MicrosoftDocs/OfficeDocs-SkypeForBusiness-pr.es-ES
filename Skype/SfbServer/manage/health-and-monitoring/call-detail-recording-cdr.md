# <a name="call-detail-recording-cdr-in-skype-for-business-server"></a>Grabación de detalles de llamadas (CDR) en Skype empresarial Server
 
**Resumen:** Obtenga más información sobre los registros de grabación de detalles de llamadas (CDR) que se usan en Skype empresarial Server.
  
El registro detallado de llamadas (CDR) registra la información de uso y diagnóstico sobre actividades punto a punto, como la mensajería instantánea, las llamadas de voz sobre IP (VoIP), el uso compartido de aplicaciones, la transferencia de archivos y las reuniones. Los datos de uso pueden servir para calcular el rendimiento de la inversión y los datos de diagnóstico se pueden emplear para solucionar problemas de reuniones y actividades punto a punto. Al instalar Skype empresarial Server, también instalará una colección predefinida de opciones de configuración global para CDR. Use los temas de esta sección para configurar CDR.
