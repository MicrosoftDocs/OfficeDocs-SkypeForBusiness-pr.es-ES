# <a name="archiving-planning-tool"></a>Herramienta de planeación de archivado
 
El archivado en Skype Empresarial Server proporciona una forma de archivar las comunicaciones que se envían a través de Skype Empresarial Server.
  
Para obtener más información sobre cómo preparar el entorno para el servidor de archivado, vea [Plan for archiving in Skype Empresarial Server 2015](../../plan-your-deployment/archiving/archiving.md).
