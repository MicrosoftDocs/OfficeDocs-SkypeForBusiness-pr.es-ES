# <a name="dial-in-conferencing-planning-tool"></a>Marcado de conferencias (herramienta de planeación)
 
Skype para Business Server 2015 marcado en conferencias permite a los usuarios a unirse a la parte de audio de un Skype para conferencia Business Server 2015 mediante el uso de un teléfono de telefónica pública conmutada (PSTN) de la red sin necesidad de un proveedor de conferencia de audio de terceros.
  
Para obtener información sobre la planeación de la habilitación de la conferencia de acceso telefónico en la organización, consulte el siguiente tema y los temas a los que se vincula: 
  
- [Plan para conferencias en Skype para Business Server 2015](../../plan-your-deployment/conferencing/conferencing.md)
