# <a name="federation-planning-tool"></a>Federación (herramienta de planeación)
 
Los servidores perimetrales se pueden configurar para permitir que los usuarios internos y externos tengan acceso a contactos de organizaciones o servicios asociados.
  
 La Federación funciona como en el caso de Microsoft Lync 2013. Para obtener más información acerca de la preparación del entorno para la Federación, vea los siguientes temas:
  
- [Planeación de la Federación de Lync Server 2013 y Office Communications Server](https://technet.microsoft.com/library/jj205335%28v=ocs.15%29.aspx)
    
- [Planeación de la conectividad de mensajería instantánea pública en Lync Server 2013](https://technet.microsoft.com/library/jj205349%28v=ocs.15%29.aspx)
    
- [Planeación de la Federación de protocolo extensible de mensajería y presencia (XMPP) en Lync Server 2013](https://technet.microsoft.com/library/jj205107%28v=ocs.15%29.aspx)
