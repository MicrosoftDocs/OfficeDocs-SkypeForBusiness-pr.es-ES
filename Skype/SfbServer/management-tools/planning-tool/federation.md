# <a name="federation-planning-tool"></a>Herramienta de planeación de federación
 
Los servidores perimetrales se pueden configurar para permitir que los usuarios internos y externos accedan a los contactos en organizaciones o servicios asociados.
  
 La federación funciona como lo hacía con Microsoft Lync 2013. Para obtener más información sobre cómo preparar el entorno para federación, consulte los siguientes temas:
  
- [Planeación de la federación de Lync Server 2013 y Office Communications Server](/previous-versions/office/lync-server-2013/lync-server-2013-planning-for-lync-server-and-office-communications-server-federation)
    
- [Planeación de la conectividad de mensajería instantánea pública en Lync Server 2013](/previous-versions/office/lync-server-2013/lync-server-2013-planning-for-public-instant-messaging-connectivity)
    
- [Planeación de la federación de protocolo de presencia y mensajería extensible (XMPP) en Lync Server 2013](/previous-versions/office/lync-server-2013/lync-server-2013-planning-for-extensible-messaging-and-presence-protocol-xmpp-federation)
