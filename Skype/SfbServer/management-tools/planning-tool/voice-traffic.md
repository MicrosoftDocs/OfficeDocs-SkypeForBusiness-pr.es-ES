# <a name="voice-traffic-planning-tool"></a>Herramienta de planeación de tráfico de voz
 
La herramienta de planeación usa información sobre el tráfico de usuarios para determinar el número de puertos que se usarán en la implementación. Este número, a su vez, determina el número de servidores de mediación y puertas de enlace que se necesitan. Para obtener más información, vea [Estimating voice usage and traffic for Skype Empresarial Server 2015](../../plan-your-deployment/capacity/estimating-voice-traffic.md).
