# <a name="voice-traffic-planning-tool"></a>Tráfico de voz (herramienta de planeación)
 
La herramienta de planificación utiliza información acerca del tráfico de usuario para determinar el número de puertos para uso en la implementación. A su vez, este número determina el número de servidores de mediación y puertas de enlace que serán necesarios. Para obtener más información, consulte [uso de la voz de estimación y tráfico de Skype para Business Server 2015](../../plan-your-deployment/capacity/estimating-voice-traffic.md).
