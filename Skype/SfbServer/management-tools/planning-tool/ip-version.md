# <a name="ip-version-planning-tool"></a>Versión IP (herramienta de planeación)
 
Tiene tres opciones al configurar direcciones IP en Skype para Business Server 2015. Puede configurar Skype para Business Server 2015 admitir sólo IP versión 4 (IPv4), sólo IP versión 6 (IPv6), o una combinación de ambos (conocido como pila doble).
  
Para obtener más información acerca de cómo elegir su infraestructura IP, consulte [Plan para IPv6 en Skype para el negocio](../../plan-your-deployment/network-requirements/ipv6.md).
