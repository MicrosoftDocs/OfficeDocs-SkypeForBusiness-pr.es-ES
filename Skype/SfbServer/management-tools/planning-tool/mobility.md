# <a name="mobility-planning-tool"></a>Herramienta de planeación de movilidad
 
La característica de movilidad de Skype Empresarial Server 2015 admite Skype Empresarial funcionalidad en dispositivos móviles.
  
La funcionalidad de movilidad es la misma que en Lync Server 2013. Para obtener más información acerca de cómo preparar el entorno para la movilidad, vea [Plan for Mobility for Skype Empresarial Server](../../plan-your-deployment/mobility.md).
