# <a name="mobility-planning-tool"></a>Mobility (Planning Tool)
 
La característica de movilidad de Skype para Business Server 2015 admite Skype para la funcionalidad empresarial en dispositivos móviles.
  
Funcionalidad de movilidad es la misma que en Lync Server 2013. Para obtener más información acerca de cómo preparar el entorno para la movilidad, consulte [Plan para la movilidad de Skype para Business Server](../../plan-your-deployment/mobility.md).
