# <a name="mediation-collocation-planning-tool"></a>Colocación de mediación (herramienta de planeación)
 
El servidor de mediación está ubicado en el servidor Front-End de forma predeterminada. El servidor de mediación también puede implementarse en un grupo independiente por motivos de rendimiento, o si implementa trunking SIP, en cuyo caso el grupo independiente es muy recomendable. Colocación en Skype para las funciones de servidor de negocios al igual que ocurría en Lync Server 2013. Para obtener más información, consulte los temas siguientes:
  
- [Colocación de servidores compatibles en Lync Server 2013](https://technet.microsoft.com/en-us/library/gg425885%28v=ocs.15%29.aspx)
    
- [Componente de servidor de mediación de Skype para Business Server 2015](../../plan-your-deployment/enterprise-voice-solution/mediation-server.md)
