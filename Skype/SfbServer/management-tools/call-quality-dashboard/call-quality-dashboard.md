# <a name="call-quality-dashboard-for-skype-for-business-server-2015"></a>Panel de calidad de la llamada de Skype para Business Server 2015
 
**Resumen:** Obtener información sobre el panel de calidad llamar, que es una herramienta de reporting de Skype para Business Server 2015.
  
## 

El Skype para Business Server 2015 llamar calidad Dashboard (CQD) es una capa de informes sobre la calidad de experiencia de base de datos en el servidor de supervisión en Skype para Business Server 2015 y Lync Server 2013. CQD utiliza Microsoft SQL Server Analysis Services para proporcionar uso adicional y llamar a información de calidad así como filtrado y desplazamiento en el conjunto de datos.
