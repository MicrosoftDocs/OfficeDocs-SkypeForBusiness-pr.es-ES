# <a name="call-quality-dashboard-for-skype-for-business-server"></a>Panel de calidad de llamadas para Skype Empresarial Server
 
**Resumen:** Obtenga más información sobre el panel de calidad de llamadas, que es una herramienta de creación de informes para Skype empresarial Server.
 
El panel de calidad de llamadas de Skype empresarial Server (CQD) es un nivel de informes que se encuentra en la parte superior de la base de datos de la calidad de la experiencia en el servidor de supervisión de Skype empresarial Server y Lync Server 2013. El CQD usa Microsoft SQL Server Analysis Services para proporcionar información sobre la calidad de las llamadas y el uso agregado, así como el filtrado y la dinamización del conjunto de datos.
  
Los siguientes temas se incluyen en esta sección y deben revisarse antes de implementar el panel de calidad de llamadas.
