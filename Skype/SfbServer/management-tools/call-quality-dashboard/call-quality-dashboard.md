# <a name="call-quality-dashboard-for-skype-for-business-server"></a>Panel de calidad de llamadas para Skype Empresarial Server
 
**Resumen:** Obtenga información sobre el Panel de calidad de llamadas, que es una herramienta de informes para Skype Empresarial Server.
 
El Skype Empresarial Server de calidad de llamadas (CQD) es una capa de informes sobre la base de datos de calidad de la experiencia en el servidor de supervisión en Skype Empresarial Server y Lync Server 2013. CQD usa Microsoft SQL Server Analysis Services para proporcionar información de calidad de llamadas y uso agregados, así como filtrar y pivotar en el conjunto de datos.
  
Los siguientes temas se incluyen en esta sección y deben revisarse antes de implementar el Panel de calidad de llamadas.
