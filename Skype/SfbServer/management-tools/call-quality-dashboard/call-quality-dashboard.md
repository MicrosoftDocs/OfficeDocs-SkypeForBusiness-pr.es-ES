# <a name="call-quality-dashboard-for-skype-for-business-server"></a>Panel de calidad de llamada de Skype para Business Server
 
**Resumen:** Obtenga información sobre el panel de calidad de llamadas, que es una herramienta de informes de Skype para Business Server.
 
El Skype para servidor de llamadas calidad panel (CQD) de negocio es una capa informes sobre la calidad de experiencia de base de datos en el servidor de supervisión en Skype para Business Server. CQD utiliza Microsoft SQL Server Analysis Services para proporcionar uso agregado e información sobre la calidad, así como el filtrado y creación de tablas dinámicas en el conjunto de datos de llamadas.
