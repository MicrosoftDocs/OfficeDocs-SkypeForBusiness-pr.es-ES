# <a name="develop-call-quality-dashboard-for-skype-for-business-server-2015"></a>Desarrollar llamada Quality Dashboard para Skype para Business Server 2015
 
**Resumen:** Aprenda a desarrollar soluciones para llamar al panel de calidad. Panel de calidad de la llamada es una herramienta de Skype para Business Server 2015.
  
Este tema proporciona la documentación de desarrollo para llamar a calidad Dashboard (CQD) de Skype para Business Server 2015.
  
## <a name="develop-call-quality-dashboard-for-skype-for-business-server-2015"></a>Desarrollar llamada Quality Dashboard para Skype para Business Server 2015

CQD se puede ampliar e integrado a su entorno de creación de informes mediante desarrollo personalizado. 
  
Existen dos conjuntos de APIs de web, uno para el acceso a los datos del cubo (a través de las API de datos) y otro para el acceso a los metadatos de informe (a través de la API de repositorio). 
  
- [API de datos de llamada Quality Dashboard (CQD) en Skype para Business Server 2015](data-api.md)
    
- [API del depósito para el panel de calidad de la llamada (CQD) en Skype para Business Server 2015](repository-api.md)
    
- [Ejemplos de desarrollo CQD](cqd-development-samples.md)
