# <a name="develop-call-quality-dashboard-for-skype-for-business-server-2015"></a>Desarrollar el panel de calidad de llamada de Skype para Business Server 2015
 
**Resumen:** Obtenga información sobre cómo desarrollar soluciones para llamar al panel de calidad. Panel de calidad de llamada es una herramienta de Skype para Business Server 2015.
  
En este tema se proporciona documentación de desarrollo para el panel de calidad de llamadas (CQD) de Skype para Business Server 2015.
  
## <a name="develop-call-quality-dashboard-for-skype-for-business-server-2015"></a>Desarrollar el panel de calidad de llamada de Skype para Business Server 2015

CQD se pueden extender e integrado en su entorno informes mediante desarrollo personalizado. 
  
Dos conjuntos de API web están disponibles, uno para obtener acceso a datos del cubo (a través de las API de datos) y otro para obtener acceso a los metadatos de informe (a través de la API de repositorio). 
  
- [API de datos para el panel de calidad de llamada (CQD) en Skype para Business Server 2015](data-api.md)
    
- [API del depósito para el panel de calidad de llamada (CQD) en Skype para Business Server 2015](repository-api.md)
    
- [Ejemplos de desarrollo de CQD](cqd-development-samples.md)
