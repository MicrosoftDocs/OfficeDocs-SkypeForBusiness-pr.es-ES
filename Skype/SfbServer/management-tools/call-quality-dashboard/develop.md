# <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>Desarrollar el panel de calidad de llamadas para Skype empresarial Server
 
**Resumen:** Más información sobre cómo desarrollar soluciones para el panel de calidad de llamadas. El panel de calidad de llamadas es una herramienta para Skype empresarial Server.
  
En este tema se proporciona documentación de desarrollo para el panel de calidad de llamadas (CQD) para Skype empresarial Server.
  
## <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>Desarrollar el panel de calidad de llamadas para Skype empresarial Server

El CQD puede ampliarse e integrarse en su entorno de creación de informes mediante programación personalizada. 
  
Hay dos conjuntos de API Web disponibles, uno para acceder a los datos del cubo (a través de las API de datos) y otro para acceder a los metadatos del informe (mediante la API del repositorio). 
  
- [API de datos para el panel de calidad de llamadas (CQD) en Skype empresarial Server](data-api.md)
    
- [API de repositorio para el panel de calidad de llamadas (CQD) en Skype empresarial Server](repository-api.md)
    
- [Ejemplos de desarrollo de CQD](cqd-development-samples.md)
