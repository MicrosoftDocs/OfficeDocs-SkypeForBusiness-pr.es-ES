# <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>Desarrollar panel de calidad de llamadas para Skype Empresarial Server
 
**Resumen:** Obtenga información sobre cómo desarrollar soluciones para el Panel de calidad de llamadas. Panel de calidad de llamadas es una herramienta para Skype Empresarial Server.
  
En este tema se proporciona documentación de desarrollo para el Panel de calidad de llamadas (CQD) para Skype Empresarial Server.
  
## <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>Desarrollar panel de calidad de llamadas para Skype Empresarial Server

CQD se puede extender e integrar en el entorno de informes mediante el desarrollo personalizado. 
  
Hay dos conjuntos de API web disponibles, uno para obtener acceso a datos en el cubo (a través de API de datos) y otro para obtener acceso a los metadatos del informe (a través de la API de repositorio). 
  
- [API de datos para panel de calidad de llamadas (CQD) en Skype Empresarial Server](data-api.md)
    
- [API de repositorio para panel de calidad de llamadas (CQD) en Skype Empresarial Server](repository-api.md)
    
- [Ejemplos de desarrollo de CQD](cqd-development-samples.md)
