# <a name="skype-for-business-server-2015-management-tools"></a>Herramientas de administración de Skype Empresarial Server 2015
 
**Resumen:** Obtenga información sobre las herramientas de administración de servicio de Skype para Business Server 2015.
  
Skype para el software de comunicaciones de Business Server 2015 (anteriormente Lync Server), ofrece mensajería instantánea (IM), presencia, conferencias y soluciones de telefonía que son compatibles con los requerimientos de colaboración de nivel empresarial. Las herramientas para administrar estos servicios son flexibles y avanzadas.
