# <a name="skype-for-business-server-2015-management-tools"></a>Herramientas de administración de Skype Empresarial Server 2015
 
**Resumen:** Obtenga información acerca de las herramientas de administración de servicio de Skype para Business Server 2015.
  
Skype para el software de comunicaciones empresariales Server 2015 (anteriormente Lync Server), ofrece mensajería instantánea (IM), presencia, conferencias y soluciones de telefonía que admiten los requisitos de colaboración de la empresa. Las herramientas para administrar estos servicios son flexibles y avanzadas.
