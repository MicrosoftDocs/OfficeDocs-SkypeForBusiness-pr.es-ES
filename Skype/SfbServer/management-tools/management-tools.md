# <a name="skype-for-business-server-2015-management-tools"></a>Herramientas de administración de Skype Empresarial Server 2015
 
**Resumen:** Obtenga más información sobre las herramientas de administración de servicios de Skype empresarial Server 2015.
  
El software de comunicaciones de Skype empresarial Server 2015 (anteriormente Lync Server) ofrece mensajería instantánea, presencia, Conferencia y soluciones de telefonía que admiten los requisitos de colaboración a nivel empresarial. Las herramientas para administrar estos servicios son flexibles y avanzadas.
