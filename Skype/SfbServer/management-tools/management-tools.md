# <a name="skype-for-business-server-2015-management-tools"></a>Herramientas de administración de Skype empresarial Server 2015
 
**Resumen:** Obtenga información sobre las herramientas de administración de servicios en Skype empresarial Server 2015.
  
El software de comunicaciones de Skype empresarial Server 2015 (anteriormente Lync Server) ofrece mensajería instantánea (mi), presencia, conferencias y soluciones de telefonía que admiten los requisitos de colaboración en el nivel empresarial. Las herramientas para administrar estos servicios son flexibles y eficaces.
