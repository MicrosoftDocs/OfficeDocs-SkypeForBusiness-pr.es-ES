# <a name="plan-for-your-skype-for-business-server-2015-deployment"></a>Planificar la implementación de Skype Empresarial Server 2015
 
**Resumen:** Explore estos vínculos para aprender a desarrollar conceptuales, lógicos y diseños físicos para configurar Skype para características Business Server 2015, servidores y topologías.
  
Skype para Business Server ofrece mensajería instantánea (IM), presencia, conferencias, vídeo y soluciones de telefonía que pueden soportar los requerimientos de colaboración de nivel empresarial. Este tema proporciona vínculos a información acerca de cómo planear una exitosa Skype para la implementación de Business Server. 
  
Los autores de los artículos los actualizan de forma continua a medida que se encuentra disponible nueva información y que los usuarios proporcionan comentarios. Esta página se actualiza con frecuencia, de modo que compruébela con regularidad.
