# <a name="requirements-for-your-skype-for-business-2015-environment"></a>Requisitos para el entorno de Skype empresarial 2015
 
**Resumen:** esquema de los temas de los requisitos del sistema y los requisitos de infraestructura o de red de Skype Empresarial Server 2015. Si tiene que instalar o configurar algo como requisito previo a la instalación e implementación de Skype Empresarial Server 2015, estará aquí.
  
Esta es la página de inicio de artículos sobre requisitos sobre una implementación de Skype empresarial Server 2015. Esto incluye tanto los requisitos del servidor como todo lo que se debe preparar, incluido el dominio y la estructura de red y los certificados.
  
## <a name="whats-in-this-topic"></a>Contenido del tema

Los dos artículos de la sección requisitos para su entorno describen lo que necesita tener y/o configurar antes de que pueda continuar e implementar correctamente o actualizar a Skype empresarial Server 2015 en el entorno de servidor de su organización.
  
### <a name="server-requirements"></a>Requisitos del servidor

[Requisitos de servidor para Skype empresarial Server 2015](server-requirements.md) son lo que necesita para prepararse en los servidores individuales que está intentando usar en su implementación.
  
### <a name="environmental-requirements"></a>Requisitos del entorno

[Los requisitos ambientales de Skype empresarial Server 2015](environmental-requirements.md) son una variedad más amplia de cosas que deben prepararse en su infraestructura antes de implementar Skype empresarial Server 2015.
