# <a name="requirements-for-your-skype-for-business-2015-environment"></a>Requisitos para su entorno de Skype Empresarial 2015
 
**Resumen:** Esquema de temas sobre los requisitos de sistemas y los requisitos de red o infraestructura para Skype Empresarial Server 2015. If you need to install or configure anything as a prerequisite to installing and deploying Skype for Business Server 2015, it will be here.
  
Esta es la página de aterrizaje de artículos sobre los requisitos de una implementación de Skype Empresarial Server 2015. Esto incluye tanto los requisitos del servidor como los aspectos más generales para prepararse, incluidos el dominio y la estructura de red, y los certificados.
  
## <a name="whats-in-this-topic"></a>¿Qué hay en este tema?

Los dos artículos de la sección de requisitos de su entorno describen lo que necesita tener o configurar antes de poder continuar e implementar correctamente o actualizar a Skype Empresarial Server 2015 en el entorno de servidor de su organización.
  
### <a name="server-requirements"></a>Requisitos de servidor

Los requisitos de servidor para [Skype Empresarial Server 2015](server-requirements.md) son los elementos que necesita para prepararse en los servidores individuales que desea usar en su implementación.
  
### <a name="environmental-requirements"></a>Requisitos del entorno

Los requisitos del entorno para Skype Empresarial [Server 2015](environmental-requirements.md) son una gama más amplia de cosas que deben estar listas en su infraestructura antes de implementar Skype Empresarial Server 2015.
