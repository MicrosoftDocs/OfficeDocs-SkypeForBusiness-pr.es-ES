# <a name="requirements-for-your-skype-for-business-2015-environment"></a>Requisitos para su Skype para entorno empresarial 2015
 
**Resumen:** esquema de los temas de los requisitos del sistema y los requisitos de infraestructura o de red de Skype Empresarial Server 2015. Si tiene que instalar o configurar algo como requisito previo a la instalación e implementación de Skype Empresarial Server 2015, estará aquí.
  
Esta es la página de inicio para artículos sobre los requisitos de alrededor de un Skype para la implementación empresarial Server 2015. Esto incluye los requisitos de servidor y las cosas más generales para preparar, incluido su dominio y estructura de redes y certificados.

Los dos artículos en los requisitos para la sección de entorno de esquema lo que necesita tener o configurar antes de poder con antelación y correctamente implementar o actualizar a Skype para Business Server 2015 en el entorno de servidor de su organización.

[Requisitos de servidor de Skype para Business Server 2015](server-requirements.md) son las cosas que debe prepararse en los servidores individuales que está va a usar en su implementación.
  
### <a name="environmental-requirements"></a>Requisitos del entorno

[Requisitos de entorno para Skype para Business Server 2015](environmental-requirements.md) es una mayor variedad de cosas que tienen que estar preparados en su infraestructura antes de implementar Skype para Business Server 2015.
