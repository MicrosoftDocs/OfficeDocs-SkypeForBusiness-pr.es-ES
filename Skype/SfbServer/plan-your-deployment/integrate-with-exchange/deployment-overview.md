# <a name="deployment-process-overview-for-integrating-on-premises-unified-messaging-and-skype-for-business"></a>Información general sobre el proceso de implementación para integrar la mensajería unificada local y Skype Empresarial
 
**Resumen:** Revise este tema mientras se planea integrar Skype para Business Server 2015 con Exchange de 2013.
  
 Si desea integrar Exchange Unified Messaging (UM) con Skype para Business Server 2015, debe realizar las tareas descritas en este tema. Asegúrese también de que revise las prácticas recomendadas de diseño e implementación descritas en el [Plan para la integración de mensajería unificada de Exchange en Skype para el negocio](unified-messaging.md). Este tema se supone que ha implementado Skype para el año 2015 de servidor empresariales con un servidor de mediación colocadas y que ha habilitado a los usuarios de Skype para Business Server 2015, pero quizás no haya realizado todos los pasos de implementación y configuración para habilitar la empresa Voz, como se describe en [Implementar Telefonía IP empresarial en Skype para el año 2015 de Business Server](../../deploy/deploy-enterprise-voice/deploy-enterprise-voice.md) en la documentación de implementación.
