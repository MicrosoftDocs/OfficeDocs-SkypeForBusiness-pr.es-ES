# <a name="deployment-process-overview-for-integrating-on-premises-unified-messaging-and-skype-for-business"></a>Información general sobre el proceso de implementación para integrar la mensajería unificada local y Skype Empresarial
 
**Resumen:** Revise este tema cuando planee integrar Skype para Business Server con Exchange 2013 o 2016.
  
 Si desea integrar la mensajería unificada de Exchange (UM) con Skype para Business Server, debe realizar las tareas que se describen en este tema. También debe asegurarse de que revise las prácticas recomendadas de planeación e implementación descritas en el [Plan para la integración de mensajería unificada de Exchange en Skype para la empresa](unified-messaging.md). En este tema se da por supuesto que ha implementado Skype para Business Server con un servidor de mediación combinado y que ha habilitado a los usuarios de Skype para Business Server, pero es posible que no se han realizado todos los pasos de implementación y configuración para habilitar Enterprise Voice, como se describe en [Implementar Enterprise Voice en Skype para Business Server](../../deploy/deploy-enterprise-voice/deploy-enterprise-voice.md) en la documentación de implementación.
 
> [!NOTE]
> Mensajería unificada de Exchange como conocidos anteriormente ya no está disponible en Skype para Business Server 2019, que usa el sistema telefónico para registrar los mensajes de correo de voz y, a continuación, deje la grabación en el buzón de Exchange del usuario. Para obtener más información, vea [servicio de planeación de correo de voz en la nube](../../../SfBServer2019/hybrid/plan-cloud-voicemail.md) .
