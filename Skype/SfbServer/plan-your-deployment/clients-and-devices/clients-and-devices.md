# <a name="plan-for-clients-and-devices"></a>Planeación de clientes y dispositivos

**Resumen:** Revisión de los clientes y aplicaciones compatibles para Skype Empresarial.

La plantilla actual está en constante movimiento. Los empleados deben comunicarse y colaborar ya sea trabajando desde la oficina corporativa, en ubicaciones regionales, en oficinas en casa o en la carretera. Skype Empresarial Server estas necesidades a través de una colección de interfaces de cliente que puede implementar para los usuarios de la organización. Una planeación cuidadosa garantiza que los empleados obtengan lo que necesitan y que Skype Empresarial esté disponible para ellos donde sea que estén.

## <a name="available-clients"></a>Clientes disponibles

Skype Empresarial Server admite varios tipos de clientes, incluido el software de cliente instalado en el equipo, los clientes basados en web y los clientes para dispositivos móviles. Los clientes principales se presentan en esta sección, para obtener una descripción detallada de todos los clientes admitidos, vea Comparación de características de cliente de escritorio para [Skype Empresarial Server 2015](desktop-feature-comparison.md) o Comparación de características de cliente de [escritorio para Skype Empresarial Server 2019](../../../SfBServer2019/plan/feature-comparison.md). Si ha usado anteriormente una combinación de clientes de Lync, [](clients-and-devices.md#Legacy) tenga en cuenta que hay clientes heredados no compatibles que son incompatibles con Skype Empresarial Server 2019. Las actualizaciones se llevan a cabo con regularidad, por lo que consulte este tema periódicamente para obtener la información del cliente más reciente.

### <a name="skype-for-business-2019"></a>Skype Empresarial (2019)

Skype Empresarial (2019) es el cliente completo recomendado para Skype Empresarial Server 2015 y 2019. Consulte [Follow the latest updates in Skype Empresarial](https://support.office.com/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781) para obtener una descripción de las nuevas características. La compatibilidad con características de cliente se detalla en la comparación de características de cliente de escritorio para Skype Empresarial [y](desktop-feature-comparison.md)la documentación del usuario está en [Skype Empresarial ayuda](https://support.office.com/Skype-for-business). Este cliente se incluye cuando un usuario instala Microsoft 365 o Office 365.

También hay disponible un cliente básico gratuito que admite menos características. Ambas versiones están disponibles para su descarga en [Descargar Skype Empresarial todos los dispositivos](https://products.office.com/skype-for-business/download-app?tab=tabs-3). Las diferencias entre los clientes Completo y Básico se describen en la [sección Limitaciones básicas del](desktop-feature-comparison.md#Full-Basic) cliente.

Para obtener un resumen de los cambios, vea [Lync is now Skype Empresarial — see what's new](https://support.office.com/en-in/article/aba02d7e-c801-4a82-bccd-e7207240f612). ### <a name="skype-for-business-for-mobile-devices"></a>Skype Empresarial para dispositivos móviles

Los clientes están disponibles para Windows Phone, iPhone/iPad y Android. Los usuarios pueden obtenerlos [en Descargar Skype Empresarial todos los dispositivos](https://products.office.com/skype-for-business/download-app?tab=tabs-3). La compatibilidad con características para estos clientes se detalla en [La comparación de](mobile-feature-comparison.md)características de cliente móvil para Skype Empresarial .

> [!NOTE]
> La compatibilidad con MCX (Mobility Service) para clientes móviles heredados ya no está disponible en Skype Empresarial Server 2019. Todos los clientes Skype Empresarial móviles ya usan la API web de comunicaciones unificadas (UCWA) para admitir mensajería instantánea (MI), presencia y contactos. Los usuarios con clientes heredados que usen MCX tendrán que actualizar a un cliente actual.

### <a name="online-meeting-add-in-for-skype-for-business"></a>Complemento de reunión en línea para Skype Empresarial

El complemento de reunión en línea para Skype Empresarial la administración de reuniones desde el cliente de colaboración y mensajería de Microsoft Outlook en Windows. El complemento de reunión en línea para Skype Empresarial software se instala automáticamente con Skype Empresarial.

### <a name="skype-for-business-web-app-and-skype-meetings-app"></a>aplicación web de Skype Empresarial y Skype de reuniones

Si Skype Empresarial no está instalado en el equipo de un usuario y el usuario hace clic en un vínculo de reunión en una solicitud de reunión en un equipo de Windows, se instalará y se abrirá la aplicación de reuniones de Skype o aplicación web de Skype Empresarial.  Skype Meetings App es el cliente elegido para los participantes fuera de la organización. (Tenga en cuenta que en un Mac, Skype Empresarial en Mac se instalará y se abrirá). Consulta [Plan for Meetings clients (Web App and Meetings App)](meetings-clients.md) para conocer los requisitos para usar estos clientes.


### <a name="skype-for-business-web-scheduler"></a>Programador web de Skype Empresarial

[Programador web de Skype Empresarial](https://sched.lync.com) es una herramienta de administración y programación de reuniones basada en web para usuarios de Skype Empresarial Online que no tienen acceso a Microsoft Outlook o que están en un sistema operativo que no se basa en Windows. Con Programador web de Skype Empresarial, los usuarios pueden crear nuevas reuniones, modificar reuniones existentes y enviar invitaciones con su programa de correo electrónico preferido. Programador web de Skype Empresarial documentación [proporciona](https://support.office.com/article/Skype-for-Business-Web-Scheduler-3b24a211-6470-4a2d-81b7-22d9399d0fec?ui=en-US&amp;rs=en-US&amp;ad=US) más detalles.

### <a name="vdi-plugins"></a>Complementos VDI

Un Infraestructura de escritorio virtual (VDI) se usa en algunas organizaciones donde los problemas de seguridad y cumplimiento son especialmente confidenciales. El Skype Empresarial con audio y vídeo completos en una conexión como esa requiere una gran cantidad de procesamiento de audio y vídeo en el cliente que se encuentra en un escritorio virtual. Hay disponible un software de complemento VDI adicional que descarga ese procesamiento en la máquina local del usuario final y reduce la carga en el escritorio virtual. Consulta [Plan for Skype Empresarial en entornos VDI](vdi-environments.md) para obtener más información sobre el uso de estos complementos.

### <a name="microsoft-teams-rooms"></a>Salas de Microsoft Teams

Salas de Microsoft Teams es la solución de conferencia más reciente de Microsoft, que usa una interfaz familiar y se implementa y administra fácilmente, aprovechando los equipos existentes, como los paneles LCD, para facilitar la instalación. Salas de Microsoft Teams usa una aplicación para UWP integrada que se ejecuta en un Surface Pro 4 o Surface Pro en un modo de consola (una vez implementada la aplicación para UWP es la única aplicación que se ejecutará en el dispositivo) y requiere su propia cuenta de dispositivo en la implementación. El software se actualiza a través de Windows store y Windows Update. Consulta https://aka.ms/MTRDocs para obtener más información sobre el uso de estas consolas de sala en la implementación. 

### <a name="skype-for-business-on-surface-hub"></a>Skype Empresarial en Surface Hub

Microsoft Surface Hub es un dispositivo de productividad todo en uno destinado a la lluvia de ideas, la colaboración y las presentaciones. Tiene su propia iteración del cliente Skype Empresarial, documentado en la Microsoft Surface Hub [de administración.](/surface-hub/)

## <a name="choosing-your-organizations-preferred-client"></a>Elegir el cliente preferido de la organización
<a name="BK_client_choose"> </a>

Si su organización compró las licencias adecuadas, elija el cliente completo, de lo contrario, elija el cliente básico.

Los usuarios pueden instalar el cliente por sí mismos desde [Descargar Skype Empresarial en todos los dispositivos](https://products.office.com/skype-for-business/download-app?tab=tabs-3). El cliente también se instala cuando los usuarios instalan Microsoft 365 o Office 365 en Windows. Si algunos de los usuarios tienen Mac, estos usuarios tendrán un conjunto diferente de características como se describe en las secciones anteriores.

Algunas características disponibles con Skype Empresarial Server 2015 no están disponibles en Skype Empresarial Online o Skype Empresarial Server 2019, consulte [Online or Hybrid user account limitations for 2015](desktop-feature-comparison.md#Online-Hybrid) o Online or Hybrid user account [limitations for 2019](desktop-feature-comparison.md#Online-Hybrid) para obtener información específica. Skype Empresarial Es posible que los administradores en línea quieran consultar [Skype Empresarial descripción](/office365/servicedescriptions/skype-for-business-online-service-description/skype-for-business-online-service-description) del servicio en línea para obtener información sobre los diferentes planes disponibles para ellos.

 Antes de implementar o actualizar a Skype Empresarial, compruebe qué clientes ya están en uso en su organización. Use la [comparación de características de](desktop-feature-comparison.md) cliente de escritorio Skype Empresarial para comprender el impacto de la compatibilidad de características en esos clientes. Esto puede ayudarle a comunicar los cambios a los usuarios, dar un ritmo al proceso de implementación y comprender completamente las ventajas de actualizar al cliente más reciente.

## <a name="ways-to-deploy-the-client-to-your-users"></a>Formas de implementar el cliente para los usuarios
<a name="BK_User_Deploy"> </a>

Los instaladores de cliente están disponibles para instaladores de tipo MSI y hacer clic y ejecutar. La Skype Empresarial de mantenimiento del cliente puede afectar a sus opciones, por lo que debe comprender los siguientes puntos:

- En general Skype Empresarial no agrega nuevas características a clientes publicados anteriormente

- En general Skype Empresarial no planea el envío de nuevas características en el Skype Empresarial MSI después de su versión inicial. Las mejoras de MSI entre las versiones serán principalmente de calidad y seguridad en la naturaleza.

- La experiencia de cliente más Skype Empresarial y más reciente se encontrará en el instalador Skype Empresarial hacer clic y ejecutar de 2019.

Puede realizar una implementación personalizada del cliente como se describe en Personalizar Windows [instalación de cliente en Skype Empresarial Server](../../deploy/deploy-clients/customize-windows-client-installation.md). Los métodos de instalación se describen con más detalle en [Implementar clientes para Skype Empresarial Server](../../deploy/deploy-clients/deploy-clients.md)

### <a name="click-to-run"></a>Hacer clic y ejecutar

Hacer clic para ejecutar es una tecnología de streaming y virtualización de Microsoft que puede usar para instalar y actualizar Office productos, incluidos Skype Empresarial. Estas capacidades de transmisión por secuencias y virtualización se basan en tecnologías de Microsoft Application Virtualization (App-V). Hacer clic y ejecutar tiene las siguientes ventajas:

- Instalación por secuencias Office conjunto de aplicaciones que da como resultado un breve tiempo de instalación

- Actualizaciones y revisiones slipstreamed

- Ocupa menos espacio en disco

- Licencias de base de usuarios: 5 instalaciones por usuario

- Personalizable a través del Editor XML para la instalación de programas independientes

Es posible que desee usar la [herramienta Office implementación para](https://www.microsoft.com/download/details.aspx?id=49117) este tipo de instalador.

Tanto las versiones de cliente básicas como las completas (con opciones de versiones de 32 y 64 bits) están disponibles con un instalador de hacer clic y ejecutar, los usuarios pueden descargar en [Descargar Skype Empresarial](https://products.office.com/skype-for-business/download-app?tab=tabs-3)en todos los dispositivos.

### <a name="msi"></a>MSI

MSI es un método de instalación más tradicional, que se usa Skype Empresarial clientes de 2015 y 2016. Le permite instalar actualizaciones y revisiones manualmente, usar licencias por volumen y activación, y se puede personalizar a través de la herramienta de personalización [Office.](https://www.microsoft.com/download/details.aspx?id=49030) Puede distribuir clientes aplicando directivas de grupo, mediante Microsoft Endpoint Configuration Manager o mediante una herramienta de terceros.
