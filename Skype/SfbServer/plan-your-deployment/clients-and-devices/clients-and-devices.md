# <a name="plan-for-clients-and-devices"></a>Planeación de clientes y dispositivos

**Resumen:** Revisión de los clientes y las aplicaciones compatibles con Skype empresarial.

El personal de hoy en día se mueve constantemente. Los empleados necesitan comunicarse y colaborar si trabajan desde la oficina corporativa, en ubicaciones regionales, en oficinas hogareñas o en la calle. Skype empresarial Server admite estas necesidades a través de una colección de interfaces de cliente que se pueden implementar en los usuarios de la organización. La planeación meditada garantiza que los empleados obtengan lo que necesitan y que Skype empresarial está a su disposición en cualquier lugar.

## <a name="available-clients"></a>Clientes disponibles

Skype empresarial Server admite varios tipos de clientes, como el software de cliente instalado en el equipo, los clientes basados en Web y los clientes para dispositivos móviles. Los clientes principales se presentan en esta sección para obtener una lista detallada de todos los clientes admitidos consulte [comparación de características de cliente de escritorio para Skype empresarial server 2015](desktop-feature-comparison.md) o [comparación de características de cliente de escritorio para skype empresarial Server 2019](../../../SfBServer2019/plan/feature-comparison.md). Si ya ha usado una combinación de clientes de Lync, tenga en cuenta que hay [clientes heredados](clients-and-devices.md#Legacy) no compatibles que no son compatibles con Skype empresarial Server 2019. Las actualizaciones se realizan regularmente, por lo que consulte periódicamente este tema para obtener la información más reciente sobre el cliente.

### <a name="skype-for-business-2019"></a>Skype empresarial (2019)

Skype empresarial (2019) es el cliente completo recomendado para Skype empresarial Server 2015 y 2019. Consulte [seguir las actualizaciones más recientes en Skype empresarial](https://support.office.com/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781) para obtener una descripción de las características nuevas. La compatibilidad con características de cliente se detalla en la [comparación de características de cliente de escritorio para Skype empresarial](desktop-feature-comparison.md)y la documentación de usuario se encuentra en [ayuda de Skype empresarial](https://support.office.com/Skype-for-business). Este cliente se incluye cuando un usuario instala Microsoft 365 u Office 365.

También hay disponible un cliente básico gratuito que admite menos características. Ambas versiones están disponibles para su descarga en [descargar Skype empresarial en todos los dispositivos](https://products.office.com/skype-for-business/download-app?tab=tabs-3). Las diferencias entre los clientes completos y básicos se describen en la sección de [limitaciones básicas de clientes](desktop-feature-comparison.md#Full-Basic) .

### <a name="skype-for-business-2016"></a>Skype Empresarial 2016

Skype empresarial 2016 es un cliente con todas las características para Skype empresarial Server 2015 o 2019. Consulte [what's New in Skype for Business 2016](https://support.office.com/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781) para obtener una descripción de las características nuevas. La compatibilidad con características de cliente se detalla en la [comparación de características de cliente de escritorio para Skype empresarial](desktop-feature-comparison.md)y la documentación de usuario se encuentra en [ayuda de Skype empresarial](https://support.office.com/Skype-for-business). Este cliente se incluye cuando un usuario instala Office 365.

También hay disponible un cliente básico gratuito que admite menos características. Ambas versiones están disponibles para su descarga en [descargar Skype empresarial en todos los dispositivos](https://products.office.com/skype-for-business/download-app?tab=tabs-3). Las diferencias entre los clientes completos y básicos se describen en la sección de [limitaciones básicas de clientes](desktop-feature-comparison.md#Full-Basic) .

### <a name="skype-for-business-2015"></a>Skype Empresarial 2015

Skype empresarial 2015 es un cliente con todas las características para Skype empresarial Server 2015 o 2019. La interfaz de usuario de Skype empresarial ha sido completamente rediseñada e incluye características recién integradas, como el monitor de llamadas, la integración del directorio de Skype, iconos gestuales, etc. Para obtener un resumen de los cambios, vea [Lync ahora es Skype empresarial; consulte what's New](https://support.office.com/en-in/article/aba02d7e-c801-4a82-bccd-e7207240f612). La compatibilidad con características de cliente se detalla en la [comparación de características de cliente de escritorio para Skype empresarial](desktop-feature-comparison.md)y la documentación de usuario se encuentra en [ayuda de Skype empresarial](https://support.office.com/Skype-for-business). Este cliente se incluye cuando un usuario instala Office 365.

### <a name="skype-for-business-on-mac"></a>Skype Empresarial en Mac

El cliente [de Skype empresarial en Mac](https://www.microsoft.com/download/details.aspx?id=54108) está disponible para su descarga. Consulte [requisitos de Skype empresarial en Mac Client](mac-requirements.md) para revisar los requisitos previos.

### <a name="skype-for-business-for-mobile-devices"></a>Skype empresarial para dispositivos móviles

Los clientes están disponibles para Windows Phone, iPhone/iPad y Android. Los usuarios pueden obtener acceso a [descargar Skype empresarial en todos los dispositivos](https://products.office.com/skype-for-business/download-app?tab=tabs-3). La compatibilidad de características para estos clientes se detalla en la [comparación de características de cliente móvil para Skype empresarial](mobile-feature-comparison.md).

> [!NOTE]
> La compatibilidad con MCX (Mobility Service) para clientes móviles heredados ya no está disponible en Skype empresarial Server 2019. Todos los clientes móviles actuales de Skype empresarial ya usan la API Web de comunicaciones unificadas (UCWA) para admitir la mensajería instantánea (mi), la presencia y los contactos. Los usuarios con clientes heredados que usen MCX deberán actualizar a un cliente actual.

### <a name="online-meeting-add-in-for-skype-for-business"></a>Complemento para reunión en línea de Skype empresarial

El complemento para reunión en línea de Skype empresarial admite la administración de reuniones desde el cliente de mensajería y colaboración de Microsoft Outlook en Windows. El complemento de conferencia en línea para el software de Skype empresarial se instala automáticamente con Skype empresarial.

### <a name="skype-for-business-web-app-and-skype-meetings-app"></a>Aplicación Web de Skype empresarial y aplicación de reuniones de Skype

Si Skype empresarial no está instalado en el equipo de un usuario y el usuario hace clic en un vínculo de reunión en una convocatoria de reunión en un equipo con Windows, se instalará y abrirá la aplicación reuniones de Skype o la aplicación Web de Skype empresarial.  La aplicación reuniones de Skype es el cliente preferido para los participantes externos a la organización. (Tenga en cuenta que en un equipo Mac, se instalará y abrirá Skype empresarial en Mac). Consulte [planeación de clientes de reuniones (aplicación web y reunión)](meetings-clients.md) para conocer los requisitos para usar estos clientes.


### <a name="skype-for-business-web-scheduler"></a>Programador web de Skype empresarial

El [programador web de Skype empresarial](https://sched.lync.com) es una herramienta de programación y administración de reuniones basada en web para los usuarios de Skype empresarial online que no tienen acceso a Microsoft Outlook o que están en un sistema operativo que no está basado en Windows. Con el programador web de Skype empresarial, los usuarios pueden crear nuevas reuniones, modificar reuniones existentes y enviar invitaciones mediante su programa de correo electrónico preferido. La [documentación](https://support.office.com/article/Skype-for-Business-Web-Scheduler-3b24a211-6470-4a2d-81b7-22d9399d0fec?ui=en-US&amp;rs=en-US&amp;ad=US) del programador web de Skype empresarial proporciona más detalles.

### <a name="vdi-plugins"></a>Complementos de VDI

Un entorno de infraestructura de escritorio virtual (VDI) se usa en algunas organizaciones donde los problemas de seguridad y cumplimiento son especialmente confidenciales. Uso de Skype empresarial con audio y vídeo completos en una conexión como esa requiere una carga pesada de procesamiento de audio y vídeo en el cliente hospedado en un escritorio virtual. Hay disponible software de complemento de VDI adicional que descarga ese procesamiento en el equipo local del usuario final y reduce la carga en el escritorio virtual. Consulte [Plan for Skype for Business in VDI Environments](vdi-environments.md) para obtener detalles sobre cómo usar estos complementos.

### <a name="microsoft-teams-rooms"></a>Salas de Microsoft Teams

Salones de Microsoft Teams es la solución de conferencias más reciente de Microsoft que usa una interfaz familiar y que se implementa y administra fácilmente, aprovechando los equipos existentes, como los paneles LCD, para facilitar la instalación. Las salas de Microsoft Teams usan una aplicación para UWP especialmente diseñada que se ejecuta en una Surface Pro 4 o Surface Pro en un modo de consola (una vez implementada la aplicación para UWP es la única aplicación que se ejecutará en el dispositivo) y requiere su propia cuenta de dispositivo en la implementación. El software se actualiza a través de la tienda Windows y de Windows Update. Consulte https://aka.ms/MTRDocs para obtener información detallada sobre el uso de estas consolas de salas en su implementación de. 

### <a name="skype-for-business-on-surface-hub"></a>Skype empresarial en Surface Hub

Microsoft Surface Hub es un dispositivo de productividad todo en uno destinado a la lluvia de ideas, colaboración y presentaciones. Tiene su propia iteración del cliente de Skype empresarial, documentada en la [Guía de administración de Microsoft Surface Hub](https://docs.microsoft.com/surface-hub/).

## <a name="choosing-your-organizations-preferred-client"></a>Elección del cliente preferido de la organización
<a name="BK_client_choose"> </a>

Si su organización ha comprado las licencias adecuadas, seleccione el cliente completo; en caso contrario, elija el cliente básico.

Los usuarios pueden instalar el cliente para que no [descarguen Skype empresarial en todos los dispositivos](https://products.office.com/skype-for-business/download-app?tab=tabs-3). El cliente también se instala cuando los usuarios instalan Microsoft 365 o Office 365 en Windows. Si algunos de los usuarios tienen equipos Mac, dichos usuarios tendrán un conjunto de características diferente, como se describe en secciones anteriores.

Algunas de las características disponibles en Skype empresarial Server 2015 no están disponibles en Skype empresarial online ni en Skype empresarial Server 2019, vea [limitaciones en la cuenta de usuario híbrido o en línea para 2015](desktop-feature-comparison.md#Online-Hybrid) o [en línea de usuario híbrido para 2019](desktop-feature-comparison.md#Online-Hybrid) para especificaciones específicas. Es posible que los administradores de Skype empresarial online consulten la [Descripción del servicio Skype empresarial online](https://technet.microsoft.com/library/skype-for-business-online-service-description.aspx) para obtener información sobre los distintos planes disponibles para ellos.

 Antes de implementar o actualizar a Skype empresarial, compruebe qué clientes están ya en uso en la organización. Use la [comparación de características de cliente de escritorio de Skype empresarial](desktop-feature-comparison.md) para comprender el impacto de la compatibilidad de características en esos clientes. Esto puede ayudarle a comunicar los cambios a los usuarios, aumentar el ritmo del proceso de implementación y comprender completamente las ventajas de la actualización al cliente más reciente.

## <a name="ways-to-deploy-the-client-to-your-users"></a>Formas de implementar el cliente para los usuarios
<a name="BK_User_Deploy"> </a>

Los instaladores de cliente están disponibles tanto para los instaladores MSI como para los de tipo hacer clic y ejecutar. La estrategia de mantenimiento del cliente de Skype empresarial puede afectar a sus opciones, por lo que debe comprender los puntos siguientes:

- En general, Skype empresarial no agrega nuevas características a los clientes publicados anteriormente

- En general, Skype empresarial no tiene previsto enviar nuevas características en el MSI de Skype empresarial después de la versión inicial. Las mejoras de MSI entre versiones serán principalmente de calidad/seguridad.

- La última y mejor experiencia de cliente de Skype empresarial se encontrará en el instalador de hacer clic y ejecutar de Skype empresarial 2019.

Puede realizar una implementación personalizada del cliente, tal como se describe en [Customize Windows Client Installation in Skype for Business Server](../../deploy/deploy-clients/customize-windows-client-installation.md). Los métodos de instalación se describen con más detalle en [deploy clients for Skype for Business Server](../../deploy/deploy-clients/deploy-clients.md)

### <a name="click-to-run"></a>Hacer clic y ejecutar

Hacer clic y ejecutar es una tecnología de Microsoft streaming and Virtualization que puede usar para instalar y actualizar productos de Office, incluido Skype empresarial. Estas capacidades de transmisión por secuencias y virtualización se basan en tecnologías de Microsoft Application Virtualization (App-V). Hacer clic y ejecutar tiene las siguientes ventajas:

- Instalación por secuencias del conjunto de aplicaciones de Office que da como resultado un breve período de instalación

- Actualizaciones y revisiones incorporadas

- Ocupa menos espacio en disco

- Licencias básicas de usuario: 5 instalaciones por usuario

- Personalizable a través del editor XML para la instalación de programas independientes

Es posible que quiera usar la [herramienta de implementación de Office](https://www.microsoft.com/download/details.aspx?id=49117) para este tipo de instalador.

Tanto las versiones de cliente básicas como completas (con una selección de versiones de 32 y 64 bits) están disponibles mediante un instalador de hacer clic y ejecutar, donde los usuarios pueden descargar en [descargar Skype empresarial en todos los dispositivos](https://products.office.com/skype-for-business/download-app?tab=tabs-3).

### <a name="msi"></a>MS

MSI es un método de instalación más tradicional que se usa en los clientes de Skype empresarial 2015 y 2016. Permite instalar manualmente actualizaciones y revisiones, usar la activación y el otorgamiento de licencias por volumen, y se puede personalizar mediante la [herramienta de personalización de Office](https://www.microsoft.com/download/details.aspx?id=49030). Puede distribuir clientes mediante la aplicación de directivas de grupo, con el administrador de configuración de Microsoft Endpoint o con una herramienta de terceros.
