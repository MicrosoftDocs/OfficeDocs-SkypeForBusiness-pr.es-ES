# <a name="plan-for-clients-and-devices"></a>Planificar los clientes y los dispositivos

**Resumen:** revisión de los clientes y aplicaciones compatibles para Skype Empresarial.

Los recursos de la actualidad se encuentran en constante movimiento. Los empleados necesitan comunicarse y colaborar independientemente de si están trabajando en la oficina corporativa, las ubicaciones regionales, las oficinas en casa o en movimiento. Skype Empresarial Server admite estas necesidades por medio de una recopilación de interfaces de cliente que puede implementar para los usuarios de la organización. Una planeación detallada garantiza que los empleados obtengan lo que necesitan y que Skype Empresarial esté disponible dondequiera que estén.

## <a name="available-clients"></a>Clientes disponibles

Skype Empresarial Server admite varios tipos de clientes, como el software de cliente instalado en el equipo, los clientes basados en web y los clientes para dispositivos móviles. En esta sección se presentan los clientes principales, pero si desea ver una lista de todos los clientes compatibles, puede ver [Comparación de características de cliente de escritorio de Skype Empresarial Server 2015](desktop-feature-comparison.md) o [Comparación de características de cliente de escritorio de Skype Empresarial Server 2019](../../../SfBServer2019/plan/feature-comparison.md). Si ha utilizado previamente una combinación de clientes de Lync, tenga en cuenta que hay [clientes heredados](clients-and-devices.md#Legacy) que no son compatibles con Skype Empresarial Server 2019. Para conocer la información del cliente más reciente, compruebe este tema con frecuencia, ya que se realizan actualizaciones a menudo.

Skype para la empresa (2019) es el cliente recomendado completas de Skype para Business Server 2015 y 2019. Para obtener una descripción de las nuevas características, vea [siga las actualizaciones más recientes de Skype para la empresa](https://support.office.com/en-us/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781) . Todos los detalles sobre la compatibilidad de las características de los clientes se encuentran en [Comparación de características de cliente de escritorio de Skype Empresarial](desktop-feature-comparison.md) y la documentación de usuario en la [Ayuda de Skype Empresarial](https://support.office.com/en-US/Skype-for-business). Este cliente se incluye cuando un usuario instala Office 365.

También hay disponible un cliente básico gratuito con menos características. Ambas versiones se pueden descargar en [Descargar Skype Empresarial en todos los dispositivos](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3). En la sección de [limitaciones del cliente básico](desktop-feature-comparison.md#Full-Basic) se describen las diferencias que existen entre los clientes básicos y los completos.

Software adicional de complemento de VDI está disponible que descarga el que el procesamiento en el equipo local del usuario final y reduce la carga en el escritorio virtual. ## <a name="choosing-your-organizations-preferred-client"></a>Selección del cliente preferido de su organización
<a name="BK_client_choose"> </a>

Si su organización ha comprado las licencias adecuadas, seleccione el cliente completo; si no, elija el cliente básico. 

Los usuarios pueden instalar al cliente para sí mismos desde [Descargar Skype para la empresa a través de todos los dispositivos](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3). También se instala el cliente cuando los usuarios instalan Office 365 en Windows. Si algunos de los usuarios dispongan de equipos Mac, los usuarios tendrán un conjunto diferente de características tal como se describe en las secciones anteriores.

Algunas características disponibles con Skype para Business Server 2015 no están disponibles en Skype para profesionales en línea o Skype para Business Server 2019, vea [cuentas limitaciones para 2015 de usuarios en línea o híbrida](desktop-feature-comparison.md#Online-Hybrid) o [en línea o limitaciones de cuenta de usuario de híbrido para 2019 ](desktop-feature-comparison.md#Online-Hybrid)para obtener información específica. Skype para los administradores en línea de negocio que desee hacer referencia a [Skype para la descripción de servicio en línea de negocio](https://technet.microsoft.com/library/skype-for-business-online-service-description.aspx) para obtener información sobre los diferentes planes disponibles para ellos.

 Antes de implementar o actualizar a Skype para la empresa, compruebe que los clientes que ya están en uso en la organización. Use la [comparación de características de cliente de escritorio de Skype para la empresa](desktop-feature-comparison.md) para comprender el impacto de compatibilidad con la característica en esos clientes. Esto puede ayudarle a comunicar los cambios a los usuarios, decidir el ritmo del proceso de lanzamiento y conocer en profundidad las ventajas que tiene actualizar al cliente más reciente.

## <a name="ways-to-deploy-the-client-to-your-users"></a>Métodos para implementar el cliente en sus usuarios
<a name="BK_User_Deploy"> </a>

Los instaladores de cliente están disponibles como instaladores del tipo MSI o hacer clic y ejecutar. El Skype para la estrategia de negocios de cliente sustainment puede afectar a las opciones disponibles, por lo que debe comprender los siguientes puntos:

- En general Skype para la empresa no agregar nuevas características a los clientes publicados anteriormente

- En general no planeación de Skype para la empresa en las nuevas características de envío en el Skype para profesionales MSI tras su publicación inicial. Las mejoras en el MSI entre versiones se centrarán principalmente en la calidad y la seguridad.

- El Skype mejor y más reciente de la experiencia del cliente empresarial se encuentran en el Skype para profesionales 2019 click-to-run installer.

Puede hacer una implementación personalizada del cliente como se describe en la [instalación del cliente Windows personalizar en Skype para Business Server](../../deploy/deploy-clients/customize-windows-client-installation.md). Métodos de instalación se describen más detalladamente en [implementar clientes de Skype para Business Server](../../deploy/deploy-clients/deploy-clients.md)

### <a name="click-to-run"></a>Hacer clic y ejecutar

Click-to-run es una transmisión por secuencias de Microsoft y la tecnología de virtualización que puede usar para instalar y actualizar los productos de Office incluidos Skype para la empresa. Estas capacidades de virtualización y transmisión por secuencias se basan en tecnologías de Microsoft Application Virtualization (App-V). Click-to-run tiene las siguientes ventajas:

- Instalación por streaming del conjunto de aplicaciones de Office en un breve espacio de tiempo

- Actualizaciones y revisiones de instalación integrada

- Ocupa menos espacio en disco

- Licencias basadas en usuarios: 5 instalaciones por usuario

- Se puede personalizar a través del Editor XML para la instalación de programas independientes

Es posible que desee usar la [Herramienta de implementación de Office](https://www.microsoft.com/en-us/download/details.aspx?id=49117) para este tipo de instalador.

Tanto las versiones de cliente básica y completa (con la opción de versiones de 32 y 64 bits) están disponibles mediante un instalador de click-to-run, los usuarios pueden descargar en [Descargar Skype para la empresa a través de todos los dispositivos](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3).

### <a name="msi"></a>MSI

MSI es un método de instalación más tradicional, utilizado en la Skype para 2015 empresarial y los clientes de 2016. Le permite instalar manualmente las actualizaciones y revisiones, las licencias por volumen de uso y activación y es personalizable a través de la [Herramienta de personalización de Office](https://www.microsoft.com/en-us/download/details.aspx?id=49030). Puede distribuir clientes mediante el System Center Configuration Manager o una herramienta de terceros.
