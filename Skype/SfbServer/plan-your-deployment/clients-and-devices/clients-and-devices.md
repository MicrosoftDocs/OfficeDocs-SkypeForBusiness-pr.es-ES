# <a name="plan-for-clients-and-devices"></a>Planeación de clientes y dispositivos

**Resumen:** Revisión de los clientes y aplicaciones compatibles para Skype Empresarial.

Los empleados actuales están constantemente en movimiento. Los empleados necesitan comunicarse y colaborar tanto si trabajan desde la oficina corporativa, en ubicaciones regionales, en oficinas en casa o de viaje. Skype Empresarial Server admite estas necesidades a través de una colección de interfaces de cliente que puede implementar para los usuarios de su organización. Una planeación cuidadosa garantiza que los empleados obtengan lo que necesitan y que Skype Empresarial esté disponible dondequiera que estén.

## <a name="available-clients"></a>Clientes disponibles

Skype Empresarial Server admite varios tipos de clientes, incluido el software cliente instalado en el equipo, los clientes basados en web y los clientes para dispositivos móviles. Los clientes principales se presentan en esta sección, para obtener una lista detallada de todos los clientes compatibles, vea la comparación de características de cliente de escritorio para [Skype Empresarial Server 2015](desktop-feature-comparison.md) o la comparación de características de cliente de escritorio para [Skype Empresarial Server 2019.](../../../SfBServer2019/plan/feature-comparison.md) Si ha usado anteriormente una combinación de clientes de Lync, [](clients-and-devices.md#Legacy) tenga en cuenta que hay clientes heredados no compatibles que son incompatibles con Skype Empresarial Server 2019. Las actualizaciones se llevan a cabo periódicamente, por lo que consulte este tema periódicamente para obtener la información más reciente del cliente.

### <a name="skype-for-business-2019"></a>Skype Empresarial (2019)

Skype Empresarial (2019) es el cliente completo recomendado para Skype Empresarial Server 2015 y 2019. Consulte [las actualizaciones más recientes de Skype](https://support.office.com/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781) Empresarial para obtener una descripción de las nuevas características. La compatibilidad con características de cliente se detalla en la comparación de características de cliente de escritorio para [Skype](desktop-feature-comparison.md)Empresarial y la documentación del usuario se encuentra en [la ayuda de Skype Empresarial.](https://support.office.com/Skype-for-business) Este cliente se incluye cuando un usuario instala Microsoft 365 u Office 365.

También hay disponible un cliente básico gratuito que admite menos características. Ambas versiones están disponibles para su descarga [en Descargar Skype Empresarial en todos sus dispositivos.](https://products.office.com/skype-for-business/download-app?tab=tabs-3) Las diferencias entre los clientes Completo y Básico se describen en la [sección Limitaciones básicas del](desktop-feature-comparison.md#Full-Basic) cliente.

### <a name="skype-for-business-2016"></a>Skype Empresarial 2016

Skype Empresarial 2016 es un cliente completo para Skype Empresarial Server 2015 o 2019. Vea [las novedades de Skype Empresarial 2016](https://support.office.com/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781) para obtener una descripción de las nuevas características. La compatibilidad con características de cliente se detalla en la comparación de características de cliente de escritorio para [Skype](desktop-feature-comparison.md)Empresarial y la documentación del usuario se encuentra en [la ayuda de Skype Empresarial.](https://support.office.com/Skype-for-business) Este cliente se incluye cuando un usuario instala Office 365.

También hay disponible un cliente básico gratuito que admite menos características. Ambas versiones están disponibles para su descarga [en Descargar Skype Empresarial en todos sus dispositivos.](https://products.office.com/skype-for-business/download-app?tab=tabs-3) Las diferencias entre los clientes Completo y Básico se describen en la [sección Limitaciones básicas del](desktop-feature-comparison.md#Full-Basic) cliente.

### <a name="skype-for-business-2015"></a>Skype Empresarial 2015

Skype Empresarial 2015 es un cliente completo para Skype Empresarial Server 2015 o 2019. La interfaz de usuario de Skype Empresarial se ha rediseñado completamente e incluye características recién integradas, como el Monitor de llamadas, la integración de directorios de Skype, los emoticonos y mucho más. Para obtener un resumen de los cambios, consulte [Lync es ahora Skype Empresarial, vea las novedades.](https://support.office.com/en-in/article/aba02d7e-c801-4a82-bccd-e7207240f612) La compatibilidad con características de cliente se detalla en la comparación de características de cliente de escritorio para [Skype](desktop-feature-comparison.md)Empresarial y la documentación del usuario se encuentra en [la ayuda de Skype Empresarial.](https://support.office.com/Skype-for-business) Este cliente se incluye cuando un usuario instala Office 365.

### <a name="skype-for-business-on-mac"></a>Skype Empresarial en Mac

El [cliente de Skype Empresarial en Mac](https://www.microsoft.com/download/details.aspx?id=54108) está disponible para su descarga. Consulte [los requisitos del cliente](mac-requirements.md) de Skype Empresarial para Mac para revisar los requisitos previos.

### <a name="skype-for-business-for-mobile-devices"></a>Skype Empresarial para dispositivos móviles

Los clientes están disponibles para Windows Phone, iPhone,iPad y Android. Los usuarios pueden obtenerlos [en Descargar Skype Empresarial en todos sus dispositivos.](https://products.office.com/skype-for-business/download-app?tab=tabs-3) La compatibilidad con características para estos clientes se detalla en la comparación de características de cliente [móvil para Skype Empresarial.](mobile-feature-comparison.md)

> [!NOTE]
> La compatibilidad con MCX (Mobility Service) para clientes móviles heredados ya no está disponible en Skype Empresarial Server 2019. Todos los clientes móviles actuales de Skype Empresarial ya usan la API web de comunicaciones unificadas (UCWA) para admitir la mensajería instantánea (MI), la presencia y los contactos. Los usuarios con clientes heredados que usen MCX tendrán que actualizar a un cliente actual.

### <a name="online-meeting-add-in-for-skype-for-business"></a>Complemento de reunión en línea para Skype Empresarial

El complemento de reunión en línea para Skype Empresarial admite la administración de reuniones desde el cliente de mensajería y colaboración de Microsoft Outlook en Windows. El complemento de reunión en línea para el software de Skype Empresarial se instala automáticamente con Skype Empresarial.

### <a name="skype-for-business-web-app-and-skype-meetings-app"></a>Aplicación web de Skype Empresarial y aplicación reuniones de Skype

Si Skype Empresarial no está instalado en el equipo de un usuario y el usuario hace clic en un vínculo de reunión en una solicitud de reunión en un equipo Windows, se instalará y se abrirá la aplicación reuniones de Skype o la aplicación web de Skype Empresarial.  La aplicación Reuniones de Skype es el cliente elegido para los participantes fuera de su organización. (Tenga en cuenta que en un Equipo Mac, Skype Empresarial en Mac se instalará y abrirá). Consulte [Plan for Meetings clients (Web App and Meetings App)](meetings-clients.md) para conocer los requisitos para usar estos clientes.


### <a name="skype-for-business-web-scheduler"></a>Programador web de Skype Empresarial

El Programador web de [Skype](https://sched.lync.com) Empresarial es una herramienta de programación y administración de reuniones basada en web para usuarios de Skype Empresarial Online que no tienen acceso a Microsoft Outlook o que están en un sistema operativo no basado en Windows. Con el Programador web de Skype Empresarial, los usuarios pueden crear nuevas reuniones, modificar reuniones existentes y enviar invitaciones mediante su programa de correo electrónico preferido. La documentación del Programador web [de](https://support.office.com/article/Skype-for-Business-Web-Scheduler-3b24a211-6470-4a2d-81b7-22d9399d0fec?ui=en-US&amp;rs=en-US&amp;ad=US)  Skype Empresarial proporciona más detalles.

### <a name="vdi-plugins"></a>Complementos de VDI

Un entorno de infraestructura de escritorio virtual (VDI) se usa en algunas organizaciones en las que los problemas de seguridad y cumplimiento son especialmente confidenciales. El uso de Skype Empresarial con audio y vídeo completos en una conexión como esta requiere cargas intensas de procesamiento de audio y vídeo en el cliente que se encuentra en un escritorio virtual. Hay disponible un software de complemento VDI adicional que descarga ese procesamiento en la máquina local del usuario final y reduce la carga en el escritorio virtual. Consulte [Plan for Skype for Business in VDI environments](vdi-environments.md) for details on using these plugins.

### <a name="microsoft-teams-rooms"></a>Salas de Microsoft Teams

Salas de Microsoft Teams es la solución de conferencias más reciente de Microsoft que usa una interfaz familiar y se implementa y administra fácilmente, lo que aprovecha los equipos existentes, como los paneles USB, para facilitar la instalación. Salas de Microsoft Teams usa una aplicación para UWP diseñada para propósito que se ejecuta en Surface Pro 4 o Surface Pro en un modo de consola (una vez implementada la aplicación para UWP es la única aplicación que se ejecutará en el dispositivo) y requiere su propia cuenta de dispositivo en la implementación. El software se actualiza a través de la Tienda Windows y Windows Update. Consulta https://aka.ms/MTRDocs los detalles sobre cómo usar estas consolas de sala en la implementación. 

### <a name="skype-for-business-on-surface-hub"></a>Skype Empresarial en Surface Hub

Microsoft Surface Hub es un dispositivo de productividad todo en uno diseñado para la lluvia de ideas, la colaboración y las presentaciones. Tiene su propia iteración del cliente de Skype Empresarial, documentada en la guía de administración de [Microsoft Surface Hub.](https://docs.microsoft.com/surface-hub/)

## <a name="choosing-your-organizations-preferred-client"></a>Elegir el cliente preferido de la organización
<a name="BK_client_choose"> </a>

Si su organización compró las licencias adecuadas, elija el cliente completo; de lo contrario, elija el cliente básico.

Los usuarios pueden instalar el cliente por sí mismos [desde Descargar Skype Empresarial en todos los dispositivos.](https://products.office.com/skype-for-business/download-app?tab=tabs-3) El cliente también se instala cuando los usuarios instalan Microsoft 365 u Office 365 en Windows. Si algunos de los usuarios tienen Mac, estos usuarios tendrán un conjunto diferente de características, como se describe en las secciones anteriores.

Some features available with Skype for Business Server 2015 are not available in Skype for Business Online or Skype for Business Server 2019, see [Online or Hybrid user account limitations for 2015](desktop-feature-comparison.md#Online-Hybrid) or Online or Hybrid user account [limitations for 2019 for](desktop-feature-comparison.md#Online-Hybrid) specifics. Es posible que los administradores de Skype Empresarial Online quieran consultar la descripción del servicio Skype Empresarial [Online](https://technet.microsoft.com/library/skype-for-business-online-service-description.aspx) para obtener información sobre los diferentes planes disponibles para ellos.

 Antes de implementar o actualizar a Skype Empresarial, compruebe qué clientes ya están en uso en su organización. Use la [comparación de características de cliente de escritorio para Skype Empresarial](desktop-feature-comparison.md) para comprender el impacto de la compatibilidad de características en esos clientes. Esto puede ayudarle a comunicar los cambios a los usuarios, al ritmo del proceso de implementación y a comprender perfectamente las ventajas de actualizar al cliente más reciente.

## <a name="ways-to-deploy-the-client-to-your-users"></a>Formas de implementar el cliente para los usuarios
<a name="BK_User_Deploy"> </a>

Los instaladores de cliente están disponibles para los instaladores de tipo MSI y hacer clic y ejecutar. La estrategia de mantenimiento del cliente de Skype Empresarial puede afectar a sus opciones, por lo que debe comprender los siguientes puntos:

- En general, Skype Empresarial no agrega nuevas características a los clientes publicados anteriormente

- En general, Skype Empresarial no tiene previsto enviar nuevas características en el MSI de Skype Empresarial después de su versión inicial. Las mejoras de MSI entre las versiones serán principalmente de calidad y seguridad por naturaleza.

- La experiencia de cliente más reciente y la mejor de Skype Empresarial se encontrará en el instalador de hacer clic y ejecutar de Skype Empresarial 2019.

Puede realizar una implementación personalizada del cliente tal como se describe en Personalizar la instalación del cliente [de Windows en Skype Empresarial Server.](../../deploy/deploy-clients/customize-windows-client-installation.md) Los métodos de instalación se describen con más detalle en [Implementar clientes para Skype Empresarial Server](../../deploy/deploy-clients/deploy-clients.md)

### <a name="click-to-run"></a>Hacer clic y ejecutar

Hacer clic y ejecutar es una tecnología de transmisión por secuencias y virtualización de Microsoft que puede usar para instalar y actualizar productos de Office, incluido Skype Empresarial. Estas capacidades de transmisión por secuencias y virtualización se basan en tecnologías de Microsoft Application Virtualization (App-V). Hacer clic y ejecutar tiene las siguientes ventajas:

- Instalación en streaming del conjunto de aplicaciones de Office que da como resultado un breve tiempo de instalación

- Actualizaciones y revisiones slipstreamed

- Ocupa menos espacio en disco

- Licencias base de usuario: 5 instalaciones por usuario

- Personalizable a través del Editor XML para la instalación de programas independientes

Es posible que desee usar la Herramienta [de implementación de Office](https://www.microsoft.com/download/details.aspx?id=49117) para este tipo de instalador.

Las versiones de cliente básicas y completas (con las versiones de 32 y 64 bits) están disponibles con un instalador de hacer clic y ejecutar, los usuarios pueden descargar en Descargar [Skype](https://products.office.com/skype-for-business/download-app?tab=tabs-3)Empresarial en todos los dispositivos.

### <a name="msi"></a>MSI

MSI es un método de instalación más tradicional, usado en los clientes de Skype Empresarial 2015 y 2016. Le permite instalar actualizaciones y revisiones manualmente, usar licencias por volumen y activación, y se puede personalizar a través de la Herramienta [de personalización de Office.](https://www.microsoft.com/download/details.aspx?id=49030) Puedes distribuir clientes aplicando directivas de grupo, con Microsoft Endpoint Configuration Manager o con una herramienta de terceros.
