# <a name="plan-for-clients-and-devices"></a>Planificar los clientes y los dispositivos
 
**Resumen:** Revisión de las aplicaciones y clientes soportados de Skype para el negocio.
  
Los recursos de la actualidad se encuentran en constante movimiento. Los empleados necesitan comunicarse y colaborar independientemente de si están trabajando desde la oficina corporativa, en las ubicaciones regionales, en las oficinas en casa o en movimiento. Skype para Business Server 2015 admite estas necesidades a través de una colección de interfaces de cliente que se pueden implementar para los usuarios de su organización. Diseño exhaustivo asegura que los empleados obtienen lo que necesitan y que está a su disposición Skype para empresas dondequiera se encuentren en. 
  
## <a name="available-clients"></a>Clientes disponibles

Skype para el año 2015 de Business Server admite varios tipos de clientes, incluidos el software instalado por el equipo cliente, los clientes basados en web y clientes para dispositivos móviles. Presentamos a cada uno de ellos en esta sección. Si previamente ha utilizado una combinación de clientes de Lync, tenga en cuenta que hay no admite [clientes heredados](clients-and-devices.md#Legacy) que no son compatibles con Skype para el negocio. Para conocer la información del cliente más reciente, compruebe este tema con frecuencia, ya que se realizan actualizaciones a menudo.
  
### <a name="skype-for-business-2016"></a>Skype para negocios de 2016

Skype para negocios 2016 es el cliente completo recomendado para Skype para Business Server 2015. Para obtener una descripción de las nuevas características, consulte [Novedades de Skype para negocios de 2016](https://support.office.com/en-us/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781) . Compatibilidad con la función de cliente que se detalla en la [comparación de características de cliente de escritorio de Skype para empresas](desktop-feature-comparison.md)y documentación de usuario está en [Ayuda Skype para el negocio](https://support.office.com/en-US/Skype-for-business). Este cliente se incluye cuando un usuario instala Office 365.
  
También hay disponible un cliente básico gratuito con menos características. Ambas versiones están disponibles para su descarga en [Descargar Skype para el negocio a través de todos los dispositivos](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3). Las diferencias entre los clientes completo y Basic se describen en la sección [limitaciones de cliente básico](desktop-feature-comparison.md#Full-Basic) .
  
### <a name="skype-for-business-2015"></a>Skype Empresarial 2015

Skype para el año 2015 del negocio es el cliente recomendado de completa de Skype para Business Server 2015. El Skype para la interfaz de usuario de negocio se ha rediseñado completamente e incluye características recién integrados, como llamar al Monitor, integración de directorio Skype, iconos gestuales y mucho más. Para obtener un resumen de los cambios, consulte [Lync es ahora Skype para empresas: novedades](https://support.office.com/en-in/article/aba02d7e-c801-4a82-bccd-e7207240f612). Compatibilidad con la función de cliente que se detalla en la [comparación de características de cliente de escritorio de Skype para empresas](desktop-feature-comparison.md)y documentación de usuario está en [Ayuda Skype para el negocio](https://support.office.com/en-US/Skype-for-business). Este cliente se incluye cuando un usuario instala Office 365.
  
También hay disponible un cliente básico gratuito con menos características. Ambas versiones están disponibles para su descarga en [Descargar Skype para el negocio a través de todos los dispositivos](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3). Las diferencias entre los clientes completo y Basic se describen en la sección [limitaciones de cliente básico](desktop-feature-comparison.md#Full-Basic) .
  
### <a name="skype-for-business-on-mac"></a>Skype Empresarial en Mac

El cliente de [Skype para el negocio en Mac](https://www.microsoft.com/en-us/download/details.aspx?id=54108) está disponible para su descarga y se incluye con Office 365 para Mac. Consulte [Skype para empresas sobre los requisitos del cliente Mac](mac-requirements.md) para revisar los requisitos previos.
  
### <a name="skype-for-business-for-mobile-devices"></a>Skype para empresas para dispositivos móviles

Los clientes están disponibles para Windows Phone, iPhone/iPad y Android. Los usuarios pueden obtenerlos [Descargar Skype para el negocio a través de todos los dispositivos](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3). Compatibilidad de la característica de estos clientes se detalla en la [comparación de características de cliente móvil de Skype para el negocio](mobile-feature-comparison.md).
  
### <a name="online-meeting-add-in-for-skype-for-business"></a>Reunión en línea complemento de Skype para empresas

El complemento de reunión en línea para Skype para empresas es compatible con administración de reunión desde dentro el cliente de mensajería y colaboración Microsoft Outlook. El complemento de reunión en línea para Skype para software empresarial se instala automáticamente con Skype para el negocio.
  
### <a name="skype-for-business-web-app-and-skype-meetings-app"></a>Skype para Business Web App y App de reuniones de Skype

Skype para el negocio de la aplicación Web es el cliente de plugin de navegador de conferencias basadas en web para Skype para reuniones de negocios servidor 2015. Los participantes de las reuniones obtienen acceso a todas las características de uso compartido y colaboración, y a todos los controles de la reunión del moderador. Este cliente también permite a los invitados desde fuera de su organización que no tienen un cliente regular para asistir a las reuniones creadas por su Skype para usuarios de negocios. 
  
Si Skype para el negocio no está instalado en el equipo del usuario y el usuario hace clic en un vínculo de reunión en una convocatoria de reunión, Skype para el negocio de la aplicación Web se instalará y abrir. También puede configurar la página para unirse a la reunión para que permita a los usuarios unirse a reuniones con versiones anteriores de los clientes. Skype para el negocio de la aplicación Web es el cliente de elección para los participantes externos a la organización. Con Skype para el negocio de la aplicación Web, instalación de cliente local no se requiere, aunque las características de audio, vídeo y uso compartidas requieren la instalación de un complemento durante el primer uso. Consulte [Planear los clientes de las reuniones (Web App y reuniones App)](meetings-clients.md) para que los requisitos para utilizar a este cliente.
  
La aplicación de las reuniones de Skype es una aplicación de Windows 10 directa que sólo utiliza un explorador durante la instalación y se ejecuta como una aplicación nativa de Windows 10. Requiere configuración de servidor específico para habilitar la distribución desde el servidor Front-End. Al igual que Skype para el negocio de la aplicación Web, este cliente también permite invitados desde fuera de su organización que no tienen un cliente regular para asistir a las reuniones creadas por su Skype para usuarios de negocios. 
  
### <a name="skype-for-business-web-scheduler"></a>Skype para el programador Web de negocio

Skype para el programador Web de negocio está basado en web herramienta de planificación y gestión para los usuarios que no tienen acceso a Microsoft Outlook, o que estén en un sistema operativo no basado en Windows de la reunión. Con Skype para el programador Web de negocio, los usuarios pueden crear reuniones nuevas, modificar reuniones existentes y enviar invitaciones usando su programa de correo preferido.
  
Documentación de [Skype para el programador Web empresarial](https://support.office.com/en-us/article/Skype-for-Business-Web-Scheduler-3b24a211-6470-4a2d-81b7-22d9399d0fec?ui=en-US&amp;rs=en-US&amp;ad=US) ofrece más información.
  
### <a name="vdi-plugins"></a>Complementos VDI

El entorno de Infraestructura de escritorio virtual (VDI) se utiliza en algunas organizaciones donde los problemas de seguridad y cumplimiento son especialmente delicados. Mediante Skype para negocios con completa de audio y vídeo en una conexión que requieren cargas pesadas de audio y vídeo de procesamiento en el cliente que se encuentran en un escritorio virtual. Hay software de complemento adicional de VDI que descarga ese proceso en el equipo local del usuario final y reduce la carga en el escritorio virtual. Para obtener información detallada sobre el uso de estos plugins, consulte [Plan de Skype para el negocio en entornos de VDI](vdi-environments.md) .
  
### <a name="skype-room-systems-v2"></a>Sistemas de salas de Skype v2

Sistemas de salas de Skype v2 es la más reciente conferencia la solución de Microsoft que utiliza un Skype familiar para la interfaz de negocio y se implementa fácilmente y administrado, aprovecha los equipos existentes como paneles LCD de fácil instalación. Sistemas de salas de Skype v2 utiliza una aplicación UWP creado especialmente para ejecutar en un Surface Pro 4 o Surface Pro en un modo de consola (una vez implementada la aplicación UWP es la única aplicación que se ejecuta en el dispositivo) y requiere su propia cuenta de dispositivo en su Skype para la implementación del negocio. El software se actualiza a través de la Tienda Windows y de Windows Update. Para obtener información detallada sobre el uso de estas consolas de sala en la implementación, consulte [Plan para sistemas de salas de Skype v2](skype-room-systems-v2-0.md) .
  
### <a name="skype-for-business-on-surface-hub"></a>Skype para el negocio en concentrador de superficie

Microsoft Surface concentrador es un dispositivo de productividad all-in-one que está pensado para la lluvia de ideas, la colaboración y presentaciones. Tiene su propio iteración de la Skype para Business client, documentado en la [Guía del Administrador de Microsoft Surface concentrador](https://docs.microsoft.com/en-us/surface-hub/).
  
## <a name="choosing-your-organizations-preferred-client"></a>Selección del cliente preferido de su organización
<a name="BK_client_choose"> </a>

Si su organización ha comprado las licencias adecuadas, seleccione el cliente completo; si no, elija el cliente básico.  
  
Los usuarios pueden instalar al cliente por sí mismos desde [Descargar Skype para el negocio a través de todos los dispositivos](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3). Si algunos de los usuarios dispongan de equipos Mac, los usuarios tendrán un conjunto diferente de características tal como se describe en las secciones anteriores. También se instala el cliente cuando los usuarios instalan Office 365.
  
Algunas características disponibles con Skype para Business Server 2015 no están disponibles en Skype para los negocios en línea, vea [limitaciones de cuenta de usuario en línea o híbrido](desktop-feature-comparison.md#Online-Hybrid) para obtener detalles específicos. Skype para los administradores de negocios en línea que desee consultar a [Skype de descripción de servicios en línea de negocio](https://technet.microsoft.com/library/skype-for-business-online-service-description.aspx) para obtener información sobre los diferentes planes disponibles para ellos.
  
 Antes de implementar o actualizar a Skype para el negocio, compruebe que los clientes ya están en uso en su organización. Utilice la [comparación de características de cliente de escritorio de Skype para empresas](desktop-feature-comparison.md) para comprender el impacto de la compatibilidad de característica en dichos clientes. Esto puede ayudarle a comunicar los cambios a los usuarios, decidir el ritmo del proceso de lanzamiento y conocer en profundidad las ventajas que tiene actualizar al cliente más reciente.
  
## <a name="ways-to-deploy-the-client-to-your-users"></a>Métodos para implementar el cliente en sus usuarios
<a name="BK_User_Deploy"> </a>

Los instaladores de cliente están disponibles como instaladores del tipo MSI o hacer clic y ejecutar. El Skype para la estrategia de sostenimiento de cliente empresarial puede afectar a las opciones disponibles, por lo que debe comprender los siguientes puntos: 
  
- En general Skype para el negocio no agrega nuevas características a clientes publicados anteriormente
    
- En general no planeado envío nuevas características en el Skype para MSI del negocio después de su lanzamiento inicial Skype para el negocio. Las mejoras en el MSI entre versiones se centrarán principalmente en la calidad y la seguridad. 
    
- La última y mejor Skype para la experiencia del cliente de negocios se encontrará en el Skype para negocios 2016 hacer clic para ejecutar instalador. 
    
Puede realizar una implementación personalizada del cliente como se describe en [instalación de cliente de personalizar Windows en Skype para Business Server 2015](../../deploy/deploy-clients/customize-windows-client-installation.md). Métodos de instalación se describen con mayor detalle en [implementar clientes de Skype para Business Server 2015](../../deploy/deploy-clients/deploy-clients.md)
  
### <a name="click-to-run"></a>Hacer clic y ejecutar

Hacer clic para ejecutar es una transmisión por secuencias de Microsoft y la tecnología de virtualización que puede utilizar para instalar y actualizar los productos de Office incluidos Skype para el negocio. Estas capacidades de virtualización y transmisión por secuencias se basan en tecnologías de Microsoft Application Virtualization (App-V). Hacer clic para ejecutar tiene las siguientes ventajas: 
  
- Instalación por streaming del conjunto de aplicaciones de Office en un breve espacio de tiempo
    
- Actualizaciones y revisiones de instalación integrada
    
- Ocupa menos espacio en disco
    
- Licencias basadas en usuarios: 5 instalaciones por usuario
    
- Se puede personalizar a través del Editor XML para la instalación de programas independientes
    
Es aconsejable utilizar la [Herramienta de implementación de 2016 de Office](https://www.microsoft.com/en-us/download/details.aspx?id=49117) para este tipo de instalador.
  
Tanto las versiones básica y completa de cliente (con la elección de las versiones de 32 y 64 bits) están disponibles mediante un instalador de hacer clic en ejecutar, los usuarios pueden descargar en [Descargar Skype para el negocio a través de todos los dispositivos](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3). 
  
### <a name="msi"></a>MSI

MSI es un método de instalación más tradicional. Permite instalar manualmente las actualizaciones y revisiones, licencias por volumen de uso y activación y puede personalizarse mediante la [Herramienta de personalización de Office](http://www.microsoft.com/en-us/download/details.aspx?id=49030). Puede distribuir clientes mediante el System Center Configuration Manager o una herramienta de terceros.
