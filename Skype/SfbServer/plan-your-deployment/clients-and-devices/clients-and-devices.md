# <a name="plan-for-clients-and-devices"></a>Planificar los clientes y los dispositivos

**Resumen:** revisión de los clientes y aplicaciones compatibles para Skype Empresarial.

Los recursos de la actualidad se encuentran en constante movimiento. Los empleados necesitan comunicarse y colaborar independientemente de si están trabajando en la oficina corporativa, las ubicaciones regionales, las oficinas en casa o en movimiento. Skype Empresarial Server admite estas necesidades por medio de una recopilación de interfaces de cliente que puede implementar para los usuarios de la organización. Una planeación detallada garantiza que los empleados obtengan lo que necesitan y que Skype Empresarial esté disponible dondequiera que estén.

## <a name="available-clients"></a>Clientes disponibles

Skype Empresarial Server admite varios tipos de clientes, como el software de cliente instalado en el equipo, los clientes basados en web y los clientes para dispositivos móviles. En esta sección se presentan los clientes principales, pero si desea ver una lista de todos los clientes compatibles, puede ver [Comparación de características de cliente de escritorio de Skype Empresarial Server 2015](desktop-feature-comparison.md) o [Comparación de características de cliente de escritorio de Skype Empresarial Server 2019](../../../SfBServer2019/plan/feature-comparison.md). Si ha utilizado previamente una combinación de clientes de Lync, tenga en cuenta que hay [clientes heredados](clients-and-devices.md#Legacy) que no son compatibles con Skype Empresarial Server 2019. Para conocer la información del cliente más reciente, compruebe este tema con frecuencia, ya que se realizan actualizaciones a menudo.

### <a name="skype-for-business-2019"></a>Skype Empresarial (2019)

Skype Empresarial (2019) es el cliente de Skype Empresarial Server 2019 que incluye todas las características y se recomienda. Consulte [Novedades en Skype Empresarial 2016](https://support.office.com/en-us/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781) para ver una descripción de las características nuevas. Todos los detalles sobre la compatibilidad de las características de los clientes se encuentran en [Comparación de características de cliente de escritorio de Skype Empresarial](desktop-feature-comparison.md) y la documentación de usuario en la [Ayuda de Skype Empresarial](https://support.office.com/en-US/Skype-for-business). Este cliente se incluye cuando un usuario instala Office 365.

También hay disponible un cliente básico gratuito con menos características. Ambas versiones se pueden descargar en [Descargar Skype Empresarial en todos los dispositivos](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3). En la sección de [limitaciones del cliente básico](desktop-feature-comparison.md#Full-Basic) se describen las diferencias que existen entre los clientes básicos y los completos.

### <a name="skype-for-business-2015"></a>Skype Empresarial 2015

En la sección de [limitaciones del cliente básico](desktop-feature-comparison.md#Full-Basic) se describen las diferencias que existen entre los clientes básicos y los completos.

### <a name="skype-for-business-on-mac"></a>Skype Empresarial en Mac

El cliente de [Skype Empresarial en Mac](https://www.microsoft.com/en-us/download/details.aspx?id=54108) se puede descargar. Consulte [Requisitos de cliente de Skype Empresarial en Mac](mac-requirements.md) para revisar los requisitos previos.

### <a name="skype-for-business-for-mobile-devices"></a>Skype Empresarial para dispositivos móviles

Los clientes están disponibles para Windows Phone, iPhone/iPad y Android. Los usuarios los pueden conseguir en [Descargar Skype Empresarial en todos los dispositivos](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3). La compatibilidad de características para estos clientes se describe en [Comparación de características de cliente móvil de Skype Empresarial](mobile-feature-comparison.md).

> [!NOTE]
> La compatibilidad de MCX (Mobility Service) con clientes móviles heredados ya no está disponible en Skype Empresarial Server 2019. Todos los clientes móviles actuales de Skype Empresarial utilizan la API web de comunicaciones unificadas (UCWA) para admitir la mensajería instantánea (IM), la presencia y los contactos. Los usuarios que tengan clientes heredados con MCX tendrán que actualizar a un cliente actual.

### <a name="online-meeting-add-in-for-skype-for-business"></a>Complemento para conferencia en línea para Skype Empresarial

El complemento para conferencias en línea de Skype Empresarial admite la administración de reuniones desde el cliente de colaboración y mensajería de Microsoft Outlook en Windows. El software de este complemento para Skype Empresarial se instala automáticamente con Skype Empresarial.

### <a name="skype-for-business-web-app-and-skype-meetings-app"></a>La aplicación web de Skype Empresarial y la aplicación Reuniones de Skype

Si Skype Empresarial no está instalado en el equipo de un usuario y este hace clic en el vínculo de una reunión de una solicitud de reunión, se instalará la aplicación Reuniones de Skype o la aplicación web de Skype Empresarial y se abrirá.  La aplicación Reuniones de Skype es el cliente que suelen preferir los participantes que no pertenecen a la organización. Consulte [Plan para los clientes de reuniones (aplicación web y aplicación Reuniones)](meetings-clients.md) para ver los requisitos que se necesitan para usar estos clientes.


### <a name="skype-for-business-web-scheduler"></a>Programador web de Skype Empresarial

[Programador web de Skype Empresarial](https://sched.lync.com) es una herramienta basada en web de programación y administración de reuniones para los usuarios de Skype Empresarial Online que no tienen acceso a Microsoft Outlook, o que utilizan un sistema operativo no basado en Windows. Con Programador web de Skype Empresarial, los usuarios pueden crear reuniones nuevas, modificar reuniones existentes y enviar invitaciones mediante su programa de correo electrónico preferido. En la documentación del [Programador web de Skype Empresarial](https://support.office.com/en-us/article/Skype-for-Business-Web-Scheduler-3b24a211-6470-4a2d-81b7-22d9399d0fec?ui=en-US&amp;rs=en-US&amp;ad=US) podrá ver más detalles.

### <a name="vdi-plugins"></a>Complementos VDI

El entorno de Infraestructura de escritorio virtual (VDI) se utiliza en algunas organizaciones donde los problemas de seguridad y cumplimiento son especialmente delicados. El uso de Skype Empresarial con vídeo y audio completo en una conexión como esa requiere altas cargas de procesamiento de audio y vídeo en el cliente alojado en un escritorio virtual. Hay disponible software adicional del complemento de VDI, que aligera ese procesamiento en el equipo local del usuario final y reduce la carga en el equipo virtual. Consulte [Planificar Skype Empresarial en entornos de VDI](vdi-environments.md) para obtener detalles sobre cómo usar estos complementos.

### <a name="skype-room-systems-v2"></a>Sistemas de salas de Skype v2

Sistemas de salas de Skype v2 es la solución más moderna de conferencias de Microsoft, donde se utiliza la ya conocida interfaz de Skype Empresarial, y se administra e implementa fácilmente para sacar partido del equipo con el que ya cuenta, como los paneles LCD, y así facilitar la instalación. Sistemas de salas de Skype v2 utiliza una aplicación para UWP diseñada específicamente para este fin, que se ejecuta en una Surface Pro 4 o una Surface Pro en modo de consola (una vez que se ha implementada, la aplicación para UWP es la única aplicación que se ejecutará en el dispositivo) y requiere su propia cuenta de dispositivo en su implementación de Skype Empresarial. El software se actualiza a través de la Tienda Windows y de Windows Update. Consulte https://aka.ms/SRSDocs para ver más detalles sobre cómo se usan estas consolas de salas en su implementación.

### <a name="skype-for-business-on-surface-hub"></a>Skype Empresarial en Surface Hub

Microsoft Surface Hub es un dispositivo de productividad todo en uno que está concebido para las lluvias de ideas, la colaboración y las presentaciones. Tiene su propia iteración del cliente de Skype Empresarial, como se explica en el [Manual del administrador de Microsoft Surface Hub](https://docs.microsoft.com/surface-hub/).

## <a name="choosing-your-organizations-preferred-client"></a>Selección del cliente preferido de su organización
<a name="BK_client_choose"> </a>

Si su organización ha comprado las licencias adecuadas, seleccione el cliente completo; si no, elija el cliente básico.

Los usuarios pueden instalar el cliente desde [Descargar Skype Empresarial en todos los dispositivos](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3). Este también se instala cuando los usuarios instalan Office 365 en Windows. Si algún usuario tiene Mac, tendrá un conjunto distinto de características, como se describe en las tablas siguientes.

Algunas características que están disponibles con Skype Empresarial Server 2015 no lo están en Skype Empresarial Online o Skype Empresarial Server 2019. Consulte [Limitaciones de las cuentas de usuario en línea o híbridas para 2015](desktop-feature-comparison.md#Online-Hybrid) o [Limitaciones de las cuentas de usuario en línea o híbridas para 2019](desktop-feature-comparison.md#Online-Hybrid) para ver más detalles. Es posible que los administradores de Skype Empresarial Online quieran consultar [Descripción del servicio Skype Empresarial Online](https://technet.microsoft.com/library/skype-for-business-online-service-description.aspx) para obtener información sobre los distintos planes que tienen a su disposición.

 Antes de implementar o actualizar a Skype Empresarial, compruebe qué clientes se están ya usando en su organización. Use la [Comparación de características de cliente de escritorio de Skype Empresarial](desktop-feature-comparison.md) para saber cómo puede influir la compatibilidad de funciones en esos clientes. Esto puede ayudarle a comunicar los cambios a los usuarios, decidir el ritmo del proceso de lanzamiento y conocer en profundidad las ventajas que tiene actualizar al cliente más reciente.

## <a name="ways-to-deploy-the-client-to-your-users"></a>Métodos para implementar el cliente en sus usuarios
<a name="BK_User_Deploy"> </a>

Los instaladores de cliente están disponibles como instaladores del tipo MSI o hacer clic y ejecutar. La estrategia de sostenimiento de clientes de Skype Empresarial puede repercutir en su elección, por lo que debe comprender los siguientes puntos:

- En general, Skype Empresarial NO agrega nuevas características a los clientes publicados anteriormente.

- En general, Skype Empresarial no incluirá nuevas características en el MSI de Skype Empresarial tras su publicación inicial. Las mejoras en el MSI entre versiones se centrarán principalmente en la calidad y la seguridad.

- La experiencia de cliente de Skype Empresarial más reciente y satisfactoria se encontrará en el instalador del tipo hacer clic y ejecutar de Skype Empresarial 2019.

La implementación del cliente se puede personalizar, como se describe en [Personalizar la instalación de cliente de Windows en Skype Empresarial Server](../../deploy/deploy-clients/customize-windows-client-installation.md). Todos los detalles de los métodos de instalación se describen en [Implementar clientes para Skype Empresarial Server](../../deploy/deploy-clients/deploy-clients.md).

### <a name="click-to-run"></a>Hacer clic y ejecutar

Hacer clic y ejecutar es una tecnología de transmisión y virtualización de Microsoft que se puede usar para instalar y actualizar productos de Office; entre ellos, Skype Empresarial. Estas funcionalidades de transmisión y virtualización se basan en las tecnologías de Microsoft Application Virtualization (App-V). Hacer clic y ejecutar tiene las siguientes ventajas:

- Instalación por streaming del conjunto de aplicaciones de Office en un breve espacio de tiempo

- Actualizaciones y revisiones de instalación integrada

- Ocupa menos espacio en disco

- Licencias basadas en usuarios: 5 instalaciones por usuario

- Se puede personalizar a través del Editor XML para la instalación de programas independientes

Es posible que quiera usar la [Herramienta de implementación de Office](https://www.microsoft.com/en-us/download/details.aspx?id=49117) para este tipo de instalador.

Las versiones de cliente básico y completo (con opciones de versiones de 32 y 64 bits) están disponibles mediante un instalador del tipo hacer clic y ejecutar. Los usuarios pueden descargarlo en [Descargar Skype Empresarial en todos los dispositivos](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3).

### <a name="msi"></a>MSI

MSI es un método de instalación más tradicional, que se utiliza en los clientes de Skype Empresarial 2015 y 2016. Con él puede instalar las actualizaciones y las revisiones manualmente, usar la activación y el programa de licencias por volumen, y personalizarlo a través de la [Herramienta de personalización de Office](https://www.microsoft.com/en-us/download/details.aspx?id=49030). Los clientes se pueden distribuir aplicando directivas de grupo mediante el System Center Configuration Manager o una herramienta de terceros.



## <a name="legacy-clients"></a>Clientes heredados
<a name="Legacy"> </a>

Skype Empresarial Server 2019 y Skype Empresarial Online son compatibles con las siguientes versiones de clientes anteriores: Skype Empresarial 2016, Skype Empresarial 2015, Lync 2013.

Skype Empresarial Server 2015 es compatible con las versiones de clientes anteriores: Lync 2013, Lync 2010, Lync 2010 Mobile, Lync Phone Edition y Lync 2010 Attendant. Para obtener información sobre estos clientes cuando se utilizan con otros servidores, consulte [Tablas de comparación de clientes para Lync Server 2013](https://technet.microsoft.com/en-us/library/gg425836%28v=ocs.15%29.aspx) y [Tablas de comparación de clientes para Lync Server 2010](https://technet.microsoft.com/en-us/library/gg425836%28v=ocs.14%29.aspx).
