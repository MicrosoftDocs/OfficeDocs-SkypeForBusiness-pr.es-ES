# <a name="plan-for-clients-and-devices"></a>Planificar los clientes y los dispositivos

**Resumen:** revisión de los clientes y aplicaciones compatibles para Skype Empresarial.

Los recursos de la actualidad se encuentran en constante movimiento. Los empleados necesitan comunicarse y colaborar independientemente de si están trabajando en la oficina corporativa, las ubicaciones regionales, las oficinas en casa o en movimiento. Skype Empresarial Server admite estas necesidades por medio de una recopilación de interfaces de cliente que puede implementar para los usuarios de la organización. Una planeación detallada garantiza que los empleados obtengan lo que necesitan y que Skype Empresarial esté disponible dondequiera que estén.

## <a name="available-clients"></a>Clientes disponibles

Skype Empresarial Server admite varios tipos de clientes, como el software de cliente instalado en el equipo, los clientes basados en web y los clientes para dispositivos móviles. En esta sección se presentan los clientes principales, pero si desea ver una lista de todos los clientes compatibles, puede ver [Comparación de características de cliente de escritorio de Skype Empresarial Server 2015](desktop-feature-comparison.md) o [Comparación de características de cliente de escritorio de Skype Empresarial Server 2019](../../../SfBServer2019/plan/feature-comparison.md). Si ha utilizado previamente una combinación de clientes de Lync, tenga en cuenta que hay [clientes heredados](clients-and-devices.md#Legacy) que no son compatibles con Skype Empresarial Server 2019. Para conocer la información del cliente más reciente, compruebe este tema con frecuencia, ya que se realizan actualizaciones a menudo.

### <a name="skype-for-business-2019"></a>Skype Empresarial (2019)

Skype empresarial (2019) es el cliente completo recomendado para Skype empresarial Server 2015 y 2019. Para obtener una descripción de las nuevas características, consulte [seguir las actualizaciones más recientes de Skype empresarial](https://support.office.com/en-us/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781) . Todos los detalles sobre la compatibilidad de las características de los clientes se encuentran en [Comparación de características de cliente de escritorio de Skype Empresarial](desktop-feature-comparison.md) y la documentación de usuario en la [Ayuda de Skype Empresarial](https://support.office.com/en-US/Skype-for-business). Este cliente se incluye cuando un usuario instala Office 365.

También hay disponible un cliente básico gratuito con menos características. Ambas versiones se pueden descargar en [Descargar Skype Empresarial en todos los dispositivos](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3). En la sección de [limitaciones del cliente básico](desktop-feature-comparison.md#Full-Basic) se describen las diferencias que existen entre los clientes básicos y los completos.

### <a name="skype-for-business-2016"></a>Skype Empresarial 2016

Skype empresarial 2016 es un cliente completo para Skype empresarial Server 2015 o 2019. Consulte [Novedades en Skype Empresarial 2016](https://support.office.com/en-us/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781) para ver una descripción de las características nuevas. Todos los detalles sobre la compatibilidad de las características de los clientes se encuentran en [Comparación de características de cliente de escritorio de Skype Empresarial](desktop-feature-comparison.md) y la documentación de usuario en la [Ayuda de Skype Empresarial](https://support.office.com/en-US/Skype-for-business). Este cliente se incluye cuando un usuario instala Office 365.

También hay disponible un cliente básico gratuito con menos características. Ambas versiones se pueden descargar en [Descargar Skype Empresarial en todos los dispositivos](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3). En la sección de [limitaciones del cliente básico](desktop-feature-comparison.md#Full-Basic) se describen las diferencias que existen entre los clientes básicos y los completos.

### <a name="skype-for-business-2015"></a>Skype Empresarial 2015

Skype Empresarial 2015 es el cliente de Skype Empresarial Server 2015 o 2019 que incluye todas las características. La interfaz de usuario de Skype Empresarial se ha rediseñado por completo e incluye características que se acaban de integrar, como el control de llamadas, la integración del directorio de Skype, emoticonos y muchas otras. Si quiere ver un resumen de todos los cambios, vea [Lync es ahora Skype Empresarial: consulte las novedades](https://support.office.com/en-in/article/aba02d7e-c801-4a82-bccd-e7207240f612). Todos los detalles sobre la compatibilidad de las características de los clientes se encuentran en [Comparación de características de cliente de escritorio de Skype Empresarial](desktop-feature-comparison.md) y la documentación de usuario en la [Ayuda de Skype Empresarial](https://support.office.com/en-US/Skype-for-business). Este cliente se incluye cuando un usuario instala Office 365.

### <a name="skype-for-business-on-mac"></a>Skype Empresarial en Mac

El cliente [de Skype empresarial para Mac](https://www.microsoft.com/en-us/download/details.aspx?id=54108) está disponible para su descarga. Para revisar los requisitos previos, consulte [requisitos del cliente de Skype empresarial en Mac](mac-requirements.md) .

### <a name="skype-for-business-for-mobile-devices"></a>Skype empresarial para dispositivos móviles

Hay clientes disponibles para Windows Phone, iPhone/iPad y Android. Los usuarios pueden [descargarlos en todos tus dispositivos](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3). La compatibilidad con las características de estos clientes se detalla en la [comparación de características de cliente móvil para Skype empresarial](mobile-feature-comparison.md).

> [!NOTE]
> La compatibilidad con MCX (Mobility Service) para clientes móviles heredados ya no está disponible en Skype empresarial Server 2019. Todos los clientes móviles actuales de Skype empresarial ya usan la API Web de comunicaciones unificadas (UCWA) para admitir la mensajería instantánea, la presencia y los contactos. Los usuarios con clientes heredados que usen MCX deberán actualizar a un cliente actual.

### <a name="online-meeting-add-in-for-skype-for-business"></a>Complemento de reuniones en línea para Skype empresarial

El complemento de reuniones en línea para Skype empresarial admite la administración de reuniones desde el cliente de mensajería y colaboración de Microsoft Outlook en Windows. El complemento de reunión en línea para el software de Skype empresarial se instala automáticamente con Skype empresarial.

### <a name="skype-for-business-web-app-and-skype-meetings-app"></a>Aplicación Web de Skype empresarial y aplicación reuniones de Skype

Si Skype empresarial no está instalado en el equipo de un usuario y el usuario hace clic en un vínculo de reunión en una convocatoria de reunión en un equipo con Windows, la aplicación reuniones de Skype o la aplicación Web de Skype empresarial se instalarán y se abrirán.  La aplicación reuniones de Skype es el cliente que elige para los participantes ajenos a la organización. (Tenga en cuenta que en un equipo Mac, se instalará y abrirá Skype empresarial en Mac). Para conocer los requisitos para usar estos clientes [, consulte planear los clientes de reuniones (aplicación web y aplicación de reuniones)](meetings-clients.md) .


### <a name="skype-for-business-web-scheduler"></a>Programador web de Skype empresarial

El [programador web de Skype empresarial](https://sched.lync.com) es una herramienta de programación y administración de reuniones basada en web para los usuarios de Skype empresarial online que no tienen acceso a Microsoft Outlook o que se encuentran en un sistema operativo que no está basado en Windows. Con el programador web de Skype empresarial, los usuarios pueden crear nuevas reuniones, modificar las reuniones existentes y enviar invitaciones mediante el programa de correo electrónico que prefiera. La [documentación](https://support.office.com/en-us/article/Skype-for-Business-Web-Scheduler-3b24a211-6470-4a2d-81b7-22d9399d0fec?ui=en-US&amp;rs=en-US&amp;ad=US) del programador web de Skype empresarial proporciona más información.

### <a name="vdi-plugins"></a>Complementos VDI

El entorno de Infraestructura de escritorio virtual (VDI) se utiliza en algunas organizaciones donde los problemas de seguridad y cumplimiento son especialmente delicados. Usar Skype empresarial con audio y vídeo completos en una conexión como esta requiere fuertes cargas de procesamiento de audio y vídeo en el cliente alojado en un escritorio virtual. Hay software complementario VDI disponible que descarga ese procesamiento en la máquina local del usuario final y reduce la carga en el escritorio virtual. Consulte [Plan for Skype for Business in VDI environments](vdi-environments.md) para obtener detalles sobre cómo usar estos complementos.

### <a name="microsoft-teams-rooms"></a>Salas de Microsoft Teams

Salas de Microsoft Teams es la solución para conferencias más reciente de Microsoft, que usa una interfaz familiar y es fácil de implementar y administrar, aprovechando los equipos existentes, como los paneles LCD para facilitar la instalación. Salas de Microsoft Teams usa una aplicación para UWP creada en Surface Pro 4 o Surface Pro en un modo de consola (una vez implementada la aplicación para UWP es la única aplicación que se ejecutará en el dispositivo) y necesita su propia cuenta de dispositivo en la implementación. El software se actualiza a través de la Tienda Windows y de Windows Update. Consulte https://aka.ms/MTRDocs para obtener información sobre cómo usar estas consolas de salas en su implementación. 

### <a name="skype-for-business-on-surface-hub"></a>Skype empresarial en Surface Hub

Microsoft Surface Hub es un dispositivo de productividad todo en uno diseñado para la lluvia de ideas, la colaboración y las presentaciones. Tiene su propia iteración del cliente de Skype empresarial, documentada en la [Guía del administrador de Surface Hub de Microsoft](https://docs.microsoft.com/surface-hub/).

## <a name="choosing-your-organizations-preferred-client"></a>Selección del cliente preferido de su organización
<a name="BK_client_choose"> </a>

Si su organización ha comprado las licencias adecuadas, seleccione el cliente completo; si no, elija el cliente básico. 

Los usuarios pueden instalar el cliente para que no [descarguen Skype empresarial en todos los dispositivos](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3). El cliente también se instalará cuando los usuarios instalen Office 365 en Windows. Si algunos de los usuarios tienen equipos Mac, esos usuarios tendrán un conjunto de características diferente, como se describe en secciones anteriores.

Algunas de las características disponibles en Skype empresarial Server 2015 no están disponibles en Skype empresarial online ni en Skype empresarial Server 2019, consulte las [limitaciones de la cuenta de usuario híbrida o en línea para 2015](desktop-feature-comparison.md#Online-Hybrid) o [en línea o las limitaciones de cuenta de usuario híbrida para 2019](desktop-feature-comparison.md#Online-Hybrid) para obtener información específica. Es posible que los administradores de Skype empresarial online deseen consultar la [Descripción del servicio de Skype empresarial online](https://technet.microsoft.com/library/skype-for-business-online-service-description.aspx) para obtener información sobre los distintos planes disponibles para ellos.

 Antes de implementar o actualizar a Skype empresarial, compruebe qué clientes ya están en uso en su organización. Use la [comparación de características de cliente de escritorio de Skype empresarial](desktop-feature-comparison.md) para comprender el impacto de la compatibilidad de características en esos clientes. Esto puede ayudarle a comunicar los cambios a los usuarios, decidir el ritmo del proceso de lanzamiento y conocer en profundidad las ventajas que tiene actualizar al cliente más reciente.

## <a name="ways-to-deploy-the-client-to-your-users"></a>Métodos para implementar el cliente en sus usuarios
<a name="BK_User_Deploy"> </a>

Los instaladores de cliente están disponibles como instaladores del tipo MSI o hacer clic y ejecutar. La estrategia de mantenimiento del cliente de Skype empresarial puede afectar a sus elecciones, por lo que debe comprender los siguientes puntos:

- En general, Skype empresarial no agrega nuevas características a los clientes lanzados previamente

- En general, Skype empresarial no planea el envío de nuevas características en el MSI de Skype empresarial después de su lanzamiento inicial. Las mejoras en el MSI entre versiones se centrarán principalmente en la calidad y la seguridad.

- La experiencia del cliente de Skype para empresas más reciente y más reciente se encuentra en el programa de instalación de Skype empresarial 2019 hacer clic y ejecutar.

Puede realizar una implementación personalizada del cliente, tal y como se describe en [personalizar la instalación del cliente de Windows en Skype empresarial Server](../../deploy/deploy-clients/customize-windows-client-installation.md). Los métodos de instalación se describen con más detalle en [implementar clientes para Skype empresarial Server](../../deploy/deploy-clients/deploy-clients.md) .

### <a name="click-to-run"></a>Hacer clic y ejecutar

Hacer clic y ejecutar es una tecnología de Microsoft streaming and Virtualization que puede usar para instalar y actualizar productos de Office, incluido Skype empresarial. Estas capacidades de streaming y virtualización se basan en tecnologías de Microsoft Application Virtualization (App-V). Hacer clic y ejecutar tiene las siguientes ventajas:

- Instalación por streaming del conjunto de aplicaciones de Office en un breve espacio de tiempo

- Actualizaciones y revisiones de instalación integrada

- Ocupa menos espacio en disco

- Licencias basadas en usuarios: 5 instalaciones por usuario

- Se puede personalizar a través del Editor XML para la instalación de programas independientes

Es posible que desee usar la [herramienta de implementación de Office](https://www.microsoft.com/en-us/download/details.aspx?id=49117) para este tipo de instalador.

Tanto la versión básica como la completa de los clientes (con varias versiones de 32 y 64 bits) están disponibles con un instalador de hacer clic y ejecutar, los usuarios pueden descargar en [descargar Skype empresarial en todos los dispositivos](https://products.office.com/en-us/skype-for-business/download-app?tab=tabs-3).

### <a name="msi"></a>MSI

MSI es un método de instalación más tradicional, que se usa en los clientes de Skype empresarial 2015 y 2016. Le permite instalar manualmente actualizaciones y revisiones, usar licencias por volumen y activación, y se puede personalizar mediante la herramienta de [Personalización de Office](https://www.microsoft.com/en-us/download/details.aspx?id=49030). Puede distribuir clientes aplicando directivas de grupo, con Microsoft Endpoint Configuration Manager o con una herramienta de terceros.



## <a name="legacy-clients"></a>Clientes heredados
<a name="Legacy"> </a>

Skype empresarial Server 2019 y Skype empresarial online son compatibles con los siguientes clientes lanzados anteriormente: Skype empresarial 2016, Skype empresarial 2015, Lync 2013.

Skype empresarial Server 2015 admite los siguientes clientes lanzados previamente: Lync 2013, Lync 2010, Lync 2010 Mobile, Lync Phone Edition y el operador de Lync 2010. Para obtener información sobre estos clientes cuando se usan con otros servidores, vea las [tablas de comparación de clientes para Lync server 2013](https://technet.microsoft.com/library/gg425836%28v=ocs.15%29.aspx) y [las tablas de comparación de clientes para Lync Server 2010](https://technet.microsoft.com/library/gg425836%28v=ocs.14%29.aspx).
