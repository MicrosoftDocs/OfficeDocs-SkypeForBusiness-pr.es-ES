# <a name="mobile-client-feature-comparison-for-skype-for-business"></a>Comparación de características de cliente móvil de Skype para empresas
 
**Resumen:** Revisar la compatibilidad de la característica para el cliente móvil al planear Skype para Business Server 2015.
  
En este artículo se compara las características y funcionalidades entre Skype para los clientes móviles de negocio y el Skype para el cliente de escritorio de negocios en las categorías siguientes:
  
- Inicio de sesión, notificaciones de inserción y características generales
    
- Presencia mejorada
    
- Contactos y grupos de contactos
    
- Mensajería instantánea (MI)
    
- Skype para negocio tenga Skype para negocios audio y vídeo
    
- Conferencia
    
- Telefonía
    
- Usuarios externos
    
- Archivado y cumplimiento
    
-  Autenticación moderna
    
En las tablas siguientes se enumeran las características que están disponibles para Skype para usuarios de negocios en una implementación local de Skype para Business Server 2015. Las mismas características también están disponibles para Skype para usuarios de negocios en línea y de Microsoft Office 365, a menos que se indique lo contrario en las notas al pie de tabla.
  
> [!NOTE]
> Para la Ayuda en línea y recursos para los usuarios finales, vea [Descubrir Skype para el negocio](https://go.microsoft.com/fwlink/p/?LinkId=528686). 
  
> [!NOTE]
> Para comparar las características disponibles en otro Skype para clientes empresariales, consulte [comparación de características de cliente de escritorio de Skype para el negocio](desktop-feature-comparison.md).
