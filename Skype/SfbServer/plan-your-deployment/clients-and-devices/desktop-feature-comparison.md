# <a name="desktop-client-feature-comparison-for-skype-for-business"></a>Comparación de características de cliente de escritorio de Skype para empresas
 
**Resumen:** Skype para Business Server 2015 o Skype para los administradores de negocios en línea puede utilizar estas tablas para comprender qué características son compatibles con los clientes.
  
 Antes de implementar o actualizar a Skype para el negocio, compruebe que los clientes ya están en uso en su organización. Utilice las tablas siguientes para comprender el impacto de la compatibilidad de característica en dichos clientes. Esto puede ayudarle a comunicar los cambios a los usuarios, decidir el ritmo del proceso de lanzamiento y conocer en profundidad las ventajas que tiene actualizar al cliente más reciente.
  
Algunas características disponibles con Skype para Business Server 2015 no están disponibles en Skype para los negocios en línea, vea [limitaciones de cuenta de usuario en línea o híbrido](desktop-feature-comparison.md#Online-Hybrid) para obtener detalles específicos. Skype para los administradores de negocios en línea que desee consultar a [Skype de descripción de servicios en línea de negocio](https://technet.microsoft.com/library/skype-for-business-online-service-description.aspx) para obtener información sobre los diferentes planes disponibles para ellos.
  
Las siguientes tablas muestran las características que están disponibles con cada cliente que funciona con Skype para Business Server 2015 o Skype para los negocios en línea. También puede hacer referencia a la [comparación de características de cliente móvil de Skype para empresas](mobile-feature-comparison.md) para teléfonos inteligentes y tablet comparaciones de características de cliente. La licencia de acceso de cliente o la licencia de suscripción de usuario que adquiera su organización también influirá en qué características están disponibles para sus usuarios. La decisión de implementar el cliente completo o el básico en los usuarios depende de la licencia o el plan que su organización decida comprar. Consulte la [Guía de licencias](https://products.office.com/en-us/skype-for-business/it-pros) para obtener más detalles.
  
> [!IMPORTANT]
> Skype para Business Server 2015 y Skype para los negocios en línea admiten los siguientes clientes publicados anteriormente: 2013 Lync, Lync 2010, Lync 2010 Mobile, Phone Edition de Lync y operador de Lync 2010. Para obtener información acerca de estos clientes cuando se utiliza con otros servidores, consulte las [tablas de comparación de cliente Lync Server 2013](https://technet.microsoft.com/en-us/library/gg425836%28v=ocs.15%29.aspx) y [tablas de comparación de cliente de Lync Server 2010](https://technet.microsoft.com/en-us/library/gg425836%28v=ocs.14%29.aspx). 
  
> [!NOTE]
> El cliente **Lync 2010 operador** no es compatible con Skype para los negocios en línea.
  
> [!NOTE]
> El Skype para cliente de explorador Web App de negocios y reuniones de Skype App Windows 10 app sólo proporcionan [soporte para las reuniones](desktop-feature-comparison.md#BKMK_Conferencing). Para obtener más información acerca de estos clientes, consulte [Plan para clientes de reuniones (Web App y App de reuniones)](meetings-clients.md) .

## <a name="client-limitations"></a>Limitaciones de cliente 
<a name="Types"> </a>

### <a name="basic-client-limitations"></a>Limitaciones del cliente básico
<a name="Full-Basic"> </a>

Las características que figuran a continuación están disponibles con el cliente completo y no con el cliente básico:  
  
- Administrar la configuración de llamada de equipo
    
- Administrar delegados
    
- Realizar llamadas en nombre de otro contacto (escenario de administrador/delegado)
    
- Administrar llamadas de otro usuario si se configura como delegado
    
- Administrar un gran volumen de llamadas
    
- Empezar una llamada a un grupo de respuesta
    
- Estacionar llamada
    
- Cambiar mensaje de saludo
    
- Atender llamada grupal
    
### <a name="online-or-hybrid-user-account-limitations"></a>Limitaciones de las cuentas de usuario en línea o híbridas
<a name="Online-Hybrid"> </a>

Las cuentas de usuario pueden existir en línea o en local, y esto puede influir en las características que estén disponibles para el usuario. Los usuarios con cuentas en Skype para los negocios en línea no tendrá acceso a las siguientes características, incluso con el cliente completo: 
  
- Presencia mejorada: usar una fotografía de cualquier sitio público para Mi imagen
    
- Contactos: buscar grupos de respuesta
    
- Compatibilidad con la mensajería instantánea: integración de chat persistente (chat grupal)
    
- Soporte IM: Escalar un salón de Chat persistente a un Skype para reuniones de negocios con un solo clic
    
- Usuarios externos: realizar llamadas de dos o más participantes con usuarios externos
