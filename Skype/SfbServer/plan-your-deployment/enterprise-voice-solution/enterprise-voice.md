<span data-ttu-id="58995-123">El Microsoft Lync Server 2013, la herramienta de planeación utiliza la métrica siguiente para estimar el tráfico de usuarios en cada sitio y el número de puertos que son necesarios para admitir ese tráfico.</span><span class="sxs-lookup"><span data-stu-id="58995-123">The Microsoft Lync Server 2013, Planning Tool uses the following metric to estimate user traffic at each site and the number of ports that are required to support that traffic.</span></span>

> <span data-ttu-id="58995-126">Para **Tráfico denso** (3 llamadas o más de RTC por usuario y hora), calcule 5 usuarios por puerto.</span><span class="sxs-lookup"><span data-stu-id="58995-126">For **Heavy traffic** (3 or more PSTN per user calls per hour), figure 5 users per port.</span></span>
