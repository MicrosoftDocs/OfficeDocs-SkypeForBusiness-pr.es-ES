# <a name="define-your-requirements-for-emergency-calls-in-skype-for-business-server"></a>Definir los requisitos para las llamadas de emergencia de Skype para Business Server
 
Se resumen los pasos necesarios para habilitar E9-1-1 en Skype para Business Server Enterprise Voice, dependiendo de si tiene un proveedor de servicios de E9-1-1 de tronco SIP o puerta de enlace ELIN.
  
Antes de comenzar una Skype para la implementación de servidor empresarial E9-1-1, primero debe ser capaz de responder a las preguntas en las secciones siguientes. La planeación que necesite dependerá del tipo de solución E9-1-1 que haya decidido implementar, ya sea un proveedor de servicios E9-1-1 por medio de un tronco SIP o una puerta de enlace de número de identificación de ubicación de emergencia (ELIN).
