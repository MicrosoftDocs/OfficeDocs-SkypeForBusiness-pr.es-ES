# <a name="choose-an-e9-1-1-service-provider-for-skype-for-business-server-2015"></a>Elegir un proveedor de servicios E9-1-1 en Skype Empresarial Server 2015
 
Elegir el proveedor de servicios para una implementación de E9-1-1 en Skype para Telefonía IP empresarial de Business Server.
  
El E9-1-1 servicio proveedor rutas llamadas de emergencia procedentes de Skype para Business Server a la correcta pública seguridad responder a punto (PSAP) basándose en la información de ubicación contenida en la llamada. 
  
Debe obtener E9-1-1 al soporte E9-1-1 como parte de un Skype para la implementación de Business Server, el servicio de enrutamiento de un programa de interoperabilidad abierta de Lync calificado proveedor de servicios de E9-1-1. Selecciona el proveedor que mejor se adapte a los requisitos de tu organización.
