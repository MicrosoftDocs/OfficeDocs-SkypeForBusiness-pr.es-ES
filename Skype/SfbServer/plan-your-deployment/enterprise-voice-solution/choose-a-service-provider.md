# <a name="choose-an-e9-1-1-service-provider-for-skype-for-business-server"></a>Elija un proveedor de servicios E9-1-1 para Skype Empresarial Server
 
Elegir el proveedor de servicios para una implementación de E9-1-1 en Skype Empresarial Server Telefonía IP empresarial.
  
El proveedor de servicios E9-1-1 enruta las llamadas de emergencia procedentes de Skype Empresarial Server al punto de respuesta de seguridad pública (PSAP) correcto en función de la información de ubicación contenida en la llamada. 
  
Para admitir E9-1-1 como parte de una implementación de Skype Empresarial Server, debe obtener el servicio de enrutamiento E9-1-1 de un proveedor de servicios E9-1-1 calificado del Programa de interoperabilidad abierta de Lync. Seleccione el proveedor que mejor se adapte a sus requisitos organizativos.

Para ver la lista de servicios de enrutamiento de E9-1-1 para Lync, vea [Servicios E-911 calificados para Lync](../../../SfbPartnerCertification/lync-cert/e-911-service-providers.md).
