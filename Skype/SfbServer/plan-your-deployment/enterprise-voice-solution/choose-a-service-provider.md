# <a name="choose-an-e9-1-1-service-provider-for-skype-for-business-server"></a>Elegir un proveedor de servicios E9-1-1 en Skype Empresarial Server
 
Selección de su proveedor de servicios para una implementación de E9-1-1 en Skype para Business Server Enterprise Voice.
  
El E9-1-1 servicio proveedor rutas las llamadas de emergencia originadas en Skype para Business Server a la correcta pública respuesta de seguridad punto (PSAP) basándose en la información de ubicación que contiene la llamada. 
  
Para admitir E9-1-1 como parte de un Skype para la implementación de Business Server, debe obtener E9-1-1 el servicio de enrutamiento desde un programa de interoperabilidad abierto de Lync compatibles con proveedor de servicios E9-1-1. Selecciona el proveedor que mejor se adapte a los requisitos de tu organización.

Para ver la lista de servicios de enrutamiento de E9-1-1 para Lync, vea [E-911 servicios compatibles con Lync](https://docs.microsoft.com/en-us/skypeforbusiness/lync-cert/e-911-service-providers).
