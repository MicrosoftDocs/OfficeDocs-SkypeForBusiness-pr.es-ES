# <a name="choose-an-e9-1-1-service-provider-for-skype-for-business-server"></a>Elegir un proveedor de servicios E9-1-1 en Skype Empresarial Server
 
Elegir el proveedor de servicios para una implementación de E9-1-1 en Skype empresarial Server Enterprise Voice.
  
El proveedor de servicios E9-1-1 enruta las llamadas de emergencia originadas en Skype empresarial Server al punto de respuesta de seguridad pública correcto (PSAP), en función de la información de ubicación que contiene la llamada. 
  
Para admitir E9-1-1 como parte de una implementación de Skype empresarial Server, debe obtener el servicio de enrutamiento E9-1-1 de un proveedor de servicios de interoperabilidad abierto de Lync E9-1-1. Selecciona el proveedor que mejor se adapte a los requisitos de tu organización.

Para ver la lista de servicios de enrutamiento de E9-1-1 para Lync, consulte [E-911 Services Qualified for Lync](https://docs.microsoft.com/en-us/skypeforbusiness/lync-cert/e-911-service-providers).
