# <a name="plan-your-enterprise-voice-solution-in-skype-for-business-server"></a>Planear la solución Telefonía IP empresarial en Skype Empresarial Server
 
**Resumen:** Obtenga información sobre las opciones para planear una solución unificada de voz y comunicaciones en Skype Empresarial Server.
  
Skype Empresarial Server ofrece dos opciones de Telefonía IP empresarial locales (Telefonía IP empresarial y Vía trabajo) para que integre el sistema telefónico de su organización con la implementación de Skype Empresarial Server, lo que ofrece una verdadera solución de comunicaciones unificadas. Ambas opciones permiten a los usuarios usar su cliente de Skype Empresarial para iniciar llamadas de voz con otros usuarios, tanto dentro como fuera de su organización.
  
También puede usar tanto Telefonía IP empresarial vía trabajo conjuntamente.
  
- Telefonía IP empresarial es la oferta de voz más completa. Es una solución de voz sobre IP (VoIP) con software que proporciona un conjunto de características muy completo, incluida una integración enriquecida con Outlook y Exchange, y muchas características eficaces, como grupos de respuesta, estacionamiento de llamadas, llamadas de equipo, atención de llamadas grupales y compatibilidad con E9-1-1 de emergencia mejorada. Con Telefonía IP empresarial, los usuarios usan un dispositivo de audio como auriculares con su equipo o un teléfono habilitado para VoIP, en lugar de un teléfono RTC o PBX tradicional.
    
- Vía trabajo ofrece una forma de integrar la solución de Skype Empresarial con los sistemas telefónicos PBX existentes. Un usuario habilitado para Vía trabajo puede hacer clic en Skype Empresarial para llamar a otro usuario, ya sea dentro de su implementación o un usuario externo. La llamada se completa con el teléfono PBX del usuario. Además, estos usuarios pueden hacer clic en Skype Empresarial para unirse a conferencias.
    
    La implementación de Vía trabajo también es una ventaja para los usuarios que aún tienen teléfonos PBX pero que se están transfirieron a Telefonía IP empresarial, ya que permite a estos usuarios usar su cliente de Skype Empresarial para controlar sus llamadas telefónicas.
    
     Vía trabajo ofrece un conjunto más limitado de características de voz que La voz de la entrada. Por ejemplo, no admite delegación, llamada de equipo, grupo de respuesta o E9-1-1 mejorado.
    
Por supuesto, puede tener algunos de sus usuarios habilitados para Telefonía IP empresarial mientras otros usuarios usan teléfonos PBX. Además, durante la transición a una solución de Telefonía IP empresarial completa, puede habilitar a los usuarios que aún tengan teléfonos PBX para Telefonía IP empresarial. Estos usuarios pueden usar su teléfono PBX mientras están en su escritorio y usar Telefonía IP empresarial para realizar o recibir llamadas con dispositivos VoIP mientras se encuentra en otras ubicaciones. Si estos usuarios están habilitados para Vía trabajo, pueden usar su cliente de Skype Empresarial para controlar su teléfono PBX mientras están en su escritorio.
  
> [!NOTE]
> El control remoto de llamadas era una característica que se ofrecía en versiones anteriores de Lync Server, lo que habilitaba a los usuarios a usar su cliente de Skype Empresarial para realizar y recibir llamadas telefónicas en sus teléfonos PBX. El control remoto de llamadas no es compatible con los usuarios que se encuentran en servidores de Skype Empresarial Server, pero se admite para los usuarios con clientes de Skype Empresarial que aún se encuentran en servidores que ejecutan Lync Server 2013.
