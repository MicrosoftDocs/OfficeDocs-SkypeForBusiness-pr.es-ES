# <a name="plan-your-enterprise-voice-solution-in-skype-for-business-server-2015"></a>Diseñe la solución de Telefonía IP empresarial en Skype para Business Server 2015
 
**Resumen:** Obtenga información acerca de las opciones para la planificación de una solución unificada de comunicaciones de voz y en Skype para Business Server 2015.
  
Skype para Business Server ofrece dos premisas opciones de Telefonía IP empresarial, Telefonía IP empresarial y llamar a través de trabajo — le permite integrar el sistema de telefonía de la organización con su Skype para la implementación de servidores empresariales, lo que para una verdadera comunicaciones unificadas solución. Ambas opciones permiten a los usuarios utilizar su Skype para Business client para iniciar llamadas de voz con otros usuarios, tanto dentro como fuera de la organización.
  
También puede beneficiarse de los servicios de PBX de nube de Microsoft, incluyendo servicios de PBX y llamadas de RTC, conectándose a la infraestructura de telefonía local a los servicios proporcionados por Skype para los negocios en línea. Para obtener más información, vea [diseñar el sistema de teléfono en la solución de Office 365 (nube PBX)](../../skype-for-business-hybrid-solutions/plan-your-phone-system-cloud-pbx-solution/plan-your-phone-system-cloud-pbx-solution.md).
  
También puede utilizar Telefonía IP empresarial y llamar a través de trabajo juntos.
  
- Telefonía IP empresarial es la voz más completa que ofrece. Es una solución de voz sobre IP (VoIP) de software que brinda un conjunto de características muy completo, que incluye la integración enriquecida con Exchange y Outlook, así como también muchas potentes características, como los grupos de respuesta, el estacionamiento de llamadas, las llamadas de equipo, la respuesta de llamadas en grupo y el soporte para la emergencia E9-1-1 mejorado. Con la Telefonía IP empresarial, los usuarios utilizar un dispositivo de audio como auriculares con su equipo, o un teléfono habilitado para VoIP, en lugar de un tradicional PSTN o teléfono PBX.
    
- Llamar a través de trabajo ofrece una forma de integrar su Skype para soluciones de negocios con los sistemas existentes de teléfono PBX. Hacer clic en un usuario habilitado para llamar a través de trabajo en Skype para empresas llamar a otro usuario, ya sea dentro de su implementación o un usuario externo. La llamada se efectúa utilizando el teléfono PBX del usuario. Además, estos usuarios pueden hacer clic en Skype para empresas para unirse a conferencias.
    
    Llamar a través de un trabajo de implementación, también es una ventaja para los usuarios que tienen teléfonos PBX aún pero se transición de Telefonía IP empresarial, ya que permite a los usuarios a que se está utilizando su Skype para Business client para controlar sus llamadas telefónicas.
    
     Llamar a través de trabajo ofrece un conjunto más limitado de características de voz thanEnterprise voz. Por ejemplo, no admite delegación, llamada de equipo, grupo de respuesta o mejorado E9-1-1.
    
Por supuesto, puede tener algunos de los usuarios habilitados para Telefonía IP empresarial mientras otros usuarios están utilizando teléfonos PBX. Además, mientras realiza la transición a una completa solución de Telefonía IP empresarial, puede permitir que usuarios que aún tengan teléfonos PBX para la Telefonía IP empresarial. Estos usuarios pueden utilizar su teléfono PBX mientras se encuentra en sus mesas y también utilizar la Telefonía IP empresarial para realizar o recibir llamadas mediante dispositivos VoIP mientras que en otras ubicaciones. Si estos usuarios están habilitados para llamar a través de trabajo, pueden utilizar su Skype para empresa cliente controlar su teléfono PBX mientras se encuentra en sus mesas.
  
> [!NOTE]
> Control de llamada remota fue una característica que se ofrece en versiones anteriores de Lync Server, permitiendo a los usuarios utilizar su Skype para cliente de negocios para hacer y recibir llamadas telefónicas en sus teléfonos PBX. Llamada remota no se admite el control de los usuarios alojados en Skype para servidores Business Server, pero se admite para usuarios con Skype para clientes empresariales que todavía están alojados en servidores que ejecutan Lync Server 2013.
