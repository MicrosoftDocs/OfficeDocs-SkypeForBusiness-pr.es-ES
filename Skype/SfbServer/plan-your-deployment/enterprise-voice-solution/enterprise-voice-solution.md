# <a name="plan-your-enterprise-voice-solution-in-skype-for-business-server"></a>Planeación de la solución de telefonía IP empresarial en Skype empresarial Server
 
**Resumen:** Obtenga información sobre las opciones para planear una solución de comunicaciones y voz unificada en Skype empresarial Server.
  
Skype empresarial Server ofrece dos opciones de telefonía IP empresarial local (telefonía IP empresarial y llamadas vía trabajo) para que pueda integrar el sistema telefónico de su organización con su implementación de Skype empresarial Server, lo que supone una verdadera solución de comunicaciones unificadas. Ambas opciones permiten a los usuarios usar su cliente de Skype empresarial para iniciar llamadas de voz con otros usuarios, tanto dentro como fuera de la organización.
  
También puede usar la telefonía IP empresarial y llamar a través del trabajo en conjunto.
  
- La telefonía IP empresarial es la oferta de voz más completa. Es una solución de voz sobre IP (VoIP) con software que proporciona un amplio conjunto de características, incluida la integración enriquecida con Outlook y Exchange, y muchas características eficaces, como grupos de respuesta, estacionamiento de llamadas, llamadas de equipo, recogida de llamadas de grupo y soporte mejorado de E9-1-1 de emergencia. Con la telefonía IP empresarial, los usuarios usan un dispositivo de audio como un casco con su equipo o un teléfono habilitado para VoIP, en lugar de un teléfono de PBX o RTC tradicional.
    
- La llamada vía trabajo ofrece una forma de integrar la solución de Skype empresarial con los sistemas de teléfono PBX existentes. Un usuario habilitado para llamadas mediante trabajo puede hacer clic en Skype empresarial para llamar a otro usuario, ya sea en su implementación o en un usuario externo. La llamada se completa con el teléfono PBX del usuario. Además, estos usuarios pueden hacer clic en Skype empresarial para unirse a conferencias.
    
    La implementación de la llamada por trabajo también es una ventaja para los usuarios que aún tienen teléfonos PBX, pero que se están cambiando a la telefonía IP empresarial, ya que permite que estos usuarios usen su cliente de Skype empresarial para controlar sus llamadas telefónicas.
    
     Call Via Work ofrece un conjunto más limitado de características de voz thanEnterprise Voice. Por ejemplo, no admite la delegación, la llamada de equipo, el grupo de respuesta o E9-1-1 mejorado.
    
Por supuesto, puede tener algunos usuarios habilitados para telefonía IP empresarial mientras otros usuarios usan teléfonos PBX. Además, mientras realiza la transición a una solución de voz empresarial completa, puede habilitar a los usuarios que aún tienen teléfonos PBX para la telefonía IP empresarial. Estos usuarios pueden usar su teléfono PBX mientras se encuentra en su escritorio y también usar la telefonía IP empresarial para realizar o recibir llamadas mediante dispositivos VoIP en otras ubicaciones. Si estos usuarios están habilitados para llamar por trabajo, pueden usar su cliente de Skype empresarial para controlar su teléfono PBX mientras se encuentra en su escritorio.
  
> [!NOTE]
> El control remoto de llamadas era una característica ofrecida en versiones anteriores de Lync Server, que permite a los usuarios usar su cliente de Skype empresarial para realizar y recibir llamadas telefónicas en sus teléfonos PBX. El control remoto de llamadas no es compatible con los usuarios hospedados en servidores de Skype empresarial Server, pero sí se admite para los usuarios con clientes de Skype empresarial que todavía están alojados en servidores que ejecutan Lync Server 2013.
