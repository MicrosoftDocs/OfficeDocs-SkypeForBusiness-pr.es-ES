# <a name="plan-for-remote-call-control-in-skype-for-business-2015"></a>Planificar el control remoto de llamadas en Skype Empresarial 2015
 
Control remoto de llamada era una característica de versiones anteriores de Lync Server que permite a los usuarios controlar sus teléfonos PBX con Lync Server. En Skype para Business Server, esta característica se ha reemplazado con llamar a través de trabajo.  *En las versiones de cliente de Skype para Business Server 2015 y va hacia delante, remoto llamada control ya no está disponible para configurar en el cliente y se ha quitado para su uso.* 
  
 Usuarios de control de llamada remota de la organización que están alojados en servidores frontales con Lync Server pueden continuar utilizando el control de llamada remota, incluso si están utilizando un Skype para cliente de empresa. Sin embargo, para los usuarios alojados en Skype para Business Server, control de llamada remota no es compatible. Consulte la siguiente tabla para ver las combinaciones de servidor o cliente y si admiten el control remoto de llamadas o Vía trabajo.
  
||**Skype para negocios 2015 cliente con Skype interfaz de usuario habilitado**|**Skype para negocios 2015 cliente con Lync interfaz de usuario habilitado**|**Skype para Business Client de 2016**|**Cliente de 2013 de Lync**|**Cliente de Lync 2010**|
|:-----|:-----|:-----|:-----|:-----|:-----|
| Skype Empresarial Server 2015 <br/> |Vía trabajo  <br/> |1 <br/> |Vía trabajo  <br/> |1 <br/> |1 <br/> |
| Lync Server 2013 <br/> |Control remoto de llamadas  <br/> |Control remoto de llamadas  <br/> |1 <br/> |Control remoto de llamadas  <br/> |Control remoto de llamadas  <br/> |
| Lync Server 2010 <br/> |Control remoto de llamadas  <br/> |Control remoto de llamadas  <br/> |1 <br/> |Control remoto de llamadas  <br/> |Control remoto de llamadas  <br/> |
   
1. Se admite ninguna característica.
  
Para obtener más información, vea [Control de llamada remota](https://go.microsoft.com/fwlink/p/?LinkId=530208) en la documentación de Lync Server 2013.
