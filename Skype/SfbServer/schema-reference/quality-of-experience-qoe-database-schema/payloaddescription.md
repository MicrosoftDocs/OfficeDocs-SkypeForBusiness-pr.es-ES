# <a name="payloaddescription-table"></a>Tabla PayloadDescription
 
La tabla PayloadDescription es una tabla auxiliar. Cada registro representa un códec que se usa en una sesión de audio o vídeo.
  
|**Columna**|**Tipo de datos**|**Clave/índice**|**Detalles**|
|:-----|:-----|:-----|:-----|
|**PayloadDescriptionKey** <br/> |Entero  <br/> |Principal  <br/> |Número único de identificación del códec.  <br/> |
|**PayloadDescription** <br/> |nvarchar(256)  <br/> |Única  <br/> |Nombre del códec.  <br/> |
