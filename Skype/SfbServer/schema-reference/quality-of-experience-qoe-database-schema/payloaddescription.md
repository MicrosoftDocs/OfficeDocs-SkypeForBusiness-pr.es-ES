# <a name="payloaddescription-table"></a>Tabla PayloadDescription
 
En la tabla PayloadDescription es una tabla de apoyo. Cada registro representa un códec, que se utiliza en una sesión de audio o vídeo.
  
|**Columna**|**Tipo de datos**|**Clave o índice**|**Detalles**|
|:-----|:-----|:-----|:-----|
|**PayloadDescriptionKey** <br/> |int  <br/> |Primary  <br/> |Número único que identifica el códec.  <br/> |
|**PayloadDescription** <br/> |nvarchar(256)  <br/> |Único  <br/> |Nombre del códec.  <br/> |
