# <a name="networkconfigurationsettings-view"></a>Vista NetworkConfigurationSettings
 
La vista NetworkConfigurationSettings almacena información sobre la configuración de red. Esta vista se presentó en Microsoft Lync Server 2013.
  
|**Columna**|**Tipo de datos**|**Detalles**|
|:-----|:-----|:-----|
|SubnetIP  <br/> |int  <br/> |Dirección IP de subred (solo se admiten las direcciones IPv4).  <br/> |
|UserSiteName  <br/> |nvarchar(128  <br/> |Nombre del sitio del usuario.  <br/> |
|RegionName  <br/> |nvarchar(128  <br/> |Nombre del país o de la región del usuario.  <br/> |
