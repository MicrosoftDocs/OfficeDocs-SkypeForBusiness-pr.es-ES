# <a name="macaddress-table"></a>Tabla MacAddress
 
La tabla MacAddress es una tabla de soporte. Cada registro representa un origen.
  
|**Columna**|**Tipo de datos**|**Clave o índice**|**Detalles**|
|:-----|:-----|:-----|:-----|
|**MacAddressKey** <br/> |int  <br/> |Primary  <br/> |Número único que identifica la dirección Mac.  <br/> |
|**MacAddress** <br/> |VARCHAR (256)  <br/> |Solo  <br/> |Cadena de dirección Mac.  <br/> |
