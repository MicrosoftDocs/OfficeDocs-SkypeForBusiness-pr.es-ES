# <a name="devicedriver-table"></a>Tabla DeviceDriver
 
La tabla DeviceDriver es una tabla auxiliar. Cada registro representa un controlador que usa un dispositivo de captura o un dispositivo de representación.
  
|**Columna**|**Tipo de datos**|**Clave/índice**|**Detalles**|
|:-----|:-----|:-----|:-----|
|**DeviceDriverKey** <br/> |Entero  <br/> |Principal  <br/> |Número único de identificación de este registro de controlador de dispositivo.  <br/> |
|**DeviceDriver** <br/> |varchar(256)  <br/> |único  <br/> |Nombre de controlador de dispositivo.  <br/> |
