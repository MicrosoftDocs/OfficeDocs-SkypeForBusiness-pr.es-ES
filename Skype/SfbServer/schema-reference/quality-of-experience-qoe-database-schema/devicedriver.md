# <a name="devicedriver-table"></a>Tabla DeviceDriver
 
La tabla DeviceDriver es una tabla de soporte. Cada registro representa un controlador usado por un dispositivo de captura o un dispositivo de representación.
  
|**Columna**|**Tipo de datos**|**Clave o índice**|**Detalles**|
|:-----|:-----|:-----|:-----|
|**DeviceDriverKey** <br/> |int  <br/> |Primary  <br/> |Número único que identifica este registro de controlador de dispositivo.  <br/> |
|**DeviceDriver** <br/> |VARCHAR (256)  <br/> |solo  <br/> |Nombre del controlador del dispositivo.  <br/> |
