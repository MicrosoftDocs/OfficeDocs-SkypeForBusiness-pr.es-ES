# <a name="tblsystemrevision"></a>tblSystemRevision
 
tblSystemRevision contiene el número de revisión que se utiliza con la tabla de tblAdminLock para conseguir coherencia entre varios clientes de administrador.
  
**Columnas**

|**Columna**|**Tipo**|**Descripción**|
|:-----|:-----|:-----|
|sysRevision  <br/> |bigint, no nulo  <br/> |Número de revisión.  <br/> |
