# <a name="tblsystemrevision"></a>tblSystemRevision
 
tblSystemRevision contiene el número de revisión que se usa con la tabla tblAdminLock para conseguir coherencia entre varios clientes de administrador.
  
**Columnas**

|**Columna**|**Tipo**|**Descripción**|
|:-----|:-----|:-----|
|sysRevision  <br/> |bigint, no es nulo  <br/> |Número de revisión.  <br/> |
