# <a name="list-of-cdr-views"></a>Lista de vistas de CDR
 
Las vistas proporcionan una manera fácil de tener acceso a información acerca de los escenarios más comunes que se utilizan para devolver los datos de la base de datos de CDR. Se recomienda utilizar vistas para generar informes personalizados en lugar de utilizar al CDR real tablas de base de datos; eso es porque las vistas de base de datos son más propensos a mantener la compatibilidad con versiones futuras al revés.
  
|**Nombre de la vista**|**Descripción**|
|:-----|:-----|
|[Vista de ClientVersions](clientversions-0.md) <br/> |Devuelve información acerca del cliente de software o dispositivos utilizados en una sesión de comunicación.  <br/> |
|[Vista de ConferenceMessageCount](conferencemessagecount-0.md) <br/> |Devuelve información sobre el número de mensajes enviados por los usuarios en una conferencia.  <br/> |
|[Vista de conferencias](conferences-0.md) <br/> |Devuelve información de la conferencia, incluyendo la hora de inicio y finalización y organizador de la conferencia.  <br/> |
|[Vista de ConferenceSessionDetails](conferencesessiondetails.md) <br/> |Devuelve los detalles de la sesión para todas las sesiones de la conferencia, incluyendo inicio y hora de finalización, identificadores de usuario, códigos de respuesta y diagnóstico IDs.  <br/> |
|[Vista de ConferenceUris](conferenceuris-0.md) <br/> |Devuelve información sobre la conferencia que URI utilizan en una conferencia  <br/> |
|[Vista de ErrorReport](errorreport-0.md) <br/> |Devuelve información sobre los errores que se produjeron durante una sesión.  <br/> |
|[Vista de Bloqueandote](filetransfers.md) <br/> |Devuelve información acerca de las sesiones de transferencia de archivos, incluyendo el nombre del archivo y si se aceptó la transferencia, rechazado o cancelado.  <br/> |
|[Vista de FocusJoinsAndLeaves](focusjoinsandleaves-0.md) <br/> |Devuelve información acerca de conferencia actividades de unión y deja.  <br/> |
|[Vista de McuJoinsAndLeaves](mcujoinsandleaves-0.md) <br/> |Devuelve combina información acerca de la incorporación a la conferencia y deja las actividades (cada combinación de conferencia se corresponde con la correspondiente licencia de conferencia).  <br/> |
|[Vista de MCU](mcus-0.md) <br/> |Devuelve información acerca de servidores de conferencia.  <br/> |
|[Vista de elementos multimedia](media-0.md) <br/> |Devuelve información sobre los tipos de medios utilizados en las sesiones de comunicación peer-to-peer.  <br/> |
|[Vista de ProgressReport](progressreport-0.md) <br/> |Devuelve información acerca de sesiones completadas.  <br/> |
|[Vista de registro](registration-0.md) <br/> |Devuelve información acerca de registros con Skype para Business Server 2015.  <br/> |
|[Vista de SessionDetails](sessiondetails-0.md) <br/> |Devuelve información acerca de las sesiones de peer-to-peer, incluyendo llamadas de teléfono VoIP VoIP, sesiones de mensajería instantánea de dos partes u otras sesiones de comunicación peer-to-peer.  <br/> |
|[Vista de usuario](user.md) <br/> |Devuelve información sobre los usuarios que han participado en las sesiones de comunicación.  <br/> |
|[Vista de VoIPDetails](voipdetails.md) <br/> |Devuelve información de sesiones de peer-to-peer relacionadas con al menos un usuario de VoIP (voz a través de IO).  <br/> |
