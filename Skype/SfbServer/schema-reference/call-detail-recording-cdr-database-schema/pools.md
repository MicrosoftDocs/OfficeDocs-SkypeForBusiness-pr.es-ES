# <a name="pools-table"></a>Tabla Pools
 
La tabla Pools es una tabla de soporte técnico que almacena información sobre los distintos grupos de servidores. Cada registro de la tabla representa un grupo.
  
|**Columna**|**Tipo de datos**|**Clave/índice**|**Detalles**|
|:-----|:-----|:-----|:-----|
|**PoolId** <br/> |Entero  <br/> |Principal  <br/> |Número único con el que se identifica a este grupo.  <br/> |
|**PoolFQDN** <br/> |nvarchar(256)  <br/> | <br/> |FQDN del grupo.  <br/> |
