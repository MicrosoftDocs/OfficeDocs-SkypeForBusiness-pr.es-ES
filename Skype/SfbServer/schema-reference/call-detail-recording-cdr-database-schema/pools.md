# <a name="pools-table"></a>Tabla Pools
 
En la tabla de grupos de servidores es una tabla de apoyo que almacena información sobre el grupo de servidores distintos. Cada registro de la tabla representa un grupo de servidores.
  
|**Columna**|**Tipo de datos**|**Clave o índice**|**Detalles**|
|:-----|:-----|:-----|:-----|
|**PoolId** <br/> |int  <br/> |Primary  <br/> |Número único que identifica a este grupo de servidores.  <br/> |
|**PoolFQDN** <br/> |nvarchar(256)  <br/> | <br/> |FQDN del grupo.  <br/> |
