# <a name="pools-table"></a>Tabla Pools
 
La tabla de grupos es una tabla de soporte que almacena información sobre la agrupación de varios. Cada registro de la tabla representa un grupo de servidores.
  
|**Columna**|**Tipo de datos**|**Clave o índice**|**Detalles**|
|:-----|:-----|:-----|:-----|
|**PoolId** <br/> |int  <br/> |Primary  <br/> |Número único que identifica este grupo.  <br/> |
|**PoolFQDN** <br/> |nvarchar(256)  <br/> | <br/> |FQDN del grupo.  <br/> |
