# <a name="mediationservers-table"></a>Tabla MediationServers
 
La tabla MediationServers es una tabla de soporte técnico. Cada registro almacena información sobre un servidor de mediación que está implicado en las llamadas que tienen registros en la base de datos.
  
|**Columna**|**Tipo de datos**|**Clave/índice**|**Detalles**|
|:-----|:-----|:-----|:-----|
|**MediationServerId** <br/> |Entero  <br/> |Principal  <br/> |Número único que identifica este servidor de mediación.  <br/> |
|**MediationServer** <br/> |nvarchar(256)  <br/> | <br/> |Nombre del servidor de mediación.  <br/> |
