# <a name="mediationservers-table"></a>Tabla MediationServers
 
En la tabla MediationServers es una tabla de apoyo. Cada registro almacena información acerca de un servidor de mediación que está implicado en las llamadas que tienen registros en la base de datos.
  
|**Columna**|**Tipo de datos**|**Clave o índice**|**Detalles**|
|:-----|:-----|:-----|:-----|
|**MediationServerId** <br/> |int  <br/> |Primary  <br/> |Número único que identifica este servidor de mediación.  <br/> |
|**MediationServer** <br/> |nvarchar(256)  <br/> | <br/> |Nombre del servidor de mediación.  <br/> |
