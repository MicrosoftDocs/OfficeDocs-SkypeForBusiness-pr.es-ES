# <a name="uritypes-table"></a>Tabla UriTypes
 
La tabla UriTypes contiene los diferentes tipos URI (identificador uniforme de recursos) supervisados en Skype para Business Server 2015.
  
|**Columna**|**Tipo de datos**|**Clave o índice**|**Detalles**|
|:-----|:-----|:-----|:-----|
|**UriTypeId** <br/> |tinyint  <br/> |Primary  <br/> |Identificador único asignado a un tipo URI.  <br/> |
|**UriType** <br/> |nvarchar(256)  <br/> || Descripciones de los distintos tipos URI. Los valores permitidos son: <br/>  0 - Uri del teléfono <br/>  1 - Uri del usuario <br/> |
