# <a name="edgeservers-table-in-skype-for-business-server-2015"></a>Tabla EdgeServers en Skype empresarial Server 2015
 
La tabla EdgeServers es una tabla de soporte. Cada registro almacena información acerca de un servidor perimetral implicado en llamadas que tienen registros en la base de datos.
  
|**Columna**|**Tipo de datos**|**Clave o índice**|**Detalles**|
|:-----|:-----|:-----|:-----|
|**EdgeServerId** <br/> |int  <br/> |Primary  <br/> |Número único que identifica este servidor perimetral.  <br/> |
|**EdgeServer** <br/> |nvarchar(256)  <br/> | <br/> |Nombre del servidor perimetral.  <br/> |
