# <a name="edgeservers-table-in-skype-for-business-server-2015"></a>Tabla EdgeServers en Skype Empresarial Server 2015
 
La tabla EdgeServers es una tabla compatible. Cada registro almacena información sobre un servidor perimetral que está implicado en las llamadas que tienen registros en la base de datos.
  
|**Columna**|**Tipo de datos**|**Clave/índice**|**Detalles**|
|:-----|:-----|:-----|:-----|
|**EdgeServerId** <br/> |Entero  <br/> |Principal  <br/> |Número único que identifica este servidor perimetral.  <br/> |
|**EdgeServer** <br/> |nvarchar(256)  <br/> | <br/> |Nombre del servidor perimetral.  <br/> |
