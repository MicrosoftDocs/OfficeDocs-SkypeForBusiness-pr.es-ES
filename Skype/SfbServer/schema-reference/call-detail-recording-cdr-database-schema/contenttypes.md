# <a name="contenttypes-table-in-skype-for-business-server-2015"></a>Tabla ContentTypes de Skype para Business Server 2015
 
En la tabla ContentTypes es una tabla de apoyo que almacena una lista de los tipos de contenido que se utiliza en las sesiones de punto a punto y las sesiones de conferencia. Cada registro de la tabla representa un tipo de contenido.
  
|**Columna**|**Tipo de datos**|**Clave o índice**|**Detalles**|
|:-----|:-----|:-----|:-----|
|**ContentTypeId** <br/> |int  <br/> |Primary  <br/> |Número único que identifica el tipo de contenido.  <br/> |
|**ContentType** <br/> |nvarchar(256)  <br/> ||Nombre de tipo de contenido.  <br/> |
