# <a name="mcus-view"></a>Vista Mcus
 
La vista de MCU almacena información sobre las MCU que han participado en sesiones de conferencia. Esta vista se introdujo en Microsoft Lync Server 2013.
  
|**Columna**|**Tipo de datos**|**Detalles**|
|:-----|:-----|:-----|
|**McuId** <br/> |Entero  <br/> |Número único que identifica la MCU.  <br/> |
|**McuUri** <br/> |nvarchar(450)  <br/> |URI de la MCU.  <br/> |
|**McuUriType** <br/> |nvarchar(256)  <br/> |Tipo de URI de la MCU. Vea la [tabla UriTypes](uritypes.md) para obtener más información. <br/> |
