# <a name="gateways-table-in-skype-for-business-server-2015"></a>Tabla Puertas de enlace en Skype Empresarial Server 2015
 
La tabla Puertas de enlace es una tabla de soporte técnico. Cada registro almacena información sobre una puerta de enlace que está implicada en llamadas de red telefónica conmutada (RTC) que tienen registros en la base de datos.
  
|**Columna**|**Tipo de datos**|**Clave/índice**|**Detalles**|
|:-----|:-----|:-----|:-----|
|**GatewayId** <br/> |Entero  <br/> |Principal  <br/> |Número único que identifica esta puerta de enlace.  <br/> |
|**Puerta de enlace** <br/> |nvarchar(256)  <br/> | <br/> |Nombre de la puerta de enlace.  <br/> |
