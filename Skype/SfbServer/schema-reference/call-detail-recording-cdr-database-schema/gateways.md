# <a name="gateways-table-in-skype-for-business-server-2015"></a>Tabla Gateways en Skype Empresarial Server 2015
 
La tabla Puertas de enlace es una tabla compatible. Cada registro almacena información acerca de una puerta de enlace que participa en llamadas de red telefónica conmutada (RTC) que tienen registros en la base de datos.
  
|**Columna**|**Tipo de datos**|**Clave/índice**|**Detalles**|
|:-----|:-----|:-----|:-----|
|**GatewayId** <br/> |entero  <br/> |Principal  <br/> |Número único que identifica esta puerta de enlace.  <br/> |
|**Puerta de enlace** <br/> |nvarchar(256)  <br/> | <br/> |Nombre de la puerta de enlace.  <br/> |
