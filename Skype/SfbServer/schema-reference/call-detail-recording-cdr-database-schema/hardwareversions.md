# <a name="hardwareversions-table-in-skype-for-business-server-2015"></a>Tabla HardwareVersions en Skype para Business Server 2015
 
La tabla HardwareVersions es un auxiliar. Cada registro almacena información sobre la versión del hardware de un dispositivo.
  
|**Columna**|**Tipo de datos**|**Clave o índice**|**Detalles**|
|:-----|:-----|:-----|:-----|
|**VersionId** <br/> |int  <br/> |Primary  <br/> |Número único que identifica la versión del hardware.  <br/> |
|**Versión** <br/> |nvarchar(256)  <br/> | <br/> |Versión de hardware.  <br/> |
