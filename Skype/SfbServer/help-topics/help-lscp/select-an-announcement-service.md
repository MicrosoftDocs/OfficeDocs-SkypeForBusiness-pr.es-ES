# <a name="select-an-announcement-service"></a>Seleccionar servicio de anuncio
 
Use el cuadro de diálogo **Seleccionar un servicio de anuncio** para seleccionar un servicio de anuncio que controlará las llamadas entrantes a un intervalo de números sin asignar.
  
En el cuadro Buscar, escriba todo o parte del nombre del servicio de anuncio que se va a usar. Para devolver todos los servicios de anuncio disponibles, deje vacío el cuadro de búsqueda y haga clic en el icono de búsqueda.
  
En la lista de servicios de anuncio devueltos por la búsqueda, seleccione el servicio que desea usar para controlar llamadas entrantes en el intervalo de números sin asignar.
