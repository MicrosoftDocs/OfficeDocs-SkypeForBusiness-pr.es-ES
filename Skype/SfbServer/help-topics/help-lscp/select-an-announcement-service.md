# <a name="select-an-announcement-service"></a>Seleccionar servicio de anuncio
 
Use el cuadro de diálogo **Seleccionar un servicio de anuncio** para seleccionar un servicio de anuncio que controlará las llamadas entrantes en un intervalo de números sin asignar.
  
En el cuadro de búsqueda, escriba el nombre completo o parcial del servicio de anuncio que desea usar. Para devolver todos los servicios de anuncio disponibles, deje el cuadro de búsqueda en blanco y haga clic en el icono de búsqueda.
  
En la lista de servicios de anuncio devueltos por la búsqueda, seleccione el servicio que desee usar para controlar llamadas entrantes en el intervalo de números sin asignar.
