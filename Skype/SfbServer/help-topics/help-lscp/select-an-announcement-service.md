# <a name="select-an-announcement-service"></a>Seleccionar servicio de anuncio
 
Use el cuadro de diálogo **seleccionar un servicio de anuncios** para seleccionar un servicio de anuncios que controle las llamadas entrantes a un intervalo de números no asignados.
  
En el cuadro de búsqueda, escriba todo o parte del nombre del servicio de anuncios que desea usar. Para devolver todos los servicios de presentación disponibles, deje el cuadro de búsqueda vacío y haga clic en el icono de búsqueda.
  
En la lista de servicios de anuncios devueltos por la búsqueda, seleccione el servicio que desea usar para controlar las llamadas entrantes al intervalo de números no asignados.
