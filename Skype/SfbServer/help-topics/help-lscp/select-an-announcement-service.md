# <a name="select-an-announcement-service"></a>Seleccione un servicio de anuncio
 
Utilice el cuadro de diálogo **Seleccione un servicio de anuncio** para seleccionar un servicio de anuncio que controlará las llamadas entrantes a un intervalo de números sin asignar.
  
En el cuadro Buscar, escriba todo o parte del nombre del servicio de anuncio que desea utilizar. Para devolver todos los servicios disponibles de anuncio, deje vacío el cuadro de búsqueda y haga clic en el icono de búsqueda.
  
En la lista de servicios de anuncio devueltos por la búsqueda, seleccione el servicio que desea utilizar para controlar las llamadas entrantes para el intervalo de números sin asignar.
