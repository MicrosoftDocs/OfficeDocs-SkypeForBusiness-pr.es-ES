# <a name="set-pin"></a>Establecer PIN

Para generar el PIN del usuario, **seleccione Generar automáticamente un PIN válido.**

Para crear su propio PIN, seleccione **Introducir manualmente un PIN determinado** y, a continuación, escriba en el cuadro de texto un PIN que cumpla con los requisitos de PIN especificados en su configuración de directiva de PIN.

Para obtener información detallada sobre los diferentes procedimientos que puede realizar mediante el Panel de control de Skype Empresarial Server, vea [Manage Skype for Business Server 2015](../../manage/manage.md).
