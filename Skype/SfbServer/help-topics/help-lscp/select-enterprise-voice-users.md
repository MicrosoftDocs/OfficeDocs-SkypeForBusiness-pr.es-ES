# <a name="select-enterprise-voice-users"></a>Seleccione los usuarios de Telefonía IP empresarial
 
Utilice el cuadro de diálogo **Seleccionar usuarios de Telefonía IP empresarial** para encontrar usuarios que están habilitados para Telefonía IP empresarial. En el cuadro Buscar, escriba la dirección SIP o mostrar el nombre del usuario que desea buscar y, a continuación, haga clic en **Buscar**. Para seleccionar un usuario, haga clic en el usuario en la lista de resultados de búsqueda y, a continuación, haga clic en **Aceptar**. Si no puede encontrar el usuario que está buscando, ese usuario no puede habilitarse para Telefonía IP empresarial. 
  
> [!NOTE]
> Para recuperar todos los usuarios habilitados para Telefonía IP empresarial, no escriba nada en el cuadro de búsqueda. También puede encontrar a todos los usuarios cuyos nombres de presentación o direcciones SIP empiezan por una determinada cadena. Por ejemplo, para buscar todos los usuarios cuya dirección SIP comienza con ken, escriba sip: ken. Esto devolverá los usuarios como sip:kenmyer@fabrikam.com, sip:kenkwok.fabrikam.com y sip:kenmallit@fabrikam.com. 
  
Para obtener más información acerca de los distintos procedimientos que se pueden realizar utilizando el Skype para Business Server Control Panel, vea [Administrar Skype para Business Server 2015](../../manage/manage.md).
