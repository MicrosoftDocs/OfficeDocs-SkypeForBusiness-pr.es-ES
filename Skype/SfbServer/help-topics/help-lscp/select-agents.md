# <a name="select-agents"></a>Seleccionar agentes
 
Los agentes son los usuarios que se designan para responder a las llamadas de grupo de respuesta. Los grupos de respuesta necesitan tener un grupo de agentes asignados que identifique los agentes que pueden recibir llamadas para el grupo de respuesta. Una forma de crear un grupo de agentes es definir un grupo personalizado seleccionando los usuarios aptos. Usuarios están habilitados para Skype para Business Server y Telefonía IP empresarial. 
  
Use el cuadro de diálogo **Seleccionar agentes** para seleccionar los usuarios que se van a agregar a un grupo de agentes.
  
## <a name="ui-reference"></a>Referencia de interfaz de usuario

En la siguiente lista se describen los controles del cuadro de diálogo **Seleccionar agentes**.
  
- **Buscar** Busca el SIP direcciones o mostrar el nombre de un usuario. Escriba una parte o la totalidad de la dirección o el nombre. Deje el cuadro de búsqueda en blanco para mostrar todos los usuarios que están habilitados para Skype para Business Server y Telefonía IP empresarial.
    
- **Número máximo de usuarios para mostrar** Cambia el número de resultados devueltos que se muestran. Use este contador para limitar la búsqueda si espera muchos resultados.
    
En la siguiente lista se describen los campos del cuadro de diálogo **Seleccionar agentes**.
  
- **Agente** Muestra los nombres de usuario devueltos por la búsqueda.
    
- **Dirección SIP** Muestra las direcciones SIP de usuario devueltas por la búsqueda.
    
- **Telefonía** Muestra el valor del campo de **telefonía** definido para los usuarios.
    
- **Habilitado** Muestra el valor del campo **habilitado para Lync Server** definido para los usuarios.
    
Para obtener información detallada sobre cómo trabajar con grupos de agentes, vea [Administrar grupos de agente](http://technet.microsoft.com/library/36084cdc-38f1-4c45-922f-f81c7e86210c.aspx) en la documentación de las operaciones.
