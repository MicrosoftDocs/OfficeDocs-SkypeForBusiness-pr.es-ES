# <a name="view-gateways-import-configuration"></a>Ver puertas de enlace (Importar configuración)
 
En **Puerta de enlace implementada**, haga clic en la puerta de enlace donde desee importar el archivo de configuración de voz que ha seleccionado.
  
Para obtener información detallada sobre los diferentes procedimientos que puede realizar mediante el Panel de control de Skype Empresarial Server, vea [Manage Skype Empresarial Server 2015](../../manage/manage.md).
