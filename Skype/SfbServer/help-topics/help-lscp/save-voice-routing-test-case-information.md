# <a name="save-voice-routing-test-case-information"></a>Guardar información de caso de prueba de enrutamiento de voz
 
Puede usar el cuadro de diálogo **Guardar información de caso de prueba de enrutamiento de voz** para guardar el caso de prueba de la ruta de voz de atemporary. En **Guardar información de caso de prueba de enrutamiento de voz**, escriba el nombre que desea asignar a este caso de prueba en **nombre** y, a continuación, haga clic en **Aceptar**. 
  
> [!NOTE]
> Cuando guarda un caso de prueba temporal, no lo está guardando en el almacén de administración central. En su lugar, hace que el caso de prueba esté disponible en la pestaña **probar el enrutamiento de voz** de la página de enrutamiento de **voz** . El caso de prueba aparecerá con un estado de no **confirmado**. Para guardar el caso de prueba en el almacén de administración central, debe confirmar la configuración de la voz haciendo clic en **confirmar todo** en cualquier menú de **acciones** de la página de **enrutamiento de voz** .
  
Para obtener más información sobre los distintos procedimientos que puede realizar con el panel de control de Skype empresarial Server, consulte [Manage Skype empresarial server 2015](../../manage/manage.md).
