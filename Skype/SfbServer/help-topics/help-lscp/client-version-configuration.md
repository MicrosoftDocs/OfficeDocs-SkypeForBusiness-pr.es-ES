# <a name="client-version-configuration"></a>Configuración de la versión de cliente
 
Además de especificar la versión de clientes que desea admitir en su entorno, también puede especificar una acción predeterminada para los clientes que aún no tiene una directiva de versión definida. Esto le permite restringir las versiones de cliente se utilizan en su entorno, que puede ayudar a controlar los costos asociados con el soporte de varias versiones de cliente.
  
## <a name="tasks-you-can-perform"></a>Tareas que puede realizar

En la página **Configuración de la versión de cliente** puede realizar las siguientes tareas:
  
- Modificar la configuración predeterminada ( **Global**) de la versión de cliente.
    
- Crear configuración de la versión de cliente para un sitio determinado.
    
- Habilitar y deshabilitar las configuraciones de versión de cliente existentes.
    
> [!NOTE]
> Porque los usuarios anónimos no están asociados a un sitio, sólo directivas de nivel global afecta a los usuarios anónimos. 
  
## <a name="ui-reference"></a>Referencia de interfaz de usuario

En las siguientes listas se describen los menús, comandos, campos y propiedades de la página.
  
- **Nuevo** Puede crear una configuración de versión de cliente de un sitio determinado.
    
- **Editar** Puede cambiar las opciones de cualquiera de las directivas de la versión de cliente. Al usar esta opción, puede hacer lo siguiente:
    
  - **Mostrar detalles** Esta opción abre un cuadro de diálogo en el que puede cambiar las opciones de configuración de versión de un cliente.
    
  - **Seleccionar todo** Esta opción selecciona todas las configuraciones de la versión de cliente de la lista.
    
  - **Eliminar** Esta opción elimina todas las configuraciones de la versión de cliente seleccionado.
    
- **Actualizar** Puede actualizar la lista de configuración de la versión de cliente para comprobar el estado de las opciones de todas las configuraciones de la versión de cliente.
    
Para obtener más información acerca de la interoperabilidad entre los clientes y las versiones de cliente, vea [Interoperabilidad de cliente en la vista previa de 2013 Lync](http://technet.microsoft.com/library/0f126571-91a2-45d5-855c-1e4ddb45fc04.aspx) en la documentación de planeamiento. Para obtener más información sobre cómo trabajar con las configuraciones de versión de cliente, vea [modificar la acción predeterminada para clientes no admitidas explícitamente o restringido](http://technet.microsoft.com/library/548dd0f5-62fe-4c3f-8952-2b9fd4c5fff3.aspx) en la documentación de las operaciones.
