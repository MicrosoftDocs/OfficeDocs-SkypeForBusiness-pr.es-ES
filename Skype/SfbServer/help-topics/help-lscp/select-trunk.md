# <a name="select-trunk"></a>Seleccione el tronco
 
Utilice el cuadro de diálogo **Seleccionar tronco** para encontrar un tronco existente que puede utilizar.
  
> [!NOTE]
> Para recuperar todos los troncos disponibles, no escriba nada en el cuadro de búsqueda y, a continuación, haga clic en el icono de búsqueda. 
  
Para obtener más información acerca de los distintos procedimientos que se pueden realizar utilizando el Skype para Business Server Control Panel, vea [Administrar Skype para Business Server 2015](../../manage/manage.md).
