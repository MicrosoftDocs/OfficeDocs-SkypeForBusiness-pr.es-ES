# <a name="prefix-edit"></a>Edición de prefijo
 
Para definir prefijos o dígitos iniciales, agregue prefijos numéricos que esta ruta deba controlar, incluido el signo + si es necesario. Por ejemplo, puede hacer coincidir todos los números empezando por + 123.
  
Para obtener más información sobre los distintos procedimientos que puede realizar con el panel de control de Skype empresarial Server, consulte [Manage Skype empresarial server 2015](../../manage/manage.md).
