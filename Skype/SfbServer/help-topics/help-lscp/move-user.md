# <a name="move-user"></a>Mover usuario

En el cuadro de diálogo **Mover usuarios** puede seleccionar un grupo de destino al que mover el usuario seleccionado.

Para obtener más información sobre cómo mover usuarios, consulte [Mover usuarios a otro grupo de servidores](https://technet.microsoft.com/library/e7b4968c-0e9d-4d56-b5f1-9edf0f7206f8.aspx).
