# <a name="select-file-type-extensions"></a>Seleccionar extensiones de tipo de archivo
 
Use el cuadro de diálogo **Seleccionar extensiones de tipos de archivo** para seleccionar las extensiones de tipos de archivo que desea bloquear o permitir: marque (para bloquear) o desmarque (para permitir) las casillas junto a las extensiones en las categorías de **Extensiones de tipos de archivo**. Haga clic en la flecha para ampliar una categoría y ver todas las extensiones de tipos de archivo de esa categoría. Si desea seleccionar todas las extensiones de tipos de archivo de una categoría, seleccione el nombre de la categoría. 
  
Si no ve la extensión de un tipo de archivo que desea seleccionar en **Extensiones de tipos de archivo**, escriba la extensión en el cuadro de texto de **Agregar nuevas extensiones de archivo a la lista** y, a continuación, haga clic en **Agregar**. 
  
Después de agregar todas las extensiones de tipos de archivo que desee a la lista **Extensiones de tipos de archivo seleccionadas**, haga clic en **Aceptar**.
  
Para obtener más información sobre los distintos procedimientos que puede realizar con el Panel de control de Skype Empresarial Server, consulte Administrar Skype Empresarial [Server 2015.](../../manage/manage.md)
