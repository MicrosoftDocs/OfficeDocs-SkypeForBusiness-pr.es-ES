# <a name="select-file-type-extensions"></a>Seleccionar extensiones de tipo de archivo
 
Use el cuadro de diálogo **seleccionar extensiones de tipo de archivo** para seleccionar las extensiones de tipo de archivo que desea bloquear o permitir seleccionando (para bloquear) o desactivando (para permitir) las casillas junto a las extensiones de **extensiones de tipo de archivo**. Haga clic en la flecha para expandir una categoría y ver todas las extensiones de tipo de archivo de esa categoría. Si desea seleccionar todas las extensiones de tipo de archivo para una categoría, seleccione el nombre de la categoría. 
  
Si no ve la extensión de un tipo de archivo que desea seleccionar en **las extensiones de tipo de archivo**, escriba la extensión en el cuadro de texto en **Agregar nuevas extensiones de archivo a la lista**y, a continuación, haga clic en **Agregar**. 
  
Una vez que haya agregado todas las extensiones de tipo de archivo que desee a la lista de **extensiones de tipos de archivos** , haga clic en **Aceptar**.
  
Para obtener más información sobre los distintos procedimientos que puede realizar con el panel de control de Skype empresarial Server, consulte [Manage Skype empresarial server 2015](../../manage/manage.md).
