# <a name="select-time-out-queue"></a>Seleccionar cola de tiempo de espera
 
Use el cuadro de diálogo **Seleccione una cola de tiempo de espera** para seleccionar una cola donde se desvíen las llamadas cuando se agote el tiempo de espera.
  
En el cuadro de búsqueda, escriba todo o parte del nombre de la cola que desea usar. Para devolver todas las colas disponibles, deje el cuadro de búsqueda vacío y haga clic en el icono de búsqueda.
  
En la lista de colas devueltas por la búsqueda, seleccione la cola que quiere usar para las llamadas que se han agotado.
