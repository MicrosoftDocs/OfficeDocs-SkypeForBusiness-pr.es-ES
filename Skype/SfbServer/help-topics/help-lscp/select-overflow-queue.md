# <a name="select-overflow-queue"></a>Seleccione cola de desbordamiento
 
Utilice el cuadro de diálogo **Seleccione una cola de desbordamiento** para seleccionar una cola donde las llamadas se reenvían cuando se llena una cola del grupo de respuesta.
  
En el cuadro Buscar, escriba todo o parte del nombre de la cola que desea utilizar. Para devolver todas las colas disponibles, deje vacío el cuadro de búsqueda y haga clic en el icono de búsqueda.
  
En la lista de colas devuelta por la búsqueda, seleccione la cola que desea utilizar para reenviar las llamadas cuando se llena una cola del grupo de respuesta.
