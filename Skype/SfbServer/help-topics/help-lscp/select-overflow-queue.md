# <a name="select-overflow-queue"></a>Seleccionar cola de desbordamiento
 
Use el cuadro de diálogo **seleccionar una cola de desbordamiento** para seleccionar una cola en la que se desvíen las llamadas cuando una cola de grupo de respuesta esté llena.
  
En el cuadro de búsqueda, escriba todo o parte del nombre de la cola que desea usar. Para devolver todas las colas disponibles, deje el cuadro de búsqueda vacío y haga clic en el icono de búsqueda.
  
En la lista de colas devueltas por la búsqueda, seleccione la cola que quiere usar para desviar las llamadas cuando una cola de grupo de respuesta esté llena.
