# <a name="select-overflow-queue"></a>Seleccionar cola de desbordamiento
 
Use el cuadro de diálogo **Seleccionar una cola de desbordamiento** para seleccionar una cola en la que las llamadas se reenvíen cuando se llene una cola de grupo de respuesta.
  
En el cuadro de búsqueda, escriba el nombre completo o parcial de la cola que desea usar. Para devolver todas las colas disponibles, deje el cuadro de búsqueda en blanco y haga clic en el icono de búsqueda.
  
En la lista de colas devueltas por la búsqueda, seleccione la cola que desea usar para reenviar las llamadas cuando una cola de grupo de respuesta se llene.
