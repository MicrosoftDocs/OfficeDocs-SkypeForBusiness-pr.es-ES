# <a name="select-exchange-um-auto-attendant"></a>Seleccionar Operador automático de MU de Exchange
 
Use el cuadro de diálogo **Seleccionar operador automático de mensajería unificada de Exchange** para seleccionar un operador automático de MU de Exchange que controlará las llamadas entrantes en un intervalo de números sin asignar.
  
En el cuadro de búsqueda, escriba el nombre completo o parcial del operador automático de mensajería unificada de Exchange que desea usar. Para devolver todos los operadores automáticos de UM de Exchange disponibles, deje el cuadro de búsqueda en blanco y haga clic en el icono de búsqueda.
  
En la lista de operadores automáticos de UM de Exchange devueltos por la búsqueda, seleccione el operador automático que desee usar para controlar llamadas entrantes en el intervalo de números sin asignar.
