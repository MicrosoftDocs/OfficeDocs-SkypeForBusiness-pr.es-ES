# <a name="select-exchange-um-auto-attendant"></a>Seleccionar Operador automático de MU de Exchange
 
Use el cuadro de diálogo **seleccionar operador automático de mensajería unificada de Exchange** para seleccionar un operador automático de MU de Exchange que controle las llamadas entrantes a un intervalo de números no asignados.
  
En el cuadro de búsqueda, escriba todo o parte del nombre del operador automático de MU de Exchange que desea usar. Para devolver todos los operadores automáticos de MU disponibles de Exchange, deje el cuadro de búsqueda vacío y haga clic en el icono de búsqueda.
  
En la lista de operadores automáticos de mensajería unificada de Exchange devueltos por la búsqueda, seleccione el operador automático que desea usar para controlar las llamadas entrantes al intervalo de números no asignados.
