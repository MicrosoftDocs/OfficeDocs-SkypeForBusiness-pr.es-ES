# <a name="select-exchange-um-auto-attendant"></a>Seleccionar Operador automático de MU de Exchange
 
Use el cuadro de diálogo **Seleccione Exchange operador automático de UM** para seleccionar a un Exchange operador automático de UM que controlará las llamadas entrantes a un intervalo de números sin asignar.
  
En el cuadro Buscar, escriba todo o parte del nombre de la mensajería unificada de operador automático de Exchange que desea usar. Para devolver a todos los disponibles Exchange automáticos de mensajería unificada, deje vacío el cuadro de búsqueda y haga clic en el icono de búsqueda.
  
En la lista de Exchange automáticos de mensajería unificada devueltos por la búsqueda, seleccione al operador automático que desea usar para controlar llamadas entrantes en el intervalo de números sin asignar.
