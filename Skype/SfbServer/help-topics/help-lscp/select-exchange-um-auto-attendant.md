# <a name="select-exchange-um-auto-attendant"></a>Seleccione a de Operador automático de mensajería unificada de Exchange
 
Utilice el cuadro de diálogo **Seleccionar Exchange UM Operador automático** para seleccionar a un Exchange UM Operador automático que controlará las llamadas entrantes a un intervalo de números sin asignar.
  
En el cuadro Buscar, escriba todo o parte del nombre de la Exchange UM Operador automático que desea utilizar. Para devolver a todos los disponibles Exchange UM operadores automáticos, deje vacío el cuadro de búsqueda y haga clic en el icono de búsqueda.
  
En la lista de Exchange UM Auto Attendants devueltos por la búsqueda, seleccione al Operador automático que desea utilizar para controlar las llamadas entrantes para el intervalo de números sin asignar.
