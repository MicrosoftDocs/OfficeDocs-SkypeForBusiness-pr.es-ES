# <a name="translation-rule-type-a-regular-expression"></a>Regla de conversión: Escribir una expresión regular
 
En el campo **Hacer coincidir este patrón**, especifique el patrón que se usará para hacer coincidir los números objeto de la conversión. En el campo **Regla de conversión**, especifique un patrón para el formato de los números convertidos. Por ejemplo, si escribe ^\+(\d  {9}\d+)$ en el campo Coincidir este patrón y 011$1 en el campo Regla de traducción, la regla traducirá +441235551010 a 011441235551010.
  
Para obtener información detallada sobre los diferentes procedimientos que puede realizar mediante el Panel de control de Skype Empresarial Server, vea [Manage Skype Empresarial Server 2015](../../manage/manage.md).
