# <a name="translation-rule-type-a-regular-expression"></a>Regla de conversión: Escribir una expresión regular
 
En el campo **coincidir con este patrón** , especifique el patrón que se usará para coincidir con los números que se van a traducir. En el campo **regla de traducción** , especifique un patrón para el formato de los números traducidos. Por ejemplo,\+si escribe ^ (\d{9}\d +) $ en el campo **coincidir con este patrón** y 011 $1 en el campo regla de **traducción** , la regla traducirá + 441235551010 a 011441235551010.
  
Para obtener más información sobre los distintos procedimientos que puede realizar con el panel de control de Skype empresarial Server, consulte [Manage Skype empresarial server 2015](../../manage/manage.md).
