# <a name="pin-set-successfully"></a>PIN establecido correctamente
 
Para usar el correo electrónico para enviar el PIN al usuario, haga clic en **Abrir mi aplicación de correo electrónico para enviar el PIN nuevo al usuario**. Para copiar el PIN en el mensaje de correo electrónico, seleccione **Mostrar PIN**. 
  
Para obtener información detallada sobre los diferentes procedimientos que puede realizar mediante el Panel de control de Skype Empresarial Server, vea [Manage Skype Empresarial Server 2015](../../manage/manage.md).
