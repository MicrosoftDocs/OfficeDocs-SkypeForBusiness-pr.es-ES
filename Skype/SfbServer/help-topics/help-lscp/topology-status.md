# <a name="topology-status"></a>Topología: Estado

La **página Topología:** **Estado** muestra el estado de los servidores de la topología de Skype Empresarial.

## <a name="tasks-you-can-perform"></a>Tareas que puede realizar

Puede realizar las siguientes tareas en la página **Topología**: **Estado**:

- [Ver una lista de equipos que ejecutan Lync Server 2010](/previous-versions/office/lync-server-2013/lync-server-2013-view-a-list-of-computers-running-lync-server-2013)

- [Ver detalles sobre un servicio](/previous-versions/office/lync-server-2013/lync-server-2013-view-details-about-a-service)

- [Iniciar o detener servicios de Lync Server 2010](/previous-versions/office/lync-server-2013/lync-server-2013-start-or-stop-lync-server-services)

- [Impedir nuevas conexiones a servicios](/previous-versions/office/lync-server-2013/lync-server-2013-prevent-sessions-for-services)

Para obtener información detallada sobre los diferentes procedimientos que puede realizar mediante el Panel de control de Skype Empresarial Server, vea [Manage Skype for Business Server 2015](../../manage/manage.md).
