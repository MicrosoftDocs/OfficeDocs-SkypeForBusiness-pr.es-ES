# <a name="select-contacts-from-active-directory"></a>Seleccionar contactos de Active Directory
 
Use el cuadro **de diálogo Seleccionar de Active** Directory para buscar los contactos que desea habilitar para Skype empresarial. Escriba en el cuadro de búsqueda la dirección SIP o nombre para mostrar del contacto que quiera localizar. Para recuperar todos los contactos disponibles, deje vacío el cuadro de búsqueda.
  
Para obtener más información sobre los distintos procedimientos que puede realizar con el panel de control de Skype empresarial Server, consulte [Manage Skype empresarial server 2015](../../manage/manage.md).
