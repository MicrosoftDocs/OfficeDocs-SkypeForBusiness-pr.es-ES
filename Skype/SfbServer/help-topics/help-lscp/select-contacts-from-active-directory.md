# <a name="select-contacts-from-active-directory"></a>Seleccionar contactos de Active Directory
 
Utilice el cuadro de diálogo **Seleccionar de Active Directory** para buscar los contactos que desea habilitar para Skype para el negocio. Escriba en el cuadro de búsqueda la dirección SIP o nombre para mostrar del contacto que quiera localizar. Para recuperar todos los contactos disponibles, deje vacío el cuadro de búsqueda.
  
Para obtener más información acerca de los distintos procedimientos que se pueden realizar utilizando el Skype para Business Server Control Panel, vea [Administrar Skype para Business Server 2015](../../manage/manage.md).
