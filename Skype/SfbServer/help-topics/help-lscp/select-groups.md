# <a name="select-groups"></a>Seleccionar grupos

Las colas de grupo de respuesta mantienen llamadas a grupos de respuesta hasta que un agente responde a la llamada. Las colas tienen grupos de agentes asignados. Al crear o editar una cola, use el cuadro de diálogo **Seleccionar grupos** para seleccionar uno o varios grupos de agentes para que se asignen a la cola.

## <a name="ui-reference"></a>Referencia de la interfaz de usuario

La siguiente lista describe los controles y campos del cuadro de diálogo **Seleccionar grupos**.

- Cuadro de búsqueda Busca el nombre de un grupo de agentes. Especifica todo el nombre o parte de él. Deje el cuadro de búsqueda en blanco para mostrar todos los grupos de agentes de su organización.

- **Nombre** Muestra los nombres de los grupos de agentes devueltos.

- **Directiva de participación** Muestra la directiva de participación definida para el grupo de agentes: formal o informal.

- **Método de enrutamiento** Muestra el tipo de enrutamiento definido para el grupo de agentes.

Para obtener detalles sobre cómo trabajar con colas, vea [Managing Response Group Queues](/previous-versions/office/lync-server-2013/lync-server-2013-managing-response-group-queues) en la documentación de operaciones.
