# <a name="select-groups"></a>Seleccionar grupos
 
Las colas de respuesta grupo contienen llamadas a grupos de respuesta hasta que un agente responde a la llamada. Colas de asignar a grupos de agente. Al crear o editar una cola, utilice el cuadro de diálogo **Seleccionar grupos** para seleccionar uno o más grupos de agentes que se asignará a la cola.
  
## <a name="ui-reference"></a>Referencia de interfaz de usuario

La lista siguiente describen los controles y campos en el cuadro de diálogo **Seleccionar grupos** .
  
- Cuadro busca el nombre de un grupo de agentes de búsqueda. Escriba todo o parte del nombre. Deje el cuadro de búsqueda en blanco para mostrar todos los grupos de agente para su organización. 
    
- **Nombre** Muestra los nombres de los grupos de agentes devuelta.
    
- **Directiva de participación** Muestra la política de participación definida para el grupo de agentes: formal o informal.
    
- **Método de enrutamiento** Muestra el tipo de enrutamiento definido para el grupo de agentes.
    
Para obtener más información acerca de cómo trabajar con colas, consulte [Administrar colas de grupo de respuesta](http://technet.microsoft.com/library/1e91720c-ab67-4dfb-b30c-0ef2a8012310.aspx) en la documentación de las operaciones.
