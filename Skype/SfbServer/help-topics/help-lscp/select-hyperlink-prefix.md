# <a name="select-hyperlink-prefix"></a>Seleccione prefijo de hipervínculo
 
Utilice el cuadro de diálogo **Seleccione el prefijo de hipervínculo** para seleccionar los tipos de hipervínculos que desee bloquear, enviar un mensaje de advertencia o permiten seleccionar (para bloquear o enviar un mensaje de advertencia) o desactivando (para permitir) las casillas de verificación de las extensiones de la categorías en **prefijos de hipervínculo**. Haga clic en la flecha para expandir una categoría y ver todos los prefijos de hipervínculo para esa categoría. Si desea seleccionar todos los prefijos de hipervínculo de una categoría, seleccione el nombre de categoría. 
  
Si no aparece un prefijo de hipervínculo que desea seleccionar en **prefijos de hipervínculo**, escriba la extensión en el cuadro de texto en **Agregar nuevos prefijos de hipervínculo a la lista**y, a continuación, haga clic en **Agregar**. 
  
Después de agregar todos los prefijos de hipervínculo que desee a la lista de **prefijos de hipervínculo seleccionado** , haga clic en **Aceptar**.
  
Para obtener más información acerca de los distintos procedimientos que se pueden realizar utilizando el Skype para Business Server Control Panel, vea [Administrar Skype para Business Server 2015](../../manage/manage.md).
