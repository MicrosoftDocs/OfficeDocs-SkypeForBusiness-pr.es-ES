# <a name="pin-policy"></a>Directiva de PIN
 
¿Qué desea hacer? haga clic en el enlace de la tarea que desee llevar a cabo.
  
 **Trabajo con directivas de NIP para los teléfonos IP** Esta directiva de identificación personal numérica (PIN) admite la autenticación para los usuarios que se conectan a Skype para Business Server utilizando un teléfono IP.
- [Crear una nueva directiva de NIP en Skype para Business Server 2015](../../manage/authentication/create-a-new-pin-policy.md)
    
- [Modificar una directiva existente de PIN en Skype para Business Server 2015](../../manage/authentication/modify-an-existing-pin-policy.md)
    
- [Eliminar una directiva de NIP en Skype para Business Server 2015](../../manage/authentication/delete-a-pin-policy.md)
    
 **Trabajo con directivas de NIP para conferencias de acceso telefónico** Esta política PIN permite a los usuarios que tienen credenciales de servicios de dominio de Active Directory de la organización para unirse a conferencias de acceso telefónico como usuarios autenticados escribiendo un PIN.
- [Crear o modificar la configuración de conferencia telefónica de PIN para un sitio o grupo de usuarios](http://technet.microsoft.com/library/c29bab5c-2b93-48e0-ae0b-29564daaff9a.aspx)
    
- [Modificar la configuración de PIN de conferencias de acceso telefónico predeterminada](http://technet.microsoft.com/library/2d110e94-ad29-4755-b17f-d8c2da9b78a4.aspx)
    
- [Eliminar la configuración de conferencia telefónica de PIN para un sitio o grupo de usuarios](http://technet.microsoft.com/library/15a9faee-d024-4c0e-b2a0-fe7e7dc00589.aspx)
    
Para obtener más información acerca de los distintos procedimientos que se pueden realizar utilizando el Skype para Business Server Control Panel, vea [Administrar Skype para Business Server 2015](../../manage/manage.md).
