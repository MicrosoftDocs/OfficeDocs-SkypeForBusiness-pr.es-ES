Utilice al Asistente para mover usuarios para mover cuentas de usuario que se encuentran en un grupo local para ser alojados en Skype para los negocios en línea. Para obtener más información, consulte [usar el Asistente para la administración híbrida](http://technet.microsoft.com/library/d777f79b-a740-4aba-a9e2-c91f0315b6f8.aspx).
