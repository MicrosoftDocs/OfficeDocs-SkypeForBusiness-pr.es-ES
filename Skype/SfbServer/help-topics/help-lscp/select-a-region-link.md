# <a name="select-a-region-link"></a>Seleccionar un vínculo de región
 
Use el **cuadro de diálogo Vínculo** de región para buscar vínculos de región disponibles en su entorno. Para usar un vínculo de región existente, haga clic en un vínculo de región de la lista y, a continuación, haga clic en **Aceptar**.
  
> [!NOTE]
> Para recuperar todos los vínculos de región disponibles, no escriba nada en el cuadro de búsqueda y, a continuación, haga clic en el icono de búsqueda. 
  
Para obtener información detallada sobre los diferentes procedimientos que puede realizar mediante el Panel de control de Skype Empresarial Server, vea [Manage Skype Empresarial Server 2015](../../manage/manage.md).
