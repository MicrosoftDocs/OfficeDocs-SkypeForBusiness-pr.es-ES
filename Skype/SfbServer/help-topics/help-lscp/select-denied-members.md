# <a name="select-denied-members"></a>Seleccionar miembros denegados
 
Un administrador de charla persistente puede crear y administrar categorías de salón de chat. Como parte de la creación y administración de categorías de salón de chat, un administrador de charla persistente puede configurar a identidades (servicios de dominio de Active Directory grupos/contenedores/usuarios) que tienen acceso a ser miembros o creadores de salas de chat de una categoría determinada. Un administrador de charla persistente también puede agregar DeniedMembers a una categoría y se convierten en exclusiones explícitas a la lista de permitidos. DeniedMembers reemplazar lo que está en AllowedMembers.
  
## <a name="tasks-that-you-can-perform"></a>Tareas que puede realizar

En la página **Seleccionar miembros denegados** puede realizar las siguientes tareas:
  
- [Configurar categorías](http://technet.microsoft.com/library/4547f514-f0c0-404d-890f-092ddeeac852.aspx)
    
- [Nuevas características de servidor de charla persistente](http://technet.microsoft.com/library/c3ec6f33-6261-4bf5-aa31-baa8ab2a87d8.aspx)
    
Para obtener más información acerca de los distintos procedimientos que se pueden realizar utilizando el Skype para Business Server Control Panel, vea [Administrar Skype para Business Server 2015](../../manage/manage.md).
  
## <a name="to-configure-categories-for-chat-rooms"></a>Para configurar categorías para salones de chat

En la **suscripción**, en la sección **miembros denegados** , agregar o quitar usuarios y otras entidades de Active Directory asociados con miembros denegados de la sala.
