# <a name="select-from-active-directory"></a>Seleccionar de Active Directory
 
Utilice el cuadro de diálogo **Seleccionar de Active Directory** para buscar los usuarios que desea habilitar para Skype para Business Server. Escriba en el cuadro de búsqueda la dirección SIP o nombre para mostrar del usuario que quiera localizar. Para recuperar todos los usuarios disponibles, deje vacío el cuadro de búsqueda.
  
Para obtener más información acerca de los distintos procedimientos que se pueden realizar utilizando el Skype para Business Server Control Panel, vea [Administrar Skype para Business Server 2015](../../manage/manage.md).
