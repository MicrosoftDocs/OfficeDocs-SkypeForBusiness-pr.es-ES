Con RBAC, puede conceder privilegios administrativos asignando usuarios a roles administrativos predefinidos. Skype Empresarial Server incluye un amplio conjunto de roles administrativos integrados y también le permite crear nuevos roles y especificar una lista personalizada de cmdlets para cada rol nuevo. También puede añadir scripts de cmdlets a las tareas permitidas tanto de los roles RBAC predefinidos como personalizados. Para obtener más información, incluidos los roles administrativos predefinidos disponibles en Skype Empresarial Server, consulte Role-Based Access Control.
