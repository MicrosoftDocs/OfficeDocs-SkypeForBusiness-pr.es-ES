Con RBAC, puede conceder privilegios administrativos asignando usuarios a roles administrativos predefinidos. Skype para Business Server incluye un amplio conjunto de funciones administrativas integradas y también le permite crear nuevos roles y especificar una lista personalizada de cmdlets para cada nueva función. También puede agregar scripts de cmdlets a las tareas permitidas de roles de RBAC tanto predefinidos como personalizados. Para obtener información detallada, incluidas las funciones administrativas predefinidas disponibles en Skype para Business Server, vea Control de acceso basado en funciones.
