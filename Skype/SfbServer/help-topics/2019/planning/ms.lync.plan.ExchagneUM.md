# <a name="exchange-unified-messaging-planning-tool-in-skype-for-business-2015"></a>Mensajería (herramienta de planeación) en Skype para 2015 empresarial unificada de Exchange

> [!NOTE]
> Mensajería unificada de Exchange como conocidos anteriormente ya no está disponible en Exchange 2019, pero puede seguir usando el sistema telefónico a los mensajes de correo de voz de registro y, a continuación, deje la grabación en el buzón de Exchange del usuario. Para obtener más información, vea [servicio de planeación de correo de voz en la nube](../../../../SfBServer2019/hybrid/plan-cloud-voicemail.md) .

Skype para Business Server 2015 admite la integración con Exchange mensajería unificada (UM) de la combinación de mensajería de voz y mensajería de correo electrónico en una sola infraestructura de mensajería.

Para obtener detalles, consulte:

- "Planeación para mensajería unificada" en [https://go.microsoft.com/fwlink/?LinkId=285344](https://go.microsoft.com/fwlink/?LinkId=285344).
