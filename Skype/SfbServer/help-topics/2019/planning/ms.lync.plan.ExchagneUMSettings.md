# <a name="exchange-unified-messaging-settings-planning-tool-in-skype-for-business-2015"></a>Exchange Mensajería unificada Configuración (herramienta de planeación) en Skype Empresarial 2015

> [!NOTE]
> Exchange La mensajería unificada como se conocía anteriormente ya no está disponible en Exchange 2019, pero aún puede usar Sistema telefónico para grabar mensajes de correo de voz y, a continuación, dejar la grabación en el buzón de correo Exchange de un usuario. Vea [Plan Correo de voz en la nube service](../../../../sfbhybrid/hybrid/plan-cloud-voicemail.md) para obtener más información.

Skype Empresarial Server 2015 admite la integración con Exchange mensajería unificada (UM) para combinar la mensajería de voz y la mensajería de correo electrónico en una única infraestructura de mensajería.
