# <a name="disaster-recovery-planning-tool"></a>Recuperación ante desastres (herramienta de planeación)
 
Skype para Business Server incluye ante desastres medidas de recuperación, ya que permite a los grupos de servidores Front-End del par que se encuentran en dos centros de datos.
  
Para obtener información detallada acerca de cómo preparar el entorno de recuperación ante desastres, consulte [Plan de alta disponibilidad y recuperación ante desastres en Skype para Business Server](../../../plan-your-deployment/high-availability-and-disaster-recovery/high-availability-and-disaster-recovery.md).
