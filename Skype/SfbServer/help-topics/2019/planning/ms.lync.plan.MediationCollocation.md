# <a name="mediation-collocation-planning-tool"></a>Mediation Collocation (Planning Tool)
 
El servidor de mediación se encuentra en el servidor front-end de forma predeterminada. El servidor de mediación también se puede implementar en un grupo independiente por razones de rendimiento o si implementa la Troncalización SIP, en cuyo caso se recomienda encarecidamente el grupo independiente. Collocation en Skype empresarial Server funciona igual que en Lync Server 2013. Para obtener más información, consulte los temas siguientes:
  
- [Instalación de servidores compatibles en Lync Server 2013](https://technet.microsoft.com/en-us/library/gg425885%28v=ocs.15%29.aspx)
    
- [Componente de servidor de mediación en Skype empresarial Server](../../../plan-your-deployment/enterprise-voice-solution/mediation-server.md)
