# <a name="dial-in-conferencing-planning-tool"></a>Conferencia de acceso telefónico local (herramienta de planeación)
 
Skype Empresarial Server conferencia de acceso telefónico local permite a los usuarios unirse a la parte de audio de una conferencia de Skype Empresarial Server mediante un teléfono de red telefónica conmutada (RTC) sin necesidad de un proveedor de audioconferencia de terceros.
  
Para obtener información sobre cómo habilitar conferencias de acceso telefónico local para su organización, consulte el siguiente tema y los temas a los que se vincula: 
  
- [Planear conferencias en Skype Empresarial Server](../../../plan-your-deployment/conferencing/conferencing.md)
