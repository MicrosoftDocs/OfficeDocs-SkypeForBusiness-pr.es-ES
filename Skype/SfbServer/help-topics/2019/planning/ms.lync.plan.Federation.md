# <a name="federation-planning-tool"></a>Federación (herramienta de planeación)
 
Los servidores perimetrales pueden configurarse para permitir que a los usuarios internos y externos acceso a los contactos pertenecientes a organizaciones asociadas o servicios.
  
 Federación funciona como lo hizo Microsoft Lync 2013. Para obtener más detalles sobre la preparación del entorno para la federación, consulte los siguientes temas:
  
- [Planear la federación de Lync Server 2013 y Office Communications Server](https://technet.microsoft.com/en-us/library/jj205335%28v=ocs.15%29.aspx)
    
- [Planeación para la conectividad de mensajería instantánea pública en Lync Server 2013](https://technet.microsoft.com/en-us/library/jj205349%28v=ocs.15%29.aspx)
    
- [Planear la federación de protocolo (XMPP) extensible de mensajería y presencia en Lync Server 2013](https://technet.microsoft.com/en-us/library/jj205107%28v=ocs.15%29.aspx)
