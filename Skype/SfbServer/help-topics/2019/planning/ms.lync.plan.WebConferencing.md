# <a name="web-conferencing-planning-tool"></a>Conferencia web (herramienta de planeación)
 
Las conferencias web de Skype Empresarial Server incluyen colaboración en documentos, uso compartido de aplicaciones y uso compartido de escritorio.
  
> [!NOTE]
> Skype Empresarial Server usa Office Web Apps y Office Web Apps Server para controlar el uso compartido y la representación de presentaciones de PowerPoint. Las características de conferencia web funcionan exactamente igual que en Microsoft Lync 2013. Para obtener más información sobre cómo instalar y configurar Office Web Apps Server, vea [Configuring integration with Office Web Apps Server and Lync Server 2013](/previous-versions/office/lync-server-2013/lync-server-2013-enabling-office-web-apps-server-and-lync-server-2013).
