# <a name="ip-version-planning-tool"></a>Versión IP (herramienta de planeación)
 
Dispone de tres opciones al configurar las direcciones IP en Skype para Business Server. Puede configurar Skype para Business Server admitir sólo IP versión 4 (IPv4), sólo IP versión 6 (IPv6), o una combinación de ambos (conocido como una pila dual).
  
Para obtener más información acerca de cómo elegir la infraestructura de IP, consulte [Plan para IPv6 en Skype para la empresa](../../../plan-your-deployment/network-requirements/ipv6.md).
