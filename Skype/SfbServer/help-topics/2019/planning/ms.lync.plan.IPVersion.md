# <a name="ip-version-planning-tool"></a>Versión IP (herramienta de planeación)
 
Tiene tres opciones al configurar direcciones IP en Skype Empresarial Server. Puede configurar Skype Empresarial Server para admitir solo IP versión 4 (IPv4), solo ip versión 6 (IPv6) o una combinación de ambos (conocida como pila dual).
  
Para obtener más información sobre cómo elegir la infraestructura IP, vea [Plan for IPv6 in Skype Empresarial](../../../plan-your-deployment/network-requirements/ipv6.md).
