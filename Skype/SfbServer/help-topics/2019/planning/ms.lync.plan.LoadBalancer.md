# <a name="load-balancer-planning-tool"></a>Load Balancer (Planning Tool)
 
Si tiene grupos de servidores front-end, grupos de directores o grupos de servidores perimetrales, necesita implementar el equilibrio de carga para estos grupos. El equilibrio de carga distribuye el tráfico entre los servidores de los grupos de servidores. Para obtener detalles, consulte:
  
- [Requisitos del equilibrio de carga para Skype Empresarial](../../../plan-your-deployment/network-requirements/load-balancing.md)
