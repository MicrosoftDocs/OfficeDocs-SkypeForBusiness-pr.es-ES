# <a name="media-bypass-planning-tool"></a>Media Bypass (Planning Tool)
 
La omisión de elementos multimedia hace referencia a quitar el servidor de mediación de la ruta multimedia siempre que sea posible para las llamadas cuya señalización atraviese el servidor de mediación. Para obtener más información, vea [planear la omisión de medios en Skype empresarial](../../../plan-your-deployment/enterprise-voice-solution/media-bypass.md).
