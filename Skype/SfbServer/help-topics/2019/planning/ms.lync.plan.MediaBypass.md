# <a name="media-bypass-planning-tool"></a>Desvío de medios (herramienta de planeación)
 
Desvío de medios hace referencia a quitar el servidor de mediación de la ruta de acceso de medios siempre que sea posible para las llamadas con señalización que atraviese el servidor de mediación. Para obtener más detalles, vea [Plan para los medios de desvío en Skype para la empresa](../../../plan-your-deployment/enterprise-voice-solution/media-bypass.md).
