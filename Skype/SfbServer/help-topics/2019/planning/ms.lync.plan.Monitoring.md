# <a name="monitoring-planning-tool"></a>Supervisión (herramienta de planeación)
 
En Skype para Business Server, la supervisión se usa para recopilar información sobre el uso y los datos de calidad de la experiencia (QoE) acerca de las sesiones de comunicación que los usuarios participan en.
  
Para obtener más información acerca de la planeación para el servidor de supervisión, consulte [Planning for supervisión](http://technet.microsoft.com/library/26cead5a-183c-42f1-a4b0-0e8d61c6159d.aspx).
