# <a name="voice-infrastructure-planning-tool"></a>Infraestructura de voz (Herramienta de planeación)
 
Skype Empresarial Server proporciona una conectividad RTC confiable y escalable mediante troncos SIP, conexiones SIP directas a una puerta de enlace RTC o conexiones SIP directas a una PBX. Para obtener más información, consulte [Plan for PSTN connectivity in Skype for Business Server](../../../plan-your-deployment/enterprise-voice-solution/pstn-connectivity-0.md).
