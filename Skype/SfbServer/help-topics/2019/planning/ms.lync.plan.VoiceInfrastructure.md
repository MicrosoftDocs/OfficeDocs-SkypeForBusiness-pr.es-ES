# <a name="voice-infrastructure-planning-tool"></a>Voice Infrastructure (Planning Tool)
 
Skype para Business Server ofrece una conectividad de RTC confiable y escalable mediante troncos SIP, conexiones SIP directas a una puerta de enlace RTC o conexiones SIP directas a un sistema PBX. Para obtener información detallada, consulte [Plan para la conectividad de RTC en Skype para Business Server](../../../plan-your-deployment/enterprise-voice-solution/pstn-connectivity-0.md).
