# <a name="setup-or-remove-skype-for-business-server-components-intro"></a>Instalar o quitar componentes de Skype Empresarial Server (introducción)
 
Está preparando la instalación y activación o desactivación y desinstalación de Skype Empresarial Server componentes. Los procesos que debe ejecutar dependen de la configuración actual de la definición publicada en el almacén de administración central. Para completar correctamente las tareas, debe haber iniciado sesión como miembro del grupo de administradores local y, como mínimo, debe poder leer Servicios de dominio de Active Directory.
  
Haga clic en **Siguiente** para continuar con la pantalla **Ejecutando comandos**.
