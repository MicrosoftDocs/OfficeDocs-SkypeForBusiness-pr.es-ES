# <a name="prepare-current-forrest-intro"></a>Preparar bosque actual (introducción)
 
Para empezar la tarea **Preparar el bosque**, haga clic en **Siguiente**. Ejecución de la tarea de **Preparar el bosque** crea la configuración global y los grupos universales para su uso por Skype Business Server.
