# <a name="pending-certificate-intro"></a>Certificado pendiente (introducción)
 
Para procesar una solicitud de certificado en línea pendiente, es decir, una solicitud enviada a su entidad de certificación en línea pero que administrativamente se ha definido para que pase por un proceso de aprobación manual, haga clic en **Siguiente**.
