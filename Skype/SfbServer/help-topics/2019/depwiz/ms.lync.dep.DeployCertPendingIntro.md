# <a name="pending-certificate-intro"></a>Certificado pendiente (introducción)
 
Para procesar una solicitud de certificado en línea pendiente, es decir, una solicitud que se ha emitido a su entidad de certificación en línea pero que está configurada administrativamente para requerir un proceso de aprobación manual, haga clic en **siguiente**.
