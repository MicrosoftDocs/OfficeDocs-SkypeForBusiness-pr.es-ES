# <a name="prepare-current-forrest-invoke"></a>Preparar bosque actual (invocar)
 
La página **Ejecutando comandos** muestra un resumen de las tareas a medida que se ejecutan. Examine la pantalla de resumen, asegúrese de que las tareas se hayan ejecutado correctamente y que no haya errores. Examine las advertencias para comprobar que sean las previstas. Debajo del panel de resumen, debería ver: **Estado de la tarea: completado**. En la lista desplegable, debería estar visible el archivo de registro **Preparar el bosque**. Si no está, seleccione el archivo de registro y haga clic en **Ver registro** para examinar las acciones en detalle y el resultado de las tareas. Haga clic en **Finalizar** para completar la preparación del bosque.
