# <a name="certificate-request-certificate-authority-account"></a>Solicitud de certificado (cuenta de entidad de certificación)
 
Para enviar una solicitud, la entidad de certificación (CA) puede requerir credenciales distintas de las del usuario con el que inició sesión actualmente. Para permitir una solicitud de certificado como un usuario diferente, active la casilla **especificar credenciales alternativas para la entidad emisora de certificados**y, a continuación, escriba el nombre de usuario o el nombre de usuario del _dominio_\ __ de un usuario que pueda solicitar el certificado . En el cuadro de texto **contraseña** , escriba la contraseña del usuario que especificó. El nombre de usuario y la contraseña se envían como parte del proceso de solicitud de certificado, pero no en la solicitud de certificado real, a una entidad de certificación en línea.
