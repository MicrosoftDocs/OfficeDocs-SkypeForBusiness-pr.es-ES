# <a name="certificate-request-certificate-authority-account"></a>Solicitud de certificados (cuenta de entidad de certificación)
 
Para enviar una solicitud, es posible que la entidad de certificación solicite credenciales diferentes a las del usuario que ha iniciado sesión. Para permitir una solicitud de certificado como un usuario diferente, active la casilla Especificar credenciales alternativas para la entidad de certificación y, a continuación, escriba el nombre de usuario o el nombre de usuario del dominio para un usuario que pueda solicitar \   el certificado. En el cuadro de texto **Contraseña**, escriba la contraseña del usuario que ha especificado. El nombre de usuario y la contraseña se envían como parte del proceso de solicitud del certificado, y no en la solicitud real del certificado, a una entidad de certificación en línea.
