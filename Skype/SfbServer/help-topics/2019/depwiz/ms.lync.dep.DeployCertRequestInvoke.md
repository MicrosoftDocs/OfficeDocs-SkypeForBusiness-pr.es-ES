# <a name="certificate-request-invoke"></a><span data-ttu-id="c03fe-109">Solicitud de certificados (invocar)</span><span class="sxs-lookup"><span data-stu-id="c03fe-109">Certificate Request (Invoke)</span></span>
 
<span data-ttu-id="c03fe-p102">La página **Ejecutando comandos** muestra un panel con el estado de la tarea a medida que los comandos se ejecutan para generar la solicitud de certificado. Una vez finalizadas las tareas, la lista desplegable que hay debajo del panel de resumen muestra una lista con archivos de registro. En la lista desplegable, seleccione el archivo de registro que desee ver y haga clic en **Ver registro**. Internet Explorer se iniciará y mostrará en pantalla el archivo de registro. Si no se puede atender una solicitud, examine el archivo o los archivos de registro para determinar el motivo del error. Ejecute de nuevo la solicitud de certificado para resolver el problema y corregir el error que figuraba en el archivo de registro. Vaya pasando por las páginas de solicitud de certificado y vuelva a solicitarlo.</span><span class="sxs-lookup"><span data-stu-id="c03fe-p102">The **Executing Commands** page displays a task status pane as the commands run for the generation of the certificate request. After the tasks are completed, the drop-down list below the summary pane contains a list of log files. Use the drop-down list to select a log file to view, and then click **View Log**. Internet Explorer will start and display the log file. In case of a failed request, you should review the log file(s) to determine the cause of the failure. You can rerun the certificate request to resolve the issue, correcting the problem indicated in the log file. Continue though the certificate request pages and request the certificate again.</span></span>
