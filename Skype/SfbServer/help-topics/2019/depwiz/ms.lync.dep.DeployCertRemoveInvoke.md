# <a name="remove-certificate-invoke"></a>Quitar certificado (invocar)
 
La página **Ejecutando comandos** proporciona información para quitar el certificado. Consulte la información de resumen de eliminación del certificado. Asegúrese de que el archivo de registro Quitar certificados aparezca en la lista desplegable y haga clic en **Ver registro** para examinar los datos de la eliminación del certificado. Una vez examinados el resumen y el archivo de registro, haga clic en  **Finalizar**.
