# <a name="sql-server-reporting-services-groups"></a>SQL Server Reporting Services (grupos)

En el **Grupo de Read-Only especificar**, especifique el grupo al que desea conceder acceso de solo lectura a los informes de supervisión. Utilice el formato &lt;dominio&gt;\\&lt;grupo de acceso de&gt;. Por ejemplo: contoso\RTCUniversalReadOnlyAdmins.
  
Asignación de un grupo de acceso de sólo lectura es opcional. Si no desea conceder acceso de solo lectura ahora, puede configurar más adelante directamente en SQL Server Reporting Services.
