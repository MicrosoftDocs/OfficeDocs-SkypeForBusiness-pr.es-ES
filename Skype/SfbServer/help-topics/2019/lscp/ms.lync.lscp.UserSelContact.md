# <a name="select-contacts-from-active-directory"></a>Seleccionar contactos de Active Directory
 
Use el **cuadro de diálogo Seleccionar de Active Directory** para buscar contactos que desee habilitar para Skype Empresarial. En el cuadro de búsqueda, escriba la dirección SIP o nombre para mostrar del contacto que desee localizar. Para recuperar todos los contactos disponibles, deje vacío el cuadro de búsqueda.
