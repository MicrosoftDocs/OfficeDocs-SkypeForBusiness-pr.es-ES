# <a name="select-contacts-from-active-directory"></a>Seleccionar contactos de Active Directory
 
Use el cuadro de diálogo **Seleccionar de Active Directory** para buscar contactos que desea habilitar para Skype para la empresa. Escriba en el cuadro de búsqueda la dirección SIP o nombre para mostrar del contacto que quiera localizar. Para recuperar todos los contactos disponibles, deje vacío el cuadro de búsqueda.
