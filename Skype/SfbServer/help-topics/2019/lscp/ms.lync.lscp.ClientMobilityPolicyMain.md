# <a name="clients-mobility-policy"></a>Clientes: directiva de movilidad

La página **Directiva de movilidad** para **clientes** muestra las directivas de movilidad disponibles en su implementación.

## <a name="tasks-you-can-perform"></a>Tareas que puede realizar

Puede realizar las siguientes tareas en la página de la **Directiva de movilidad** :

- Ver las directivas de movilidad existentes

- Crear una nueva Directiva de movilidad

- Editar una directiva de movilidad

## <a name="ui-reference"></a>Referencia de interfaz de usuario

En las siguientes listas se describen los menús, comandos, campos y propiedades de la página.



- **Campo de búsqueda** Escriba una cadena en el cuadro de texto para filtrar las directivas que se muestran.

- **Nuevo** Haga clic en **nuevo**y, a continuación, seleccione **sitio** o **usuario** para crear una nueva Directiva de movilidad.

- **Editar** Seleccione una directiva y, a continuación, haga clic en **Editar** para editar las propiedades de la Directiva seleccionada.

- **Actualizar** Actualiza la lista de directivas mostradas. Al hacer clic en **Actualizar**, se borra la cadena de filtro de búsqueda y se muestran todas las directivas existentes.

Para obtener más información sobre cómo configurar directivas de movilidad, consulte Configuración de la [Directiva de movilidad](https://technet.microsoft.com/library/595536e0-9bb3-49a3-8d13-1a77351ebc62.aspx).
