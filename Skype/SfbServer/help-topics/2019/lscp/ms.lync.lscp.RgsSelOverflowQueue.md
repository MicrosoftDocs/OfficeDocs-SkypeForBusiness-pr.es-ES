# <a name="select-overflow-queue"></a>Seleccionar cola de desbordamiento
 
Use el cuadro de diálogo **Seleccionar una cola de desbordamiento** para seleccionar una cola que se reenvíen las llamadas cuando una cola de grupo de respuesta está lleno.
  
En el cuadro Buscar, escriba todo o parte del nombre de la cola que desea usar. Para devolver todas las colas disponibles, deje vacío el cuadro de búsqueda y haga clic en el icono de búsqueda.
  
En la lista de colas devueltas por la búsqueda, seleccione la cola que desea usar para reenviar las llamadas cuando una cola de grupo de respuesta está lleno.
