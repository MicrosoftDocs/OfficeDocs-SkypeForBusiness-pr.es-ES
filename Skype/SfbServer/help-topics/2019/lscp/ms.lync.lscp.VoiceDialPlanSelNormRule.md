# <a name="select-normalization-rules"></a>Seleccionar reglas de normalización
 
Las reglas de normalización .NET Framework expresiones regulares para traducir los números de teléfono marcados al formato E.164. Cada plan de marcado debe tener asignadas una o más reglas de normalización.
  
Para usar una regla de normalización existente, haga clic en una regla de la lista y, a continuación, haga clic en **Aceptar**.
