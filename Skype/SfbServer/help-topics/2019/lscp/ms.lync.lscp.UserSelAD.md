# <a name="select-from-active-directory"></a>Seleccionar de Active Directory
 
Use el cuadro de diálogo **Seleccionar de Active Directory** para buscar los usuarios que desea habilitar para Skype para Business Server. Escriba en el cuadro de búsqueda la dirección SIP o nombre para mostrar del usuario que quiera localizar. Para recuperar todos los usuarios disponibles, deje vacío el cuadro de búsqueda.
  
Para obtener información detallada sobre los distintos procedimientos que puede realizar mediante el uso de la Skype para el Panel de Control de servidor empresarial, vea [Administrar Skype para Business Server 2015](../../manage/manage.md).
