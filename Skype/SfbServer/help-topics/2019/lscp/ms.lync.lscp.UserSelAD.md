# <a name="select-from-active-directory"></a>Seleccionar de Active Directory
 
Use el cuadro de diálogo Seleccionar **de Active Directory** para buscar usuarios que desee habilitar para Skype Empresarial Server. En el cuadro de búsqueda, escriba la dirección SIP o nombre para mostrar del usuario que desee localizar. Para recuperar todos los usuarios disponibles, deje vacío el cuadro de búsqueda.
