# <a name="translation-rule-type-a-regular-expression"></a>Regla de conversión: Escriba una expresión Regular
 
En el campo **hacer coincidir este patrón** , especifique el patrón que se usará para hacer coincidir los números que se va a traducir. En el campo **regla de conversión** , especifique un patrón para el formato de los números convertidos. Por ejemplo, si especifica ^\+(\d{9}\d+)$ en el campo **coincidir este patrón** y 011$ 1 en el campo **regla de conversión** , la regla convertirá + 441235551010 a 011441235551010.
  
Para obtener información detallada sobre los distintos procedimientos que puede realizar mediante el uso de la Skype para el Panel de Control de servidor empresarial, vea [Administrar Skype para Business Server 2015](../../../manage/manage.md).
