# <a name="save-voice-routing-test-case-information"></a>Guardar información de caso de prueba de enrutamiento de voz
 
Puede usar el cuadro de **diálogo Guardar** información de casos de prueba de enrutamiento de voz para guardar un caso de prueba de enrutamiento de voz moderno. En **Información para guardar caso de prueba de enrutamiento de voz**, escriba el nombre que desee dar a este caso de prueba en **Nombre** y, a continuación, haga clic en **Aceptar**. 
  
> [!NOTE]
> Cuando guarda un caso de prueba temporal, no lo guarda en el almacén de administración central. En su lugar, hace que el caso de prueba esté disponible desde la ficha **Probar enrutamiento de voz** de la página **Enrutamiento de voz**. El caso de prueba aparecerá con el estado **No confirmado**. Para guardar el caso de prueba en el almacén de  administración central, debe confirmar la configuración de voz haciendo clic en Confirmar todo desde cualquier menú Acción dentro de la página **Enrutamiento de** voz.
