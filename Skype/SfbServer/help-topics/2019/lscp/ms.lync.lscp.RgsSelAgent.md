# <a name="select-agents"></a>Seleccionar agentes

Los agentes son usuarios designados para responder a las llamadas del grupo de respuesta. Los grupos de respuesta deben tener un grupo de agentes asignado que identifique los agentes que pueden recibir llamadas para el grupo de respuesta. Una forma de crear un grupo de agentes es definir un grupo personalizado seleccionando usuarios elegibles. Los usuarios elegibles están habilitados para Skype Empresarial Server y Telefonía IP empresarial.

Use el cuadro **de diálogo Seleccionar agentes** para seleccionar los usuarios que se agregarán a un grupo de agentes.

## <a name="ui-reference"></a>Referencia de la interfaz de usuario

En la siguiente lista se describen los controles del **cuadro de diálogo Seleccionar** agentes.

- **Buscar** Busca la dirección SIP o el nombre para mostrar de un usuario. Escriba todo o parte de la dirección o el nombre. Deje el cuadro de búsqueda vacío para mostrar todos los usuarios que están habilitados para Skype Empresarial Server y Telefonía IP empresarial.

- **Máximo de usuarios para mostrar** Cambia el número de resultados devueltos que se muestran. Use este contador para limitar la búsqueda si espera muchos resultados.

En la lista siguiente se describen los campos del **cuadro de diálogo Seleccionar** agentes.

- **Agente** Muestra los nombres de usuario devueltos por la búsqueda.

- **Dirección SIP** Muestra las direcciones SIP de usuario devueltas por la búsqueda.

- **Telefonía** Muestra el valor del **campo Telefonía** definido para los usuarios.

- **Habilitado** Muestra el valor del campo **Habilitado para Lync Server** definido para los usuarios.

Para obtener detalles sobre cómo trabajar con grupos de agentes, vea [Managing Agent Groups](/previous-versions/office/lync-server-2013/lync-server-2013-managing-response-group-agent-groups) en la documentación de operaciones.
