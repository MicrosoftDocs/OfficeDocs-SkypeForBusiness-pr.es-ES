# <a name="move-user"></a>Mover usuario

En la **página Mover usuarios** , puede seleccionar un grupo de destino al que mover el usuario seleccionado.

Para obtener más información sobre cómo mover usuarios, consulte [Move Users to Another Pool](/previous-versions/office/lync-server-2013/lync-server-2013-move-users-to-another-pool).
