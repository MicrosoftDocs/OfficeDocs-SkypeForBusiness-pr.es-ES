# <a name="select-trunk"></a>Seleccionar tronco
 
Use el cuadro de diálogo **Seleccionar tronco** para buscar un tronco existente que se puede usar.
  
> [!NOTE]
> Para recuperar todos los troncos disponibles, no escriba nada en el cuadro de búsqueda y, a continuación, haga clic en el icono de búsqueda.
