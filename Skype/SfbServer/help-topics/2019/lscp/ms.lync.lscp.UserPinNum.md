# <a name="pin-set-successfully"></a>PIN establecido correctamente
 
Para usar el correo electrónico para enviar el PIN al usuario, haga clic en **abrir mi aplicación de correo electrónico para enviar el nuevo PIN al usuario**. Para copiar el PIN en el mensaje de correo electrónico, seleccione **Mostrar PIN**.
