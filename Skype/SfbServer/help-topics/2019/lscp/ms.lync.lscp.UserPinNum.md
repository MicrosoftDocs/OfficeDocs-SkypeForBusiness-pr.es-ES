# <a name="pin-set-successfully"></a>PIN establecido correctamente
 
Para usar el correo electrónico para enviar el PIN para el usuario, haga clic en **Abrir Mi aplicación de correo electrónico para enviar el nuevo PIN para el usuario**. Para copiar el PIN en el mensaje de correo electrónico, seleccione **Mostrar PIN**. 
  
Para obtener información detallada sobre los distintos procedimientos que puede realizar mediante el uso de la Skype para el Panel de Control de servidor empresarial, vea [Administrar Skype para Business Server 2015](../../manage/manage.md).
