# <a name="move-all-users"></a>Mover todos los usuarios

En el cuadro de diálogo **Mover usuarios**, puede seleccionar los grupos de origen y destino para mover las cuentas de usuario de un grupo a otro en su implementación.

Para más información sobre cómo mover usuarios, vea [Move Users to Another Pool](https://technet.microsoft.com/library/e7b4968c-0e9d-4d56-b5f1-9edf0f7206f8.aspx)..
