# <a name="response-groups-group"></a>Grupos de respuesta: Grupo

Los grupos de respuesta constan de grupos de agentes, colas y flujos de trabajo. Un grupo de agentes es un grupo de personas designadas para responder a llamadas grupales de respuesta.

En la página**Grupo** de **grupos** - de respuesta se muestra una lista de todos los grupos de agentes definidos para su organización.

## <a name="tasks-you-can-perform"></a>Tareas que puede realizar

Puede realizar las siguientes tareas desde la página **** - **Grupo** de grupos de respuesta:

- Crear un nuevo grupo de agentes

- Cambiar un grupo de agentes existente

- Eliminar un grupo de agentes

## <a name="ui-reference"></a>Referencia de interfaz de usuario

En la siguiente lista se describen los comandos de la página.

- **Nuevo** Inicia un nuevo grupo de agentes.

- **Editar** Abre el grupo de agentes seleccionado para modificarlo, selecciona todos los grupos de agentes de la lista o elimina los grupos de agentes seleccionados.

- **Actualizar** Actualiza la lista de grupos de agentes.

En la siguiente lista se describen los campos de la página.

- **Nombre** Nombre único que identifica el grupo de agentes.

- **Servicio** El servicio **ApplicationServer** que hospeda el grupo de agentes.

- **Descripción** La descripción del grupo de agentes.

Para obtener más información sobre las funciones y características de grupos de respuesta, consulte [planear la aplicación de grupo de respuesta en Skype empresarial Server](../../../plan-your-deployment/enterprise-voice-solution/response-group.md) en la documentación de planificación. Para más información sobre cómo trabajar con grupos de agentes, mire [Managing Agent Groups](https://technet.microsoft.com/library/36084cdc-38f1-4c45-922f-f81c7e86210c.aspx) en la documentación de operaciones.
