# <a name="hybrid-manageability-wizard"></a>Asistente para la administración híbrida

Puede usar el Asistente para la administración híbrida para realizar algunas de las tareas de configuración necesarias para configurar una implementación híbrida con Skype Empresarial Online. Para obtener más información acerca de cómo configurar la implementación para híbrido, vea [Configure your environment with the Hybrid Manageability Wizard](https://technet.microsoft.com/library/d777f79b-a740-4aba-a9e2-c91f0315b6f8.aspx).
