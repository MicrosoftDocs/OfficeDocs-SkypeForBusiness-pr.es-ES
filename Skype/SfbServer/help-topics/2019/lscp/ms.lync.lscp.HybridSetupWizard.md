# <a name="hybrid-manageability-wizard"></a>Asistente de la capacidad de administración híbrida
 
Puede usar al Asistente para la administración híbrida para llevar a cabo algunas de las tareas de configuración necesarias para configurar una implementación híbrida con Skype para profesionales en línea. Para obtener más información acerca de cómo configurar la implementación para entornos híbridos, vea [Configure your environment con el Asistente para la administración híbrida](http://technet.microsoft.com/library/d777f79b-a740-4aba-a9e2-c91f0315b6f8.aspx).
