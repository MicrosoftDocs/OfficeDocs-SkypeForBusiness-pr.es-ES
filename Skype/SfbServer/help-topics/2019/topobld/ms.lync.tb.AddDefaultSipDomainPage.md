# <a name="add-default-sip-domain"></a>Agregar dominio Sip predeterminado
 
Especifique el dominio del Protocolo de inicio de sesión (SIP) principal que desea que admita la implementación. El dominio SIP principal puede ser el mismo que el dominio de los servicios de dominio de Active Directory o diferente de él. Por ejemplo, el dominio de Active Directory podría ser fabrikam.net y podría especificar que el dominio SIP principal se contoso.net.
