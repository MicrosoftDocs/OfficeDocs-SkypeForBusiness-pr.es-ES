# <a name="shared-components-scope-node"></a>Nodo de ámbito de componentes compartidos
 
Haga clic con el botón secundario en **componentes** compartidos en el árbol de jerarquía del generador de topología para crear nuevos componentes compartidos o para obtener acceso a opciones de nivel de topología.
  
![Nodo de ámbito de componentes compartidos](../../../media/Shared_Components_Scope_Node.jpg)
  
Seleccione **nuevo** para crear un nuevo almacén de aplicaciones, una nueva puerta de enlace IP/PSTN, un nuevo servidor de Office Web Apps, un nuevo almacén de SQL Server o un nuevo tronco.
  
> [!NOTE]
> Consulte los temas de ayuda para cada una de estas opciones. 
  
![Nodo de ámbito de componentes compartidos](../../../media/Shared_Components_Scope_NodeB.jpg)
  
Seleccione **topología** para hacer clic en opciones para abrir un nuevo documento de topología. Abrir un documento de topología existente, descargar la topología actual del almacén de administración central, guardar una copia del documento de topología actual, publicar el documento de topología actual, instalar una base de datos para crear la configuración de la base de datos configurada actualmente, Combine Office Communications Server 2007 R2 en la implementación actual o para quitar la implementación.
  
![Nodo de ámbito de componentes compartidos C](../../../media/Shared_Components_Scope_NodeC.jpg)
