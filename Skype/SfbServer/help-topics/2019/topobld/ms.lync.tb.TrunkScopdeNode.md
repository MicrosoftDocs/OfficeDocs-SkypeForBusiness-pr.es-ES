# <a name="trunk-scope-node"></a>Nodo de ámbito del tronco
 
Botón secundario haga clic en un tronco SIP existente en el generador de topología para realizar lo siguiente:
  
- **Editar propiedades**: inicia el asistente **Editar propiedades**, que sirve para editar o redefinir la configuración de un tronco SIP existente.
    
Además de la opción Editar propiedades, puede realizar las siguientes tareas generales:
  
- **Topología**. Esta opción permite definir una topología nueva, abrir una topología existente, descargar la topología actual, guardar una copia de la topología, publicar la topología, instalar una base de datos, combinar componentes de versiones anteriores de Lyn Server o quitar la implementación.
    
- **Eliminar**. Use esta opción para quitar el elemento seleccionado en este momento. Es posible que otras dependencias impidan la eliminación. Aparecerá un cuadro de diálogo indicando qué servicios y dependencias se deben resolver para poder eliminar el elemento.
    
- **Ayuda**. Esta opción permite tener acceso a la Ayuda del nodo activo.
