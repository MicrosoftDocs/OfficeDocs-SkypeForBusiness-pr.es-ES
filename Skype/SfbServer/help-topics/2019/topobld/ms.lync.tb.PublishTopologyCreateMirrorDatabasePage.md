# <a name="publish-topology-create-mirror-database-page"></a>Publicar la topología de creación de página de la base de datos de reflejo
 
Esta documentación es preliminar y está sujeta a cambios. Los temas en blanco que se incluyen actúan como marcadores de posición.
  
Generador de topología analiza todas las opciones de configuración en el documento de topología definido. Si ha configurado las bases de datos reflejadas de SQL Server, se presentan una pantalla que muestra el almacén de reflejo, el almacén principal, la ruta de acceso de recurso compartido de archivo y las rutas de acceso de la base de datos. Seleccione una casilla de verificación asociada con un almacén de reflejo de base de datos para crear la base de datos reflejada y la ruta de acceso del recurso compartido de archivo. Desactive una casilla de verificación de un reflejo de la base de datos si no desea crear e implementar la base de datos en el servidor SQL definido.
  
 Si selecciona para elegir no instalar una o todas estas bases de datos en este momento, desactive las bases de datos que no se instalará. En un momento posterior, opta por usar **Instalar base de datos** haciendo clic en la mayoría de los elementos de nodo en el generador de topología.
