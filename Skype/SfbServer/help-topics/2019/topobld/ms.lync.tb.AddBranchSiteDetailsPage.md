# <a name="add-branch-site-details"></a>Agregar detalles de sucursal

Para especificar los detalles del sitio para un sitio, haga lo siguiente:

Especifique un nombre de ciudad para la nueva sucursal. El valor de Ciudad es una entrada opcional.

Especifique un nombre de estado/provincia para la nueva sucursal. El valor de Estado o provincia es una entrada opcional.

Especifique un código de país o región para la nueva sucursal. El valor de País o región es una entrada opcional.

Para obtener más información, consulte [Sitios](/previous-versions/office/lync-server-2013/lync-server-2013-sites).
