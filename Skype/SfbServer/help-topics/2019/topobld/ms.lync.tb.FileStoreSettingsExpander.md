# <a name="file-store-settings-expander"></a>Expansor de configuración de almacén de archivos
 
Para editar las propiedades de los almacenes de archivos definido:
  
Para el nombre de recurso compartido de archivos, puede definir el nombre de un nuevo recurso compartido. A continuación, debe crear el recurso compartido en el servidor que se denomina en la propiedad de nombre (FQDN) del dominio de servidor de archivos completo.
  
> [!NOTE]
> No se puede cambiar el FQDN del servidor que se especifica en la propiedad de FQDN del servidor de archivos. Si necesita colocar el recurso compartido de archivo de almacenamiento en un servidor diferente, cree una nueva entrada del almacén de archivo.
