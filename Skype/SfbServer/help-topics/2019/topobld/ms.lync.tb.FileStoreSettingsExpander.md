# <a name="file-store-settings-expander"></a>Expansor de configuración de almacén de archivos
 
Para editar las propiedades de los almacenes de archivos definidos:
  
Para el nombre del recurso compartido de archivos, puede definir el nombre de un nuevo recurso compartido. A continuación, debe crear el recurso compartido en el servidor denominado en la propiedad nombre de dominio completo (FQDN) del servidor de archivos.
  
> [!NOTE]
> No puede cambiar el FQDN del servidor especificado en la propiedad FQDN del servidor de archivos. Si necesita colocar el recurso compartido de archivos del almacén de archivos en un servidor diferente, cree una nueva entrada en el almacén de archivos.
