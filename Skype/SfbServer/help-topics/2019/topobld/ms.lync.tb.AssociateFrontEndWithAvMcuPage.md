# <a name="associate-front-end-with-av-mcu"></a>Asociar front-end con MCU AV
 
Las conferencias A/V permiten comunicaciones de audio y vídeo en tiempo real entre los usuarios (es decir, si tienen dispositivos cliente apropiados, como videollamadas y cámaras Web para videoconferencias). Si su implementación admite conferencias y habilita las conferencias web y A/V, puede Collocate el servidor de conferencia A/V con el servidor front-end o puede implementar uno o varios servidores de conferencia A/v independientes (grupo de conferencia A/V). Si elige la opción de implementar un servidor de conferencia A/V independiente, debe definirlo en el generador de topología.
  
Todas las agrupaciones en un sitio y los grupos de varios sitios centrales pueden usar el mismo servidor de conferencia A/V, si el uso no supera la capacidad del servidor de conferencia A/V.
