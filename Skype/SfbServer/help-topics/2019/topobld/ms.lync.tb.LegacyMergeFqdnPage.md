# <a name="legacy-merge-fqdn"></a>FQDN de combinación de información heredada
 
El **FQDN interno del grupo de servidores de acceso perimetral** se utiliza para una variedad de escenarios donde los usuarios internos comunicarse con usuarios externos para la federación, el acceso de usuarios remotos y la conectividad de mensajería instantánea pública. Si se ha implementado un servidor perimetral con equilibrio de carga en el entorno heredado, escriba el nombre de dominio completo (FQDN) del equilibrador de carga interno.
  
El valor de **puerto SIP interno de acceso** de **5061** es el puerto de protocolo de Control de transmisión (TCP) SIP predeterminado para la comunicación con los clientes, grupos de servidores Front-End y servidores heredados. Si no se utilizó el valor predeterminado, actualizar la **puerto SIP interno de acceso:** valor.
