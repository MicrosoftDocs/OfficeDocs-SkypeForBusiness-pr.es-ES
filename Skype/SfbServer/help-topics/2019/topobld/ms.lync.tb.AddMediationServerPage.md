# <a name="add-mediation-server"></a>Agregar servidor de mediación
 
Para definir el servidor de mediación, especifique un nombre de dominio completo (FQDN) para el grupo de servidores de mediación. Si va a crear un grupo de servidores de mediación, deberá especificar el nombre del grupo de servidores de mediación que va a implementar. Definirá los FQDN de los servidores individuales en otro paso. Si implementa un único servidor de mediación, el FQDN que especifique será el FQDN del único servidor de mediación. Para obtener más información, consulte [implementar un servidor de mediación en el generador de topología en Skype empresarial Server](../../../deploy/deploy-enterprise-voice/deploy-a-mediation-server.md).
