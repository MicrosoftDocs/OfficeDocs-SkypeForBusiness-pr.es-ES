# <a name="associate-front-end-with-office-web-apps-server-page"></a>Página Asociar front-end con servidor Office Web Apps

El Office Web Apps Server proporciona servicios similares a un moderador que carga un documento para usarlo en una reunión, pero habilita características adicionales que las versiones anteriores no lo hacían.

Seleccione un servidor Office Web Apps server para su implementación si ha elegido conferencia como una de las opciones de características para la configuración Skype Empresarial Server usuario.

Las opciones que se incluyen en este cuadro de diálogo son:

1. **Asociar grupo con un Office Web Apps Server:** active esta casilla si va a proporcionar e implementar un servidor Office web apps para los usuarios de conferencia.

2. Si ha activado esta casilla, use la lista para seleccionar un Servidor de Office Web Apps ya definido.

3. Haga clic en **Nuevo** para definir un Servidor de Office Web Apps nuevo.
