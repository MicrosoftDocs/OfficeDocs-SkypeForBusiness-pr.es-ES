# <a name="associate-front-end-with-office-web-apps-server-page"></a>Página Asociar front-end con servidor Office Web Apps

Office Web Apps Server proporciona servicios similares a los de un moderador que carga un documento para usarlo en una reunión, pero habilita características adicionales que no hacían las versiones anteriores.

Seleccione un servidor de Office Web Apps para su implementación si ha elegido conferencia como una de las opciones de características para la configuración de Skype Empresarial Server.

Las opciones que se incluyen en este cuadro de diálogo son:

1. **Asociar un grupo con un servidor de Office Web Apps:** active esta casilla si va a proporcionar e implementar un servidor de Office Web Apps para los usuarios de conferencia.

2. Si ha activado esta casilla, use la lista para seleccionar un Servidor de Office Web Apps ya definido.

3. Haga clic en **Nuevo** para definir un Servidor de Office Web Apps nuevo.
