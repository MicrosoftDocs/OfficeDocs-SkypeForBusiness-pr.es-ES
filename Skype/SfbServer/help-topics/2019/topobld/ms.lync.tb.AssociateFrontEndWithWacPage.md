# <a name="associate-front-end-with-office-web-apps-server-page"></a>Página Asociar front-end con servidor de Office Web Apps

El servidor de Office Web Apps proporciona servicios similares a los de un presentador que carga un documento para usarlo en una reunión, pero habilita otras características que las versiones anteriores no habilitaban.

Seleccione un servidor de Office Web Apps para su implementación si ha elegido las conferencias como una de sus opciones de características para la configuración de Skype empresarial Server.

Las opciones que se incluyen en este cuadro de diálogo son:

1. **Asociar grupo con un servidor de Office Web Apps** : Active esta casilla si va a proporcionar e implementar un servidor de Office Web Apps para sus usuarios de conferencia.

2. Si ha activado esta casilla, use la lista para seleccionar un servidor de Office Web Apps ya definido.

3. Haga clic en **Nuevo** para definir un servidor de Office Web Apps nuevo.
