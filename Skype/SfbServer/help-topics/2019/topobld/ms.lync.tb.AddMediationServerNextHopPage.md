# <a name="add-mediation-server-nexthop"></a>Agregar próximo salto de servidor de mediación
 
Para definir el próximo salto de un servidor de mediación que está definiendo, seleccione de la lista desplegable **grupo de próximos saltos** el servidor Enterprise Edition o el servidor Standard Edition que actuará como próximo salto para el servidor de mediación. Haga clic en **Siguiente** para continuar.
