# <a name="add-mediation-server-nexthop"></a>Agregar próximo salto de servidor de mediación
 
Para definir el próximo salto para un servidor de mediación que se está definiendo, seleccione de la lista desplegable **grupo del próximo salto** del servidor Enterprise Edition o servidor Standard Edition que van a actuar como el próximo salto para el servidor de mediación. Haga clic en **Siguiente** para continuar.
