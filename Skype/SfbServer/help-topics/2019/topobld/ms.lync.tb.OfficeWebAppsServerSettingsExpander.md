<span data-ttu-id="2f122-105">Para editar las propiedades de Office Web Apps Server configurada.</span><span class="sxs-lookup"><span data-stu-id="2f122-105">You edit the properties of the configured Office Web Apps Server.</span></span> <span data-ttu-id="2f122-106">Pueden modificarse las siguientes propiedades:</span><span class="sxs-lookup"><span data-stu-id="2f122-106">The following properties are available to edit:</span></span>
