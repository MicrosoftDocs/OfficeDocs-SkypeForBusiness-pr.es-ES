# <a name="add-front-end-sql-store-page-for-lync-server-2010"></a>Agregar página de almacén de Sql de Front-End de Lync Server 2010
 
Definir las propiedades de los almacenes de SQL Server para el nuevo grupo de servidores Front-End que va a configurar.
  
> [!IMPORTANT]
> Si va a configurar un servidor Standard Edition, se mostrará este cuadro de diálogo, pero no podrá configurar las opciones. Haga clic en **Siguiente** para continuar.
  
Configuración que se puede configurar:
  
- **Usar un almacén de SQL Server definido previamente**: seleccione un almacén de SQL Server existente en la lista para el almacén del grupo de servidores Front-End.
    
- Seleccione el **almacén de definir un nuevo SQL Server** para crear un nuevo almacén de SQL Server.
    
    Escriba el nombre de dominio completo del servidor SQL en **FQDN de SQL Server**
    
    La **instancia predeterminada** está seleccionada inicialmente. Si lo prefiere definir una instancia, seleccione **instancia con nombre** y escriba el nombre de la instancia que se debe crear.
    
 **Atrás**: vuelve al cuadro de diálogo anterior
  
 **Siguiente**: permite confirmar los cambios y se mueve al siguiente cuadro de diálogo de paso
  
 **Cancelar**: descarta los cambios y cancela la configuración
  
 **Ayuda**: muestra información, como por ejemplo en este tema de ayuda
