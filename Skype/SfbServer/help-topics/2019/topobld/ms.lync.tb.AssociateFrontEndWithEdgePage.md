# <a name="associate-front-end-with-edge"></a>Asociar front-end con servidor perimetral

Un grupo de servidores front-end puede tener asociado un solo servidor perimetral o grupo de servidores perimetrales. Al habilitar el acceso de usuarios externos para un sitio, puede permitir el acceso a usuarios remotos. También puede permitir el acceso a usuarios federados, que pueden ser usuarios de determinados proveedores de conectividad para mensajería instantánea (como Windows Live), y a usuarios anónimos.

Todos los grupos de servidores de un sitio y los grupos de servidores de varios sitios centrales pueden usar el mismo servidor perimetral, siempre que el uso no sobrepase la capacidad del servidor perimetral. Para obtener información detallada acerca de la supervisión, incluidos el ajuste de escala, consulte [Planning for acceso de usuarios externos](https://technet.microsoft.com/library/ea098933-eff5-461e-aba3-e7f128784dc2.aspx) en la documentación de planeación. Para obtener información detallada acerca del diseño de una topología para admitir el acceso de usuarios externos, consulte [Define Your Edge Topology](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx) en la documentación de implementación.
