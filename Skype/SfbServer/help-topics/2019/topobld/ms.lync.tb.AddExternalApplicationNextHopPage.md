# <a name="add-nexthop-trusted-application-server"></a>Agregar servidor de aplicaciones de confianza de próximo salto
 
Para establecer el próximo salto para el servidor de aplicaciones de confianza o un grupo de servidores, haga lo siguiente:
  
En la lista desplegable **grupo de servidores del próximo salto** , seleccione el grupo de servidores Enterprise Edition o Standard Edition servidor Front-End que desea que el grupo de aplicaciones de confianza que se utilizará como próximo salto.
  
Haga clic en **Finalizar** para completar la creación del grupo de servidores de aplicaciones de confianza.
