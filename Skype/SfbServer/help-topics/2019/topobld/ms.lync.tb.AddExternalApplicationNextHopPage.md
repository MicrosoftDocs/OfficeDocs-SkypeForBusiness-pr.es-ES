# <a name="add-nexthop-trusted-application-server"></a>Agregar servidor de aplicaciones de confianza de próximo salto
 
Para definir el próximo salto del servidor o grupo de servidores de aplicaciones de confianza, efectúe los pasos siguientes:
  
En la lista desplegable **Grupo de servidores del próximo salto**, seleccione el grupo de servidores Enterprise Edition o el servidor front-end de Standard Edition que el grupo de servidores de aplicaciones de confianza usará como próximo salto.
  
Haga clic en **Finalizar** para completar la creación del grupo de servidores de aplicaciones de confianza.
