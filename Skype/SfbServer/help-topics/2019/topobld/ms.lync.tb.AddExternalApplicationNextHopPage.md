# <a name="add-nexthop-trusted-application-server"></a>Agregar servidor de aplicaciones de confianza de próximo salto
 
Para establecer el próximo salto del servidor o grupo de aplicaciones de confianza, haga lo siguiente:
  
En la lista desplegable **grupo de próximos saltos** , seleccione el grupo de servidores Enterprise Edition o el servidor front-end Standard Edition que quiere que el grupo de aplicaciones de confianza use como próximo salto.
  
Haga clic en **Finalizar** para completar la creación del grupo de aplicaciones de confianza.
