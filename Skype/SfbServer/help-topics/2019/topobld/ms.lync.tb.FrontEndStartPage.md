# <a name="front-end-start-page"></a>Página de inicio del front-end

Use este asistente para definir un nuevo grupo de servidores Front-End para un sitio. Cada sitio debe tener al menos un servidor Standard Edition o grupo de servidores Enterprise Edition. Para obtener información detallada sobre los requisitos para usar al Asistente para crear un nuevo grupo de servidores Front-End, consulte [definir y configurar un grupo de servidores Front-End](https://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx) en la documentación de implementación.
