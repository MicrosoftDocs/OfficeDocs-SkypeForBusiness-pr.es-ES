# <a name="add-edge-machine-external-ip-2010"></a>Agregar IP externa del equipo perimetral 2010
 
Puede configurar las propiedades de esta página para apuntar el servidor perimetral a las ubicaciones externas del servicio perimetral de acceso, servicio perimetral de conferencia Web y / servicio perimetral A/v.
  
- En **direcciones IPv4 externas**, escriba las direcciones IPv4 externas del servidor perimetral en el **servicio de servidor perimetral de acceso**, **servicio perimetral de conferencia Web**, y **A / servicio perimetral A/v**.
    
- Haga clic en **Aceptar** para guardar y confirmar los cambios en el documento de topología.
    
- Haga clic en **Cancelar** para descartar los cambios y cerrar al Asistente para **Agregar servidor al grupo de servidores perimetrales** .
    
- Haga clic en **Ayuda** para leer este tema de ayuda.
    
Para obtener información detallada sobre la adición de servidores perimetrales a una topología existente, vea [Define Your Edge Topology](http://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx). Para obtener información acerca de la migración de servidores perimetrales, consulte [fase 5: agregar Lync Server 2013 Preview servidor perimetral a un grupo piloto de](http://technet.microsoft.com/library/d2fd0a2f-8fec-4f86-a5b0-7f7d03172016.aspx).
