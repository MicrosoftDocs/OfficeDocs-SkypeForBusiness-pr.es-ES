# <a name="associate-av-mcu-with-front-end"></a>Asociar MCU AV con Front-End
 
Todos los grupos de servidores Front-End de Enterprise Edition con la conferencia habilitado deben tener un servicio de conferencia A/v combinado, o debe usar independiente o grupo de servidores de conferencia A/v. Cualquier existente Enterprise Edition Front-End de grupos que disponen actualmente de un / conferencia A/v servicio combina puede seleccionarse para usar este nuevo independiente A grupo de conferencia A/v en su lugar.
