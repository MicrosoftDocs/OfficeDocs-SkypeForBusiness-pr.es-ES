# <a name="add-edge-server-internal-ip"></a>Agregar IP interna del servidor perimetral

La dirección IP interna es la dirección IP de la interfaz interna del servidor perimetral.

Puede añadir compatibilidad para el acceso de usuarios externos al implementar la topología inicial o más adelante. Para obtener más información sobre cómo agregar servidores perimetrales a una topología ya creada, consulte [Define Your Edge Topology](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx) en la documentación sobre implementación de servidores perimetrales.
