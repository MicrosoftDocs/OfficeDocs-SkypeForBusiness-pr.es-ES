# <a name="legacy-merge-next-hop"></a>Próximo salto de combinación heredada
 
**Número de dominio completo del próximo salto** identifica el servidor del próximo salto al que el servidor perimetral heredado enruta el tráfico interno. Escriba el nombre de dominio completo (FQDN) del servidor del próximo salto al que este servidor perimetral enruta el tráfico interno y, a continuación, seleccione si es un grupo de servidores **front-end** o un **director.**
