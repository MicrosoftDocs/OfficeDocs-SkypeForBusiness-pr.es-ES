# <a name="skype-for-business-server-create-new-expander"></a>Crear nuevo expansor de Skype empresarial Server
 
Puede crear nuevos roles de servidor en el generador de topología haciendo clic con el botón secundario en **Skype empresarial Server**, seleccionando **nuevo**y seleccionando el rol de servidor que desea crear.
