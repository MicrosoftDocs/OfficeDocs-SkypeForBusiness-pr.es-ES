# <a name="add-file-store"></a>Agregar almacén de archivos
 
Para definir un nuevo almacén de archivos, especifique lo siguiente:
  
- El nombre de dominio completo (FQDN) del servidor que se va a hospedar el nuevo almacén de archivos. El FQDN del servidor debe existir en el sistema de nombres de dominio (DNS) antes de publicar la topología.
    
- El nombre de recurso compartido de archivos en el servidor host. El recurso compartido de archivos debe existir antes de publicar la topología.
