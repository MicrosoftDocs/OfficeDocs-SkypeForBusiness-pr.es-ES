# <a name="external-application-scope-node"></a>Nodo de ámbito de aplicación externa
 
Este nodo muestra el nombre de dominio completo (FQDN) del grupo de servidores, la opción de replicación y el grupo de servidores del próximo salto para este grupo de servidores de aplicaciones de confianza. Puede realizar las acciones siguientes:
  
- **Nuevo servidor**. Inicia el asistente **Definir nuevo servidor**, que puede utilizar para agregar a otro servidor en este grupo de servidores de aplicaciones de confianza.
    
- **Editar propiedades**. Abre la página **Editar propiedades** para este servidor; en ella, puede cambiar el puerto y nombre de dominio completo del grupo de servidores externos, habilitar y deshabilitar la federación y el traductor de direcciones de red, y especificar el próximo salto.
    
- **Eliminar**. Elimina el grupo de servidores de la topología.
    
Además de estas opciones, puede realizar las tareas generales siguientes:
  
- **Topología**. Puede usar esta opción para definir una topología nueva, abrir una topología existente, descargar la topología actual, guardar una copia de la topología, publicar la topología, instalar una base de datos o quitar la implementación.
    
- **Ver**. Use esta opción para personalizar la vista del Generador de topologías seleccionando los elementos que se deben mostrar y ocultar en las vistas de árbol de consola.
    
- **Ayuda**. Use esta opción para obtener acceso a la Ayuda del nodo activo.
