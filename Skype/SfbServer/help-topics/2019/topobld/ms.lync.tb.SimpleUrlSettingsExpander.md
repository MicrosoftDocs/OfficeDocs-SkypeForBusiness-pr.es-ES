# <a name="simple-url-settings-expander"></a>Expansor de configuración de dirección URL simple

Para especificar nuevas, quitar las existentes o cambiar las direcciones URL simples, haga lo siguiente:

## <a name="simple-urls"></a>Direcciones URL simples

Para modificar o definir las direcciones URL simples, primero debe decidir cuál de las tres direcciones URL simples desea modificar o cambiar. Puede elegir entre la dirección URL de acceso telefónico, la dirección URL de la reunión y la dirección URL de acceso administrativo.

Para modificar la dirección URL de acceso del teléfono o la dirección URL de la reunión, seleccione la dirección URL que necesita cambiar. Haga clic en **Editar URL**. Edite la dirección URL y haga clic en **Aceptar** para guardar la dirección URL. Haga clic en **Cancelar** para descartar los cambios.

Para agregar una nueva dirección URL, haga clic en **Agregar**. En el cuadro de diálogo **Agregar dirección URL simple** , especifique la dirección URL y, a continuación, haga clic en **Aceptar** para guardar la dirección URL. Seleccione **Convertir esta dirección en la URL activa para el dominio seleccionado** si quiere configurar la nueva dirección URL como dirección URL activa. Haga clic en **Cancelar** para descartar los cambios.

Para convertir una dirección URL diferente en la dirección URL activa (como se indica en la marca de verificación verde junto a la dirección URL), seleccione la dirección URL y haga clic en **Activar**.

> [!NOTE]
> Solo puede haber una dirección URL activa para cada dominio del Protocolo de inicio de sesión (SIP).

Para quitar una dirección URL, selecciónela y haga clic en **quitar**.

> [!CAUTION]
> Asegúrese de leer la información de la página de diálogo Configuración de direcciones URL simples. Quitar una dirección URL de reunión puede representar las reuniones existentes programadas por los usuarios que ya no tienen acceso. Considere la posibilidad de dejar la URL existente después de activar la nueva dirección URL de la reunión. Cuando esté seguro de que los usuarios ya no usan la dirección URL de la reunión anterior, puede quitarla de forma segura.

Para editar o cambiar la dirección URL de acceso administrativo, edite la entrada.

Cuando haya terminado con todos los cambios en la página Editar propiedades, haga clic en **Aceptar** para guardar los cambios. Haga clic en **Cancelar** para descartar los cambios.
