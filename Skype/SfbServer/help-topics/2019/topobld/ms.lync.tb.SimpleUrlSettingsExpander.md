# <a name="simple-url-settings-expander"></a>Expansor de configuración de dirección URL simple

Para especificar nuevas, quitar las ya existentes, o para cambiar las direcciones URL sencillas de la activa, haga lo siguiente:

## <a name="simple-urls"></a>Direcciones URL simples

Para modificar o definir las direcciones URL sencillas, primero debe decidir cuál de las tres direcciones URL sencillas que desee editar o cambiar. Puede elegir entre la dirección URL de acceso telefónico, la dirección URL de la reunión y la dirección URL de acceso administrativo.

Para modificar la dirección URL de acceso de teléfono o la dirección URL de la reunión, seleccione la dirección URL que se debe cambiar. Haga clic en **Editar dirección URL**. Editar la dirección URL y haga clic en **Aceptar** para guardar la dirección URL. Haga clic en **Cancelar** para descartar los cambios.

Para agregar una nueva dirección URL, haga clic en **Agregar**. En el cuadro de diálogo **dirección URL sencilla de agregar** , especifique la dirección URL y, a continuación, haga clic en **Aceptar** para guardar la dirección URL. Seleccione **Convertir esta dirección en la URL activa para el dominio seleccionado** si quiere configurar la nueva dirección URL como dirección URL activa. Haga clic en **Cancelar** para descartar los cambios.

Para hacer que la dirección URL activa (lo que se indica mediante la marca de verificación verde junto a la dirección URL) de una dirección URL diferente, seleccione la dirección URL y haga clic en **Convertir en activa**.

> [!NOTE]
> Sólo puede haber una dirección URL activa para cada dominio de protocolo de inicio de sesión (SIP).

Para quitar una dirección URL, seleccione la dirección URL y haga clic en **Quitar**.

> [!CAUTION]
> Asegúrese de leer la información en la página de cuadro de diálogo de configuración de las direcciones URL simple. Eliminación de una dirección URL de la reunión puede representar las reuniones existentes programadas por los usuarios ya no se puede tener acceso. Considere la posibilidad de salir de la dirección URL existente después de realizar la nueva dirección URL de reunión activo. Cuando esté seguro de que los usuarios ya no están utilizando la dirección URL de reunión antigua, puede quitar sin problemas.

Para editar o cambiar la dirección URL de acceso administrativo, edite la entrada.

Una vez que haya terminado con todos los cambios en la página Editar propiedades, haga clic en **Aceptar** para guardar los cambios. Haga clic en **Cancelar** para descartar los cambios.
