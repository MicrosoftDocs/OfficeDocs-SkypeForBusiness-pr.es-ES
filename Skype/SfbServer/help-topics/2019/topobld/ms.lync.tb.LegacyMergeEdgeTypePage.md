# <a name="legacy-merge-edge-type"></a>Tipo de perímetro combinación de información heredada
 
Para un **servidor perimetral único** o un servidor **perimetral consolidado con equilibrio de carga** , haga clic en **siguiente** para especificar la información de configuración. Para un **borde más amplio**, la configuración de cada servidor debe cambiarse a una configuración de límite consolidada antes de que se pueda fusionar.
