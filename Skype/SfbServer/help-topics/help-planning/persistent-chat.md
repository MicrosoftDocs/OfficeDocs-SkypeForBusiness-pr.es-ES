# <a name="persistent-chat-planning-tool"></a>Persistent Chat (Planning Tool)
 
El servidor de chat persistente permite a los usuarios participar en conversaciones de varias partes y basadas en temas que persisten a lo largo del tiempo.
  
Para obtener más información sobre cómo preparar el entorno para el servidor de chat persistente, consulte [planear el servidor de chat persistente en Skype empresarial server 2015](../../plan-your-deployment/persistent-chat-server/persistent-chat-server.md).
