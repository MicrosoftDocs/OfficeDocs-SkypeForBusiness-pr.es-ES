# <a name="media-bypass-planning-tool"></a>Omisión de medios (Herramienta de planeación)
 
El desvío de medios consiste en quitar el servidor de mediación de la ruta de acceso a medios siempre que sea posible, en las llamadas cuya señalización atraviese el servidor de mediación. Para obtener más información, consulte [Plan for media bypass in Skype for Business 2015](../../plan-your-deployment/enterprise-voice-solution/media-bypass.md).
