# <a name="dial-in-conferencing-planning-tool"></a>Dial-in Conferencing (Planning Tool)
 
Las conferencias de acceso telefónico local de Skype empresarial Server 2015 permiten a los usuarios unirse a la parte de audio de una conferencia de Skype empresarial Server 2015 con un teléfono de red telefónica conmutada (RTC) sin necesidad de usar un proveedor de servicios de audioconferencia de terceros.
  
Para obtener información sobre la planeación de la habilitación de la conferencia de acceso telefónico en la organización, consulte el siguiente tema y los temas a los que se vincula: 
  
- [Plan for conferencing in Skype for Business Server 2015](../../plan-your-deployment/conferencing/conferencing.md)
