# <a name="ip-version-planning-tool"></a>IP Version (Planning Tool)
 
Tiene tres opciones para configurar las direcciones IP en Skype empresarial Server 2015. Puede configurar Skype empresarial Server 2015 para que admita solo IP versión 4 (IPv4), solo IP versión 6 (IPv6) o una combinación de ambos (conocido como una pila doble).
  
Para obtener más información sobre cómo elegir la infraestructura IP, consulte [planear IPv6 en Skype empresarial](../../plan-your-deployment/network-requirements/ipv6.md).
