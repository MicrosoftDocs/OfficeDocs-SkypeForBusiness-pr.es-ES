# <a name="federation-planning-tool"></a>Federation (Planning Tool)
 
Los servidores perimetrales se pueden configurar para permitir a los usuarios externos y internos el acceso a los contactos de las organizaciones o servicios asociados.
  
 La Federación funciona de la misma manera que en Microsoft Lync 2013. Para obtener más detalles sobre la preparación del entorno para la federación, consulte los siguientes temas:
  
- [Planificación de la Federación de Lync Server 2013 y Office Communications Server](https://technet.microsoft.com/en-us/library/jj205335%28v=ocs.15%29.aspx)
    
- [Planeamiento de la conectividad de mensajería instantánea pública en Lync Server 2013](https://technet.microsoft.com/en-us/library/jj205349%28v=ocs.15%29.aspx)
    
- [Planeamiento de la Federación de protocolo de presencia y mensajería extensible (XMPP) en Lync Server 2013](https://technet.microsoft.com/en-us/library/jj205107%28v=ocs.15%29.aspx)
