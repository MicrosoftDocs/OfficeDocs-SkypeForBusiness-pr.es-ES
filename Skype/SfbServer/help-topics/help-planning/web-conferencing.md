# <a name="web-conferencing-planning-tool"></a>Conferencias Web (herramienta de planeación)
 
Skype para conferencias web de Business Server 2015 incluye la colaboración en documentos, compartir aplicaciones y uso compartido de escritorio.
  
> [!NOTE]
> Skype para Business Server 2015 utiliza Office Web Apps y el servidor de Office Web Apps para controlar el uso compartido y el procesamiento de las presentaciones de PowerPoint. Características de conferencias Web funcionan exactamente igual como lo hacen en Microsoft Lync 2013. Para obtener más información sobre cómo instalar y configurar el servidor de Office Web Apps, vea [Configurar la integración con Office Web Apps Server y Lync Server 2013](https://technet.microsoft.com/en-us/library/jj204792%28v=ocs.15%29.aspx).
