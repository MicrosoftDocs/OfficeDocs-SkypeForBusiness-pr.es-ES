# <a name="certificate-assignment-invoke"></a>Asignación de certificados (Invoke)
 
La página **Ejecutar comandos** muestra las acciones realizadas por las tareas para asignar el certificado al servidor. Cuando se completan las tareas, puede ver el registro que se detalla el proceso de asignación de certificado. Para visualizar el archivo de registro, haga clic en **Ver registro**.
