# <a name="pending-certificate-intro"></a>Certificado pendiente (Intro)
 
Para procesar una solicitud de certificado en línea pendiente, es decir, una solicitud que se ha emitido a la autoridad de certificación en línea pero administrativamente está establecida para requerir un proceso de aprobación manual, haga clic en **siguiente**.
