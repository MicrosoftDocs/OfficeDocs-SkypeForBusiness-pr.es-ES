# <a name="setup-or-remove-skype-for-business-server-components-intro"></a>Instalar o quitar componentes de Skype Empresarial Server (introducción)
 
Se están preparando para instalar y activar o desactivar y desinstalar Skype para los componentes de servidor de negocios 2015. Los procesos que debe ejecutar dependen de la configuración actual de la definición publicada en el almacén de administración central. Para realizar correctamente las tareas, debe iniciar sesión como miembro del grupo de administradores locales y, como mínimo, debe ser capaz de leer los servicios de dominio de Active Directory.
  
Haga clic en **Siguiente** para continuar con la pantalla **Ejecución de comandos**.
