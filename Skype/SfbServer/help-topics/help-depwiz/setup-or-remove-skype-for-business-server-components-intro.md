# <a name="setup-or-remove-skype-for-business-server-components-intro"></a>Instalar o quitar componentes de Skype Empresarial Server (introducción)
 
Se está preparando para instalar y activar o desactivar y desinstalar los componentes de Skype empresarial Server 2015. Los procesos que debe ejecutar dependen de la configuración actual de la definición publicada en el almacén de administración central. Para completar las tareas correctamente, debe haber iniciado sesión como miembro del grupo de administradores locales y, como mínimo, debe poder leer los servicios de dominio de Active Directory.
  
Haga clic en **Siguiente** para continuar con la pantalla **Ejecución de comandos**.
