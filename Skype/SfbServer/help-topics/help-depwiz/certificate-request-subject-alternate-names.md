# <a name="certificate-request-subject-alternate-names"></a>Solicitud de certificado (nombres alternativos de asunto)
 
El **nombre de asunto o nombres alternativos del sujeto** página presenta un resumen del nombre del sujeto y todos los nombres alternativos de asunto que llenará el certificado. Los nombres que se presentan para el nombre del sujeto y los nombres alternativos del sujeto se recopilan desde el documento de definición, ya que se creó cuando se define la implementación de generador de topología.
  
Si debe realizar cambios en el nombre del sujeto o cualquiera de los nombres alternativos del sujeto, haga clic en **Cancelar** para salir del Asistente para solicitud de certificado y vuelva a ejecutar el generador de topología. A continuación, volver a publicar la nueva configuración con el asunto definido correctamente y nombres alternativos del sujeto.
