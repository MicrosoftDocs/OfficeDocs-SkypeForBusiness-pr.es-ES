# <a name="import-certificate-invoke"></a>Importar certificado (Invoke)
 
En la página **Ejecutar comandos** , revise la información de resumen para el certificado importado. En la lista desplegable, seleccione el archivo de registro de certificado de importación. Asegúrese de que el archivo de registro de certificado de importación está visible en la lista desplegable. Haga clic en **Ver registro** para revisar los detalles de la importación del certificado. Cuando haya terminado de revisar el resumen y el archivo de registro, haga clic en **Finalizar**.
