# <a name="startservices-intro"></a>StartServices (introducción)
 
Para iniciar servicios en el Skype para servidor de rol Business Server 2015 que va a implementar, revisar la pantalla de introducción. Observe que los comandos emitidos para iniciar el Skype para Business Server espera para que iniciar el servicio. Es el modo más eficaz de iniciar los servicios; se recomienda comprobar el estado actual de los servicios usando Windows Services Microsoft Management Console (MMC). Haga clic en **Siguiente** para comenzar la tarea **Iniciar servicios**.
