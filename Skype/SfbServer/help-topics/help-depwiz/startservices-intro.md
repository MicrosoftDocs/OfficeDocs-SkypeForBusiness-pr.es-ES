# <a name="startservices-intro"></a>StartServices (introducción)
 
Para iniciar servicios en el Skype para servidor de funciones de Business Server 2015 que va a implementar, revise la pantalla de introducción. Tenga en cuenta que los comandos emitidos para iniciar la Skype para Business Server no espere a que el servicio se inicie. Es el modo más eficaz de iniciar los servicios; se recomienda comprobar el estado actual de los servicios usando Windows Services Microsoft Management Console (MMC). Haga clic en **Siguiente** para comenzar la tarea **Iniciar servicios**.
