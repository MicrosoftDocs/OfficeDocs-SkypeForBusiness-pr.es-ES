# <a name="startservices-intro"></a>StartServices (introducción)
 
Para iniciar servicios en el servidor de roles de Skype Empresarial Server 2015 que está implementando, revise la pantalla de introducción. Observe que los comandos emitidos para iniciar Skype Empresarial Server no esperan a que se inicie el servicio. Es el modo más eficaz de iniciar los servicios; se recomienda comprobar el estado actual de los servicios usando Windows Services Microsoft Management Console (MMC). Haga clic en **Siguiente** para comenzar la tarea **Iniciar servicios**.
