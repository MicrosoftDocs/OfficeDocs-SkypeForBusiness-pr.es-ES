# <a name="certificate-request-invoke"></a>Solicitud de certificado (Invoke)
 
La página **Comandos de ejecución** muestra un panel de estado de tareas como los comandos ejecutar para la generación de la solicitud de certificado. Una vez completadas las tareas, la lista desplegable situada debajo del panel Resumen contiene una lista de archivos de registro. Utilice la lista desplegable para seleccionar un archivo de registro para ver y, a continuación, haga clic en **Ver registro**. Internet Explorer iniciará y mostrar el archivo de registro. En el caso de un error en las peticiones, debe revisar los archivos de registro para determinar la causa del error. Puede ejecutar de nuevo la solicitud de certificado para resolver el problema, corrija el problema indicado en el archivo de registro. Continuar aunque las páginas de solicitud de certificado y volver a solicitar el certificado.
