Use la lista desplegable para seleccionar un archivo de registro para ver y, a continuación, haga clic en Ver registro. Internet Explorer se iniciará y mostrar el archivo de registro. En el caso de una solicitud con errores, debe revisar los archivos de registro para determinar la causa del error. Puede volver a ejecutar la solicitud de certificado para resolver el problema, corregir el problema que se indica en el archivo de registro. Continuar aunque las páginas de solicitud de certificado y volver a solicitar el certificado.
