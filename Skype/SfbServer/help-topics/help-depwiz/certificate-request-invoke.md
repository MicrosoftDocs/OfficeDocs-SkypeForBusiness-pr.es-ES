# <a name="certificate-request-invoke"></a>Solicitud de certificados (invocar)
 
La página **Ejecutando comandos** muestra un panel con el estado de la tarea a medida que los comandos se ejecutan para generar la solicitud de certificado. Una vez finalizadas las tareas, la lista desplegable que hay debajo del panel de resumen muestra una lista con archivos de registro. En la lista desplegable, seleccione el archivo de registro que desee ver y haga clic en **Ver registro**. Internet Explorer se iniciará y mostrará en pantalla el archivo de registro. Si no se puede atender una solicitud, examine el archivo o los archivos de registro para determinar el motivo del error. Ejecute de nuevo la solicitud de certificado para resolver el problema y corregir el error que figuraba en el archivo de registro. Vaya pasando por las páginas de solicitud de certificado y vuelva a solicitarlo.
