Una vez completadas las tareas, la lista desplegable situada bajo el panel Resumen contiene una lista de archivos de registro. Use la lista desplegable para seleccionar un archivo de registro que desee ver y, a continuación, haga clic en Ver registro. Internet Explorer se iniciará y mostrará el archivo de registro. En caso de que se produzca un error en una solicitud, debe revisar los archivos de registro para determinar la causa del error. Puede volver a ejecutar la solicitud de certificado para resolver el problema, corrigiendo el problema indicado en el archivo de registro. Continúe con las páginas de solicitud de certificado y vuelva a solicitar el certificado.
