# <a name="certificate-request-certificate-signing-request-generated"></a>Solicitud de certificado (certificado de firma de solicitud generada)
 
Después de haber completado una solicitud para un certificado sin conexión, se abrirá la página de **Archivo de solicitud de certificado** . Para mostrar la solicitud de certificado y prepararlo para su envío a la autoridad de certificación sin conexión (CA) o una CA pública, haga clic en el botón **vista** . La solicitud de certificado se muestra en el Bloc de notas de Windows (de forma predeterminada). Puede copiar y pegar la solicitud de certificado codificado de esta pantalla o finalizar el procesamiento de la solicitud de certificado siguiendo las instrucciones de su elegido entidad emisora sin conexión o la entidad emisora de certificados pública. Haga clic en **Finalizar** para continuar.
