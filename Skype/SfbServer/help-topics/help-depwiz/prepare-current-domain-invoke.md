# <a name="prepare-current-domain-invoke"></a>Preparar el dominio actual (Invoke)
 
La página **Comandos de ejecución** muestra un resumen de tareas mientras se ejecutan. Revise la pantalla Resumen y asegúrese de que las tareas tuvieron éxito y que no hay ningún error. Revisar las advertencias para asegurarse de que se esperan. A continuación el panel Resumen, debería ver **estado de la tarea: completar**. En la lista desplegable, el archivo de registro **Preparar dominio** debe ser visible. Si no es así, seleccione el archivo de registro y, a continuación, haga clic en **Ver registro** para revisar las acciones detalladas y el resultado de las tareas. Haga clic en **Finalizar** para completar la preparación del dominio.
