# <a name="prepare-current-forest-universal-group-location"></a>Preparar el bosque actual (ubicación del grupo Universal)
 
Debe indicar que necesita crear los grupos universales en el bosque. La página **Ubicación del grupo Universal** utiliza el dominio del que el equipo que ha iniciado sesión en un miembro de forma predeterminada. Si prefiere crear los grupos universales en un dominio distinto del **dominio Local**predeterminado, seleccione el **FQDN del dominio** y escriba el nombre de dominio completo (FQDN) en el cuadro de texto. Crear un dominio alternativo es útil cuando se ejecuta la tarea de preparación del bosque de un dominio distinto de donde desea que los grupos universales. Haga clic en **Siguiente** para continuar.
