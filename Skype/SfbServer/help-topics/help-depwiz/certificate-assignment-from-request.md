# <a name="certificate-assignment-from-request"></a>Asignación de certificados (a partir de solicitud)
 
La **página Asignación de** certificados inicia el proceso de asignación de un certificado a los usos solicitados en el servidor de roles. Haga clic en **Ver detalles del certificado** para ver el certificado. Puede seleccionar la pestaña **General,** la **pestaña Detalles** o la pestaña **Ruta de** certificación. La **pestaña General** mostrará información básica sobre el certificado, como la entidad de certificación emisora y el estado de una clave privada. La pestaña Detalles mostrará información del certificado, como el nombre del sujeto, los nombres alternativos del sujeto y otros detalles. La **pestaña Ruta de** certificación mostrará la "cadena de confianza" del certificado a través de cualquier entidad emisora o de certificación intermedia (si están presentes) a la entidad de certificación raíz.
  
Haga clic en **Siguiente** para continuar.
