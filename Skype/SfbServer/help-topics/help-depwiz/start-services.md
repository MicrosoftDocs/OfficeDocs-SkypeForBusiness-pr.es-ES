# <a name="start-services"></a>Iniciar servicios
 
Para iniciar los servicios en el rol de servidor implementado, haga clic en **Ejecutar** en **paso 4: iniciar servicios**. Debe iniciar sesión como miembro del grupo local Administradores o equivalente. 
  
> [!NOTE]
> Si se ha ejecutado **Iniciar servicios** , puede volver a ejecutar el comando haciendo clic en **Ejecutar de nuevo**.
