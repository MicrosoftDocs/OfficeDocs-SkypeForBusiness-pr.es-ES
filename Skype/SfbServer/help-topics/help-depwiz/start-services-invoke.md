# <a name="start-services-invoke"></a>Iniciar servicios (invocar)
 
El panel de resumen del panel **comandos en ejecución** muestra el estado de las tareas emitidas para iniciar los servicios del servidor de rol de Skype empresarial Server 2015 que está implementando. El resumen del panel de tareas no representa ninguna indicación en tiempo real de inicio del servicio. Algunos de los servicios de Skype empresarial pueden tardar mucho tiempo en comenzar el proceso de inicio inicial. Las tareas indican al comando que se inicie, pero no esperan para determinar si el servicio se ha iniciado correctamente. Si debe supervisar el inicio y el estado de los servicios, se recomienda usar Windows Services Microsoft Management Console (MMC).
  
Debajo del panel de tareas, hay una lista desplegable que muestra el archivo de registro **Iniciar servicios**. Para visualizar el archivo de registro, haga clic en **Ver registro**. Haga clic en **Finalizar** para completar la tarea.
