# <a name="install-or-update-skype-for-business-server-system"></a>Instalar o actualizar el sistema Skype Empresarial Server
 
Para comenzar la instalación o actualización de una implementación existente de Skype para Business Server 2015, haga clic en **instalar o actualizar Skype para Business Server**. Los pasos descritos en esta sección instalación componentes específicos a un rol de servidor determinado, basado en el documento de generador de topología. Antes de comenzar esta serie de pasos, debe definir la implementación de generador de topología. Debe estar iniciado en el equipo que va a implementar como administrador local y como miembro del grupo Administradores del dominio en el dominio donde se encuentra el servidor. El equipo debe estar unido al dominio antes de completar este paso.
  
Si está actualizando un servidor implementado actualmente, debe realizar los cambios que desea realizar en el generador de topología.
