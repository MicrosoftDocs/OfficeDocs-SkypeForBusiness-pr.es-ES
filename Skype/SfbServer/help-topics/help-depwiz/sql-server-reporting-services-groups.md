# <a name="sql-server-reporting-services-groups"></a>SQL Server Reporting Services (grupos)

**Especificar el grupo de sólo lectura**, especifique el grupo al que desea conceder acceso de sólo lectura a los informes de supervisión. Utilice el formato &lt;dominio&gt;\\&lt;grupo de acceso de&gt;. Por ejemplo: contoso\RTCUniversalReadOnlyAdmins.
  
Asignar un grupo de acceso de sólo lectura es opcional. Si no desea conceder acceso de sólo lectura ahora, puede configurar más adelante directamente en SQL Server Reporting Services.
