# <a name="sql-server-reporting-services-credentials"></a>SQL Server Reporting Services (credenciales)
 
Para especificar las credenciales para el SQL Server Reporting Services (SSRS) para tener acceso a la base de datos de supervisión, haga lo siguiente:
  
En la **Página credenciales especificar**, especifique las credenciales del usuario que desea utilizar para tener acceso a la base de datos de supervisión (QoEMetrics y LcsCDR). 
  
> [!NOTE]
> Para obtener más información, consulte "especificar credenciales y conexión información para informe de datos orígenes (SSRS)" en [https://go.microsoft.com/fwlink/p/?LinkId=263431](https://go.microsoft.com/fwlink/p/?LinkId=263431). 
  
Para obtener más información acerca de cómo implementar los informes de Reporting Services para el servidor de supervisión, consulte [Instalar informes de supervisión en Skype para Business Server 2015](../../deploy/deploy-monitoring/install-monitoring-reports.md).
