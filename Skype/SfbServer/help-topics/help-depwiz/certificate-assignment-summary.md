# <a name="certificate-assignment-summary"></a>Asignación de certificados (resumen)
 
Antes de que finalice la asignación de certificados, se presenta un **Resumen de asignación de certificados** que se enumeran los detalles del certificado que va a asignar. Revise la información para asegurarse de que el nombre de sujeto y el nombre alternativo de sujeto es correcto. Cuando haya terminado de revisar el resumen de certificados, haga clic en **siguiente**.
