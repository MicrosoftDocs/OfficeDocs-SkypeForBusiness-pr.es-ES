# <a name="certificate-assignment-summary"></a>Asignación de certificados (resumen)
 
Antes de que finalice la asignación de certificado, se le presenta un **Resumen de asignación de certificado** que muestra los detalles del certificado que se asignará. Revise la información para asegurarse de que el nombre del sujeto y el asunto sean correctos. Cuando haya terminado de revisar el resumen del certificado, haga clic en **siguiente**.
