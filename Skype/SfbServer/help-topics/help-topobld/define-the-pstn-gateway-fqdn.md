# <a name="define-the-pstn-gateway-fqdn"></a>Definir el FQDN de puerta de enlace de RTC

Especifique el nombre de dominio completo (FQDN) de la puerta de enlace de red telefónica conmutada (RTC) que conectará los componentes de Telefonía IP empresarial a una RTC o una central de conmutación (PBX).
