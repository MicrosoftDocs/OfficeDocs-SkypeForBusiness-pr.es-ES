# <a name="legacy-merge-summary-page"></a>Página de resumen de combinación heredados
 
Migración de Microsoft Office Communications Server 2007 R2 a Lync Server 2013 utiliza características del generador de topología para combinar las topologías existentes. La pantalla de resumen especifica todas las acciones y las selecciones que ha realizado durante la configuración de la fusión de Office Communications Server 2007 R2 en la configuración de generador de topología de Lync Server 2013.
  
Revise el resumen y determinar si se han definido correctamente las selecciones. Continúe con la fusión si está satisfecho con las selecciones que se reflejan en el resumen.
