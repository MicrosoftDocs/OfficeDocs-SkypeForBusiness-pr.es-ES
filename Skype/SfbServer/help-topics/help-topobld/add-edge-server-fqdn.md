# <a name="add-edge-server-fqdn"></a>Agregar FQDN de servidor perimetral
 
Debe especificar el nombre de dominio completo para el servicio perimetral de acceso. Si no seleccionó la opción Usar una sola dirección **IP FQDN &amp;** en la página  Seleccionar características, también debe especificar un FQDN para el servicio perimetral de conferencia web y para el servicio perimetral A/V.
  
Además, si seleccionó la opción Usar una sola dirección **IP de FQDN&amp;**, debe especificar un número de puerto diferente para cada uno de los servicios perimetrales (configuración de puerto recomendada: 444 para el servicio perimetral de acceso, 8057 para el servicio perimetral de conferencia web y 443 para el servicio perimetral A/V). Si no ha seleccionado la opción, puede usar el mismo número de puerto (por ejemplo, 443) para los tres servicios.
