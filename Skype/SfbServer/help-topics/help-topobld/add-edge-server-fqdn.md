# <a name="add-edge-server-fqdn"></a>Agregue el FQDN del servidor de borde
 
Debe especificar un nombre de dominio completo (FQDN) para el servicio de servidor perimetral de acceso. Si no seleccionó la **utilizar un FQDN único &amp; dirección IP** opción en la página **Seleccionar características** , también debe especificar un FQDN para el servicio perimetral de conferencia Web y en lugar de A / servicio V perimetral.
  
Además, si ha seleccionado la **utilizar un FQDN único &amp; dirección IP** opción, debe especificar un número de puerto diferente para cada uno de los servicios de borde (puerto configuración recomendada: 444 para el servidor perimetral de acceso de servicio, 8057 para el servicio perimetral de conferencia Web y 443 para A / V Servicio de borde). Si no seleccionó la opción, puede utilizar el mismo número de puerto (por ejemplo, 443) para los tres servicios.
