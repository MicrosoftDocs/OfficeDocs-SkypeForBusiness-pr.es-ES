# <a name="monitoring-scope-node"></a>Nodo de ámbito de supervisión
 
En el generador de topología de Lync Server 2013, puede realizar tareas relacionadas con un servidor individual ejecutando supervisión pulsando el servidor ejecuta la supervisión en el árbol de consola, haga clic en **acción** en la barra de herramientas o haciendo clic en una tarea de las acciones panel y, a continuación, haga clic en una de las siguientes opciones para el servidor de supervisión:
  
- **Editar propiedades**. Abre la página **Editar propiedades** para el servidor de supervisión, donde puede cambiar el nombre de dominio completo (FQDN) del servidor que ejecuta la base de datos de SQL Server supervisión o de supervisión.
    
- **Eliminar**. Elimina el servidor de supervisión de la topología y la base de datos de SQL Server supervisión.
    
Aparte de las opciones del servidor que ejecuta la supervisión que se han descrito anteriormente en este tema, existen opciones generales del nodo de supervisión que también están disponibles desde otros nodos del árbol de la consola. Son las siguientes:
  
- **Topología**. Esta opción permite definir una topología nueva, abrir una topología existente, descargar la topología actual, guardar una copia de la topología, publicar la topología, instalar una base de datos o quitar la implementación.
    
- **Ayuda**. Esta opción permite tener acceso a la Ayuda del nodo activo.
