# <a name="add-edge-machine-external-ip-2010"></a>Agregar IP externa de equipo perimetral 2010

Puede configurar las propiedades de esta página para que apunten el servidor perimetral a las ubicaciones externas del servicio perimetral de acceso, el servicio perimetral de conferencia web y el servicio perimetral A/V.

- En **Direcciones IPv4** externas , escriba las direcciones IPv4 externas del servidor perimetral en el servicio perimetral de **acceso,** el servicio perimetral de conferencia **web** y el servicio **perimetral A/V.**

- Haga clic en **Aceptar** para guardar y confirmar los cambios en el documento de la topología.

- Haga clic en **Cancelar** para descartar los cambios y cerrar el asistente **Añadir un servidor al grupo de servidores perimetrales**.

- Haga clic en **Ayuda** para leer este tema de la Ayuda.

Para obtener más información acerca de cómo agregar servidores perimetrales a una topología existente, consulte [Definir la topología perimetral.](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx) Para obtener información acerca de la migración de servidores perimetrales, consulte Fase 5: Agregar servidor perimetral de vista previa de [Lync Server 2013 al grupo piloto.](https://technet.microsoft.com/library/d2fd0a2f-8fec-4f86-a5b0-7f7d03172016.aspx)
