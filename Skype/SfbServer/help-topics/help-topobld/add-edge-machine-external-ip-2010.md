# <a name="add-edge-machine-external-ip-2010"></a>Agregar IP externa de equipo perimetral 2010

Puede configurar las propiedades de esta página para apuntar el servidor perimetral a las ubicaciones externas del servicio perimetral de acceso, el servicio perimetral de conferencia web y el servicio perimetral a/V.

- En **direcciones IPv4 externas**, escriba las direcciones IPv4 externas del servidor perimetral en el **servicio perimetral de acceso**, el **servicio perimetral de conferencia web**y el **servicio perimetral a/V**.

- Haga clic en **Aceptar** para guardar y confirmar los cambios en el documento de la topología.

- Haga clic en **Cancelar** para descartar los cambios y cerrar el asistente **Añadir un servidor al grupo de servidores perimetrales**.

- Haga clic en **Ayuda** para leer este tema de la Ayuda.

Para obtener información detallada sobre cómo agregar servidores perimetrales a una topología existente, consulte [define Your Edge Topology](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx). Para obtener información acerca de la migración de servidores perimetrales, consulte [Phase 5: Add Lync server 2013 Preview Edge Server to Pilot Pool](https://technet.microsoft.com/library/d2fd0a2f-8fec-4f86-a5b0-7f7d03172016.aspx).
