# <a name="add-edge-machine-external-ip-2010"></a>Agregar IP externa de equipo perimetral 2010

Puede configurar las propiedades de esta página para que apunten al servidor perimetral a las ubicaciones externas del servicio perimetral de acceso, el servicio perimetral de conferencias web y el servicio perimetral a/V.

- En **direcciones IPv4 externas**, escriba las direcciones IPv4 externas del servidor perimetral en el **servicio perimetral de acceso**, el **servicio perimetral de conferencias web**y el **servicio perimetral a/V**.

- Haga clic en **Aceptar** para guardar y confirmar los cambios en el documento de topología.

- Haga clic en **Cancelar** para descartar los cambios y cerrar el asistente **para agregar servidor al grupo de bordes** .

- Haga clic en **ayuda** para leer este tema de ayuda.

Para obtener más información sobre cómo agregar servidores perimetrales a una topología existente, consulte [definir la topología de borde](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx). Para obtener información sobre cómo migrar servidores perimetrales, vea [fase 5: agregar el servidor perimetral de Lync Server 2013 en una agrupación piloto](https://technet.microsoft.com/library/d2fd0a2f-8fec-4f86-a5b0-7f7d03172016.aspx).
