# <a name="legacy-merge-fqdn"></a>FQDN de combinación heredado
 
El **FQDN interno del grupo de borde de acceso** se utiliza para una variedad de escenarios donde los usuarios internos comunican con usuarios externos para la federación, el acceso de usuarios remotos y la conectividad con IM pública. Si se ha implementado un servidor perimetral de equilibrio de carga en el entorno heredado, escriba el nombre de dominio completo (FQDN) del equilibrador de carga interno.
  
El valor del **puerto de acceso SIP internos** de **5061** es el puerto de protocolo de Control de transmisión (TCP) SIP predeterminado para la comunicación con los clientes, heredados Front-End y grupos de servidores. Si no se utilizó el valor predeterminado, actualizar el **puerto de acceso SIP internos:** valor.
