# <a name="lync-server-2013-create-new-expander"></a>Lync Server 2013 Crear nuevo expansor
 
Puede crear nuevos roles de servidor en el nodo de 2013 de Lync Server de generador de topología haciendo clic con el botón secundario del ratón en **Lync server 2013**, seleccione **nuevo**y, a continuación, seleccione el rol de servidor que desea crear.
