# <a name="add-front-end-features"></a>Agregar características de Front-End
 
Seleccione cada característica que desea habilitar para el grupo de servidores Front-End. Como mínimo, debe implementar mensajería instantánea (IM), que siempre está habilitada y no se puede deseleccionar. También tiene la opción de implementar otras características, como las conferencias, la Telefonía IP empresarial y llamar al control de admisión (CAC). Algunas características dependen de la topología. 
  
- Conferencia
    
- Telefonía IP empresarial
    
- Control de admisión de llamadas (CAC)
    
- Archivado
    
- Supervisión
    
> [!NOTE]
> Si desea habilitar CAC en su implementación, debe habilitar CAC en exactamente un grupo por cada sitio central. Recomendamos CAC si va a implementar las funciones de voz o A / conferencias audiovisuales. 
  
Para obtener más información acerca de cómo especificar las características de un grupo de servidores Enterprise Edition, vea [definir y configurar un grupo de servidores frontales](http://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx) en la documentación de implementación.
