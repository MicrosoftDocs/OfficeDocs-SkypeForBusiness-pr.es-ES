# <a name="associate-front-end-with-office-web-apps-server-page"></a>Página Asociar front-end con servidor Office Web Apps

El servidor de Office Web Apps proporciona servicios similares a un moderador que carga un documento para usarlo en una reunión, pero habilita características adicionales que no se realizaron en las versiones anteriores.

Seleccione un servidor de Office Web Apps para la implementación si ha elegido la Conferencia como una de las opciones de características para su configuración de Skype empresarial Server 2015.

Las opciones que se incluyen en este cuadro de diálogo son:

1. **Asociar grupo con un servidor de Office Web Apps** : Active esta casilla si va a proporcionar e implementar un servidor de Office Web Apps para los usuarios de conferencias.

2. Si ha activado esta casilla, use la lista para seleccionar un Servidor de Office Web Apps ya definido.

3. Haga clic en **Nuevo** para definir un Servidor de Office Web Apps nuevo.
