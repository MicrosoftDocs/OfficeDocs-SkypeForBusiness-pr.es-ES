# <a name="add-edge-server-associations"></a>Agregar asociaciones de servidor perimetral
 
Utilice esta página para seleccionar qué grupos de Front-End y los servidores Standard Edition va a utilizar este grupo de borde para un / tráfico V. 
  
Cada grupo de servidores Front-End puede ser asociada con sólo una sección de borde. Si selecciona una agrupación que ya está asociada con otro grupo de borde, cambiará la asociación de la piscina de borde anterior.
  
También puede asociar este servidor perimetral o un grupo de servidores de borde con servidores de mediación o grupos de servidor de mediación para configurar y habilitar la Telefonía IP empresarial para los usuarios externos.
