# <a name="add-edge-server-associations"></a>Agregar asociaciones de servidor perimetral
 
Use esta página para seleccionar los grupos de servidores Front-End y servidores Standard Edition usará este grupo de servidores perimetrales para A y el tráfico A/v. 
  
Cada grupo de servidores Front-End se puede asociar con un solo grupo de servidores perimetrales. Si selecciona un grupo de servidores que ya está asociado a otro grupo de servidores perimetrales, cambiará la asociación de grupo de servidores perimetrales anterior.
  
También puede elegir asociar este servidor perimetral o grupo de servidores perimetrales a los servidores de mediación o grupos de servidores de mediación para configurar y habilitar Enterprise Voice para los usuarios externos.
