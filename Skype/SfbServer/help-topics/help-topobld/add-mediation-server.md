# <a name="add-mediation-server"></a>Agregar servidor de mediación
 
Para definir el servidor de mediación, especifique un nombre de dominio completo (FQDN) para el grupo de servidores de mediación. Si está creando un grupo de servidores de mediación, especificará el nombre del grupo para los servidores de mediación que esté implementando. Definirá los FQDN de los servidores individuales en otro paso. Si está implementando un solo servidor de mediación, el FQDN que especifique será el FQDN del único servidor de mediación. Para obtener más información, vea [Deploy a Mediation Server in Topology Builder in Skype Empresarial Server 2015](../../deploy/deploy-enterprise-voice/deploy-a-mediation-server.md).
