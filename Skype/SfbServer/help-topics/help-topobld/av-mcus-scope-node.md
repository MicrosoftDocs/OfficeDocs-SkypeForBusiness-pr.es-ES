# <a name="av-mcus-scope-node"></a>Nodo de ámbito MCU AV
 
El nodo **Grupos de servidores de conferencia A/V** contiene los grupos de servidores de conferencia A/V de la topología. Puede efectuar la acción siguiente:
  
- **Nuevo grupo de servidores de conferencia A/V**. Inicia el asistente **Definir nuevo grupo de servidores de conferencia A/V**, que sirve para agregar un nuevo grupo de servidores de conferencia A/V a la topología.
    
Además de las opciones de los grupos de servidores de conferencia A/V, puede efectuar las tareas generales siguientes:
  
- **Topología**. Esta opción permite definir una topología nueva, abrir una topología existente, descargar la topología actual, guardar una copia de la topología, publicar la topología, instalar una base de datos, combinar componentes o quitar la implementación.
    
- **Ver**. Esta opción permite personalizar la vista del Generador de topologías seleccionando los elementos que desea que se muestren y los que desea que queden ocultos en las vistas del árbol de la consola.
    
- **Ayuda**. Esta opción permite tener acceso a la Ayuda del nodo activo.
