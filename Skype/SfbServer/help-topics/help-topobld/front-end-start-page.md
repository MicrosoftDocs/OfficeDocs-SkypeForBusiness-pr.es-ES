# <a name="front-end-start-page"></a>Página de inicio del front-end

Use este asistente para definir un nuevo grupo de servidores front-end para un sitio. Cada sitio debe tener al menos un grupo de servidores Enterprise Edition o un servidor Standard Edition. Para obtener más información sobre los requisitos para usar el Asistente para crear un nuevo grupo de servidores front-end, vea [definir y configurar un grupo de servidores front-end](https://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx) en la documentación de implementación.
