# <a name="add-monitoring-server"></a>Agregar servidor de supervisión

Para incorporar un servidor de supervisión en el diseño de topología, debe especificar el nombre de dominio completo (FQDN) del servidor en el que desea implementar el servidor de supervisión. Antes de publicar una topología que incluya el servidor de supervisión e instalar Skype Empresarial Server en el servidor, debe completar todos los requisitos previos para implementar la supervisión. Para obtener más información sobre estos requisitos previos, consulte [Implementar la supervisión](/previous-versions/office/lync-server-2013/lync-server-2013-deploying-monitoring) en la documentación sobre implementación.
