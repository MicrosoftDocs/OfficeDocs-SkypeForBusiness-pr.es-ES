# <a name="associate-branch-office-appliance-with-front-end"></a>Asociar el equipo de sucursal a Front-End
 
Para asociar el dispositivo de sucursal que sobreviven o el servidor de sucursal que sobreviven a un grupo de servidores Front-End o un servidor Standard Edition, en la página **Seleccionar el grupo de servidores Front-End** , seleccione un servidor Standard Edition o el grupo de servidores de Front-End del **grupo de servidores Front-End** lista desplegable. Haga clic en **Siguiente** para continuar.
