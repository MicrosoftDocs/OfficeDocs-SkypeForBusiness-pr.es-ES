# <a name="add-director-association-for-lync-server-2010"></a>Agregar asociación de directores para Lync Server 2010
 
Defina si el director o el grupo de directores tendrán asociado un servidor de supervisión. Active la casilla para asociar un servidor de supervisión con el grupo de directores o directores.
  
> [!TIP]
> Si aún no ha creado una base de datos del servidor de supervisión (e instancia opcional), tendrá la oportunidad de crearla más adelante si selecciona esta casilla para asociar un servidor de supervisión con el grupo de directores o directores.
