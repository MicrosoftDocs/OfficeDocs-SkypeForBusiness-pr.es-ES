# <a name="add-director-association-for-lync-server-2010"></a>Agregar Asociación de director para Lync Server 2010
 
Defina si el director o el grupo de directores tendrá un servidor de supervisión asociado a él. Seleccione la casilla para asociar un servidor de supervisión al director o al grupo de directores.
  
> [!TIP]
> Si aún no ha creado una base de datos del servidor de supervisión (e instancia opcional), tendrá la oportunidad de crearla más adelante si activa esta casilla para asociar un servidor de supervisión con el grupo de directores o directores.
