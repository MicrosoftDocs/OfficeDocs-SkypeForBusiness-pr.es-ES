# <a name="add-director-association-for-lync-server-2010"></a>Agregar asociación de Director para Lync Server 2010
 
Define si el Director o el Director de grupo de servidores será un servidor de supervisión asociado a él. Seleccione la casilla de verificación para asociar un servidor de supervisión con el Director o el Director de grupo de servidores.
  
> [!TIP]
> Si aún no ha creado una base de datos de servidor de supervisión (y una instancia opcional), se le ofrecerá la oportunidad de crear más adelante si selecciona esta casilla de verificación para asociar un servidor de supervisión con el Director o el Director de grupo de servidores.
