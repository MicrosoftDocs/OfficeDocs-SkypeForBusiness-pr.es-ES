# <a name="add-director-association-for-lync-server-2010"></a>Agregar asociación de Director de Lync Server 2010
 
Defina si el Director o el Director de grupo será tienen un servidor de supervisión asociados con él. Active la casilla de verificación para asociar un servidor de supervisión con el Director o el Director de grupo.
  
> [!TIP]
> Si aún no ha creado una base de datos del servidor de supervisión (y opcional de la instancia), tendrá la oportunidad de crear más adelante si selecciona esta casilla de verificación para asociar un servidor de supervisión con el Director o el Director de grupo.
