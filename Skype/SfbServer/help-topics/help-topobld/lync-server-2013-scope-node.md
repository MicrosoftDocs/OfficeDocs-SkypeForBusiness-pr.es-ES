# <a name="lync-server-2013-scope-node"></a>Nodo del ámbito de Lync Server 2013
 
Los pasos necesarios para definir un nuevo VIS o una puerta de enlace de vídeo en el generador de topologías se describen en línea en [Create a vis Pool in Skype for Business Server 2015](../../deploy/deploy-video-interop-server/create-a-vis-pool.md).
  
Los pasos necesarios para asignar un rol de servidor de interoperabilidad de vídeo en un servidor mediante Windows PowerShell se describen en línea en [deploy The Vis Server Role in Skype for Business Server 2015](../../deploy/deploy-video-interop-server/deploy-the-vis-server-role.md)
  
Los pasos de configuración para un nuevo servidor de interoperabilidad de vídeo mediante Windows PowerShell se describen en línea en [Configure the video Interop Server in Skype for Business Server 2015](../../deploy/deploy-video-interop-server/configure-the-vis.md).
  
 La configuración de una puerta de enlace de vídeo que debe establecerse en la interfaz gráfica de usuario de Cisco para interoperar con un VIS se describe en línea en [Configure CUCM for interoperate with Skype for Business Server 2015](../../deploy/deploy-video-interop-server/configure-cucm-for-interoperation.md).
  
 La configuración de un VTC que debe establecerse en la interfaz gráfica de usuario de Cisco para interoperar con una puerta de enlace de vídeo y un VIS se describen en línea en [Configure a VTC for interoperate with Skype for Business Server 2015](../../deploy/deploy-video-interop-server/configure-a-vtc-for-interoperation.md).
