# <a name="add-simple-url"></a>Agregar dirección URL Simple
 
Para agregar direcciones URL simples adicionales a la definición de la implementación, especifique la dirección URL de la nueva dirección URL simple. El formato de la dirección URL es http:// _\<tipo de dirección URL simple\>.\< nombre de dominio\> _ o _http://\<nombre de dominio\>/\<tipo de dirección URL simple\>_, donde el tipo de dirección URL simple es uno de los siguientes:
  
- Reunión
    
- Acceso telefónico
    
> [!IMPORTANT]
> Puede tener un marcado en URL simple y una URL simple satisfacer activo en cualquier momento. Agregar una nueva dirección URL simple y lo que es la dirección URL activa establecerá todas las solicitudes futuras para utilizar la nueva dirección URL simple. Deje la URL simple no activa para controlar conferencias y otras solicitudes que se realizaron cuando la anterior URL simple estaba activa. 
  
Para obtener más información acerca de cómo agregar una dirección URL simple, vea [requisitos de DNS para direcciones URL simples en Skype para Business Server 2015](../../plan-your-deployment/network-requirements/simple-urls.md).
