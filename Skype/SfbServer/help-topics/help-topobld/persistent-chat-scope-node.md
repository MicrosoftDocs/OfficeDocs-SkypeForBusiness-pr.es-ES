# <a name="persistent-chat-scope-node"></a>Nodo de ámbito del chat persistente
 
Para comenzar la creación de un nuevo servidor de Chat persistentes o grupo de servidores de charla persistente, en el generador de topología, haga clic con el botón secundario en **grupos de Chat persistentes** y, a continuación, seleccione **Nuevo grupo de Chat persistentes**
  
- **Topología**. Puede utilizar esta opción para definir una nueva topología, abrir una topología existente, descargue la topología actual, guardar una copia de la topología, publicar la topología, instalar una base de datos, combinar componentes de versiones anteriores de Office Communications Server o quitar el implementación.
    
- **Ayuda**. Esta opción permite tener acceso a la Ayuda del nodo activo.
