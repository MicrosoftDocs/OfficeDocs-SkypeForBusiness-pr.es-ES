# <a name="publish-topology"></a>Publicar la topología
 
Cuando comience el proceso de publicación, se muestra la página **publicar la topología** . Lea y confirme que las tareas enumeradas se completa antes de continuar. Cuando esté listo para continuar, haga clic en **siguiente**. Si debe cerrar el proceso de publicación, haga clic en **Cancelar**.
