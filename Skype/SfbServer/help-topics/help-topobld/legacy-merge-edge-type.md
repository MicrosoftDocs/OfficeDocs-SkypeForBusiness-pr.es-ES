# <a name="legacy-merge-edge-type"></a>Tipo de perímetro combinación de información heredada
 
En caso de **Un solo servidor perimetral** o un **Servidor perimetral consolidado con equilibrio de carga**, haga clic en **Siguiente** para especificar la información de configuración. Para un **perímetro expandido**, la configuración de cada servidor se debe cambiar a una configuración perimetral consolidada antes de poder combinarla con Lync Server 2013.
