# <a name="legacy-merge-edge-type"></a>Tipo de perímetro de combinación heredada
 
En caso de **Un solo servidor perimetral** o un **Servidor perimetral consolidado con equilibrio de carga**, haga clic en **Siguiente** para especificar la información de configuración. Para un **servidor perimetral expandido,** la configuración de cada servidor debe cambiarse a una configuración perimetral consolidada para poder combinarse con Lync Server 2013.
