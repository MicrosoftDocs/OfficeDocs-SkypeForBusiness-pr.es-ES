# <a name="federation-route-settings-expander"></a>Ampliador de configuración de ruta de federación
 
Para establecer una asignación de ruta de la federación de sitio, primero debe tener habilitada en el servidor perimetral o un grupo de servidores de servidor perimetral de federación. Si la federación no está habilitada en el servidor perimetral o un grupo, la configuración de asignación de ruta de federación para el sitio no estará disponible para su modificación.
  
## 

Si se ha configurado la configuración de la federación en el servidor perimetral o un grupo, puede configurar las siguientes opciones: 
  
- **Asignaciones de ruta de la federación de permitir a todos los sitios** Esta configuración afectará a todos los sitios. Asegúrese de que la configuración que se está configurando en este sitio es adecuada para todos los sitios.
    
- **Federación de habilitar SIP** Seleccione esta opción para habilitar una ruta de federación SIP y, a continuación, seleccione un grupo de Director o borde como la ruta de la federación.
    
- **Federación de habilitar XMPP** Seleccione esta opción para habilitar una ruta de federación XMPP y, a continuación, seleccione un grupo de Director o borde como la ruta de la federación.
