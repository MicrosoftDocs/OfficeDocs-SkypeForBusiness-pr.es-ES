# <a name="add-mediation-server-nexthop"></a>Agregar próximo salto de servidor de mediación
 
Para definir el siguiente salto para un servidor de mediación que se está definiendo, seleccione en la lista desplegable **grupo de salto siguiente** el servidor Enterprise Edition o el servidor Standard Edition que actuará como el próximo salto para el servidor de mediación. Haga clic en **Siguiente** para continuar.
