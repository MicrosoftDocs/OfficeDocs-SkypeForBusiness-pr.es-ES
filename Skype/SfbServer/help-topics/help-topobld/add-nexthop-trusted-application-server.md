# <a name="add-nexthop-trusted-application-server"></a>Agregar servidor de aplicaciones de confianza NextHop
 
Para establecer el siguiente salto para el servidor de aplicaciones de confianza o un grupo de servidores, haga lo siguiente:
  
En la lista desplegable **grupo de siguiente salto** , seleccione el grupo de servidores Enterprise Edition o Standard Edition servidor Front-End que desea que el grupo de aplicaciones de confianza que desee utilizar como próximo salto.
  
Haga clic en **Finalizar** para completar la creación del grupo de aplicaciones de confianza.
