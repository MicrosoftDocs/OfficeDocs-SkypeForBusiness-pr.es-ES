# <a name="edge-server-add-machine"></a>Equipo agregado del servidor perimetral

Especifique el nombre de dominio completo (FQDN) de cada equipo que desee agregar como servidor perimetral en este grupo de servidores. Tras haber agregado un equipo a la lista, puede actualizar el nombre de dominio completo del equipo o quitarlo del grupo de servidores en cualquier momento antes de publicar la topología. Una vez publicada la topología, para cambiar el FQDN se debe eliminar el servidor de Topology Builder y, a continuación, agregar un nuevo servidor al grupo de servidores con el nuevo FQDN. Para obtener más información sobre cómo designar la topología de un servidor perimetral, consulte [Define Your Edge Topology](/previous-versions/office/lync-server-2013/lync-server-2013-define-your-edge-topology) en la documentación sobre implementación de servidores perimetrales.
