Después de publicar la topología, cambiar el FQDN requiere que se elimine el servidor en el generador de topología y que, después, se agregue un nuevo servidor al grupo con el nuevo FQDN. Para obtener detalles sobre el diseño de la topología para un grupo de servidores perimetrales, consulte definir la topología de borde en la documentación de implementación.
