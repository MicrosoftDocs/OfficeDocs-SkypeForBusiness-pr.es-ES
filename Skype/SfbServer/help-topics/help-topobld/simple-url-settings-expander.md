# <a name="simple-url-settings-expander"></a>Dirección URL sencilla configuración Expander
 
Para especificar nuevos, quitar o cambiar las direcciones URL simples de la activa, haga lo siguiente:
  
## <a name="simple-urls"></a>Direcciones URL simples

Para modificar o definir la URL simple, primero debe decidir cuál de las tres direcciones URL simples que desea editar o cambiar. Puede elegir entre la dirección URL de acceso telefónico, la dirección URL de la reunión y la dirección URL de acceso administrativo.
  
Para modificar la dirección URL de acceso de teléfono o la dirección URL de la reunión, seleccione la dirección URL que debe cambiar. Haga clic en **modificar la dirección URL**. Modificar la dirección URL y haga clic en **Aceptar** para guardar la dirección URL. Haga clic en **Cancelar** para descartar los cambios.
  
Para agregar una nueva dirección URL, haga clic en **Agregar**. En el cuadro de diálogo **Agregar simple URL** , especifique la dirección URL y, a continuación, haga clic en **Aceptar** para guardar la dirección URL. Seleccione **Convertir esta dirección en la URL activa para el dominio seleccionado** si quiere configurar la nueva dirección URL como dirección URL activa. Haga clic en **Cancelar** para descartar los cambios.
  
Para hacer una dirección URL diferente de la dirección URL activa (como se indica por la marca de verificación verde junto a la dirección URL), seleccione la dirección URL y haga clic en **Activar**.
  
> [!NOTE]
> Sólo puede haber una dirección URL activa para cada dominio de protocolo de inicio de sesión (SIP). 
  
Para quitar una dirección URL, seleccione la dirección URL y haga clic en **Quitar**.
  
> [!CAUTION]
> Asegúrese de leer la información en la página de diálogo de configuración de direcciones URL simple. Quitar una dirección URL de la reunión puede representar las reuniones existentes programadas por usuarios que ya no es accesibles. Considerar dejar la dirección URL después de activar la nueva dirección URL de reunión. Cuando esté seguro de que los usuarios dejen de utilizar la anterior dirección URL de reunión, puede quitar de forma segura. 
  
Para editar o cambiar la dirección URL de acceso administrativo, edite la entrada.
  
Una vez que haya terminado con todos los cambios en la página Editar propiedades, haga clic en **Aceptar** para guardar los cambios. Haga clic en **Cancelar** para descartar los cambios.
