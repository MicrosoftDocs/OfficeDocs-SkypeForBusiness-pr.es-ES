# <a name="av-mcu-general-settings-expander"></a>Botón de expansión de AV MCU configuración General
 
Para editar las propiedades de una A / grupo V Conferencing Server que ya ha definido, puede hacer lo siguiente:
  
## <a name="general-settings"></a>Configuración general

Actualizar el nombre de dominio completo (FQDN) del A / grupo V Conferencing Server escribiendo el nombre de la agrupación, que se define en el sistema de nombres de dominio (DNS).
  
## <a name="av-conferencing-service"></a>Servicio de conferencia A/V

Para asociar una A / grupo V Conferencing Server, seleccione el grupo de servidores Front-End o agrupa que tiene que asociar con la letra A o grupo V Conferencing Server.
  
> [!NOTE]
> Si no hay grupos de Front-End se muestran en el **Seleccione los grupos de Front-End para asociar este A / grupo de conferencias audiovisuales** panel, o bien ha colocado el / V Conferencing Server con el grupo de servidores Front-End o no ha seleccionado para agregar conferencia en las características de **y funcionalidad** sección de las propiedades de grupo de servidores Front-End.
  
## 

Cuando haya terminado, haga clic en **Aceptar** para completar la edición del A / V Conferencing Server grupo de configuración. Haga clic en **Cancelar** para descartar los cambios.
