# <a name="add-survivable-branch-appliance-pool"></a>Agregar grupo de servidores de aplicación de sucursal con funciones de supervivencia
 
Para **definir el FQDN de la aplicación de rama superviviente**, escriba el nombre de dominio completo (FQDN) (según se define en el sistema de nombres de dominio (DNS)) de la aplicación de rama superviviente o servidor de sucursal superviviente en el cuadro de texto **FQDN** . Haga clic en **Siguiente** para continuar.
  
> [!NOTE]
> A menos que se indique lo contrario, la definición de un equipo de sucursal o de un servidor de sucursal con la supervivencia es el mismo procedimiento.
