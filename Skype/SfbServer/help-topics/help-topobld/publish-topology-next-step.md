# <a name="publish-topology-next-step"></a>Publicar la topología siguiente paso
 
Al final de la publicación de la topología, se muestra la página del **Asistente para la publicación completa** .
  
Cuando termine de pasos durante el proceso de publicar la topología, se crean registros que contienen información sobre las tareas que se realizaron. Para ver un archivo de registro relacionadas con un paso determinado, seleccione el paso que se debe revisar el archivo de registro y, a continuación, haga clic en **Ver registros**. 
  
> [!NOTE]
> No todos los pasos durante el proceso de publicar la topología tendrá asociados los archivos de registro. Si selecciona un paso y **Ver registros** está disponible, este paso ha creado un archivo de registro. Si el botón **Ver registros** no está disponible (está atenuada), no hay ningún archivo de registro se creó para este paso.
  
Para revisar los pasos sugeridos para completar la topología, haga clic en el vínculo **haga clic aquí para abrir la lista de tareas pendientes**. El archivo de texto que se abre haciendo clic en este vínculo indica los servidores que tendrá que ejecutar la instalación local y los pasos adicionales que necesita para llevar a cabo la implementación. Para cada una de las tareas enumeradas anteriormente, consulte la sección correspondiente en la documentación de implementación. Para obtener más información, consulte [Implementación de Lync Server 2013 Preview](http://technet.microsoft.com/library/b76795a4-4e71-4c70-a5c0-d1197fa8028c.aspx)y la sección relacionados para cada función de servidor que se indica en la lista de tareas pendientes.
  
Haga clic en **Finalizar**
  
Para obtener más información, consulte [publicar la topología](http://technet.microsoft.com/library/3b5a744b-b3a8-4538-a55e-e2e4f72dff47.aspx).
