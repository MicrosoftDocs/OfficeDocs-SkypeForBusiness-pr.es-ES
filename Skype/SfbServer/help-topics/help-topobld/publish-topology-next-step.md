# <a name="publish-topology-next-step"></a>Próximo paso para publicar la topología

En la finalización de la publicación de la topología, se muestra la página **Asistente para publicación completa** .

Como fin pasos durante el proceso de publicar la topología, se crean los registros que contienen información acerca de las tareas que se realizaron. Para ver un archivo de registro relacionados con un paso determinado, seleccione el paso que necesita para revisar el archivo de registro para y, a continuación, haga clic en **Ver registros**.

> [!NOTE]
> No todos los pasos durante el proceso de publicar topología habrá asociado los archivos de registro. Si selecciona un paso y **Visualización de registros** está disponible, este paso ha creado un archivo de registro. Si el botón de **Visualización de registros** no está disponible (atenuada), este paso no se creó ningún archivo de registro.

Para revisar los pasos siguientes sugeridos para llevar a cabo su topología, haga clic en el vínculo **haga clic aquí para abrir la lista de tareas pendientes**. El archivo de texto que se abre al hacer clic en este vínculo indicará los servidores que tendrá que ejecutar el programa de instalación local y los pasos adicionales que necesita para llevar a cabo la implementación. Para cada una de las tareas enumeradas, consulte la sección correspondiente en la documentación de implementación. Para obtener información detallada, vea [Implementación de Lync Server 2013 Preview](https://technet.microsoft.com/library/b76795a4-4e71-4c70-a5c0-d1197fa8028c.aspx)y hacer referencia a la sección relacionada para cada función de servidor que se indica en la lista de tareas pendientes.

Haga clic en **Finalizar**

Para obtener más información, consulte [publicar la topología](https://technet.microsoft.com/library/3b5a744b-b3a8-4538-a55e-e2e4f72dff47.aspx).
