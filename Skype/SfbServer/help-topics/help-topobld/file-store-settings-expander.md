# <a name="file-store-settings-expander"></a>Ampliador de configuración de almacén de archivos
 
Para editar las propiedades de los almacenes de archivos definidos:
  
Para el nombre de recurso compartido de archivos, puede definir el nombre de un nuevo recurso compartido. A continuación, debe crear el recurso compartido en el servidor que se menciona en la propiedad de nombre (completo FQDN) del dominio de servidor de archivos completo.
  
> [!NOTE]
> No se puede cambiar el FQDN del servidor que se especifica en la propiedad del FQDN del servidor de archivos. Si debe colocar el archivo de almacén de recurso compartido en un servidor diferente, cree una nueva entrada del almacén de archivos.
