# <a name="define-the-root-trunk-for-a-new-ip-or-pstn-gateway"></a>Definir el tronco de raíz de un nuevo IP o puerta de enlace PSTN
 
Definir el tronco de raíz para la dirección IP o la red telefónica pública conmutada (PSTN) mediante la configuración de los siguientes:
  
- **Nombre del tronco**: definir el nombre de dominio completo asociado con el tronco
    
- **Puerto de escucha para la puerta de enlace IP/PSTN**: definir el puerto que escuchará este tronco en
    
- **Protocolo de transporte SIP**: seleccione de la lista **TCP** o **TLS**, basándose en los requisitos de tronco
    
- **Servidor de mediación asociados**: seleccione de la lista de servidores de mediación disponibles en la implementación
    
- **Puerto de servidor de mediación asociados**: defina el puerto que está escuchando el servidor de mediación seleccionado
