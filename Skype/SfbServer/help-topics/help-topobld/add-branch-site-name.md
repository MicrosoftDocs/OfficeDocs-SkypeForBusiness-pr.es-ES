# <a name="add-branch-site-name"></a>Agregar nombre de sucursal

Para definir un nuevo sitio de sucursal, especifique lo siguiente:

Un nombre para la nueva sucursal. El nombre que se defina debe ser exclusivo de la implementación. Se puede usar cualquier convención de nomenclatura. Se suele recomendar que sea por ciudad, nombre de localidad o nombre de sucursal. El valor del nombre es una entrada obligatoria.

El campo de la descripción, que es opcional, brinda un espacio para especificar información adicional que no forma parte de la infraestructura del nombre y que, sin embargo, permite describir el sitio con más exactitud.

Para obtener más información, consulte [Sites](https://technet.microsoft.com/library/022cb6dd-37e2-4882-a53e-5ddfdbc6f53a.aspx).
