# <a name="publish-topology-create-mirror-database-page"></a>Publicar página de topología Crear base de datos de reflejo
 
Esta documentación es preliminar y está sujeta a cambios. Los temas en blanco que se incluyen actúan como marcadores de posición.
  
El generador de topología analiza todas las opciones de configuración en el documento de topología definido. Si ha configurado bases de datos reflejadas de SQL Server, se le presentará una pantalla que muestra el almacén de reflejos, el almacén principal, la ruta del recurso compartido de archivos y las rutas de la base de datos. Active una casilla asociada con una tienda de reflejo de base de datos para crear la base de datos reflejada y la ruta del recurso compartido de archivos. Si no desea crear e implementar la base de datos en el servidor SQL definido, desactive la casilla de verificación reflejo de la base de datos.
  
 Si elige no instalar una o todas estas bases de datos en este momento, desactive las bases de datos que no va a instalar. En otro momento, elija **instalar base de datos** haciendo clic con el botón secundario del mouse en la mayoría de los elementos del nodo en el generador de topología.
