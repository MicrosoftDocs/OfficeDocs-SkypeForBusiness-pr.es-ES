# <a name="publish-topology-create-mirror-database-page"></a>Publicar la topología Crear página de base de datos reflejada
 
Esta documentación es preliminar y está sujeta a cambios. Los temas en blanco que se incluyen actúan como marcadores de posición.
  
El generador de topología analiza todas las opciones de configuración en el documento de topología definida. Si ha configurado las bases de datos reflejadas de SQL Server, aparecerá una pantalla que muestra las rutas de acceso de la base de datos, el almacén principal, ruta del recurso compartido de archivo y el almacén de espejo. Seleccione una casilla de verificación asociada con un almacén de reflejo de base de datos para crear la base de datos reflejada y la ruta de acceso del recurso compartido de archivo. Desactive una casilla de verificación de un reflejo de la base de datos si no desea crear e implementar la base de datos en el SQL Server definido.
  
 Si opta por elegir no instalar una o todas estas bases de datos en este momento, desactive las bases de datos que no desea instalar. Más adelante, decide utilizar **Instalar la base de datos** haciendo clic en la mayoría de elementos de nodo en el generador de topología.
