# <a name="add-additional-sip-domains"></a>Agregar dominios SIP adicionales
 
Especifique el nombre de dominio de cada dominio adicional que desee admitir en la implementación. Por ejemplo, si ha especificado contoso.net como el dominio del protocolo principal de inicio de sesión (SIP) y también desea admitir fabrikam.net para el tráfico SIP de su implementación, especifique fabrikam.net como dominio admitido adicional.
