# <a name="av-mcu-settings-expander"></a>Ampliador de configuración AV MCU
 
Para editar las propiedades de una A / grupo V Conferencing Server que ya ha definido, haga lo siguiente:
  
## <a name="general-settings"></a>Configuración general

Para actualizar el nombre de dominio completo (FQDN) del A / grupo V Conferencing Server, escriba el nombre del grupo que se define en el sistema de nombres de dominio (DNS). Debe tener un registro de host (A) de DNS que se corresponde con la letra A o el nombre del grupo V Conferencing Server.
  
## <a name="av-conferencing-service"></a>A / V servicio de conferencia

Para asociar una A / grupo V Conferencing Server, seleccione el grupo de servidores Front-End o agrupa que tiene que asociar con la letra A o grupo V Conferencing Server.
  
> [!NOTE]
> Si no hay grupos de Front-End se muestran en el **Seleccione los grupos de Front-End para asociar este A / grupo de conferencias audiovisuales** panel, o bien ha colocado el / V Conferencing Server con el grupo de Front-End, o no ha seleccionado para agregar conferencia en las características de ** funcionalidad y** sección de las propiedades de grupo de servidores Front-End.
  
## 

Cuando haya terminado, haga clic en **Aceptar** para completar la edición del A / V Conferencing Server grupo de configuración. Haga clic en **Cancelar** para descartar los cambios.
