# <a name="legacy-merge-landing"></a>Inicio de mezcla heredados
 
La página **Especificar la configuración de borde** permite configurar la información del servidor de borde heredada. Haga clic en **Agregar** para especificar la versión del servidor de borde heredada de la implementación de Office Communications Server. Si no tiene ningún servidor de borde en el entorno heredado, haga clic en **siguiente** para continuar.
