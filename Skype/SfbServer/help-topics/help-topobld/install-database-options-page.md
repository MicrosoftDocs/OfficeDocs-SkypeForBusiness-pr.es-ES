# <a name="install-database-options-page"></a>Instale la página de opciones de base de datos
 
Configurar opciones avanzadas para la colocación de los archivos de registro y de base de datos en el SQL Server. Las opciones disponibles son:
  
> [!IMPORTANT]
> Seleccione la opción que mejor se adapte a sus necesidades y políticas relativas a la ubicación de archivos de datos y de registro en los equipos de SQL Server. 
  
 **Determinar automáticamente la ubicación del archivo de base de datos**: la opción predeterminado utiliza un algoritmo que determina el espacio disponible en el SQL Server y distribuye los archivos de base de datos y de registro para un rendimiento óptimo.
  
 **Valores predeterminados de instancia de SQL Server de uso**: seleccione esta opción para colocar el archivo de base de datos y registro de archivos basándose en la configuración de instancia de SQL Server. Normalmente, las opciones son administradas y configuradas por el Administrador de la base de datos.
  
 **Nos estas ruta de destino de SQL Server**: seleccione esta opción para definir sus propias rutas de acceso para los archivos de registro y de base de datos de SQL Server, escriba la ruta de acceso completa de la unidad y la carpeta donde se colocarán los archivos de base de datos y de registro.
  
> [!IMPORTANT]
> Las rutas de acceso que especifique pueden modificarse en función de algoritmos de optimización de rendimiento en la instalación. Para obtener más información, vea [Base de datos de instalación utilizando Lync Server el Shell de administración](http://technet.microsoft.com/library/c90a6449-4dd5-4b18-b21c-ea2c2a64dc3c.aspx). 
  
 **Aceptar**: haga clic en el botón Aceptar para confirmar los cambios.
  
 **Cancelar**: haga clic en Cancelar para descartar los cambios y volver a la pantalla de instalación de base de datos.
  
 **Ayuda**: haga clic en el botón Ayuda para obtener acceso a esta página de ayuda.
