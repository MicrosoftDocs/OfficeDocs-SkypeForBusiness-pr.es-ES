# <a name="add-sql-store"></a>Agregar almacén de SQL
 
Para definir un nuevo almacén de SQL, lo que significa que está especificando una base de datos de SQL Server y una instancia de SQL Server, una instancia predeterminada o una instancia con nombre, especifique lo siguiente.
  
Especifique el nombre de dominio completo (FQDN) de la de SQL Server que alojará la instancia de base de datos que está definiendo.
  
Especifique la instancia de SQL Server que alojará los datos. Puede especificar la instancia predeterminada, o puede especificar una instancia con nombre.
  
Colocación de las bases de datos en casos específicos debe entenderse muy claramente. Para obtener más información acerca de la colocación de servidores y la colocación de la instancia de base de datos, consulte [Colocación de servidores en una implementación de grupo de extremo frontal](http://technet.microsoft.com/library/0516b18d-14c0-4237-9279-0f92e341b1bd.aspx) y [Colocación de servidores en una implementación de servidor Standard Edition](http://technet.microsoft.com/library/0763ffab-4fd6-463a-8e62-d97876b376d3.aspx).
