# <a name="add-sql-store"></a>Agregar almacén SQL

Para definir un nuevo almacén de SQL, lo que significa que se están especificando una base de datos basada en SQL Server y una instancia de SQL Server, una instancia predeterminada o una instancia con nombre: especifique lo siguiente.

Especifique el nombre de dominio completo (FQDN) del servidor SQL que se va a hospedar la instancia de base de datos que se está definiendo.

Especifique la instancia de SQL Server que se va a hospedar los datos. Puede especificar la instancia predeterminada, o puede especificar una instancia con nombre.

Debe entender claramente muy combinación de bases de datos en instancias específicas. Para obtener información detallada acerca de la combinación de servidor y combinación de la instancia de base de datos, vea [Combinación de servidor en una implementación de grupo de servidores Front-End](https://technet.microsoft.com/library/0516b18d-14c0-4237-9279-0f92e341b1bd.aspx) y [Combinación de servidor en una implementación de servidor Standard Edition](https://technet.microsoft.com/library/0763ffab-4fd6-463a-8e62-d97876b376d3.aspx).
