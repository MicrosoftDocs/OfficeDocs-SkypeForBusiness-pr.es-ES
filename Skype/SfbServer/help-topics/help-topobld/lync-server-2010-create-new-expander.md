# <a name="lync-server-2010-create-new-expander"></a>Lync Server 2010 Crear nuevo expansor
 
Puede crear nuevos roles de servidor en el nodo de 2010 de Lync Server de generador de topología haciendo clic con el botón secundario del ratón en **Lync server 2010**, seleccione **nuevo**y, a continuación, seleccione el rol de servidor que desea crear.
