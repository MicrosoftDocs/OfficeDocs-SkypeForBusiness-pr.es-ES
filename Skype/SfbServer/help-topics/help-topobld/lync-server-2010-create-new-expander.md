# <a name="lync-server-2010-create-new-expander"></a>Lync Server 2010 crear nuevo expansor
 
Puede crear nuevos roles de servidor en el nodo de Lync Server 2010 del generador de topología haciendo clic en con el botón secundario en **Lync Server 2010**, seleccione **nuevo**y luego seleccione el rol de servidor que desea crear.
