# <a name="add-edge-machine-nat-ip"></a>Agregar borde equipo NAT IP
 
Si la dirección IP pública de este grupo de borde se traduce por una traducción de direcciones de red (NAT), debe especificar la dirección IP pública a traducir. Debe ser la dirección IP externa del A / servicio V perimetral.
  
Para determinar si la piscina de borde utiliza traducción NAT, haga clic en el nombre de la agrupación de borde y haga clic en **Editar propiedades**.
