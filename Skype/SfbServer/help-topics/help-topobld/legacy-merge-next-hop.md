# <a name="legacy-merge-next-hop"></a>Siguiente salto de combinación heredados
 
El **FQDN de salto siguiente** identifica el servidor del próximo salto al que el servidor perimetral heredados enruta tráfico interno. Escriba el nombre de dominio completo (FQDN) del servidor del próximo salto al que este servidor perimetral enruta tráfico interno y, a continuación, seleccione si es un **Director**o un **grupo de servidores Front-End** .
