# <a name="legacy-merge-next-hop"></a>Próximo salto de combinación de información heredada
 
El **siguiente salto FQDN** identifica el servidor del próximo salto al que el servidor perimetral heredado enrutará el tráfico interno. Escriba el nombre de dominio completo (FQDN) del servidor del próximo salto al que el servidor perimetral va a enrutar el tráfico interno y, a continuación, seleccione si es un **grupo de servidores front-end** o un **Director**.
