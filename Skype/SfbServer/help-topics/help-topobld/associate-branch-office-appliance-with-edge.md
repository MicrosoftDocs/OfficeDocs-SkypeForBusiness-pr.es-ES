# <a name="associate-branch-office-appliance-with-edge"></a>Asociar la aplicación de la sucursal con el perímetro
 
Si tiene servidores perimetrales implementados y debe proporcionar servicios multimedia para uso externo, elija el servidor perimetral o grupo de servidores que se asociará con el equipo de la sucursal o el servidor de sucursal que siga funcionando, seleccionándolo en la lista desplegable. Haga clic en **Siguiente** para continuar.
  
> [!NOTE]
> Si no tiene un grupo de servidores perimetrales o perimetrales definidos en su topología, puede omitir este paso. Podrá asociar el equipo de sucursal o el servidor de sucursal que sea sobreviviente a un servidor perimetral o un grupo de servidores perimetrales cuando defina un nuevo servidor perimetral o un grupo de servidores perimetrales, o cuando edite las páginas de propiedades de la aplicación de la rama superviviente o que requieran El servidor de sucursal después de un grupo de servidores perimetrales o perimetrales se define en la topología.
