# <a name="legacy-merge-add-expanded-edge"></a>Combinación heredada del perímetro expandido agregado
 
Skype para Business Server no admite la combinación de un heredado Office Communications Server 2007 R2 configuración del servidor perimetral expandida. Antes, hay que convertir el servidor perimetral expandido a una configuración de servidor perimetral consolidado para poder combinar la información del servidor perimetral.
