# <a name="publish-topology-select-cms-page"></a>Publicar la página Select CMS de topología
 
Publicar la topología que se ha configurado mediante el generador de topología. Deberá seleccionar de una lista en la que el servidor Front-End o Front-End de grupo asumirá la función de mantener el almacén de Administración Central. Sólo un grupo de servidor Front-End o Front-End puede contener esta función en un momento dado. 
  
Para obtener más información en el almacén de Administración Central de qué es, consulte [mover el servidor de Administración Central de Lync Server 2010 para Lync Server 2013](http://technet.microsoft.com/library/30cc98f2-1916-4dbe-99d0-8df5368ed3ec.aspx).
