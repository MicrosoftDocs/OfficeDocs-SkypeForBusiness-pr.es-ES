# <a name="define-the-ip-address-of-the-ip-or-pstn-gateway"></a>Definir la dirección IP de la dirección IP o puerta de enlace PSTN
 
Definir las direcciones IPv4 o IPv6 de la IP o teléfono telefónica conmutada (RTC) de red puerta de enlace, seleccione **Habilitar IPv4** o **IPv6 de habilitar**. Puede **utilizar todas las direcciones IP** o **uso del servicio de límite a las direcciones IP seleccionadas**.
  
Si selecciona **utilizar todas las direcciones IP**, se utilizará todas las direcciones configuradas en el gateway. Opcionalmente, seleccione **consumo de servicio del límite que las direcciones IP seleccionadas**, lo que permite la entrada de **dirección IP de RTC** . Escriba la dirección IPv4 o IPv6 que no se ha asignado esta puerta de enlace.
