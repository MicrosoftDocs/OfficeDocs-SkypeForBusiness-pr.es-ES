# <a name="add-av-mcu-machine"></a>Agregar equipo MCU A/V
 
Especifique el nombre de dominio completo de cada equipo que desee agregar como servidor de conferencia A/V en este grupo de servidores. Tras haber agregado un equipo a la lista, puede actualizar el nombre de dominio completo del equipo o quitarlo del grupo de servidores en cualquier momento antes de publicar la topología. Una vez publicada la topología, para cambiar el nombre de dominio se debe eliminar el servidor del Generador de topologías y, a continuación, agregar un nuevo servidor al grupo de servidores con el nuevo nombre de dominio completo.
