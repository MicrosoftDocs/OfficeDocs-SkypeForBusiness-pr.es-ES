# <a name="install-database-create-mirror-database-page"></a>Base de datos de instalación crear página de base de datos reflejada
 
Esta documentación es preliminar y está sujeta a cambios. Los temas en blanco que se incluyen actúan como marcadores de posición.
  
Si decide no instalar las bases de datos de duplicación de SQL Server durante el proceso de publicación, instalarlos mediante **Instalar la base de datos** de un clic derecho en cualquier nodo del generador de topología. Seleccione el almacén de espejo para instalar. Haga clic en **configuración** para definir la configuración avanzada.
