# <a name="remove-deployment-introduction"></a>Quitar la introducción de la implementación
 
Ejecute el **Asistente para implementación de quitar** sólo si desea retirar toda la implementación. Este asistente crea una topología vacía que consta de un único servidor de Front-End. Después de ejecutar a este asistente y publicar la topología vacía, no podrá volver a la topología original.
  
> [!IMPORTANT]
> Antes de hacer clic en **siguiente**, asegúrese de quitar los equipos inactivos, deshabilitar todos los usuarios y los extremos de las comunicaciones unificadas y quitar todos los directorios de conferencia como se especifica en la primera página del asistente.
