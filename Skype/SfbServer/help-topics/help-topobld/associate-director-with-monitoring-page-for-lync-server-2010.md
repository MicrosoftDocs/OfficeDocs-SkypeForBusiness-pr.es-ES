# <a name="associate-director-with-monitoring-page-for-lync-server-2010"></a>Asociar director con la página de supervisión para Lync Server 2010
 
Para configurar un servidor de supervisión y asociarlo con el Director o el Director de grupo de servidores:
  
En **El servidor de supervisión** seleccione de la lista de un servidor de supervisión.
  
Si no hay ningún servidor de supervisión, y desea crear un servidor de supervisión ahora, haga clic en **nuevo**.
  
 **Atrás**: vuelve al cuadro de diálogo anterior
  
 **Finalizar**: permite confirmar los cambios y finalizar el Asistente
  
 **Cancelar**: descarta los cambios y cancela la configuración
  
 **Ayuda**: muestra información, como por ejemplo en este tema de ayuda
