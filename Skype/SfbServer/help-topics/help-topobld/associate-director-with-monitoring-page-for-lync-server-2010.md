# <a name="associate-director-with-monitoring-page-for-lync-server-2010"></a>Asociar director con la página de supervisión para Lync Server 2010
 
Para configurar un servidor de supervisión y asociarlo con el director o grupo de directores:
  
En **Servidor de supervisión,** seleccione en la lista un servidor de supervisión.
  
Si no hay ningún servidor de supervisión y desea crear un servidor de supervisión ahora, haga clic en **Nuevo**.
  
 **Atrás**: permite volver al cuadro de diálogo anterior
  
 **Finalizar**: permite confirmar los cambios y finalizar el asistente
  
 **Cancelar**: permite descartar los cambios y cancelar la configuración
  
 **Ayuda**: muestra información de ayuda, como este tema
