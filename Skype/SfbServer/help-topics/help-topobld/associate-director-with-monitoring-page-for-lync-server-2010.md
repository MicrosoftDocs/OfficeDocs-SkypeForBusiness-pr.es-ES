# <a name="associate-director-with-monitoring-page-for-lync-server-2010"></a>Director asociado con la supervisión de la página para Lync Server 2010
 
Para configurar un servidor de supervisión y asociarlo con el Director o el Director de grupo:
  
En **El servidor de supervisión** se seleccione en la lista un servidor de supervisión.
  
Si no hay ningún servidor de supervisión, y desea crear ahora un servidor de supervisión, haga clic en **nuevo**.
  
 **Atrás**: vuelve al cuadro de diálogo anterior
  
 **Finalizar**: confirma los cambios y finaliza el Asistente
  
 **Cancelar**: descarta los cambios y cancela la configuración
  
 **Ayuda**: muestra información, como por ejemplo este tema de ayuda
