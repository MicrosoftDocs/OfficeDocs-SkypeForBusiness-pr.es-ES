# <a name="persistent-chat-machines-expander"></a>Expansor de equipos de chat persistente
 
Puede activar o desactivar un servidor de chat persistente o un grupo de servidores de chat persistente implementados mediante los servidores o grupos enumerados en la configuración **de estado del equipo.** Seleccione un servidor o grupo de  servidores en la lista y haga clic en el botón Activar para establecer un servidor o grupo como activo.
  
Para establecer un servidor de chat persistente implementado o un grupo de servidores de chat persistente en inactivos, seleccione el servidor o grupo de servidores de la lista y haga clic **en Activar.** Al menos un servidor debe establecerse en activo.
  
 **Aceptar** Acepta y confirma los cambios realizados en el cuadro de diálogo.
  
 **Cancelar** Descarta los cambios y cierra el cuadro de diálogo.
  
 **Ayuda** Muestra la ayuda de esta pantalla.
