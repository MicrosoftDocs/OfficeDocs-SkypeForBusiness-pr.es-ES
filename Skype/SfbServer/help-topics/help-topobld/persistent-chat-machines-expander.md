# <a name="persistent-chat-machines-expander"></a>Expansor de equipos de chat persistente
 
Para activar o desactivar un servidor de chat persistente o un grupo de servidores de chat persistente, use los servidores o grupos que se muestran en **configuración de estado del equipo**. Seleccione un servidor o un grupo de servidores en la lista y haga clic en el botón **Establecer como activo** para establecer un servidor o un grupo de servidores como activo.
  
Para establecer un servidor de chat persistente o un grupo de servidores de chat persistentes en inactivo, seleccione el servidor o grupo de servidores en la lista y haga clic en **hacer inactivo**. Debe haber al menos un servidor definido como activo.
  
 **Aceptar**   Se aceptan y confirman los cambios en el cuadro de diálogo.
  
 **Cancelar.** Se descartan los cambios y se cierra el cuadro de diálogo.
  
 **Ayuda.** Abre esta pantalla de ayuda.
