# <a name="add-front-end-sql-store-page-for-lync-server-2010"></a>Agregar página de almacén de SQL front-end para Lync Server 2010
 
Defina las propiedades de los almacenes de SQL Server para el nuevo grupo de servidores front-end que está configurando.
  
> [!IMPORTANT]
> Si está configurando un servidor Standard Edition, se mostrará este cuadro de diálogo, pero no podrá establecer la configuración. Haga clic en **Siguiente** para continuar.
  
Configuración que puede configurar:
  
- **Usar un almacén de SQL Server definido anteriormente**: Seleccione un almacén de SQL Server existente de la lista para el almacén de la agrupación de servidores front-end.
    
- Seleccione **definir un nuevo almacén de SQL Server** para crear un nuevo almacén de SQL Server.
    
    Escribe el nombre de dominio completo de SQL Server en FQDN de **SQL Server**
    
    La **instancia predeterminada** se selecciona inicialmente. Si prefiere definir una instancia, seleccione **instancia con nombre** y escriba el nombre de la instancia que necesita crear.
    
  **Atrás**: vuelve al cuadro de diálogo anterior
  
  **Siguiente**: confirma los cambios y pasa al cuadro de diálogo siguiente.
  
  **Cancelar**: descarta los cambios y cancela la configuración.
  
  **Ayuda**: muestra información de ayuda, como este tema
