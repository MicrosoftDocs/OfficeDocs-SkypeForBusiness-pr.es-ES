# <a name="add-front-end-sql-store-page-for-lync-server-2010"></a>Agregar página de almacén de Sql de extremo frontal para Lync Server 2010
 
Definir las propiedades de los almacenes de SQL Server para el nuevo pool de Front-End que está configurando.
  
> [!IMPORTANT]
> Si está configurando un servidor Standard Edition, se mostrará este cuadro de diálogo, pero no será capaz de configurar las opciones. Haga clic en **Siguiente** para continuar.
  
Opciones que puede configurar:
  
- **Utilizar un almacén de SQL Server definido anteriormente**: seleccione un almacén de SQL Server existente en la lista para el almacén de grupo de servidores Front-End.
    
- Seleccione **definir un nuevo SQL Server almacenar** para crear un nuevo almacén de SQL Server.
    
    Escriba el nombre de dominio completo de la de SQL Server en **SQL Server FQDN**
    
    La **instancia predeterminada** se selecciona inicialmente. Si prefiere definir una instancia, seleccione **instancia con nombre** y escriba el nombre de la instancia que debe crear.
    
 **Atrás**: vuelve al cuadro de diálogo anterior
  
 **Siguiente**: confirma los cambios y se mueve al siguiente cuadro de diálogo de paso
  
 **Cancelar**: descarta los cambios y cancela la configuración
  
 **Ayuda**: muestra información, como por ejemplo este tema de ayuda
