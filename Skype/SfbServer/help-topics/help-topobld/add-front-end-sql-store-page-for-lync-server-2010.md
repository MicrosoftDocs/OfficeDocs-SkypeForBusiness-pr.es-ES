# <a name="add-front-end-sql-store-page-for-lync-server-2010"></a>Agregar página de almacén de SQL front-end para Lync Server 2010
 
Defina las propiedades de los almacenes de SQL Server para el nuevo grupo de servidores front-end que va a configurar.
  
> [!IMPORTANT]
> Si está configurando un servidor Standard Edition, se mostrará este cuadro de diálogo, pero no podrá establecer la configuración. Haga clic en **Siguiente** para continuar.
  
Opciones que se pueden configurar:
  
- **Usar un almacén de SQL Server definido anteriormente**: Seleccione un almacén de SQL Server existente de la lista para el almacén del grupo de servidores front-end.
    
- Seleccione **definir un nuevo almacén de SQL Server** para crear un nuevo almacén de SQL Server.
    
    Escribe el nombre de dominio completo del SQL Server en FQDN de **SQL Server**
    
    La **Instancia predeterminada** se selecciona inicialmente. Si prefiere definir una instancia, seleccione **Instancia con nombre** y escriba el nombre de la instancia que debe crear.
    
  **Atrás**: permite volver al cuadro de diálogo anterior
  
  **Siguiente**: permite confirmar los cambios y pasar al cuadro de diálogo del siguiente paso
  
  **Cancelar**: permite descartar los cambios y cancelar la configuración
  
  **Ayuda**: muestra información de ayuda, como este tema
