# <a name="monitoring-servers-scope-node"></a>Nodo de ámbito de los servidores de supervisión
 
El nodo **Servidores de supervisión** contiene los servidores de supervisión de la topología. Puede efectuar la acción siguiente:
  
- **Nuevo servidor de supervisión**. Inicia el asistente para **definir nuevo servidor de supervisión**, que se emplea para incorporar a la topología un nuevo servidor de supervisión.
    
Además de las opciones del servidor de supervisión, puede efectuar las tareas generales siguientes:
  
- **Topología**. Esta opción es válida para definir una topología nueva, abrir una topología ya creada, descargar la topología actual, guardar una copia de la topología, publicar la topología, instalar una base de datos, combinar componentes de varias versiones anteriores de Office Communications Server o quitar la implementación.
    
- **Ver**. Use esta opción para personalizar la vista del Generador de topologías seleccionando los elementos que se deben mostrar y ocultar en las vistas de árbol de consola.
    
- **Ayuda**. Use esta opción para obtener acceso a la Ayuda del nodo activo.
