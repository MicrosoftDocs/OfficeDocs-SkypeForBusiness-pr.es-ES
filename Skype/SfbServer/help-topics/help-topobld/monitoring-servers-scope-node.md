# <a name="monitoring-servers-scope-node"></a>Nodo de ámbito de los servidores de supervisión
 
El nodo **servidores de supervisión** enumera los servidores de supervisión de su topología. You can perform the following action:
  
- **Nuevo servidor de supervisión**. Inicia el asistente **definir nuevo servidor de supervisión** , que se usa para agregar un nuevo servidor de supervisión a la topología.
    
Además de las opciones del servidor de supervisión, puede realizar las siguientes tareas generales:
  
- **Topología**. Puede usar esta opción para definir una nueva topología, abrir una topología existente, descargar la topología actual, guardar una copia de la topología, publicar la topología, instalar una base de datos, combinar componentes de versiones anteriores de Office Communications Server o quitar el implementación.
    
- **Ver**. Esta opción permite personalizar la vista del Generador de topologías seleccionando los elementos que desea que se muestren y los que desea que queden ocultos en las vistas del árbol de la consola.
    
- **Ayuda**. Esta opción permite tener acceso a la Ayuda del nodo activo.
