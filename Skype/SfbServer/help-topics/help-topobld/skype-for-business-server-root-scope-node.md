# <a name="skype-for-business-server-root-scope-node"></a>Nodo de ámbito raíz de Skype Empresarial Server
 
El nodo **Lync Server** enumera los sitios centrales de la topología. Puede realizar las acciones siguientes:
  
- **Nuevo sitio central**. Inicia el Asistente para definir un nuevo sitio central, que permite agregar un nuevo sitio central a la topología.
    
- **Editar propiedades**. Permite editar los parámetros que afectan a la topología completa, como los dominios del protocolo de inicio de sesión (SIP) compatibles y las direcciones URL sencillas.
    
- **Nueva topología**. Permite empezar a crear una nueva topología para la implementación. Esta opción resulta útil cuando se planea la implementación, antes de instalar lo servidores. No haga clic en esta opción si ya ha usado la topología actual para instalar la implementación.
    
- **Abrir topología**. Abre un archivo de topología guardado.
    
- **Descargar topología**. Carga la topología implementada desde el almacén de administración central. Si ya ha definido una topología y ha establecido el almacén de administración central, siempre debe seleccionar esta opción para cargar la topología.
    
- **Guardar una copia de la topología como**. Guarda la topología actual en un archivo que se puede cargar más adelante en el generador de topologías.
    
- **Publicar topología**. Use esta opción para publicar esta topología en el almacén de administración central, de modo que pueda empezar a implementar los servidores.
    
- **Instalar base de datos**. Permite crear las bases de datos necesarias para la implementación.
    
- **Combinar topología de Office Communications Server 2007 R2**. Inicia el asistente de Topology Builder. Esta herramienta recopila información sobre el entorno anterior de Office Communications Server, incluida la información del servidor perimetral, y publica dicha información en una base de datos compartida. 
    
- **Quitar implementación**. Inicia el Asistente para quitar la implementación, que permite empezar a quitar la implementación completa.
    
Además de estas opciones, puede realizar las siguientes tareas generales:
  
- **Ver**. Esta opción le permite personalizar la vista del Topology Builder seleccionando los elementos que se deben mostrar y ocultar en las vistas de árbol de consola.
    
- **Ayuda**. Esta opción le permite acceder a la Ayuda del nodo activo.
