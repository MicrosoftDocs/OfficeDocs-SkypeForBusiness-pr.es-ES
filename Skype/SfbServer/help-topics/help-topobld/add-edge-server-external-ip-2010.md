# <a name="add-edge-server-external-ip-2010"></a>Agregar IP externa del servidor perimetral 2010

Puede configurar las propiedades de esta página para que apunten el servidor perimetral a las ubicaciones externas del servicio perimetral de acceso, el servicio perimetral de conferencia web y el servicio perimetral A/V.

- En **Direcciones IPv4** externas , escriba las direcciones IPv4 externas del servidor perimetral en el servicio perimetral de **acceso,** servicio perimetral de conferencia **web** y servicio perimetral **A/V**.

- Haga clic en **Aceptar** para guardar y confirmar los cambios en el documento de la topología.

- Haga clic en **Cancelar** para descartar los cambios y cerrar el asistente **Añadir un servidor al grupo de servidores perimetrales**.

- Haga clic en **Ayuda** para leer este tema de la Ayuda.

Para obtener más información sobre cómo agregar servidores perimetrales a una topología existente, vea [Define Your Edge Topology](/previous-versions/office/lync-server-2013/lync-server-2013-define-your-edge-topology). Para obtener más información acerca de la migración de servidores perimetrales, vea [Phase 5: Add Lync Server 2013 Preview Edge Server to Pilot Pool](/previous-versions/office/lync-server-2013/phase-5-add-lync-server-2013-edge-server-to-pilot-pool).
