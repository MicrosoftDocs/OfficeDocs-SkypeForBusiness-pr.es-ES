# <a name="add-edge-server-external-ip-2010"></a>Agregar IP externa del servidor de borde 2010
 
Puede configurar las propiedades de esta página para que señale el servidor perimetral a las ubicaciones externas del servicio de servidor perimetral de acceso, el servicio perimetral de conferencia Web y A / servicio V perimetral.
  
- En **direcciones IPv4 externa**, escriba las direcciones de IPv4 externas del servidor Edge en **servicio de servidor perimetral de acceso**, **el servicio perimetral de conferencia Web**, y **A / servicio perimetral V**.
    
- Haga clic en **Aceptar** para guardar y confirmar los cambios en el documento de la topología.
    
- Haga clic en **Cancelar** para descartar los cambios y cerrar al Asistente para **Agregar servidor al grupo de borde** .
    
- Haga clic en **Ayuda** para leer este tema de ayuda.
    
Para obtener más información sobre cómo agregar servidores de borde a una topología existente, consulte [Definir la topología de la periferia](http://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx). Para obtener más información acerca de cómo migrar servidores perimetrales, consulte [fase 5: agregar Lync Server 2013 Preview servidor perimetral al grupo piloto](http://technet.microsoft.com/library/d2fd0a2f-8fec-4f86-a5b0-7f7d03172016.aspx).
