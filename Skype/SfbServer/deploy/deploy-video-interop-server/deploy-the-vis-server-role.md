# <a name="deploy-the-vis-server-role-in-skype-for-business-server"></a>Implementar el rol de servidor VIS en Skype Empresarial Server
 
**Resumen:** Implemente el rol Servidor de interoperabilidad de vídeo (VIS) en Skype Empresarial Server.
  
Para configurar el servicio VIS en el servidor recién creado en el Generador de topologías, inicie el Asistente para la implementación de Skype Empresarial Server, presione Instalar o actualizar **Skype Empresarial Server System** y siga estos pasos en el asistente:
  
1.  Seleccione **Instalar almacén de configuración local**.
    
2. Seleccione **Configurar o Quitar Skype Empresarial Server componentes**.
    
3. Seleccione **Solicitar, instalar o asignar certificados**.
    
4. Seleccione **Iniciar servicios**.
    
El software de este servicio ya está instalado y en ejecución. Puede abrir la herramienta mmc servicios para ver si el servicio de servidor de interoperabilidad de vídeo de **Skype Empresarial Server** se está ejecutando junto con otros servicios Skype Empresarial Server web. A continuación, debe configurar el servidor o grupo vis.
