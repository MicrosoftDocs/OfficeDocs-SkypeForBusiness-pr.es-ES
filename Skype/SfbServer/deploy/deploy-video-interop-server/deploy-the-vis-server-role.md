# <a name="deploy-the-vis-server-role-in-skype-for-business-server"></a>Implementar el rol de servidor de respecto de Skype para Business Server
 
**Resumen:** Implementar el rol de servidor de interoperabilidad de vídeo (VISIBLES) de Skype para Business Server.
  
Para configurar el servicio VISIBLES en el servidor que acaba de crear en el generador, iniciar el Skype para el Asistente para la implementación de servidor empresarial, presione **instalación o actualización de Skype para Business Server System** y siga estos pasos en el Asistente para:
  
1.  Seleccione **Instalar almacén de configuración local**.
    
2. Seleccione **instalar o quitar Skype para los componentes de servidor empresariales**.
    
3. Seleccione **Solicitar, instalar o asignar certificados**.
    
4. Seleccione **Iniciar servicios**.
    
The software for this service is now installed and running. Puede abrir la herramienta de mmc de servicios para ver si se está ejecutando el servicio de **Skype para Business Server vídeo interoperabilidad Server** junto con otro Skype para servicios de Business Server. Next, you must configure the VIS server or pool.
