# <a name="deploy-the-vis-server-role-in-skype-for-business-server-2015"></a>Implementar el rol de servidor VIS en Skype Empresarial Server 2015
 
**Resumen:** Implemente la función del servidor de interoperabilidad de vídeo (VIS) en Skype para Business Server 2015.
  
Para configurar el servicio VIS en el servidor que acaba de crear en el generador de topología, iniciar el Skype para el Asistente para implementación de Business Server, presione la **instalación o actualización de Skype para el sistema de servidor empresarial** y siga estos pasos en el asistente:
  
1.  Seleccione **Instalar almacén de configuración local**.
    
2. Seleccione **instalar o quitar Skype para los componentes de servidor empresariales**.
    
3. Seleccione **Solicitar, instalar o asignar certificados**.
    
4. Seleccione **Iniciar servicios**.
    
El software de este servicio ya está instalado y en funcionamiento. Puede abrir la herramienta mmc de servicios para ver si está ejecutando el servicio de **Skype de Business Server interoperabilidad de vídeo** junto con otro Skype para Business Server services. A continuación, debe configurar el servidor o grupo de servidores de VIS.
