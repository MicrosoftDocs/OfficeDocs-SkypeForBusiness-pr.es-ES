# <a name="deploy-video-interop-server-in-skype-for-business-server"></a>Implementar el servidor de interoperabilidad de vídeo en Skype Empresarial Server
 
**Resumen:** Implemente el rol de servidor VIS en Skype Empresarial Server.
  
Skype Empresarial Server ahora puede integrarse directamente con los sistemas de teleconferencia (VTC) de Cisco, como Cisco C60 o Cisco MX300. Esto requiere la introducción de un nuevo rol de servidor denominado servidor de interoperabilidad de vídeo (VIS) y la configuración correcta tanto del VIS como del equipo con el que interoperará. Un VTC se registra con la infraestructura de Cisco existente, como Cisco Unified Communication Manager (CUCM), y se usa un tronco SIP de vídeo entre CUCM y el grupo vis.
  
## <a name="in-this-section"></a>En esta sección

La configuración de la interoperabilidad entre un servidor o grupo de servidores VIS y sistemas VTC requiere realizar los cinco procedimientos siguientes: 
  
- [Crear un grupo vis en Skype Empresarial Server](create-a-vis-pool.md)
    
- [Implementar el rol de servidor VIS en Skype Empresarial Server](deploy-the-vis-server-role.md)
    
- [Configurar el servidor de interoperabilidad de vídeo en Skype Empresarial Server](configure-the-vis.md)
    
- [Configurar CUCM para interoperación con Skype Empresarial Server](configure-cucm-for-interoperation.md)
    
- [Configurar una VTC para interoperación con Skype Empresarial Server](configure-a-vtc-for-interoperation.md)
