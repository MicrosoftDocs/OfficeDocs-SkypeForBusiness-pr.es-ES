# <a name="deploy-video-interop-server-in-skype-for-business-server"></a>Implementar el servidor de interoperabilidad de vídeo en Skype Empresarial Server
 
**Resumen:** Implemente el rol de servidor VIS en Skype Empresarial Server.
  
Skype Empresarial Server ahora se puede integrar directamente con sistemas de teleconferencia de Cisco (VTC), como Cisco C60 o Cisco MX300. Esto requiere la introducción de un nuevo rol de servidor denominado Servidor de interoperabilidad de vídeo (VIS) y la configuración correcta del VIS y el equipamiento con el que interoperará. Un VTC se registra con la infraestructura de Cisco existente, como Cisco Unified Communication Manager (CUCM), y se usa un tronco SIP de vídeo entre CUCM y el grupo vis.
  
## <a name="in-this-section"></a>En esta sección

Para configurar la interoperabilidad entre un servidor o grupo vis y sistemas VTC, es necesario realizar los siguientes cinco procedimientos: 
  
- [Crear un grupo de vis en Skype Empresarial Server](create-a-vis-pool.md)
    
- [Implementar el rol de servidor VIS en Skype Empresarial Server](deploy-the-vis-server-role.md)
    
- [Configurar el servidor de interoperabilidad de vídeo en Skype Empresarial Server](configure-the-vis.md)
    
- [Configurar CUCM para interoperación con Skype Empresarial Server](configure-cucm-for-interoperation.md)
    
- [Configurar un VTC para interoperación con Skype Empresarial Server](configure-a-vtc-for-interoperation.md)
