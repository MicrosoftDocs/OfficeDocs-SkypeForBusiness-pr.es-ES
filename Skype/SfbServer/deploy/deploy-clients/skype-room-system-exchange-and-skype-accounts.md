# <a name="provisioning-of-skype-room-system-exchange-and-skype-accounts"></a>Aprovisionamiento de cuentas de Skype Room System Exchange y Skype
 
Lea estos temas para obtener información sobre cómo aprovisionar cuentas de Exchange y Skype para El sistema de salón de Skype. 

> [!NOTE]
> Microsoft Teams Rooms es un producto diferente con diferentes dependencias y procedimientos de implementación. Para obtener información sobre salas de Microsoft Teams, vea la introducción a la implementación de Salas [de](/MicrosoftTeams/rooms/rooms-deploy)Microsoft Teams .
  
> [!NOTE]
> El aprovisionamiento de cuentas del sistema de salón de Skype depende del tipo de topología que tenga su organización. Para obtener más información acerca de las topologías de Active Directory, vea [Environmental requirements for Skype for Business Server 2015](../../plan-your-deployment/requirements-for-your-environment/environmental-requirements.md).
