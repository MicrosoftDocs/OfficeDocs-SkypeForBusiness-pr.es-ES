# <a name="provisioning-of-skype-room-system-exchange-and-skype-accounts"></a>Aprovisionamiento de cuentas de Exchange y Skype del sistema de salas de Skype
 
Lea estos temas para obtener información sobre cómo aprovisionar cuentas de Exchange y Skype para el sistema de salas de Skype. 

> [!NOTE]
> Salas de Microsoft Teams es un producto diferente con diferentes dependencias y procedimientos de implementación. Para obtener información sobre las salas de Microsoft Teams, vea la [información general sobre la implementación](room-systems-v2.md)de salas de Microsoft Teams.
  
> [!NOTE]
> El aprovisionamiento de la cuenta del sistema de salas de Skype depende del tipo de topología que tenga su organización. Para obtener más información acerca de las topologías de Active Directory, consulte [Environmental Requirements for Skype for Business Server 2015](../../plan-your-deployment/requirements-for-your-environment/environmental-requirements.md).
