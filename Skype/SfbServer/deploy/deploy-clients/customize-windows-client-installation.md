# <a name="customize-windows-client-installation-in-skype-for-business-server"></a>Personalizar Windows de cliente en Skype Empresarial Server
 
**Resumen:** Información general sobre los métodos de instalación y las herramientas para Skype Empresarial.
  
> [!NOTE]
> Para obtener información sobre la instalación Skype Empresarial que incluye Microsoft 365 y Office 365, vea [Deploy the Skype Empresarial client in Microsoft 365 or Office 365](https://support.office.com/article/8c563b81-22c9-4024-9efe-9fe28c7bbc96). 
  
Enterprise los administradores pueden personalizar la instalación basada en Windows Installer (.msi) de las versiones de licencia por volumen de Skype Empresarial mediante los métodos que se definieron en esta sección. Dado que ninguna herramienta única proporciona todas las opciones de personalización, es probable que use una combinación de estos métodos en la Skype Empresarial implementación. Puede usar las herramientas que se describen en las secciones siguientes:
  
- [Use la herramienta Office personalización (OCT)](use-the-office-customization-tool-oct.md) de Skype Empresarial Server para personalizar las opciones de configuración y las características de Skype Empresarial otros Office programa.
    
- [Use Config.xml para realizar tareas](use-config-xml-to-perform-installation-tasks.md) de instalación en Skype Empresarial Server para especificar la ruta de acceso del punto de instalación de red y realizar una instalación silenciosa.
    
- [Use las opciones de línea de](use-setup-command-line-options.md) comandos del programa de instalación Skype Empresarial Server para especificar el archivo Config.xml que se usará durante la instalación.
    
- [Configure las directivas de arranque de cliente Skype Empresarial Server](configure-client-bootstrapping-policies.md) mediante el complemento MMC del Editor de objetos de directiva de grupo.
    
Probablemente habrá otras opciones que querrá configurar a medida que implemente el conjunto Office de productos. Los temas de esta sección proporcionan información general sobre estas herramientas de personalización y analizan las consideraciones específicas de Skype Empresarial. Se incluyen vínculos a información Office ayuda para cada herramienta.
