# <a name="install-persistent-chat-components-in-skype-for-business-server-2015"></a>Instalar componentes de chat persistente en Skype Empresarial Server 2015
 
**Resumen:** Lea este tema para obtener información sobre cómo usar el Asistente Skype Empresarial Server implementación para instalar Skype Empresarial Server componentes y servicios de 2015.
  
Antes de instalar componentes de chat persistente, asegúrese de que ya ha instalado hardware y software previos y creó la topología adecuada para admitir el servidor de chat persistente. Para obtener más información sobre la planeación y los requisitos, vea [Requisitos](../../plan-your-deployment/requirements-for-your-environment/requirements-for-your-environment.md) para el entorno de Skype Empresarial y [Plan for Persistent Chat Server in Skype Empresarial Server 2015](../../plan-your-deployment/persistent-chat-server/persistent-chat-server.md). Para obtener información sobre cómo actualizar y publicar la topología para incluir el servidor de chat persistente, vea [Add Persistent Chat Server to your Skype Empresarial Server 2015 topology](add-persistent-chat-server.md).
  
> [!NOTE] 
> El chat persistente está disponible en Skype Empresarial Server 2015, pero ya no se admite en Skype Empresarial Server 2019. La misma funcionalidad está disponible en Teams. Para obtener más información, vea [Getting started with your Microsoft Teams upgrade](/microsoftteams/upgrade-start-here). Si necesita usar el chat persistente, las opciones son migrar usuarios que requieren esta funcionalidad a Teams, o bien seguir usando Skype Empresarial Server 2015. 

## <a name="install-and-start-services"></a>Instalar e iniciar servicios

Con el Asistente Skype Empresarial Server implementación, elija Instalar o actualizar Skype Empresarial Server System para realizar los siguientes pasos: 
  
1. Instalar almacén de configuración local
    
2. Instalar o quitar componentes de Skype Empresarial Server
    
3. Solicitar, instalar o asignar certificados
    
4. Iniciar servicios
    
Para obtener más información sobre cómo usar el Asistente para la implementación para instalar componentes, asignar certificados e iniciar servicios, vea [Install Skype Empresarial Server 2015](../../deploy/install/install.md) e [Install Skype Empresarial Server 2015 on servers in the topology](../../deploy/install/install-skype-for-business-server.md).
