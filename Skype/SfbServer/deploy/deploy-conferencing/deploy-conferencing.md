# <a name="deploy-conferencing-in-skype-for-business-server"></a>Implementar conferencias en Skype Empresarial Server

**Resumen:** Lea este tema para obtener información sobre cómo implementar conferencias en Skype Empresarial Server.

Hay cuatro tipos de conferencias disponibles en Skype Empresarial Server: conferencia web, conferencia de audio y vídeo (A/V), conferencia de acceso telefónico local y conferencia de mensajes instantáneos (MI). Puede elegir habilitar todos los tipos de conferencia o usar solo un tipo, según sus necesidades.

Al implementar Skype Empresarial Server, las funcionalidades de conferencia de mensajería instantánea se implementan automáticamente. Al crear y publicar una nueva topología mediante el Generador de topologías, se especifica si se va a implementar la web, A/V y las conferencias de acceso telefónico local, como se describe en las siguientes listas de comprobación:

- [Lista de comprobación de implementación para conferencias web y de audio y vídeo](deploy-conferencing.md#BKMK_ChecklistWebConferencing)

- [Diagrama de flujo de implementación y lista de comprobación para conferencias de acceso telefónico local](deploy-conferencing.md#BKMK_DialinConferencing)

Antes de implementar las conferencias, debe leer los siguientes temas de planeación:

- [Planear conferencias en Skype Empresarial Server](../../plan-your-deployment/conferencing/conferencing.md)

- [Requisitos de hardware y software para conferencias en Skype Empresarial Server](../../plan-your-deployment/conferencing/hardware-and-software-requirements.md)

- [Planear la topología de conferencias para Skype Empresarial Server](../../plan-your-deployment/conferencing/conferencing-topology.md)

- [Planear conferencias de acceso telefónico local en Skype Empresarial Server](../../plan-your-deployment/conferencing/dial-in-conferencing.md)

- [Planear reuniones grandes en Skype Empresarial Server](../../plan-your-deployment/conferencing/large-meetings.md)

## <a name="deployment-flowchart-and-checklist-for-dial-in-conferencing"></a>Diagrama de flujo de implementación y lista de comprobación para conferencias de acceso telefónico local
<a name="BKMK_DialinConferencing"> </a>

 Las conferencias de acceso telefónico local permiten a los usuarios llamar desde la red telefónica conmutada (RTC) para unirse a una conferencia de audio y vídeo.

Algunos de los componentes necesarios para las conferencias de acceso telefónico local también se usan para Telefonía IP empresarial. Por ejemplo, si va a implementar Telefonía IP empresarial, también debe implementar un servidor de mediación y una puerta de enlace RTC, componentes que también son necesarios para las conferencias de acceso telefónico local. Por lo tanto, la forma en que se implementan las conferencias de acceso telefónico local depende de si también está implementando una Telefonía IP empresarial de acceso telefónico local.

El diagrama de flujo de conferencia de acceso telefónico local muestra los pasos que debe seguir en función de si también está implementando una Telefonía IP empresarial personalizada. La tabla siguiente al diagrama de flujo proporciona información general sobre los pasos necesarios y recomendados para implementar conferencias de acceso telefónico local. También se incluyen vínculos a la documentación de planeación y procedimientos asociada. Para obtener más información acerca de la planeación de una solución Telefonía IP empresarial completa, consulte [Plan your Telefonía IP empresarial solution in Skype for Business Server](../../plan-your-deployment/enterprise-voice-solution/enterprise-voice-solution.md).
