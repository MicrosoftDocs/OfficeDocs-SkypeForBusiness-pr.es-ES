# <a name="deploy-instant-messaging-and-presence-in-skype-for-business-server-2015"></a>Implementar la presencia y la mensajería instantánea en Skype Empresarial Server 2015
 
**Resumen:** información sobre los pasos necesarios para implementar la mensajería instantánea y la presencia en Skype Empresarial Server 2015.
  
## <a name="deployment-steps-for-instant-messaging-and-presence"></a>Pasos para implementar la mensajería instantánea y la presencia

El artículo de planificación para presencia y mensajería instantánea le guía por la información necesaria para planificar su implementación. Mensajería instantánea y presencia está habilitado cuando instala Skype para Business Server 2015. Para más información acerca de cómo planear la presencia y mensajería instantánea, consulte [Plan para mensajería instantánea y presencia en Skype para Business Server 2015](../../plan-your-deployment/instant-messaging-and-presence.md).
  
En esta sección que aprenderá acerca de algunos de los pasos de implementación que se pueden personalizar para un preciso ajustar los escenarios de presencia y mensajería instantáneos.
