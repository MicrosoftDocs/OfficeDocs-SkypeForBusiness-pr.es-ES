# <a name="start-cclogging"></a>Start-CcLogging
 
El cmdlet Start-CcLogging genera el registro de llamadas entrantes y salientes para un Skype for Business Edición de conector de nube dispositivo. 
  
```powershell
Start-CcLogging
```

## <a name="parameters"></a>Parámetros

Ninguno
  
## <a name="examples"></a>Ejemplos
<a name="Examples"> </a>

### <a name="example-1"></a>Ejemplo 1

En el ejemplo siguiente se genera el registro de llamadas entrantes y salientes:
  
```powershell
Start-CcLogging
```

## <a name="detailed-description"></a>Descripción detallada
<a name="DetailedDescription"> </a>

El cmdlet Start-CcLogging proporciona una forma de que los administradores comiencen a iniciar el registro de llamadas entrantes y salientes en un dispositivo de Cloud Connector. De forma predeterminada, el registro se detendrá automáticamente después de cuatro horas.
  
## <a name="input-types"></a>Tipos de entrada
<a name="InputTypes"> </a>

Ninguno. El cmdlet Start-CcLogging no acepta entradas canalizadas.
  
## <a name="return-types"></a>Tipos de valores devueltos
<a name="ReturnTypes"> </a>

Ninguno
